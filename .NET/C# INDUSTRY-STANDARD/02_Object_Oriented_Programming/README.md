# 📘 Object-Oriented Programming (OOP)

Welcome to the OOP module! This folder covers the fundamental concepts of object-oriented programming in C#.

## 🎯 Learning Objectives

- Understand classes and objects
- Master constructors and access modifiers
- Learn encapsulation, inheritance, and polymorphism
- Work with interfaces and abstract classes

## Core OOP Concepts

```
┌─────────────────────────────────────────────────────────────────┐
│                      OOP CONCEPTS                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ┌───────────────┐                                             │
│   │  ENCAPSULATION │  ← Data + Methods in one unit              │
│   │               │    ← Control access to data                  │
│   └───────┬───────┘                                             │
│           │                                                      │
│           ▼                                                      │
│   ┌───────────────┐                                             │
│   │   INHERITANCE │  ← Reuse code from parent class            │
│   │               │    ← Create is-a relationship              │
│   └───────┬───────┘                                             │
│           │                                                      │
│           ▼                                                      │
│   ┌───────────────┐                                             │
│   │ POLYMORPHISM  │  ← One name, many forms                    │
│   │               │    ← Method overriding                     │
│   └───────────────┘                                             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```
