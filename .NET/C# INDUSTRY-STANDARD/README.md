## 📚 How to Use This Repository

### Step 1: Environment Setup

```bash
# Check .NET installation
dotnet --version

# Create your first project
dotnet new console -n MyFirstApp
cd MyFirstApp
dotnet run
```

### Step 2: Follow the Path

Each folder contains:

- **README.md** - Overview and learning objectives
- **Topic_*.md** - Detailed explanations with examples
- **Code/**/*.cs** - Runnable code examples
- **Exercises/** - Practice problems with solutions

### Step 3: Practice Daily

1. Read the concept explanation
2. Study the code examples
3. Run the examples in your environment
4. Complete the exercises
5. Review interview questions

## 🛠️ Environment Setup Guide

### 1. Install .NET SDK

```bash
# Windows (using winget)
winget install Microsoft.DotNet.SDK.8

# macOS (using Homebrew)
brew install dotnet

# Linux (Ubuntu)
sudo apt-get install dotnet-sdk-8.0
```

### 2. Install Visual Studio Code

Download from: https://code.visualstudio.com/

### 3. Install C# Extension

1. Open VS Code Extensions (Ctrl+Shift+X)
2. Search "C#"
3. Install "C# by Microsoft"

### 4. Verify Setup

```bash
dotnet --version  # Should show 8.0.x or higher
dotnet new console -n TestProject
cd TestProject
dotnet run  # Should print "Hello, World!"
```
