IIFE Pattern

IIFE stands for Immediately Invoked Function Expression. It's a function that executes immediately after being defined.

Syntax

// Basic IIFE
(function() {
    // code here
})();

// With parameters
(function(param) {
    // code here
})(argument);

Examples

Example 1: Basic IIFE

(function() {
    var message = "IIFE executed!";
    document.getElementById("output1").innerHTML = message;
})();

Example 2: IIFE with Return Value

var sum = (function(a, b) {
    return a + b;
})(5, 3);

document.getElementById("output2").innerHTML = "Sum: " + sum;

Example 3: Encapsulation - Private Variables

var counter = (function() {
    var count = 0;
    return {
        getCount: function() { return count; },
        increment: function() { count++; },
        decrement: function() { count--; }
    };
})();

counter.increment();
counter.increment();
counter.increment();

document.getElementById("output3").innerHTML = "Count: " + counter.getCount();

Example 4: Multiple IIFEs

var person = (function(name) {
    return "Name: " + name;
})("John");

var calculator = (function(a, b, operation) {
    if (operation === "add") return a + b;
    if (operation === "sub") return a - b;
    return 0;
})(10, 5, "add");

document.getElementById("output4").innerHTML = person + "<br>Calculator: " + calculator;

Example 5: Initialization Pattern

var config = (function() {
    var settings = {
        theme: "dark",
        language: "en",
        maxItems: 100
    };
    return settings;
})();

document.getElementById("output5").innerHTML = "Theme: " + config.theme + ", Language: " + config.language;

Key Points

  • IIFE executes immediately after being defined
  • Creates a new scope - useful for encapsulation
  • Helps create private variables
  • Commonly used for initialization and module pattern
