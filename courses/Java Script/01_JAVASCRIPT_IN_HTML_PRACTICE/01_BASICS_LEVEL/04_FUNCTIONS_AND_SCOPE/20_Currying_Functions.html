Currying Functions

Currying is transforming a function with multiple arguments into a sequence of functions, each taking a single argument.

Currying Syntax

// Regular function
function add(a, b, c) { return a + b + c; }

// Curried version
function curriedAdd(a) {
    return function(b) {
        return function(c) {
            return a + b + c;
        };
    };
}

Examples

Example 1: Basic Currying

function add(a) {
    return function(b) {
        return function(c) {
            return a + b + c;
        };
    };
}

var result = add(1)(2)(3);
document.getElementById("output1").innerHTML = "1 + 2 + 3 = " + result;

Example 2: Curried Multiplier

function multiply(a) {
    return function(b) {
        return a * b;
    };
}

var double = multiply(2);
var triple = multiply(3);

document.getElementById("output2").innerHTML = 
    "double(5) = " + double(5) + "<br>" +
    "triple(5) = " + triple(5);

Example 3: Arrow Function Currying

var add = a => b => c => a + b + c;

document.getElementById("output3").innerHTML = 
    "add(1)(2)(3) = " + add(1)(2)(3) + "<br>" +
    "add(10)(20)(30) = " + add(10)(20)(30);

Example 4: Converting Regular Function

function convertToCurry(fn) {
    return function(a) {
        return function(b) {
            return fn(a, b);
        };
    };
}

function subtract(a, b) { return a - b; }

var curriedSub = convertToCurry(subtract);
var subFrom10 = curriedSub(10);

document.getElementById("output4").innerHTML = "10 - 3 = " + subFrom10(3);

Example 5: Curried Greeting

function greet(greeting) {
    return function(name) {
        return greeting + ", " + name + "!";
    };
}

var sayHello = greet("Hello");
var sayHi = greet("Hi");

document.getElementById("output5").innerHTML = 
    sayHello("John") + "<br>" +
    sayHi("Jane");

Key Points

  • Currying converts multi-arg functions into nested single-arg functions
  • Each call returns a new function waiting for the next argument
  • Useful for creating specialized functions from general ones
  • Arrow functions provide concise currying syntax
