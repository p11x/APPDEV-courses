Recursion - Basic

Recursion is a technique where a function calls itself to solve a problem by breaking it into smaller subproblems.

Recursion Structure

function recursiveFunction(param) {
    // Base case - stops recursion
    if (baseCondition) {
        return result;
    }
    // Recursive case - calls itself with different param
    return recursiveFunction(smallerProblem);
}

Examples

Example 1: Basic Recursive Countdown

function countdown(n) {
    if (n <= 0) {
        return "Done!";
    }
    return n + ", " + countdown(n - 1);
}

document.getElementById("output1").innerHTML = countdown(5);

Example 2: Sum of Numbers

function sum(n) {
    if (n === 0) {
        return 0;
    }
    return n + sum(n - 1);
}

document.getElementById("output2").innerHTML = "Sum of 1-5: " + sum(5);

Example 3: Count Array Elements

function countArray(arr) {
    if (arr.length === 0) {
        return 0;
    }
    arr.pop();
    return 1 + countArray(arr);
}

var numbers = [1, 2, 3, 4, 5];
document.getElementById("output3").innerHTML = "Count: " + countArray(numbers);

Example 4: Reverse String

function reverseString(str) {
    if (str === "") {
        return "";
    }
    return reverseString(str.substr(1)) + str.charAt(0);
}

document.getElementById("output4").innerHTML = "Reverse of 'hello': " + reverseString("hello");

Example 5: Calculate Power

function power(base, exponent) {
    if (exponent === 0) {
        return 1;
    }
    return base * power(base, exponent - 1);
}

document.getElementById("output5").innerHTML = "2^3 = " + power(2, 3) + "<br>5^2 = " + power(5, 2);

Key Points

  • Recursion involves a function calling itself
  • Base case stops the recursion
  • Recursive case calls with a smaller problem
  • Always ensure there's a base case to prevent infinite loops
