Function Expression

A function expression assigns a function to a variable. Unlike declarations, expressions are not hoisted.

Syntax

var functionName = function(parameter1, parameter2) {
    // function body
    return result;
};

Examples

Example 1: Basic Function Expression

// Function expression assigned to a variable
var greet = function(name) {
    return "Hello, " + name + "!";
};

var message = greet("JavaScript");
document.getElementById("output1").innerHTML = message;

Example 2: Function Expression with Parameters

var multiply = function(a, b) {
    return a * b;
};

var product = multiply(4, 6);
document.getElementById("output2").innerHTML = "4 × 6 = " + product;

Example 3: Hoisting Difference

// This will cause an error - cannot call before assignment
try {
    var result = divide(10, 2);
    document.getElementById("output3").innerHTML = result;
} catch(e) {
    document.getElementById("output3").innerHTML = "Error: " + e.message;
}

// Function expression - must be defined BEFORE calling
var divide = function(a, b) {
    return a / b;
};

Example 4: Named Function Expression

// Named function expression - useful for recursion
var factorial = function computeFactorial(n) {
    if (n <= 1) {
        return 1;
    }
    return n * computeFactorial(n - 1);
};

var fact = factorial(5);
document.getElementById("output4").innerHTML = "5! = " + fact;

Example 5: Function Expression as Callback

var numbers = [1, 2, 3, 4, 5];

// Function expression used as callback
var doubled = numbers.map(function(num) {
    return num * 2;
});

document.getElementById("output5").innerHTML = "Original: " + numbers.join(", ") + "
Doubled: " + doubled.join(", ");

Example 6: Passing Function Expression as Argument

function executeOperation(a, b, operation) {
    return operation(a, b);
}

var result = executeOperation(8, 4, function(x, y) {
    return x - y;
});

document.getElementById("output6").innerHTML = "8 - 4 = " + result;

Key Points

  • Function expressions are NOT hoisted - must be defined before calling
  • Can be anonymous (no name) or named
  • Can be passed as arguments to other functions
  • Can be assigned to variables
