Function Declaration

Function declarations use the function keyword and are hoisted to the top of their scope.

Syntax

function functionName(parameter1, parameter2) {
    // function body
    return result;
}

Examples

Example 1: Simple Greeting Function

// Function declaration - can be called before definition due to hoisting
function greet(name) {
    return "Hello, " + name + "!";
}

// Call the function
var message = greet("World");
document.getElementById("output1").innerHTML = message;

Example 2: Addition Function

function add(a, b) {
    return a + b;
}

var result = add(5, 3);
document.getElementById("output2").innerHTML = "5 + 3 = " + result;

Example 3: Function with No Parameters

function getCurrentDate() {
    return new Date().toLocaleDateString();
}

var date = getCurrentDate();
document.getElementById("output3").innerHTML = "Today's date: " + date;

Example 4: Hoisting Demonstration

// Calling function BEFORE its declaration - this works due to hoisting
var doubled = double(7);
document.getElementById("output4").innerHTML = "Double of 7: " + doubled;

// Function declaration - hoisted to top of scope
function double(num) {
    return num * 2;
}

Example 5: Multiple Functions

function calculateArea(width, height) {
    return width * height;
}

function calculatePerimeter(width, height) {
    return 2 * (width + height);
}

var area = calculateArea(5, 3);
var perimeter = calculatePerimeter(5, 3);
document.getElementById("output5").innerHTML = "Area: " + area + ", Perimeter: " + perimeter;

Key Points

  • Function declarations are hoisted - they can be called before they are defined
  • They start with the function keyword
  • They must have a name
  • They can accept parameters and return values
