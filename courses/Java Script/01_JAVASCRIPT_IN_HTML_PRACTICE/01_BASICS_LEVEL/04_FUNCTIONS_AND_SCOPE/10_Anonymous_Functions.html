Anonymous Functions

An anonymous function is a function without a name. They are often used as function expressions or callbacks.

Syntax

// Function expression
var myFunction = function(parameter) {
    // function body
    return result;
};

// Arrow function (also anonymous)
var myArrowFunction = (parameter) => result;

Examples

Example 1: Function Expression

var greet = function(name) {
    return "Hello, " + name + "!";
};

document.getElementById("output1").innerHTML = greet("World");

Example 2: Anonymous Functions as Callbacks

var numbers = [1, 2, 3, 4, 5];

var squared = numbers.map(function(num) {
    return num * num;
});

document.getElementById("output2").innerHTML = squared.join(", ");

Example 3: Arrow Functions (Anonymous)

var add = (a, b) => a + b;
var multiply = (a, b) => a * b;

document.getElementById("output3").innerHTML = "5 + 3 = " + add(5, 3) + "<br>4 * 7 = " + multiply(4, 7);

Example 4: Anonymous Function in Event

var message = "Button clicked!";
document.getElementById("output4").innerHTML = "Click the button to see action<br><button onclick='document.getElementById(\"output4\").innerHTML = message'>Click Me";

Example 5: Immediately Invoked Anonymous Function

var result = (function() {
    var x = 10;
    var y = 20;
    return x + y;
})();

document.getElementById("output5").innerHTML = "IIFE Result: " + result;

Key Points

  • Anonymous functions have no name
  • Function expressions assign anonymous functions to variables
  • Arrow functions are a shorter syntax for anonymous functions
  • Commonly used as callbacks in array methods
