Array.flat()

The flat() method creates a new array with all sub-array elements flattened.

1. Flatten One Level

const nested = [1, [2, 3], [4, 5]];
const flat = nested.flat();

2. Flatten Multiple Levels

const deep = [1, [2, [3, [4]]]];
const flat = deep.flat(2);

3. Flatten Completely (Infinity)

const veryDeep = [1, [2, [3, [4, [5]]]]];
const flat = veryDeep.flat(Infinity);
