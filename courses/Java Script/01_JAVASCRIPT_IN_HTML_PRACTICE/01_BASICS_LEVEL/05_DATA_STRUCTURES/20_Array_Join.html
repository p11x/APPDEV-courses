Array.join()

The join() method joins array elements into a string.

1. Join with Comma

const fruits = ["apple", "banana", "cherry"];
const str = fruits.join();

2. Join with Custom Separator

const arr = ["a", "b", "c"];
const str = arr.join("-");

3. Join Numbers

const nums = [1, 2, 3];
const str = nums.join(" ");
