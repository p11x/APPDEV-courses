WeakMap and WeakSet

Weak references - allow garbage collection of keys/values.

1. WeakMap (Keys must be objects)

const wm = new WeakMap();
const obj = { name: "test" };
wm.set(obj, "value");
wm.get(obj);

2. WeakSet (Stores objects only)

const ws = new WeakSet();
const obj1 = { a: 1 };
ws.add(obj1);
ws.has(obj1);
