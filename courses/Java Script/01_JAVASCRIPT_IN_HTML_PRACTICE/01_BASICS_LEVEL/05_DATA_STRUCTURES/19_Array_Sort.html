Array.sort()

The sort() method sorts elements (mutates original array).

1. Sort Strings

const fruits = ["banana", "apple", "cherry"];
fruits.sort();

2. Sort Numbers (Wrong - String Sort)

const nums = [10, 2, 30, 1];
nums.sort(); // "10" comes before "2"!

3. Sort Numbers (Correct - Comparator)

const nums = [10, 2, 30, 1];
nums.sort((a, b) => a - b);

4. Reverse Sort

const nums = [1, 5, 3, 9, 2];
nums.sort((a, b) => b - a);
