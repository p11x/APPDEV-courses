Array.find()

The find() method returns the first element that satisfies a condition.

1. Find Number Greater Than 10

const numbers = [5, 12, 8, 21, 3];
const result = numbers.find(num => num > 10);

2. Find Object by Property

const users = [
  { name: "John", age: 25 },
  { name: "Jane", age: 30 },
  { name: "Bob", age: 35 }
];
const result = users.find(user => user.age > 28);

3. Find String Starting With

const fruits = ["apple", "banana", "cherry", "date"];
const result = fruits.find(f => f.startsWith("b"));

4. Find Returns undefined if Not Found

const numbers = [1, 2, 3, 4, 5];
const result = numbers.find(num => num > 100);
