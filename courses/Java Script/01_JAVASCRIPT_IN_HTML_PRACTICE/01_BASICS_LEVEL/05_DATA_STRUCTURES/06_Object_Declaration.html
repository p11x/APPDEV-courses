Object Declaration Patterns

This file demonstrates different ways to create objects in JavaScript.

1. Object Literal

// Most common and simple way
const person = {
    name: "John",
    age: 30,
    city: "New York"
};
console.log(person);

2. Using new Object()

// Using Object constructor
const car = new Object();
car.brand = "Toyota";
car.model = "Camry";
car.year = 2023;
console.log(car);

3. Object with Methods

// Object can have functions as values
const calculator = {
    num1: 10,
    num2: 5,
    add: function() {
        return this.num1 + this.num2;
    },
    subtract: function() {
        return this.num1 - this.num2;
    }
};
console.log("Add:", calculator.add());
console.log("Subtract:", calculator.subtract());

4. Object.create() Method

// Create object with prototype
const animal = {
    speak: function() {
        return "Some sound";
    }
};
const dog = Object.create(animal);
dog.breed = "Golden Retriever";
console.log("Dog:", dog);
console.log("Prototype:", dog.speak());

5. Dynamic Property Names

// Computed property names
const key = "email";
const user = {
    name: "Alice",
    [key]: "alice@example.com"
};
console.log(user);

6. Object.assign() - Merge Objects

// Merge multiple objects into one
const obj1 = {a: 1};
const obj2 = {b: 2};
const obj3 = {c: 3};
const merged = Object.assign({}, obj1, obj2, obj3);
console.log(merged);

7. Object Destructuring

// Extract properties into variables
const employee = {
    name: "Bob",
    role: "Developer",
    salary: 50000
};
const {name, role} = employee;
console.log("Name:", name);
console.log("Role:", role);
