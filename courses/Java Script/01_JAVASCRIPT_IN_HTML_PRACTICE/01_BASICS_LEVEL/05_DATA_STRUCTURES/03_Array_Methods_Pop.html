Array Methods: pop() and shift()

These methods remove elements from an array.

1. pop() - Remove from End

// pop() removes and returns the LAST element
const fruits = ["apple", "banana", "orange"];
const removed = fruits.pop();
console.log("Removed:", removed);
console.log("Remaining:", fruits);

2. pop() on Empty Array

// pop() returns undefined for empty array
const empty = [];
const result = empty.pop();
console.log("Result:", result);

3. shift() - Remove from Start

// shift() removes and returns the FIRST element
const colors = ["red", "green", "blue"];
const removed = colors.shift();
console.log("Removed:", removed);
console.log("Remaining:", colors);

4. shift() Multiple Times

// Can call shift() multiple times
const nums = [1, 2, 3, 4, 5];
const first = nums.shift();
const second = nums.shift();
console.log("First:", first, "Second:", second);
console.log("Remaining:", nums);

5. Queue Implementation

// Using pop and shift together
const queue = ["A", "B", "C"];
// Remove from front (dequeue)
const served = queue.shift();
// Add to back (enqueue)
queue.push("D");
console.log("Served:", served);
console.log("Queue:", queue);
