Array.lastIndexOf()

The lastIndexOf() method returns the last index of an element.

1. Find Last Index

const fruits = ["apple", "banana", "cherry", "banana"];
const idx = fruits.lastIndexOf("banana");

2. Search from End

const arr = [1, 2, 3, 2, 1];
const idx = arr.lastIndexOf(2);

3. Returns -1 if Not Found

const nums = [1, 2, 3];
const idx = nums.lastIndexOf(5);
