Array.some()

The some() method checks if ANY element passes a test and returns true/false.

1. Check if Any Number is Even

const numbers = [1, 3, 5, 8, 9];
const hasEven = numbers.some(n => n % 2 === 0);

2. Check if Any Object Has Property

const users = [
  { name: "John", active: false },
  { name: "Jane", active: true }
];
const hasActive = users.some(u => u.active);

3. Returns false for Empty Array

const empty = [];
const result = empty.some(x => x > 0);
