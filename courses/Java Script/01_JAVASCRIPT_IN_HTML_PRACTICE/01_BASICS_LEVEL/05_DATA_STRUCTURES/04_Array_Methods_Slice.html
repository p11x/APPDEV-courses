Array Methods: slice() and splice()

slice() extracts portion; splice() adds/removes elements.

1. slice() - Extract Portion

// slice(start, end) - returns portion without modifying original
// end is exclusive
const fruits = ["apple", "banana", "orange", "mango", "grape"];
const selected = fruits.slice(1, 4);
console.log("Selected:", selected);
console.log("Original:", fruits);

2. slice() - Without End Parameter

// If end not specified, goes to end of array
const colors = ["red", "green", "blue", "yellow"];
const part = colors.slice(2);
console.log("Part:", part);
console.log("Original:", colors);

3. slice() - Negative Index

// Negative index counts from end
const nums = [1, 2, 3, 4, 5];
const lastThree = nums.slice(-3);
console.log("Last three:", lastThree);

4. splice() - Remove Elements

// splice(start, deleteCount) - removes and returns elements
// Modifies original array
const animals = ["cat", "dog", "elephant", "bird", "fish"];
const removed = animals.splice(1, 2);
console.log("Removed:", removed);
console.log("Remaining:", animals);

5. splice() - Insert Elements

// splice(start, deleteCount, items...) - insert at position
const letters = ["a", "b", "e", "f"];
letters.splice(2, 0, "c", "d");
console.log("Result:", letters);

6. splice() - Replace Elements

// splice can replace elements (delete + insert)
const nums = [10, 20, 30, 40, 50];
const replaced = nums.splice(1, 2, 25, 35);
console.log("Replaced:", replaced);
console.log("Result:", nums);
