Array.concat()

The concat() method merges arrays into a new array.

1. Merge Two Arrays

const arr1 = [1, 2];
const arr2 = [3, 4];
const merged = arr1.concat(arr2);

2. Merge Multiple Arrays

const a = [1], b = [2], c = [3];
const result = a.concat(b, c);

3. Add Values Directly

const arr = [1, 2];
const result = arr.concat(3, 4);
