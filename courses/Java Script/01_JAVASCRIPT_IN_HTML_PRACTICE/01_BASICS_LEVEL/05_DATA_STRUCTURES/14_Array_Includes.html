Array.includes()

The includes() method checks if an array contains a specific element.

1. Check if Array Contains Value

const fruits = ["apple","banana","orange"];
const hasBanana = fruits.includes("banana");

2. Check with Start Index

const arr = [1,2,3,4,5];
const has3 = arr.includes(3, 2);

3. Returns false for Non-Existent

const nums = [1,2,3];
const has5 = nums.includes(5);
