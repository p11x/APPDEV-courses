Object Merge Methods

This file demonstrates different ways to merge objects in JavaScript.

1. Using Spread Operator (...)

const obj1 = { a: 1, b: 2 };
const obj2 = { b: 3, c: 4 };
const merged = { ...obj1, ...obj2 };
// { a: 1, b: 3, c: 4 }

2. Using Object.assign()

const target = { a: 1 };
const source = { b: 2, c: 3 };
const result = Object.assign(target, source);
// { a: 1, b: 2, c: 3 }

3. Using Object.assign() - Multiple Sources

const obj1 = { a: 1 };
const obj2 = { b: 2 };
const obj3 = { c: 3 };
const merged = Object.assign({}, obj1, obj2, obj3);
// { a: 1, b: 2, c: 3 }

4. Merge with Nested Objects

const user = { name: "John", profile: { age: 25 } };
const updates = { profile: { age: 26, city: "NYC" } };
// Note: Spread only shallow merge
const shallow = { ...user, ...updates };
// profile is completely replaced
