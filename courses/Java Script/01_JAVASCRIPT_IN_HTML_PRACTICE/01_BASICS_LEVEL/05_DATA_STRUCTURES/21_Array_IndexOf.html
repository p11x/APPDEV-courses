Array.indexOf()

The indexOf() method returns the first index of an element.

1. Find Index of Element

const fruits = ["apple", "banana", "cherry", "banana"];
const idx = fruits.indexOf("banana");

2. Start Search from Index

const arr = ["a", "b", "c", "b"];
const idx = arr.indexOf("b", 2);

3. Returns -1 if Not Found

const nums = [1, 2, 3];
const idx = nums.indexOf(5);
