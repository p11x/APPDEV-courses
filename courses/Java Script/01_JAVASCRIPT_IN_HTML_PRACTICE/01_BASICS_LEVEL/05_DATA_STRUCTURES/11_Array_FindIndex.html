Array.findIndex()

The findIndex() method returns the index of the first element that satisfies a condition.

1. Find Index of Number

const numbers = [5, 12, 8, 21, 3];
const idx = numbers.findIndex(num => num > 10);

2. Find Index of Object

const users = [{name:"John"},{name:"Jane"},{name:"Bob"}];
const idx = users.findIndex(u => u.name === "Jane");

3. Returns -1 if Not Found

const fruits = ["apple","banana"];
const idx = fruits.findIndex(f => f === "orange");
