Object Deep Copy Methods

This file demonstrates different ways to create deep copies of objects in JavaScript.

1. Using Spread Operator (...) - Shallow Copy

// Spread operator - shallow copy only
const original = { name: "John", address: { city: "NYC" } };
const shallowCopy = { ...original };
shallowCopy.address.city = "LA";
console.log(original.address.city); // "LA" - also changed!

2. Using JSON.parse(JSON.stringify()) - Deep Copy

// JSON methods - creates true deep copy
const original2 = { name: "Jane", address: { city: "Boston" } };
const deepCopy = JSON.parse(JSON.stringify(original2));
deepCopy.address.city = "Seattle";
console.log(original2.address.city); // "Boston" - unchanged

3. Using Structured Clone (Modern)

// Structured clone - native deep copy
const original3 = { name: "Mike", hobbies: ["reading", "coding"] };
const clone = structuredClone(original3);
clone.hobbies.push("gaming");
console.log(original3.hobbies); // ["reading", "coding"]

4. Manual Deep Copy Function

// Custom recursive deep copy function
function deepCopy(obj) {
    if (obj === null || typeof obj !== 'object') return obj;
    if (Array.isArray(obj)) return obj.map(deepCopy);
    const copy = {};
    for (let key in obj) copy[key] = deepCopy(obj[key]);
    return copy;
}
