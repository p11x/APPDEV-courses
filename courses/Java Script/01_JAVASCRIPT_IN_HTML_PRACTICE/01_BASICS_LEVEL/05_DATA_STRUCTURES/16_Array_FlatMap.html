Array.flatMap()

The flatMap() method maps elements and flattens them into a new array.

1. Map and Flatten

const arr = [1, 2, 3];
const result = arr.flatMap(x => [x, x * 10]);

2. Split Sentence into Words

const sentences = ["hello world", "foo bar"];
const words = sentences.flatMap(s => s.split(" "));

3. Remove and Flatten

const nums = [1, 2, 3, 4];
const result = nums.flatMap(x => x % 2 === 0 ? [] : [x]);
