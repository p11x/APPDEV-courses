Array Methods: push() and unshift()

These methods add elements to an array.

1. push() - Add to End

// push() adds elements to the END of array
// Returns new length of array
const animals = ["cat", "dog"];
const newLength = animals.push("elephant");
console.log(animals);
console.log("New length:", newLength);

2. push() Multiple Elements

// push() can add multiple elements at once
const nums = [1, 2];
nums.push(3, 4, 5);
console.log(nums);

3. unshift() - Add to Start

// unshift() adds elements to the START of array
// Returns new length of array
const colors = ["blue", "green"];
const newLen = colors.unshift("red");
console.log(colors);
console.log("New length:", newLen);

4. unshift() Multiple Elements

// unshift() can add multiple elements at start
const letters = ["x", "y"];
letters.unshift("a", "b", "c");
console.log(letters);

5. Combined Example

// Using both push and unshift together
const queue = ["Person3", "Person4"];
queue.unshift("Person1", "Person2"); // Add to front
queue.push("Person5"); // Add to end
console.log(queue);
