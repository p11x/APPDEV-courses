Map Declaration and Methods

Map is a collection of key-value pairs with any key type.

1. Create Map with new Map()

const map = new Map();
map.set("name", "John");
map.set("age", 30);

2. Initialize with Array

const map = new Map([
  ["a", 1],
  ["b", 2]
]);

3. Map Methods - get, has, delete

const map = new Map([["x", 10]]);
map.get("x");
map.has("x");
map.delete("x");
