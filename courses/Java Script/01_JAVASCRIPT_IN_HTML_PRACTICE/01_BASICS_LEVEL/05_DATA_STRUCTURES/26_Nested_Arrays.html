Nested Arrays

Arrays containing other arrays as elements.

1. Create 2D Array

const matrix = [[1,2,3],[4,5,6],[7,8,9]];

2. Access Elements

const arr = [[1,2],[3,4]];
arr[0][1]; // 2

3. Modify Nested Elements

const data = [[1,2],[3,4]];
data[1][0] = 99;

4. Jagged Arrays

const jagged = [[1,2,3],[4,5],[6]];
// Different lengths allowed
