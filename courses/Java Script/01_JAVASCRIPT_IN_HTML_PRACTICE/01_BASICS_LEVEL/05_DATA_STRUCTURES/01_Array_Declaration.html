Array Declaration Methods

This file demonstrates different ways to create arrays in JavaScript.

1. Array Literal Syntax []

// Most common and recommended way
const fruits = ["apple", "banana", "orange"];
console.log(fruits);

2. Using Array Constructor

// Using new Array() constructor
const colors = new Array("red", "green", "blue");
console.log(colors);

3. Empty Array with push()

// Create empty array and add elements
const numbers = [];
numbers.push(10, 20, 30);
console.log(numbers);

4. Array with Mixed Types

// Arrays can hold different data types
const mixed = [1, "hello", true, null, {name: "John"}];
console.log(mixed);

5. Array.fill() for Initialization

// Create array with default values
const zeros = new Array(5).fill(0);
console.log(zeros);

6. Array.from() Static Method

// Create array from iterable or array-like object
const strArray = Array.from("hello");
console.log(strArray);
