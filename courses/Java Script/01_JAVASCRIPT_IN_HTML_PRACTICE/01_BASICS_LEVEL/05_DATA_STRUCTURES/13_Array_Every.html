Array.every()

The every() method checks if ALL elements pass a test and returns true/false.

1. Check if All Numbers are Positive

const numbers = [1, 2, 3, 4, 5];
const allPositive = numbers.every(n => n > 0);

2. Check if All Strings Have Length > 2

const fruits = ["apple","banana","kiwi"];
const allLong = fruits.every(f => f.length > 2);

3. Returns true for Empty Array

const empty = [];
const result = empty.every(x => x > 0);
