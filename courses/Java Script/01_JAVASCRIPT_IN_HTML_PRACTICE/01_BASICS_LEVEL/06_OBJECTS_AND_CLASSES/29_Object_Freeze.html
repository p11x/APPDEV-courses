Object.freeze()

Object.freeze() prevents modifications to an object's properties.

1. Freezing an Object

const obj = { name: "John", age: 30 }; Object.freeze(obj); obj.name = "Jane"; console.log(obj.name);

2. Adding/Removing Properties

const config = { theme: "dark" }; Object.freeze(config); config.newProp = "value"; delete config.theme; console.log(config);

3. Checking if Frozen

const obj1 = { a: 1 }; const obj2 = { b: 2 }; Object.freeze(obj1); console.log(Object.isFrozen(obj1)); console.log(Object.isFrozen(obj2));
