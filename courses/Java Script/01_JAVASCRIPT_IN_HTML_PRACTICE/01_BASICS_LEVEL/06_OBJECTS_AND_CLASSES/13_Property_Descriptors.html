Property Descriptors

Property descriptors define the attributes of object properties.

1. Getting Property Descriptors

const obj = { name: "John", age: 30 }; const desc = Object.getOwnPropertyDescriptor(obj, "name"); console.log(desc);

2. Defining Property Descriptors

const obj = {}; Object.defineProperty(obj, "name", { value: "John", writable: true, enumerable: true, configurable: true }); console.log(obj.name); obj.name = "Jane"; console.log(obj.name);

3. Non-writable Property

const obj = { name: "John" }; Object.defineProperty(obj, "name", { writable: false }); obj.name = "Jane"; console.log(obj.name);

4. Non-enumerable Property

const obj = { a: 1, b: 2 }; Object.defineProperty(obj, "secret", { value: "hidden", enumerable: false }); console.log(Object.keys(obj)); console.log(obj.secret);
