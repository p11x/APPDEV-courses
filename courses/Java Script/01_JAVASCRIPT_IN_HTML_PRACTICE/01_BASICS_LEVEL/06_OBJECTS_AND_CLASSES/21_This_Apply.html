This Apply Method

apply() invokes a function with a specific 'this' and an array of arguments.

1. Basic Apply

function greet(greeting, punctuation) { return `${greeting}, I'm ${this.name}${punctuation}`; } const person = { name: "Alice" }; console.log(greet.apply(person, ["Hello", "!"]));

2. Apply with Array

function sum(a, b, c) { return a + b + c; } const nums = [1, 2, 3]; console.log(sum.apply(null, nums));

3. Apply vs Call

function describe(lang, level) { return `${this.name}: ${lang} - ${level}`; } const dev = { name: "John" }; console.log(describe.apply(dev, ["JS", "expert"])); console.log(describe.call(dev, "JS", "expert"));
