This Bind

bind() creates a new function with 'this' bound to a specific value.

1. Basic Bind

const person = { name: "John" }; function greet() { return `Hello, ${this.name}`; } const boundGreet = greet.bind(person); console.log(boundGreet());

2. Bind with Arguments

function multiply(a, b) { return a * b * this.value; } const nums = { value: 2 }; const boundMultiply = multiply.bind(nums, 5); console.log(boundMultiply(3));

3. Preserving This in Callbacks

class Button { constructor() { this.label = "Click"; this.click = this.click.bind(this); } click() { return `Clicked ${this.label}`; } } const btn = new Button(); const handler = btn.click; console.log(handler());
