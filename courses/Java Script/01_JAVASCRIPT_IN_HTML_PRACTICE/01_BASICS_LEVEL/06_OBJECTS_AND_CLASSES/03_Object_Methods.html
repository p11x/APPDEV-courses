Object Methods in JavaScript

Methods are functions stored as object properties that can perform actions on object data.

1. Basic Method Definition

// Define methods inside an object const calculator = { num1: 10, num2: 5, add: function() { return this.num1 + this.num2; }, subtract: function() { return this.num1 - this.num2; } }; console.log(calculator.add()); // 15 console.log(calculator.subtract()); // 5

2. Shorthand Method Syntax (ES6)

// Modern shorthand method syntax const mathOps = { a: 8, b: 4, // No need for "function" keyword multiply() { return this.a * this.b; }, divide() { return this.a / this.b; } }; console.log(mathOps.multiply()); // 32 console.log(mathOps.divide()); // 2

3. Method with Parameters

// Methods can accept parameters const employee = { name: "John", salary: 5000, calculateAnnualBonus: function(percentage) { return (this.salary * percentage) / 100; }, calculateMonthlyNet: function(taxRate) { return this.salary - (this.salary * taxRate / 100); } }; console.log(employee.calculateAnnualBonus(10)); // 500 console.log(employee.calculateMonthlyNet(20)); // 4000

4. Methods Returning Objects

// Methods can return new objects const rectangle = { width: 10, height: 5, getArea: function() { return { area: this.width * this.height, dimensions: `${this.width}x${this.height}` }; } }; const result = rectangle.getArea(); console.log(result);

5. Using 'this' in Methods

// 'this' refers to the object itself const bankAccount = { accountNumber: "ACC123456", balance: 1000, deposit: function(amount) { this.balance += amount; return `Deposited: $${amount}. New balance: $${this.balance}`; }, withdraw: function(amount) { if (amount <= this.balance) { this.balance -= amount; return `Withdrawn: $${amount}. Remaining: $${this.balance}`; } return "Insufficient funds"; } }; console.log(bankAccount.deposit(500)); console.log(bankAccount.withdraw(300));
