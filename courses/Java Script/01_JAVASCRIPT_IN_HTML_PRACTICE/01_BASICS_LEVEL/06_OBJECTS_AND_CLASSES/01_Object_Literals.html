Object Literals in JavaScript

Object literals are the simplest way to create objects in JavaScript using curly braces.

1. Basic Object Literal

// Creating an object with key-value pairs const person = { name: "John", age: 30, city: "New York" }; console.log(person);

2. Object with Mixed Data Types

// Object can hold various data types const student = { name: "Alice", age: 22, isEnrolled: true, grades: [85, 90, 78], address: { street: "123 Main St", city: "Boston" } }; console.log(student);

3. Computed Property Names

// Using square brackets for dynamic property names const key = "email"; const user = { name: "Mike", [key]: "mike@example.com" }; console.log(user);

4. Shorthand Property Syntax

// Shorthand when variable names match property names const firstName = "Sarah"; const lastName = "Johnson"; const age = 28; const person2 = { firstName, lastName, age }; console.log(person2);
