Object Composition

Composition combines objects to create complex behavior instead of inheritance.

1. Composition over Inheritance

const walking = (obj) => ({ walk: () => `${obj.name} is walking` }); const swimming = (obj) => ({ swim: () => `${obj.name} is swimming` }); const createDuck = (name) => { const duck = { name }; return Object.assign(duck, walking(duck), swimming(duck)); }; const duck = createDuck("Donald"); console.log(duck.walk(), duck.swim());

2. Composable Functions

const withAge = (obj) => { obj.getAge = () => 2026 - obj.birthYear; return obj; }; const withName = (obj) => { obj.greet = () => `Hello, I'm ${obj.name}`; return obj; }; const person = withAge(withName({ name: "Alice", birthYear: 1995 })); console.log(person.greet(), person.getAge());

3. Factory Pattern

const createLogger = () => ({ log: (msg) => console.log(msg) }); const createStorage = () => ({ save: (data) => localStorage.setItem("data", data) }); const createApp = () => { const components = [createLogger(), createStorage()]; return { run(data) { components[0].log("Saving..."); components[1].save(data); } }; }; const app = createApp(); app.run("test data");
