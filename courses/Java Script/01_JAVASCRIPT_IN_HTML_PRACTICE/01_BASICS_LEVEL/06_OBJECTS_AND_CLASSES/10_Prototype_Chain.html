Prototype Chain

Every JavaScript object has a prototype property that forms a chain of inheritance.

1. Understanding __proto__ Chain

const animal = { species: "Animal", breathe() { return "breathing..."; } }; const dog = { breed: "Labrador", bark() { return "Woof!"; } }; dog.__proto__ = animal; console.log(dog.species); console.log(dog.breathe()); console.log(dog.bark());

2. Prototype Chain Lookup

const grandparent = { grand: "property" }; const parent = { parent: "property" }; const child = { child: "property" }; parent.__proto__ = grandparent; child.__proto__ = parent; console.log(child.grand); console.log(child.parent); console.log(child.child);

3. Object.getPrototypeOf

function Animal(name) { this.name = name; } Animal.prototype.getName = function() { return this.name; }; const cat = new Animal("Whiskers"); const proto = Object.getPrototypeOf(cat); console.log(proto === Animal.prototype); console.log(Object.getPrototypeOf(cat).constructor.name);
