Object.seal()

Object.seal() prevents adding/removing properties but allows modifying existing ones.

1. Sealing an Object

const obj = { name: "John", age: 30 }; Object.seal(obj); obj.name = "Jane"; obj.city = "NYC"; console.log(obj.name);

2. Cannot Add/Remove Properties

const settings = { theme: "dark" }; Object.seal(settings); settings.theme = "light"; delete settings.theme; console.log(settings);

3. Seal vs Freeze

const sealed = { a: 1 }; const frozen = { b: 2 }; Object.seal(sealed); Object.freeze(frozen); sealed.a = 10; frozen.b = 20; sealed.c = 3; frozen.d = 4; console.log("Sealed:", sealed.a, sealed.c); console.log("Frozen:", frozen.b, frozen.d);
