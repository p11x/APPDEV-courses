Property Getters

Getters allow you to define computed properties in objects.

1. Basic Getter

const person = { firstName: "John", lastName: "Doe", get fullName() { return `${this.firstName} ${this.lastName}`; } }; console.log(person.fullName);

2. Getter with Computed Value

const rectangle = { width: 10, height: 5, get area() { return this.width * this.height; } }; console.log(rectangle.area);

3. Getter in Class

class Circle { constructor(radius) { this.radius = radius; } get diameter() { return this.radius * 2; } get area() { return Math.PI * this.radius ** 2; } } const circle = new Circle(5); console.log(circle.diameter); console.log(circle.area.toFixed(2));
