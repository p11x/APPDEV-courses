Static Blocks

Static blocks allow initialization of static properties in a class.

1. Basic Static Block

class Config { static baseUrl; static apiKey; static { this.baseUrl = "https://api.example.com"; this.apiKey = "abc123"; } } console.log(Config.baseUrl);

2. Static Block with Computations

class MathConsts { static pi; static e; static { this.pi = Math.PI; this.e = Math.E; } } console.log(MathConsts.pi.toFixed(4));

3. Multiple Static Blocks

class Logger { static levels = []; static { this.levels.push("INFO"); } static { this.levels.push("WARN"); } static { this.levels.push("ERROR"); } } console.log(Logger.levels);
