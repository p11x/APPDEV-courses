Object.keys()

Object.keys() returns an array of an object's own enumerable property names.

1. Basic Object.keys()

const obj = { a: 1, b: 2, c: 3 }; const keys = Object.keys(obj); console.log(keys);

2. Iterate Keys

const user = { name: "John", age: 30, city: "NYC" }; Object.keys(user).forEach(key => { console.log(key); });

3. Non-enumerable Keys

const obj = { a: 1 }; Object.defineProperty(obj, "hidden", { value: "secret", enumerable: false }); console.log(Object.keys(obj)); console.log(obj.hidden);
