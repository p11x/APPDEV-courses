Property Setters

Setters allow you to define custom logic when setting property values.

1. Basic Setter

const person = { _name: "John", set name(value) { this._name = value.toUpperCase(); }, get name() { return this._name; } }; person.name = "jane"; console.log(person.name);

2. Setter with Validation

const user = { _age: 0, set age(value) { if (value < 0) { this._age = 0; } else if (value > 150) { this._age = 150; } else { this._age = value; } }, get age() { return this._age; } }; user.age = -5; console.log(user.age); user.age = 200; console.log(user.age); user.age = 25; console.log(user.age);

3. Setter in Class

class Temperature { constructor(celsius) { this.celsius = celsius; } set fahrenheit(value) { this.celsius = (value - 32) * 5 / 9; } get fahrenheit() { return this.celsius * 9 / 5 + 32; } } const temp = new Temperature(25); console.log(temp.fahrenheit); temp.fahrenheit = 100; console.log(temp.celsius);
