Object Properties - Access and Modification

Learn how to access and modify object properties using different methods.

1. Dot Notation Access

// Access properties using dot notation const car = { brand: "Toyota", model: "Camry", year: 2022 }; console.log(car.brand); // Output: Toyota console.log(car.model); // Output: Camry console.log(car.year); // Output: 2022

2. Bracket Notation Access

// Access properties using bracket notation const product = { "product-name": "Laptop", price: 999, category: "Electronics" }; console.log(product["product-name"]); // Output: Laptop console.log(product["price"]); // Output: 999 console.log(product["category"]); // Output: Electronics

3. Modifying Properties

// Modify existing properties and add new ones const person = { name: "John", age: 25 }; person.age = 26; // Modify existing person.city = "Boston"; // Add new property person["country"] = "USA"; // Add using bracket notation console.log(person);

4. Property Descriptors

// Using Object.defineProperty for property control const user = { name: "Alice" }; Object.defineProperty(user, "id", { value: 101, writable: false, // Cannot be modified enumerable: true, configurable: false }); console.log(user.id); // user.id = 102; // Error in strict mode (silent in normal)

5. Checking and Deleting Properties

// Check if property exists and delete const book = { title: "JavaScript Guide", author: "John Doe", pages: 300 }; console.log("title" in book); // true console.log("publisher" in book); // false console.log(book.hasOwnProperty("author")); // true delete book.pages; // Delete property console.log(book);
