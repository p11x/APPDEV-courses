Object.create() for Prototypes

Object.create() creates a new object with a specified prototype.

1. Creating Object with Prototype

const personProto = { greet() { return `Hello, I'm ${this.name}`; } }; const person = Object.create(personProto); person.name = "John"; person.age = 30; console.log(person.greet());

2. Null Prototype Object

const nullProto = Object.create(null); nullProto.name = "Test"; console.log("Keys:", Object.keys(nullProto)); console.log("ToString:", nullProto.toString);

3. Deep Inheritance with Object.create

const animal = { classify() { return `I'm a ${this.species}`; } }; const mammal = Object.create(animal); mammal.hasFur = true; const dog = Object.create(mammal); dog.species = "Dog"; dog.breed = "Beagle"; console.log(dog.classify()); console.log(dog.hasFur);
