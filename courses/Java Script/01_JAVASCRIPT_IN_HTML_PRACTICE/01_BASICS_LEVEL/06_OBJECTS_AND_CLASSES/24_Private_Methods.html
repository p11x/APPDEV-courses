Private Methods

Private methods (#) cannot be called from outside the class.

1. Basic Private Method

class Calculator { #add(a, b) { return a + b; } compute(a, b) { return this.#add(a, b); } } const calc = new Calculator(); console.log(calc.compute(5, 3));

2. Private Method for Validation

class User { #validateAge(age) { return age >= 0 && age <= 150; } setAge(age) { if (this.#validateAge(age)) { this.age = age; return "Age set"; } return "Invalid age"; } } const user = new User(); console.log(user.setAge(25)); console.log(user.setAge(-5));

3. Private Static Method

class MathUtil { static #sqrt(n) { return Math.sqrt(n); } static getSqrt(n) { return this.#sqrt(n); } } console.log(MathUtil.getSqrt(16));
