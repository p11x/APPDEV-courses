This in Arrow Functions

Arrow functions do not have their own 'this' - they inherit it from the enclosing scope.

1. Arrow Function Inheritance

const person = { name: "John", regularFunc() { return this.name; }, arrowFunc: () => this.name }; console.log(person.regularFunc()); console.log(person.arrowFunc());

2. Arrow in Class Methods

class Counter { constructor() { this.count = 0; this.increment = () => { this.count++; return this.count; }; } } const counter = new Counter(); console.log(counter.increment()); console.log(counter.increment());

3. Callback with Arrow Function

const obj = { value: 10, process() { const doubled = () => this.value * 2; return doubled(); } }; console.log(obj.process());
