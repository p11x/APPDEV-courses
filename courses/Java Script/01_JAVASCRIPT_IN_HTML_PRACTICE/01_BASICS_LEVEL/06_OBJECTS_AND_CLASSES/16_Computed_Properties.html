Computed Properties

Square brackets [] allow dynamic property names in objects.

1. Basic Computed Property

const key = "name"; const obj = { [key]: "John" }; console.log(obj.name);

2. Multiple Computed Properties

const prefix = "user"; const obj = { [prefix + "Name"]: "Alice", [prefix + "Age"]: 25, [prefix + "Id"]: 101 }; console.log(obj);

3. Computed Keys with Symbols

const sym = Symbol("id"); const obj = { [sym]: 123, name: "John" }; console.log(obj[sym]);

4. Computed in Class

const propName = "fullName"; class Person { constructor(first, last) { this[propName] = `${first} ${last}`; } } const p = new Person("John", "Doe"); console.log(p.fullName);
