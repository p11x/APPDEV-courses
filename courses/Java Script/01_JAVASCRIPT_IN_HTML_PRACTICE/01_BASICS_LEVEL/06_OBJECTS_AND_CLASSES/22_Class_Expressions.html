Class Expressions

Classes can be expressed as variables, similar to function expressions.

1. Basic Class Expression

const Rectangle = class { constructor(width, height) { this.width = width; this.height = height; } area() { return this.width * this.height; } }; const rect = new Rectangle(5, 3); console.log(rect.area());

2. Named Class Expression

const Shape = class Rectangle { constructor(type) { this.type = type; } describe() { return `I am a ${this.type}`; } }; const shape = new Shape("Circle"); console.log(shape.describe());

3. Class Expression as Parameter

function createInstance(ClassRef) { return new ClassRef("Created"); } const result = createInstance(class { constructor(msg) { this.message = msg; } }); console.log(result.message);
