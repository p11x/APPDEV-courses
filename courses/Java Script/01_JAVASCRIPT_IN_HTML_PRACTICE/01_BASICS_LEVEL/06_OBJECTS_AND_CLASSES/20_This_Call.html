This Call Method

call() invokes a function with a specific 'this' value and individual arguments.

1. Basic Call

function introduce(greeting) { return `${greeting}, I'm ${this.name}`; } const person = { name: "Alice" }; console.log(introduce.call(person, "Hello"));

2. Call with Multiple Arguments

function add(lang, level) { return `${this.name} knows ${lang} at ${level} level`; } const dev = { name: "John" }; console.log(add.call(dev, "JavaScript", "advanced"));

3. Borrowing Methods

const car = { brand: "Toyota", getInfo() { return this.brand; } }; const bike = { brand: "Harley" }; console.log(car.getInfo.call(bike));
