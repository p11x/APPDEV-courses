Object.create()

Object.create() creates a new object with the specified prototype object and properties.

1. Basic Object.create()

const obj = Object.create({ name: "default" }); console.log(obj.name); obj.name = "custom"; console.log(obj.name);

2. Object.create with Methods

const mathOps = { add(a, b) { return a + b; }, subtract(a, b) { return a - b; } }; const calculator = Object.create(mathOps); console.log(calculator.add(5, 3)); console.log(calculator.subtract(10, 4));

3. Property Descriptors in Object.create

const obj = Object.create(Object.prototype, { name: { value: "John", writable: true, enumerable: true, configurable: true }, age: { value: 30, writable: false, enumerable: true, configurable: false } }); console.log(obj.name); obj.name = "Jane"; console.log(obj.name);
