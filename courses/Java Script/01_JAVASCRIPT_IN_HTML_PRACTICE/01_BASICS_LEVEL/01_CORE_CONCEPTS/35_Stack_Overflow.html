Stack Overflow

Stack overflow occurs when the call stack exceeds its limit, usually from excessive recursion.

Common Causes:

Infinite Recursion - No base case or wrong condition
Deep Recursion - Too many function calls
Circular Reference - Functions calling each other

Solution:

Add a base case and ensure progress toward it
