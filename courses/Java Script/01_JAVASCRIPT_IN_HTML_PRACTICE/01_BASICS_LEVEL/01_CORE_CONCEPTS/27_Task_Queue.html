Task Queue

The Task Queue (same as Callback Queue) stores macrotasks that are ready to be executed.

Key Points:

FIFO Order - First In, First Out processing
One at a time - Only one task runs per event loop cycle
After microtasks - Runs after all microtasks complete
Async sources - setTimeout, I/O, UI events
