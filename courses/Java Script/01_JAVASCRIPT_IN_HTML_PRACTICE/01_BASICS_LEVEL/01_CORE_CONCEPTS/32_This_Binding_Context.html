This Binding Context

The 'this' keyword in JavaScript refers to the current execution context, which varies based on how a function is called.

Binding Rules:

Global/Default - In strict mode: undefined. Non-strict: window
Method Call - Object.method(): this = object
Arrow Function - Inherits 'this' from parent scope
New Keyword - new Constructor(): this = new instance
