Comments in JavaScript

Comments are used to explain code, make notes, and temporarily disable code during development.

1. Single-line Comments

Using //

Single-line comments start with // and comment everything after it on that line.

// This is a single-line comment
var name = "John"; // This is an inline comment
// console.log("This line is commented out");

2. Multi-line Comments

Using /* */

Multi-line comments start with /* and end with */. They can span multiple lines.

/*
 This is a
 multi-line
 comment block
*/
