Promise Queue (Microtask Queue)

Promise handlers (.then, .catch, .finally) are queued in a separate microtask queue with higher priority.

How Promise Queue Works:

Separate Queue - Priority over macrotask queue
All at Once - ALL microtasks run before macrotasks
FIFO - In order they were added
