Garbage Collection

JavaScript uses automatic garbage collection to reclaim memory from objects that are no longer in use.

GC Algorithms:

Mark and Sweep - Mark reachable objects, sweep unmarked
Mark and Compact - Mark, sweep, and compact memory
Generational GC - Different rules for new/old objects
Incremental GC - Small steps to avoid pauses
