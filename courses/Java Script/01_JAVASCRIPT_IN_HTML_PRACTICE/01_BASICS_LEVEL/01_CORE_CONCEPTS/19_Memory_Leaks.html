Memory Leaks

Memory leaks occur when JavaScript holds references to objects that are no longer needed, preventing garbage collection.

Common Causes:

Global Variables - Accidental global variables in non-strict mode
Event Listeners - Not removing old event listeners
Closures - References retained in closures
timers/setInterval - Not clearing intervals
