Heap Memory

The heap is where JavaScript stores objects, arrays, and functions. It's where dynamic memory allocation happens.

Memory Areas:

Stack - Fixed size, stores primitives and references
Heap - Dynamic, stores objects and closures
Code Area - Stores executable code
Call Stack - Function execution tracking
