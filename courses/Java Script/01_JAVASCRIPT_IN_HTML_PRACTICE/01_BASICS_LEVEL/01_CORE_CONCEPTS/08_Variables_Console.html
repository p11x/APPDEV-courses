Variables and Console

JavaScript has three ways to declare variables: var, let, and const. Each has different scope and behavior.

Variable Types:

var - Function-scoped, can be re-declared and updated
let - Block-scoped, can be updated but not re-declared in same scope
const - Block-scoped, cannot be updated or re-declared

Try It:

📋 Instructions: Open browser console (F12) to see console.log output. Click buttons to test variable behaviors.
