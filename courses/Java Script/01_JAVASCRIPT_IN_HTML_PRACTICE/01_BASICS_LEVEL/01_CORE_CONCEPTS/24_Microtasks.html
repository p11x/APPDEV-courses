Microtasks

Microtasks are tasks that are executed asynchronously but with higher priority than macrotasks.

Microtask Sources:

Promise.then() - Promise callback
Promise.catch() - Error handler
Promise.finally() - Finally handler
queueMicrotask() - Explicit queue
