Network Panel

The Network panel in DevTools shows all HTTP requests made by the page, including timing and response details.

Key Features:

Request List - All HTTP requests with status, type, size, time
Timing Details - DNS, connect, SSL, TTFB, download
Request/Response - Headers, payload, response body
Filtering - Filter by type, domain, status
