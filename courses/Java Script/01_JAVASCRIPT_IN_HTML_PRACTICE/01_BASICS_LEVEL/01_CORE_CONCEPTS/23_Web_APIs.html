Web APIs

Web APIs are browser-provided interfaces that extend JavaScript, enabling async operations and DOM access.

Common Web APIs:

setTimeout/setInterval - Timer functions
fetch - Network requests
DOM APIs - Document manipulation
Storage APIs - localStorage, sessionStorage
Worker APIs - Web Workers
