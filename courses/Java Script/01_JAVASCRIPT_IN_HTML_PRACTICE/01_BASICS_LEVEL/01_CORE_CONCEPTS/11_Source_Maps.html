Source Maps

Source maps are files that map compiled/minified code back to original source code, making debugging easier.

What Are Source Maps?

Purpose: Enable debugging of minified JS as if it were the original source
How it works: Browser reads the source map file to map positions back to original source
File format: JSON file ending with .map (e.g., app.js.map)

Source Map Reference:

// In your minified JS file, add this comment at the end:
//# sourceMappingURL=app.js.map

// Or set source map in JavaScript:
//# sourceMappingURL=http://localhost/app.js.map
