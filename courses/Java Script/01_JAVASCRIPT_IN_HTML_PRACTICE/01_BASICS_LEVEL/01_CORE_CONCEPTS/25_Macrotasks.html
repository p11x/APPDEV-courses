Macrotasks

Macrotasks are async tasks that are processed after the current call stack and all microtasks are complete.

Macrotask Sources:

setTimeout - Timer callbacks
setInterval - Interval callbacks
setImmediate - Immediate callbacks (Node.js)
I/O - File/Network operations
UI rendering - Browser repaints
