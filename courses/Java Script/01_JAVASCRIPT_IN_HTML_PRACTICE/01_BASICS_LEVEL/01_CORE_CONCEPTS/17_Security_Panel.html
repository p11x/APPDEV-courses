Security Panel

The Security panel in DevTools shows security details for the current page including SSL/TLS information.

Key Features:

Connection Security - HTTPS/SSL status and certificate
Certificate Details - Validity, issuer, expiry
Secure Context - Whether page can access sensitive APIs
Subresource Security - Check all loaded resources
