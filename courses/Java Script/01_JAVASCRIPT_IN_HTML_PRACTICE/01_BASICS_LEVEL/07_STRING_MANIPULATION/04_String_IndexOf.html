4. String indexOf and lastIndexOf

The indexOf() method returns the first occurrence of a specified value. The lastIndexOf() method returns the last occurrence.

Example 1: indexOf Basic

let text = "Hello World";
let position = text.indexOf("World");
console.log(position);

Example 2: indexOf Not Found

let text = "Hello World";
let position = text.indexOf("JavaScript");
console.log(position);

Example 3: lastIndexOf

let text = "Hello World World";
let position = text.lastIndexOf("World");
console.log(position);

Example 4: indexOf with Start Position

let text = "Hello World World";
let pos1 = text.indexOf("World");
let pos2 = text.indexOf("World", pos1 + 1);
console.log("First:", pos1, "Second:", pos2);
