1. String Declaration in JavaScript

Strings can be created using single quotes, double quotes, or backticks.

Method 1: Double Quotes

let greeting1 = "Hello, World!";
console.log(greeting1);

Method 2: Single Quotes

let greeting2 = 'Hello, World!';
console.log(greeting2);

Method 3: Backticks (Template Literals)

let greeting3 = `Hello, World!`;
console.log(greeting3);

Method 4: String Constructor

let greeting4 = new String("Hello, World!");
console.log(greeting4);
