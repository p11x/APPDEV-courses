8. String Template Literals

Template literals use backticks (`) to define strings. They support multi-line strings and interpolation with ${}.

Example 1: Basic Template Literal

let name = "John";
let greeting = `Hello, ${name}!`;
console.log(greeting);

Example 2: Multi-line String

let message = `Line 1
Line 2
Line 3`;
console.log(message);

Example 3: Expression Interpolation

let a = 5;
let b = 3;
let result = `Sum: ${a + b}, Product: ${a * b}`;
console.log(result);

Example 4: Template Literal with Function

let price = 100;
let tax = 0.1;
let total = `Total: $${(price * (1 + tax)).toFixed(2)}`;
console.log(total);
