5. String Slice Method

The slice() method extracts a portion of a string and returns it as a new string.

Example 1: Slice with Start and End

let text = "Hello World";
let result = text.slice(0, 5);
console.log(result);

Example 2: Slice from Start to End

let text = "Hello World";
let result = text.slice(6);
console.log(result);

Example 3: Slice with Negative Indices

let text = "Hello World";
let result = text.slice(-5);
console.log(result);

Example 4: Slice with Negative Start and End

let text = "Hello World";
let result = text.slice(-6, -1);
console.log(result);
