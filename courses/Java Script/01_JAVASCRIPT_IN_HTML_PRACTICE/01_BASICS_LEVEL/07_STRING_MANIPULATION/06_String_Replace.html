6. String Replace and ReplaceAll

The replace() method replaces a specified value with another value. The replaceAll() method replaces all occurrences.

Example 1: replace() Basic

let text = "Hello World";
let result = text.replace("World", "JavaScript");
console.log(result);

Example 2: replace() with Regex (Global)

let text = "Hello World World";
let result = text.replace(/World/g, "JavaScript");
console.log(result);

Example 3: replaceAll()

let text = "Hello World World";
let result = text.replaceAll("World", "JavaScript");
console.log(result);

Example 4: replace() Case Insensitive

let text = "Hello World WORLD";
let result = text.replace(/world/gi, "JavaScript");
console.log(result);
