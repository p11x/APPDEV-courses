9. String charAt()

The charAt() method returns the character at a specified index.

Syntax

str.charAt(index)

Example 1: Get character at index

let text = "Hello";
let char = text.charAt(0);
console.log(char);

Example 2: Get last character

let text = "JavaScript";
let lastChar = text.charAt(text.length - 1);
console.log(lastChar);
