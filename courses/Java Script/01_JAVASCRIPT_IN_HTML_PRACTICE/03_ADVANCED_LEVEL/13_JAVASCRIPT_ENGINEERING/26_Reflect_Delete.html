Reflect.deleteProperty

Reflect.deleteProperty Method

Reflect.deleteProperty(target, propertyKey) deletes a property. Returns a boolean indicating success.

const obj = { a: 1, b: 2, c: 3 };

const result = Reflect.deleteProperty(obj, 'b');
console.log(result); // true
console.log(obj); // { a: 1, c: 3 }

// Non-configurable property returns false
const frozen = Object.freeze({ x: 1 });
const result2 = Reflect.deleteProperty(frozen, 'x');
console.log(result2); // false
