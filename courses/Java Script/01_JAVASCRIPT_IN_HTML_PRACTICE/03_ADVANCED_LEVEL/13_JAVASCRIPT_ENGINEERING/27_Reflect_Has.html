Reflect.has

Reflect.has Method

Reflect.has(target, propertyKey) returns a boolean indicating whether the target has the property (like in operator).

const obj = { x: 1, y: 2 };

console.log(Reflect.has(obj, 'x')); // true
console.log(Reflect.has(obj, 'z')); // false
console.log('y' in obj); // true (equivalent)
