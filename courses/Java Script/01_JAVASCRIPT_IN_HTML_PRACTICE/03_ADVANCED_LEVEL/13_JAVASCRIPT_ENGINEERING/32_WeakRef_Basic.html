WeakRef - Basic

WeakRef Constructor

WeakRef creates a weak reference to an object, allowing garbage collection while still referencing the object.

let obj = { name: 'Strong Object' };
const weakRef = new WeakRef(obj);

console.log(weakRef.deref()); // { name: 'Strong Object' }

obj = null;
// After GC, weakRef.deref() may return undefined
