Proxy Pattern

JavaScript Proxy Object

The Proxy object allows you to create an object that can intercept and redefine fundamental operations like property access, assignment, enumeration, and function invocation.

// Proxy with get and set traps
const handler = {
    get(target, property) {
        console.log(`Getting property: ${property}`);
        return target[property];
    },
    set(target, property, value) {
        console.log(`Setting ${property} = ${value}`);
        target[property] = value;
        return true;
    }
};

const target = { name: 'John', age: 30 };
const proxy = new Proxy(target, handler);
proxy.name; // Triggers get
proxy.age = 31; // Triggers set
