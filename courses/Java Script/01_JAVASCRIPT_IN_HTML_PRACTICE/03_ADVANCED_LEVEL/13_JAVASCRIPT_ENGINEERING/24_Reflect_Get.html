Reflect.get

Reflect.get Method

Reflect.get(target, propertyKey[, receiver]) returns the value of a property. It works like property accessor but as a function.

const obj = { x: 10, y: 20 };

console.log(Reflect.get(obj, 'x')); // 10
console.log(Reflect.get(obj, 'y')); // 20

// With receiver for getter
const parent = { value: 100 };
const child = {
    get value() { return this.value; }
};
console.log(Reflect.get(child, 'value', parent)); // 100
