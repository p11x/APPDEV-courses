FinalizationRegistry

FinalizationRegistry API

FinalizationRegistry lets you register callbacks to be invoked when an object is garbage collected.

const registry = new FinalizationRegistry((heldValue) => {
    console.log(`Cleaned up: ${heldValue}`);
});

let obj = { data: 'some data' };
registry.register(obj, 'my cleanup message');

// When obj becomes unreachable and is garbage collected,
// the callback will be called (not guaranteed when)
