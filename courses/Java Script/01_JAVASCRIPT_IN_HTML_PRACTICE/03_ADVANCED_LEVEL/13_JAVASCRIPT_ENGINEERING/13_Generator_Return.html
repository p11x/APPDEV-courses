Generator - Return

Generator Return Method

The return() method returns a given value and finishes the generator. The throw() method throws an error into the generator.

function* numbers() {
    yield 1;
    yield 2;
    yield 3;
}

const gen = numbers();
console.log(gen.next());    // { value: 1, done: false }
console.log(gen.return(0)); // { value: 0, done: true }
console.log(gen.next());    // { value: undefined, done: true }
