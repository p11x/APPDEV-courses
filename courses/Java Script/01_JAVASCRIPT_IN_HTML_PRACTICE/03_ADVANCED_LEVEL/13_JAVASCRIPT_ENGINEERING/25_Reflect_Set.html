Reflect.set

Reflect.set Method

Reflect.set(target, propertyKey, value[, receiver]) sets a property value. Returns a boolean indicating success.

const obj = {};
Reflect.set(obj, 'name', 'John');
console.log(obj.name); // "John"

// Returns true/false for success
const result = Reflect.set(obj, 'x', 100);
console.log(result); // true (property was set successfully)
