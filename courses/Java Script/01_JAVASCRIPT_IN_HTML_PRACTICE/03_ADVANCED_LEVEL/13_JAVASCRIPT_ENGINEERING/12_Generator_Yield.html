Generator - Yield Values

Yield Expressions

The yield expression pauses the generator and sends a value back. yield* delegates to another generator.

function* numberGenerator() {
    yield 10;
    yield 20;
    yield 30;
}

function* combinedGenerator() {
    yield 1;
    yield* numberGenerator(); // Delegate to another generator
    yield 40;
}

const gen = combinedGenerator();
console.log(gen.next().value); // 1
console.log(gen.next().value); // 10
console.log(gen.next().value); // 20
