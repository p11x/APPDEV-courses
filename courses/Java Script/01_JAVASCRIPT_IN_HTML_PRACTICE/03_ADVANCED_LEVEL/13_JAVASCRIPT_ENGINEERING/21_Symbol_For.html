Symbol.for and Symbol.keyFor

Global Symbol Registry

Symbol.for(key) creates a Symbol in the global registry. Symbol.keyFor(sym) retrieves the key for a global Symbol.

const sym1 = Symbol.for('shared');
const sym2 = Symbol.for('shared');

console.log(sym1 === sym2); // true - same symbol!
console.log(Symbol.keyFor(sym1)); // "shared"

// Regular Symbol (not in registry)
const sym3 = Symbol('notShared');
console.log(Symbol.keyFor(sym3)); // undefined
