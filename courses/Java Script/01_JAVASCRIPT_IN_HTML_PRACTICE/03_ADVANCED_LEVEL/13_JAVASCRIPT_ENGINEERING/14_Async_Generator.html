Async Generators

Async generators combine async operations with generator functionality. They use async function* syntax and can yield promises.

async function* asyncNumbers() {
    yield Promise.resolve(1);
    yield Promise.resolve(2);
    yield Promise.resolve(3);
}

async function demo() {
    for await (const num of asyncNumbers()) {
        console.log(num); // 1, 2, 3
    }
}
