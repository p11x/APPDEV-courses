Symbol Description

Symbol Description Property

Symbols can have an optional description that can be accessed via the description property.

const sym1 = Symbol('first');
const sym2 = Symbol('second');
const sym3 = Symbol();

console.log(sym1.description); // "first"
console.log(sym2.description); // "second"
console.log(sym3.description); // undefined
console.log(Symbol('test').description); // "test"
