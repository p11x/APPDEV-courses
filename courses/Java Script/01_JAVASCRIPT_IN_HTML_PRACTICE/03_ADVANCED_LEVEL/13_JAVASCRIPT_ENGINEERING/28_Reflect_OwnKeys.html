Reflect.ownKeys

Reflect.ownKeys Method

Reflect.ownKeys(target) returns an array of the target's own property keys (symbols included, not inherited).

const obj = { a: 1, b: 2 };
const sym = Symbol('c');
obj[sym] = 3;

console.log(Reflect.ownKeys(obj)); // ['a', 'b', Symbol(c)]

// Unlike Object.keys() which only returns strings
console.log(Object.keys(obj)); // ['a', 'b']
