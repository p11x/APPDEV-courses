24 - Routing Dynamic

What is Dynamic Routing?

Dynamic routes use parameters to match URL patterns. For example, `/user/:id` matches `/user/1`, `/user/42`, etc. The parameter value is accessible in the component.

Dynamic Route Syntax

// Route with parameter
<Route path="/user/:id" element={<UserProfile />} />

// Access parameter in component
function UserProfile() {
    const { id } = useParams();
    return <p>User ID: {id}</p>;
}

// Matches: /user/1, /user/abc, etc.
