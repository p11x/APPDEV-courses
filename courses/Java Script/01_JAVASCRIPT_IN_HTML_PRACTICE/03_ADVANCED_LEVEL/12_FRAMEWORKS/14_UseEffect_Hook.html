14 - useEffect Hook

What is useEffect?

useEffect is a Hook for handling side effects in functional components. It runs after every render by default, or when specific dependencies change.

useEffect Syntax

// Run after every render
useEffect(() => {
    console.log('Component rendered');
});

// Run only on mount (empty deps)
useEffect(() => {
    fetchData();
}, []);

// Run when 'data' changes
useEffect(() => {
    processData(data);
}, [data]);

// Cleanup function
useEffect(() => {
    const subscription = subscribe(id);
    return () => subscription.unsubscribe();
}, [id]);
