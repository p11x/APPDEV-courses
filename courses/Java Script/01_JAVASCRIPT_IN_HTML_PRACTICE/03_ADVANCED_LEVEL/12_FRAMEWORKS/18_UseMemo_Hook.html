18 - useMemo Hook

What is useMemo?

useMemo memoizes a computed value. It only recalculates when dependencies change, preventing expensive calculations on every render.

useMemo Syntax

const memoizedValue = useMemo(() => {
    // Expensive calculation
    return computeExpensiveValue(a, b);
}, [a, b]);

// Only recalculates when a or b changes!
