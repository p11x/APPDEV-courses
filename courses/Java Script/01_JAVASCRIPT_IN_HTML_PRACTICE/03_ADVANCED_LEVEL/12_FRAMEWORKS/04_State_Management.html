State Management Patterns

This example demonstrates various state management patterns in React applications.

State Management Patterns

// 1. Local State (useState) - Component-level state
// 2. Context API - Shared state across components
// 3. Reducer Pattern - Complex state logic
// 4. Custom Hooks - Reusable state logic
// 5. External Store - Global state management

Output Demonstration

5 state management patterns are demonstrated:

  1. Context API: Global state sharing via createContext/useContext
  2. useReducer: Complex state logic with dispatch/actions
  3. Custom Hooks: Reusable state logic (useLocalStorage)
  4. useMemo: Memoized computed values
  5. Derived State: Computed values from source state

Note: Check browser console to see useEffect cleanup logs and localStorage sync.
