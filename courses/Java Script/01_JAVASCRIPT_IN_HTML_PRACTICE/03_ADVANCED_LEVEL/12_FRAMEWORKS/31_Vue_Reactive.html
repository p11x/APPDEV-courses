31 - Vue Reactive

What is Reactivity?

Vue's reactivity system automatically tracks dependencies and updates the DOM when data changes. When you modify reactive data, Vue automatically re-renders the affected parts.

Reactive Data

// Vue 3 Composition API
import { ref, reactive } from 'vue';

// ref - for primitive values
const count = ref(0);
count.value++; // Access .value

// reactive - for objects
const state = reactive({
    name: 'Vue',
    version: 3
});
state.name = 'Vue.js'; // No .value needed
