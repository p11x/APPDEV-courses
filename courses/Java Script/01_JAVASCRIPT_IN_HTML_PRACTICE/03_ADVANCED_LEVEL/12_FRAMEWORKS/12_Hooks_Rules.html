12 - Hooks Rules

Rules of Hooks

  • Only call hooks at the top level - Don't call inside loops, conditions, or nested functions
  • Only call hooks from React functions - Not from regular JS functions
  • Hooks must be called in the same order - Every render must have same hook calls

Correct vs Incorrect Usage

// ❌ WRONG - Conditional hook
if (condition) {
    const [state, setState] = useState();
}

// ✅ CORRECT - Always call hooks
const [state, setState] = useState();
if (condition) {
    return null;
}

// ❌ WRONG - Hook in regular function
function handleClick() {
    useState(value);
}

// ✅ CORRECT - Hook in component
function MyComponent() {
    const [state, setState] = useState();
}
