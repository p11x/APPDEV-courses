34 - Vue Mount

What is Mounting?

Mounting is when Vue creates the DOM elements from your template and inserts them into the page. The onMounted hook runs after the component is mounted to the DOM.

Lifecycle Hooks

// Vue 3 Composition API lifecycle
import { onMounted, onUnmounted } from 'vue';

export default {
    setup() {
        // Called after component mounts
        onMounted(() => {
            console.log('Component mounted!');
            // Initialize timers, fetch data, etc.
        });

        // Called before component unmounts
        onUnmounted(() => {
            console.log('Cleaning up...');
        });
    }
}
