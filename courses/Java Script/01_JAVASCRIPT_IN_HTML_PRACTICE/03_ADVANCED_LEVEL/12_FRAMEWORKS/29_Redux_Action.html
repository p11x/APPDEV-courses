29 - Redux Action

What is a Redux Action?

An action is a plain JavaScript object that describes what happened. It must have a type property and can optionally have a payload with additional data.

Action Structure

// Simple action
{ type: 'INCREMENT' }

// Action with payload
{
    type: 'ADD_TODO',
    payload: { id: 1, text: 'Learn Redux' }
}

// Action creators (functions)
const increment = () => ({ type: 'INCREMENT' });
const addTodo = (text) => ({
    type: 'ADD_TODO',
    payload: { id: Date.now(), text }
});
