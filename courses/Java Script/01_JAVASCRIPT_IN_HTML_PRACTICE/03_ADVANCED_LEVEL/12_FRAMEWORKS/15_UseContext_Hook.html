15 - useContext Hook

What is useContext?

useContext lets you access values from a Context without passing props through every level. It's the alternative to class contextType.

Context Pattern

// Create context
const ThemeContext = React.createContext();

// Provide value at top level
<ThemeContext.Provider value="dark">
    <App />
</ThemeContext.Provider>

// Consume in any component
const theme = useContext(ThemeContext);
