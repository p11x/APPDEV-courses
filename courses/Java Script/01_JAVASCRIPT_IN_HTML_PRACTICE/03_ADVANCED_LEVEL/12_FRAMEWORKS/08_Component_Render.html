08 - Component Render

What is Render?

The render method is the only required method in a class component. It returns the JSX (JavaScript XML) that describes what should appear on the screen.

Render Process

// Component class with render
class Welcome extends Component {
    render() {
        // Return JSX - not HTML!
        return (
            <div>
                <h1>Hello, World!</h1>
                <p>This is a React component.</p>
            </div>
        );
    }
}

// Render returns a virtual DOM node
// React handles the rest!
