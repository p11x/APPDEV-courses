22 - Context Provider

What is a Provider?

The Provider is a component that accepts a value prop and makes it available to all descendant components via useContext. It wraps your application or a portion of it.

Provider Pattern

const ThemeContext = React.createContext();

// App component with Provider
function App() {
    return (
        <ThemeContext.Provider value="dark">
            <Header />
            <Main />
            <Footer />
        </ThemeContext.Provider>
    );
}

// Any child can access "dark"
function Header() {
    const theme = useContext(ThemeContext); // "dark"
}
