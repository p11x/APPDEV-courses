23 - Routing Basic

What is Routing?

Client-side routing allows navigation between different views without page reloads. The URL changes and the app renders different components based on the route.

Basic Router Setup

import { BrowserRouter, Routes, Route } from 'react-router-dom';

function App() {
    return (
        <BrowserRouter>
            <Routes>
                <Route path="/" element={<Home />} />
                <Route path="/about" element={<About />} />
                <Route path="/contact" element={<Contact />} />
            </Routes>
        </BrowserRouter>
    );
}
