Output Demonstration

6 component patterns are demonstrated:

  1. Card: Basic props (title) and children pattern
  2. Button: Props with default values and variant styling
  3. Counter: useState for internal state management
  4. UserCard: Complex object props with callbacks
  5. DataList: Children rendering and counting
  6. Toggle: useEffect for side effects and cleanup
