35 - Vue Update

What is Update?

The update phase occurs when reactive data changes. Vue automatically re-renders the component, and you can use onUpdated to run code after the DOM has been updated.

Update Lifecycle

import { onUpdated } from 'vue';

export default {
    setup() {
        // Called after component updates
        onUpdated(() => {
            // DOM is now in sync with data
            console.log('Component updated!');
        });
    }
}

// Update order:
1. Reactive data changes
2. Trigger re-render
3. Virtual DOM diff
4. Update real DOM
5. onUpdated() called
