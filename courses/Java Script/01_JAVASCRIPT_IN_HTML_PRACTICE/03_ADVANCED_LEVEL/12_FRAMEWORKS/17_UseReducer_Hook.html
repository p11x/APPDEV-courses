17 - useReducer Hook

What is useReducer?

useReducer is an alternative to useState for complex state logic. It takes a reducer function and initial state, returning current state and a dispatch function.

useReducer Syntax

const [state, dispatch] = useReducer(reducer, initialState);

// Reducer function
function reducer(state, action) {
    switch (action.type) {
        case 'increment':
            return { count: state.count + 1 };
        case 'decrement':
            return { count: state.count - 1 };
        default:
            return state;
    }
}

// Usage
dispatch({ type: 'increment' });
