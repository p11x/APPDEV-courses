16 - useRef Hook

What is useRef?

useRef returns a mutable ref object that persists across renders. It's commonly used to access DOM elements directly or hold values that don't trigger re-renders.

useRef Use Cases

// Access DOM element
const inputRef = useRef(null);
<input ref={inputRef} />
inputRef.current.focus();

// Hold mutable value (no re-render)
const countRef = useRef(0);
countRef.current++;
// Doesn't trigger re-render!

// Persist values between renders
const prevValue = useRef(value);
