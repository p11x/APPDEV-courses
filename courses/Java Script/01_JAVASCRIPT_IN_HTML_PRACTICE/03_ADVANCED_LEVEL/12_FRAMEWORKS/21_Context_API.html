21 - Context API

What is Context API?

Context API provides a way to pass data through the component tree without passing props manually at every level. It's designed for global data like themes, user info, or settings.

Creating Context

// Create context with default value
const ThemeContext = React.createContext('light');

// Context provides two components
// Provider - supplies the value
// Consumer (or useContext) - reads the value

// Default value used when no Provider
React.createContext({ theme: 'default', lang: 'en' });
