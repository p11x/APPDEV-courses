26 - Routing Params

What are Route Parameters?

Route parameters are dynamic segments in the URL that can capture variable parts of the path. They are accessed using the useParams hook in React Router.

Using useParams

// Define route with params
<Route path="/users/:userId/posts/:postId" />

// Access in component
function Post() {
    const { userId, postId } = useParams();
    // URL: /users/5/posts/10
    // userId = "5", postId = "10"
}
