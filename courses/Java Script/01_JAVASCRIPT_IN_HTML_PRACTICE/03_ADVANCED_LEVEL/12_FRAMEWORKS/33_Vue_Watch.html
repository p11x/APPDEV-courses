33 - Vue Watch

What is Watch?

watch allows you to execute a callback function whenever a reactive data source changes. It's used for side effects like API calls, DOM manipulation, or complex logic.

Watch Syntax

import { ref, watch } from 'vue';

const count = ref(0);

// Watch a ref
watch(count, (newValue, oldValue) => {
    console.log(`Changed from ${oldValue} to ${newValue}`);
});

// Watch with immediate option
watch(count, callback, { immediate: true });
