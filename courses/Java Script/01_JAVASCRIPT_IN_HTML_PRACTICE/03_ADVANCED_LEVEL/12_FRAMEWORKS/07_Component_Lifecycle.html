07 - Component Lifecycle

Lifecycle Phases

React class components go through three main lifecycle phases:

1. Mounting - Component being created and inserted into DOM

2. Updating - Component re-rendering due to state/props changes

3. Unmounting - Component being removed from DOM

Lifecycle Methods

// Mounting Phase
constructor()        → Initialize state
componentWillMount() → Before render (deprecated)
render()             → Return JSX
componentDidMount()  → After DOM inserted

// Updating Phase
componentWillReceiveProps() → New props received
shouldComponentUpdate()     → Return false to skip
componentWillUpdate()       → Before re-render
render()                    → Re-render
componentDidUpdate()        → After re-render

// Unmounting Phase
componentWillUnmount() → Cleanup before removal
