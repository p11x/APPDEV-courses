32 - Vue Computed

What are Computed Properties?

Computed properties are cached values that automatically update when their dependencies change. They only re-evaluate when a dependency changes, making them efficient.

Computed Syntax

import { ref, computed } from 'vue';

const firstName = ref('John');
const lastName = ref('Doe');

// Computed property
const fullName = computed(() => {
    return firstName.value + ' ' + lastName.value;
});

// Cached - only updates when deps change!
