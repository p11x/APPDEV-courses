11 - Hooks Basic

What are Hooks?

Hooks are functions that let you use state and lifecycle features in functional components. They "hook into" React state and lifecycle.

Built-in Hooks

// Most common hooks
useState       // Add state to functional component
useEffect      // Handle side effects
useContext     // Consume context
useRef         // Access DOM elements / persist values
useReducer     // Complex state management
useMemo        // Memoize calculations
useCallback    // Memoize functions

// Custom hooks - your own reusable logic
useCustom()    // Custom hook example
