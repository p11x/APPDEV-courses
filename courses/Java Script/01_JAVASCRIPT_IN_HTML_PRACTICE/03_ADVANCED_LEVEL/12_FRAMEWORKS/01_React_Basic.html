React Basic - React.createElement

This example demonstrates creating React elements using the React.createElement API without JSX.

Code Explanation

React.createElement(type, props, ...children)
  • type: The element type (string for HTML tags, or component function)
  • props: An object containing element properties (null for no props)
  • ...children: Child elements or text content

Output Demonstration

The following elements are rendered above:

  • A styled heading with React Blue color
  • A description paragraph
  • A bordered box containing the heading and description
  • A clickable button that triggers an alert
  • An unordered list with 3 items

Console Output: Check browser console to see the React element structure object.
