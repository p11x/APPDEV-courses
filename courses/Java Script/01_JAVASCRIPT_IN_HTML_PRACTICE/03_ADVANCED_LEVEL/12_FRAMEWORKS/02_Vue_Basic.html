Vue Basic - Reactive Data

This example demonstrates Vue.js reactive data and the Composition API.

Code Explanation

Vue 3 Composition API: setup(), reactive(), computed(), watch()
  • reactive(): Creates a reactive proxy of a plain object
  • ref(): Creates a reactive reference to a value
  • computed(): Creates a computed property
  • watch(): Watches for changes in reactive data

Output Demonstration

The following reactive features are demonstrated:

  • ref(): Simple reactive strings for title, message, author
  • reactive(): Object with reactive count and computed getter
  • computed(): Full name computed from first and last name
  • watch(): Triggers console log when message history changes
  • Reactive Array: Items list with computed total
