25 - Routing Nested

What is Nested Routing?

Nested routes are routes inside other routes. The parent component renders the child routes via an Outlet (or Switch/Routes). Child routes inherit the parent path.

Nested Routes Structure

<Route path="/dashboard" element={<Dashboard />}>
    <Route path="settings" element={<Settings />}/>
    <Route path="profile" element={<Profile />}/>
</Route>

// Full paths:
// /dashboard/settings
// /dashboard/profile
