09 - Component Update

Update Triggers

A component re-renders when:

  • State changes (setState)
  • Props change from parent
  • Parent re-renders

Update Flow

// State change triggers update
this.setState({ count: this.state.count + 1 });

// Update cycle:
1. setState() called
2. shouldComponentUpdate() 
3. render() new JSX
4. diff with old VDOM
5. componentDidUpdate()
