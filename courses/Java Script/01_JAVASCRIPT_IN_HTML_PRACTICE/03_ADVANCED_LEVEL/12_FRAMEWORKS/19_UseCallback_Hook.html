19 - useCallback Hook

What is useCallback?

useCallback memoizes a function, returning the same function reference until dependencies change. This prevents child components from re-rendering unnecessarily.

useCallback Syntax

const memoizedFn = useCallback(() => {
    return doSomething(a, b);
}, [a, b]);

// Without useCallback - new function every render
const handleClick = () => console.log(count);

// With useCallback - same function reference
const handleClick = useCallback(() => {
    console.log(count);
}, [count]);
