## Package Managers

### npm

```bash
npm init
npm install lodash
npm install -D typescript
```

### yarn

```bash
yarn init
yarn add lodash
```

### pnpm

```bash
pnpm add lodash
```

---

## Build Tools

### Vite

```bash
npm create vite@latest my-app
npm run dev
npm run build
```

### Webpack

```bash
npm install webpack webpack-cli
```

---

## Linters and Formatters

### ESLint

```bash
npm install -D eslint
npx eslint init
```

### Prettier

```bash
npm install -D prettier
npx prettier --write .
```

---

## Testing Tools

### Vitest

```bash
npm install -D vitest
```

### Jest

```bash
npm install -D jest
```
