## Career Paths

### Frontend Developer

```
Requirements:
- HTML/CSS/JavaScript
- React, Vue, or Angular
- Responsive design
- Accessibility

Salary: $60,000 - $120,000

Growth path:
Junior → Mid → Senior → Lead → Architect
```

### Full Stack Developer

```
Requirements:
- Frontend + Backend
- Database (SQL/NoSQL)
- APIs and REST
- Deployment

Salary: $70,000 - $140,000
```
