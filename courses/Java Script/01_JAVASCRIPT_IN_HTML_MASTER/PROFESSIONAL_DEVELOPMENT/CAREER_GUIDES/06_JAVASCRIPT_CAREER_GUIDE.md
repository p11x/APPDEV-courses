## Career Paths

### Junior Developer (0-2 years)

```
Responsibilities:
- Bug fixes
- Small features
- Code reviews

Skills Needed:
- HTML/CSS/JS fundamentals
- Version control
- Basic testing
- Agile basics

Salary Range: $50,000 - $80,000
```

### Mid Developer (2-5 years)

```
Responsibilities:
- Feature development
- Mentoring juniors
- Technical decisions

Skills Needed:
- Framework expertise
- Architecture patterns
- Performance optimization
- Leadership

Salary Range: $80,000 - $120,000
```

### Senior Developer (5+ years)

```
Responsibilities:
- System architecture
- Team leadership
- Technical direction

Skills Needed:
- Design patterns
- System scaling
- People management
- Strategy

Salary Range: $120,000 - $180,000
```
