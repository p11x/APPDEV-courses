## Assessment Checklist

Before proceeding to Advanced Level, ensure you can:

- [ ] Create services with proper DI
- [ ] Configure routes with parameters, guards, and resolvers
- [ ] Implement lazy loading
- [ ] Make HTTP requests with error handling
- [ ] Create reactive forms with custom validators
- [ ] Use interceptors for request/response processing
