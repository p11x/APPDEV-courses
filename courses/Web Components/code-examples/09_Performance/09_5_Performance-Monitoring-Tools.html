Performance Tips

  • FCP (First Contentful Paint): Should be under 1.5s
  • LCP (Largest Contentful Paint): Should be under 2.5s
  • CLS (Cumulative Layout Shift): Should be under 0.1
  • TTI (Time to Interactive): Should be under 3.8s

INR Context

Indian users typically access on slower 3G/4G networks. Optimize for:

  • Reduced bundle size (<100KB initial)
  • Lazy loading images
  • Code splitting by route
  • Service worker caching
