# API Reference

## OVERVIEW

Complete API reference for all Web Component specifications including Custom Elements, Shadow DOM, and ElementInternals APIs.

## Custom Elements API

| Method | Description |
|--------|-------------|
| customElements.define(name, constructor) | Register custom element |
| customElements.get(name) | Get constructor |
| customElements.whenDefined(name) | Promise when defined |

## ElementInternals API

| Method | Description |
|--------|-------------|
| attachInternals() | Attach form internals |
| setFormValue(value) | Set form value |
| setValidity(state, message, anchor) | Set validity |
