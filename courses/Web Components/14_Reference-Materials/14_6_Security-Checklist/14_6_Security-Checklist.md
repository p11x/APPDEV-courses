# Security Checklist

## OVERVIEW

Security checklist for developing secure Web Components.

## Security Requirements

- [ ] Input sanitization for user content
- [ ] No eval() with user data
- [ ] CSP headers configured
- [ ] HTTPS required
- [ ] No sensitive data in client-side code
