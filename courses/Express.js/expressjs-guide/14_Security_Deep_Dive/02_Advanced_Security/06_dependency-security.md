# Dependency Security

## 📌 What You'll Learn

- npm audit, Snyk, Dependabot
- Lock file integrity
- Supply chain attacks

## 💻 Code Example

```bash
# Run security audit
npm audit

# Fix vulnerabilities
npm audit fix

# Check for outdated packages
npm outdated

# Install with audit
npm audit --audit-level=high
```

## ✅ Quick Recap

- Run `npm audit` regularly
- Use Dependabot for automated updates
- Verify lock file integrity
- Use Snyk for additional scanning
