🚀 Bootstrap 5 Colors - Beginner's Guide

Learn Bootstrap Colors from scratch! This guide covers all color options for text, backgrounds, and more.

📚 What are Bootstrap Colors?

Bootstrap provides a predefined color system with contextual colors that convey meaning. Each color has a specific semantic purpose:

  • Primary - Main brand color (blue)
  • Secondary - Supporting color (gray)
  • Success - Positive actions (green)
  • Danger - Errors/warnings (red)
  • Warning - Caution (yellow)
  • Info - Information (cyan)
  • Dark - Dark color (black)
  • Light - Light color (white)

These colors can be applied to text, backgrounds, borders, and many other elements!

🎨 Bootstrap Color Palette

Here's an overview of all available contextual colors in Bootstrap:

Primary - .text-primary / .bg-primary
Secondary - .text-secondary / .bg-secondary
Success - .text-success / .bg-success
Danger - .text-danger / .bg-danger
Warning - .text-warning / .bg-warning
Info - .text-info / .bg-info
Light - .text-light / .bg-light
Dark - .text-dark / .bg-dark

💡 Why Use Contextual Colors?

Semantic Meaning

Bootstrap's color system follows semantic conventions that users recognize:

  • ✓ Success/Positive - Use green for confirmations
  • ✗ Danger/Error - Use red for errors
  • ⚠ Warning - Use yellow for warnings
  • ℹ Info - Use cyan for information

This improves user experience by communicating meaning through color!
