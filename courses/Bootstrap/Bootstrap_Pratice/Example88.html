📚 What is Bootstrap Navigation?

Bootstrap provides reusable navigation components for creating tabbed or pill-style interfaces:

  • Tabs - Folder-like tab navigation with borders
  • Pills - Button-like rounded navigation
  • Vertical - Stacked navigation for sidebars
  • With content - Working tabs that show/hide panels
