🖼️ Bootstrap Responsive Images

Learn how to make images scale properly with their container.

📚 What are Responsive Images?

The .img-fluid class makes images responsive by:

  • Setting max-width: 100% - image never exceeds container width
  • Setting height: auto - maintains aspect ratio
  • Preventing large images from breaking your layout

💡 How It Works

The .img-fluid CSS Properties
.img-fluid {
    max-width: 100%;  /* Never wider than container */
    height: auto;     /* Maintain aspect ratio */
}
Key Points
  • Image scales DOWN when container is smaller than image
  • Image never scales UP beyond its original size
  • Aspect ratio is always preserved

💡 When to Use Responsive Images

Best Use Cases
  • Hero images - Full-width banner images
  • Content images - Images within articles/pages
  • Any image - When you want proper scaling
Best Practice
  • Always use .img-fluid for most images
  • It ensures images work on all screen sizes
