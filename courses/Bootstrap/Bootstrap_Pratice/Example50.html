🔘 Bootstrap Block Buttons

Learn how to create full-width block buttons.

📚 What are Block Buttons?

Block buttons span the full width of their parent container:

  • Use .d-grid on parent container
  • Use .gap-* for spacing between buttons
  • Perfect for mobile-friendly designs
  • Great for call-to-action buttons

💡 When to Use Block Buttons

Best Use Cases
  • Mobile designs - Easy to tap on small screens
  • Call-to-action - Full width CTA buttons
  • Forms - Submit buttons that span full width
  • Vertical button stacks - Stack of actions
  • Landing pages - Prominent action buttons
