📋 Bootstrap Cards - Summary

Quick reference guide for all card classes.

📚 Card Classes

Class Description
.card Base card container
.card-body Main content area with padding
.card-header Card header section
.card-footer Card footer section
.card-title Card heading style
.card-subtitle Secondary heading
.card-text Paragraph text
.card-link Styled link

🖼️ Card Image Classes

Class Description
.card-img-top Image at top of card
.card-img-bottom Image at bottom of card
.card-img Full card image (for overlay)
.card-img-overlay Image as background with text overlay

🎯 Key Points

  • Cards are flexible containers for content
  • Use .card-body for the main content area
  • Add images with .card-img-top or .card-img-bottom
  • Use .card-img-overlay for text on images
  • Style with background and border color classes
  • Use .h-100 for equal-height cards in grids
