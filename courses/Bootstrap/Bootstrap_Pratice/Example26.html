🎨 Bootstrap Text Colors

Learn how to color your text using Bootstrap's .text-* utility classes.

📚 What are Text Colors?

Bootstrap provides text color utilities that allow you to color any text element using the .text-{color} class.

Syntax:

This is primary text

✨ All Text Color Classes

Here are all the available text color classes in Bootstrap:

.text-primary - Primary color (usually blue)

.text-secondary - Secondary color (usually gray)

.text-success - Success color (usually green)

.text-danger - Danger color (usually red)

.text-warning - Warning color (usually yellow)

.text-info - Info color (usually cyan)

.text-light - Light color (usually white)

.text-dark - Dark color (usually black)

.text-muted - Muted color (gray)

.text-white - White color
