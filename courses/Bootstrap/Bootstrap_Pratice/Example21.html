🚀 Text Alignment

Learn how to align and transform text in Bootstrap!

📚 Text Alignment Classes

Bootstrap provides classes to align text left, center, or right. You can also make text justified!

✨ Text Align Left

Using .text-start (default)

This text is aligned to the left. This is the default alignment for most languages like English.

✨ Text Align Center

Using .text-center

This text is aligned to the center. Perfect for headings, quotes, or important messages!

✨ Text Align Right

Using .text-end

This text is aligned to the right. Common in right-to-left languages or special designs.

✨ Text Justify

Using .text-justify

This is justified text. Bootstrap will spread the words evenly across the entire line, creating a clean block appearance. This is commonly used in newspapers and books.

✨ Responsive Text Alignment

Change alignment at different screen sizes!

I am left-aligned on mobile, but center-aligned on medium screens and larger!
I am right-aligned on mobile, but left-aligned on large screens and larger!

✨ Text Transformation

Change the case of text without rewriting it:

.text-lowercase: THIS TEXT WILL BE LOWERCASE

.text-uppercase: this text will be uppercase

.text-capitalize: this text will have capitalized words

✨ Font Weight & Italics

Change text boldness and style:

Bold text (.fw-bold)

Bolder text (.fw-bolder)

Normal text (.fw-normal)

Light text (.fw-light)

Lighter text (.fw-lighter)

Italic text (.fst-italic)

Normal style (.fst-normal)
