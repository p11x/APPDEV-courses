    🎯 Key Points

    • Badges are inline elements for small labels
    • Use .bg-* classes for colors
    • Use .rounded-pill for pill-shaped badges
    • Badges work well in headings, buttons, and lists
    • Use .text-dark with light backgrounds for contrast
