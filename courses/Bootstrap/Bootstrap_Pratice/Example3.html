🚀 Bootstrap Grid - Auto-Dividing Columns

📚 What are Auto-Dividing Columns?

When you use .col without a number, Bootstrap automatically divides the available space equally among all columns in a row. Perfect for when you want equal-width columns without specifying sizes.

Key Point: These columns are responsive and will stack on mobile devices!
