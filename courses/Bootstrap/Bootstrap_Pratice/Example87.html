📋 Bootstrap Navbar - Summary

Quick reference guide for all navbar classes.

📚 Navbar Classes

Class Description
.navbar Base navbar container
.navbar-expand-* Breakpoint for collapse (sm, md, lg, xl, xxl)
.navbar-brand Logo/brand name
.navbar-nav Navigation links container
.nav-item Individual nav link item
.nav-link Navigation link styling
.navbar-toggler Hamburger menu button
.navbar-collapse Collapsible content area
.navbar-dark Light text (for dark backgrounds)
.navbar-light Dark text (for light backgrounds)

🔻 Dropdown Classes

Class Description
.dropdown Enables dropdown
.dropdown-toggle Shows dropdown arrow
.dropdown-menu Dropdown content container
.dropdown-item Individual dropdown item
.dropdown-divider Divider line

🎨 Background Color Classes

Class Color
.bg-dark Black
.bg-primary Blue
.bg-success Green
.bg-danger Red
.bg-warning Yellow
.bg-light Light Gray
.bg-white White

🎯 Key Points

  • Navbar is a responsive navigation header
  • Use .navbar-expand-* to control when it collapses
  • Use .navbar-dark for dark backgrounds, .navbar-light for light
  • Toggler button requires data-bs-toggle="collapse"
  • Add dropdowns with .dropdown classes
  • Use .ms-auto to push elements to the right
