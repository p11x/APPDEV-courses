🚀 Nesting Grids

Create complex layouts by putting rows inside columns!

📚 What is Nesting?

Nesting means putting a .row inside a .col. Each nested row still has its own 12 columns!

Benefits:

  • Create complex, multi-level layouts
  • Each nested row has full 12-column system
  • Perfect for dashboards and content sections
