🖼️ Bootstrap Image Alignment

Learn how to align images using float and text alignment classes.

📚 Image Alignment Methods

Bootstrap provides several ways to align images:

  • Float classes - .float-start, .float-end
  • Text alignment - .text-center, .text-start, .text-end
  • Block display - .d-block with margin auto
  • Vertical alignment - .align-top, .align-middle, .align-bottom

✨ Float Alignment

Use float classes to position images left or right with text wrapping:

Float Left (.float-start)

This paragraph has an image floated to the left. The text wraps around the image on the right side. You can see how the text flows around the floated image naturally.

Float Right (.float-end)

This paragraph has an image floated to the right. The text wraps around the image on the left side. This is useful for pull quotes or featured images.

🎯 Text Alignment

Use text alignment classes to center or position block images:

Center (.text-center)
Left/Start (.text-start)
Right/End (.text-end)

📏 Vertical Alignment

Align images vertically within inline contexts:

align-top
align-middle
align-bottom
align-text-bottom

🔄 Responsive Alignment

Use responsive float classes for different screen sizes:

This image floats right on large screens but moves to the left on smaller screens. Resize your browser to see the difference!
