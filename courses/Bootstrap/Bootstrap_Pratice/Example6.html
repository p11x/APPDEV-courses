🚀 Bootstrap Grid - Responsive (Mobile First)

📚 What is Responsive Grid?

Use responsive classes like .col-md-*, .col-lg-* to control when columns should sit side-by-side or stack.

Key Point: On mobile (xs), columns always stack. On larger screens, they become side-by-side based on the breakpoint you choose.

🔍 Bootstrap Breakpoints: xs (<576px), sm (≥576px), md (≥768px), lg (≥992px), xl (≥1200px), xxl (≥1400px)
Try resizing your browser!

📝 Quick Reference
  • .col-md-* - md (768px+) and larger
  • .col-lg-* - lg (992px+) and larger
  • .col-sm-* - sm (576px+) and larger
  • .col-xl-* - xl (1200px+) and larger
  • No breakpoint = always side-by-side
