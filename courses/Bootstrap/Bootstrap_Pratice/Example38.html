🖼️ Bootstrap 5 Images - Beginner's Guide

Learn Bootstrap Images from scratch! This guide covers all image styling options.

📚 What are Bootstrap Images?

Bootstrap provides several image styling classes that make images:

  • Responsive - Scale to fit their container
  • Visually Appealing - Add rounded corners, circles, borders
  • Properly Aligned - Float or text-align positioning

Images are essential for visual content - photos, icons, graphics, and more!

🎨 Bootstrap Image Classes

Here are the main image classes in Bootstrap:

.img-fluid - Makes image responsive (scales to fit container)
.rounded - Adds rounded corners to all corners
.rounded-circle - Makes image a perfect circle
.img-thumbnail - Adds rounded corners + gray border

📋 Topics We'll Cover

  1. Responsive Images - .img-fluid for responsive scaling
  2. Rounded Images - .rounded for corner rounding
  3. Circular Images - .rounded-circle for circles
  4. Thumbnails - .img-thumbnail for bordered images
  5. Image Alignment - Float and text alignment
  6. Image Gallery - Grid layouts with images
