🎨 Bootstrap Background Colors

Learn how to add background colors to elements using Bootstrap's .bg-* utility classes.

📚 What are Background Colors?

Bootstrap provides background color utilities that allow you to set background colors using the .bg-{color} class.

Syntax:

Primary background

✨ All Background Color Classes

Here are all the available background color classes in Bootstrap:

.bg-primary - Primary background (blue)
.bg-secondary - Secondary background (gray)
.bg-success - Success background (green)
.bg-danger - Danger background (red)
.bg-warning - Warning background (yellow)
.bg-info - Info background (cyan)
.bg-light - Light background (white)
.bg-dark - Dark background (black)
.bg-white - White background
.bg-transparent - Transparent background
