🔘 Bootstrap Button Sizes

Learn how to create small, default, and large buttons.

📚 Button Size Classes

Bootstrap provides 3 button sizes:

  • .btn-sm - Small button (for compact spaces)
  • Default - Medium button (standard size)
  • .btn-lg - Large button (for emphasis)

💡 When to Use Each Size

Small Buttons (.btn-sm)
  • Toolbars and button groups
  • Cards with limited space
  • Table action buttons
  • Inline actions
Large Buttons (.btn-lg)
  • Call-to-action (CTA) buttons
  • Landing page buttons
  • Hero section buttons
  • Mobile-friendly buttons
Default Size
  • Most common use case
  • Forms and dialogs
  • General purpose buttons
