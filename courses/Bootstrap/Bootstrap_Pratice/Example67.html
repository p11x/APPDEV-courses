📚 Badge Color Variants

Bootstrap badges use background color classes:

  • Primary - Blue (default)
  • Secondary - Gray
  • Success - Green
  • Danger - Red
  • Warning - Yellow (use .text-dark)
  • Info - Cyan (use .text-dark)
  • Dark - Black

💡 When to Use Badge Colors

Color Guidelines
  • Primary - Default/general labels
  • Success - Positive states, completed
  • Danger - Errors, deleted, removed
  • Warning - Caution, pending
  • Info - Information, neutral
