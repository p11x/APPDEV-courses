📚 Vertical Button Groups

Use .btn-group-vertical to stack buttons vertically:

  • Stacked appearance - Buttons arranged top to bottom
  • Great for sidebars - Perfect for vertical layouts
  • Toolbar alternative - When horizontal space is limited

  • Split buttons - Dropdown in vertical group
