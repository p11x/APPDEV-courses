📚 Basic Button Group

Wrap buttons in a

with .btn-group class:

  • Horizontal layout - Buttons appear side by side
  • Connected appearance - Shared borders between buttons
  • role="group" - ARIA attribute for accessibility
  • aria-label - Describes the button group
