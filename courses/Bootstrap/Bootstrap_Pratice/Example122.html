📝 Bootstrap Input Groups - Summary

Complete overview of Bootstrap Input Groups.

📚 Key Input Group Classes

📦 Container
  • .input-group - Wrapper container
🏷️ Add-ons
  • .input-group-text - Text/icon addon
  • Place before input for prefix
  • Place after input for suffix
🔘 Buttons
  • .btn - Button in input group
  • Use btn btn-outline-* for styling
  • Can be before, after, or both
☑️ Checkboxes & Radios
  • .form-check-input - Input control
  • .mt-0 - Remove top margin
  • Wrapped in .input-group-text
🔻 Dropdowns
  • .dropdown-toggle - Toggle button
  • data-bs-toggle="dropdown" - Enable JS
  • .dropdown-menu - Menu items
  • .dropdown-toggle-split - Segmented
📏 Sizes
  • .input-group-sm - Small
  • .input-group - Default
  • .input-group-lg - Large

🔧 Common Patterns

  • Search Bar: Button + Input
  • Currency: Text prefix + Number input + Text suffix
  • URL: Text prefix + Input
  • Filter: Input + Dropdown
