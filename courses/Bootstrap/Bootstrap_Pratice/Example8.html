🚀 Bootstrap Grid - Column Offset

📚 What is Column Offset?

Use .offset-* classes to push columns to the right, creating empty space. The offset plus the column width should equal 12.

Key Point: Offsets create "invisible" columns to push your content to the right!

Example 1: Centered Single Column (offset-4 + col-4 = 12)

Offset of 4 + column of 4 = centered layout

📝 Quick Reference
  • .offset-* - Push column right by * columns (1-11)
  • offset + col = 12 - Must add up to 12
  • .offset-md-* - Responsive offset (applies at md+)
  • .offset-lg-* - Responsive offset (applies at lg+)
  • Useful for centering columns and creating gaps
