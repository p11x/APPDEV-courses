📝 Bootstrap Forms - Summary

Complete overview of Bootstrap Form classes.

📚 Key Form Classes

🎛️ Form Controls
  • .form-control - Base input styling
  • .form-label - Label styling
  • .form-text - Helper text
  • .form-select - Select dropdown
  • .form-control-lg - Large input
  • .form-control-sm - Small input
☑️ Checkboxes & Radios
  • .form-check - Checkbox/radio wrapper
  • .form-check-input - Checkbox/radio input
  • .form-check-label - Checkbox/radio label
  • .form-check-inline - Inline layout
  • .form-switch - Toggle switch
📐 Grid Layout
  • .row - Grid row
  • .col-* - Responsive columns
  • .g-3 - Gap spacing
🏷️ Floating Labels
  • .form-floating - Floating label wrapper
  • placeholder="" - Required for floating
📝 Input Types
  • type="text" - Text input
  • type="email" - Email input
  • type="password" - Password input
  • type="number" - Number input
  • type="file" - File upload
  • type="checkbox" - Checkbox
  • type="radio" - Radio button
⚡ Input States
  • disabled - Disabled input
  • readonly - Read-only input
  • readonly - Read-only input
  • disabled - Disabled input
  • checked - Pre-checked (checkbox/radio)
