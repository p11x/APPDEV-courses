📚 Badges with Lists

Use badges with list items for status:

  • Item counts - Number of items
  • Status - Active, Pending, etc.
  • Labels - Categories, tags

Syntax:

  • Item Count

    • ✨ Simple List with Badges

    • Inbox Messages 14
    • Sent Messages 2
    • Drafts 5
    • Spam 1
