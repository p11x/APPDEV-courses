📚 Alert Color Variants

Bootstrap provides 8 alert color variants, each with a semantic meaning:

  • Primary - General information (blue)
  • Secondary - Less important (gray)
  • Success - Successful operations (green)
  • Danger - Errors/critical (red)
  • Warning - Warnings (yellow)
  • Info - Information (cyan)
  • Light - Light theme (white)
  • Dark - Dark theme (dark gray)
