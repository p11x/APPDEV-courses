📚 What are Circular Images?

The .rounded-circle class makes an image a perfect circle:

  • Applies border-radius: 50%
  • Requires square image for best results
  • Perfect for profile pictures, avatars, icons

⚠️ Important Note

For best results with .rounded-circle:

  • Use square images (width = height)
  • Non-square images will appear as oval
  • Set explicit width and height to control size
