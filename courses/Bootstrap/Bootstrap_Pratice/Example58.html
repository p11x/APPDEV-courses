👥 Bootstrap Toggle Buttons

Learn how to create checkbox and radio button groups.

📚 Toggle Button Groups

Bootstrap allows creating toggle buttons using .btn-check:

  • Checkbox buttons - Multiple selections allowed
  • Radio buttons - Only one selection allowed
  • Hidden inputs - Using .btn-check class
  • Label styling - Labels become toggle buttons
