📊 Bootstrap Hover Tables

Learn how to add row highlighting on mouse hover for better interactivity.

📚 What are Hover Tables?

The .table-hover class adds a subtle background highlight when the mouse cursor moves over table rows. This helps users track which row they're reading.

💡 When to Use Hover Tables

Best Use Cases
  • Interactive dashboards - User can track data easily
  • Task lists - Helps identify current task
  • Data tables - Large amounts of information
  • Admin panels - Management interfaces
Common Patterns
  • Combine with .table-striped for maximum readability
  • Use with clickable rows (add cursor: pointer)
  • Pair with .table-sm for compact tables
