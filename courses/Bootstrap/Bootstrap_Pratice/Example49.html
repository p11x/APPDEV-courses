🔘 Bootstrap Button States

Learn about disabled and active button states.

📚 Button States

Bootstrap buttons can be in different states:

  • Normal - Default button appearance
  • Active - Button appears pressed/clicked
  • Disabled - Button cannot be clicked

💡 When to Use Button States

Disabled State Use Cases
  • Form validation - Disable submit until form is valid
  • Loading states - Disable button while processing
  • Permissions - Hide/disable actions user can't perform
  • Step wizards - Disable next until current step is done
Active State Use Cases
  • Toggle buttons - Show which option is selected
  • Current page - Highlight current navigation item
  • State indicators - Show pressed/selected state
