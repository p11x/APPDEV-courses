🃏 Bootstrap Cards - Images

Learn how to add images to cards.

📚 Card Images

Bootstrap provides classes for adding images to cards:

  • .card-img-top - Image at top of card (above body)
  • .card-img-bottom - Image at bottom of card (below body)
  • .card-img-overlay - Image as background with text overlay
