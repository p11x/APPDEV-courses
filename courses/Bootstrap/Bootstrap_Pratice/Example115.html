📝 Bootstrap Form Validation - Summary

Complete overview of Bootstrap Form Validation classes.

📚 Key Validation Classes

🎛️ Validation States
  • .is-valid - Green/success styling
  • .is-invalid - Red/error styling
💬 Feedback Messages
  • .valid-feedback - Inline success message
  • .invalid-feedback - Inline error message
  • .valid-tooltip - Tooltip success message
  • .invalid-tooltip - Tooltip error message
📝 Form Classes
  • .needs-validation - Marks form for validation
  • .was-validated - Shows validation after submit
  • .position-relative - Required for tooltip positioning
⚡ HTML5 Attributes
  • required - Makes field mandatory
  • novalidate - Disables browser validation
  • type="email" - Email format validation
  • minlength - Minimum character length
  • pattern - Custom regex pattern

🔧 JavaScript Implementation

(function () {
    'use strict'
    var forms = document.querySelectorAll('.needs-validation')
    Array.prototype.slice.call(forms)
        .forEach(function (form) {
            form.addEventListener('submit', function (event) {
                if (!form.checkValidity()) {
                    event.preventDefault()
                    event.stopPropagation()
                }
                form.classList.add('was-validated')
            }, false)
        })
})()
