Learn how to create closeable alerts.

📚 Dismissible Alerts

Use .alert-dismissible to create closeable alerts:

  • Close button - X button in top-right corner
  • Fade animation - Smooth close transition
  • JavaScript - Requires Bootstrap JavaScript

Components needed:

  • .alert-dismissible - On the alert div
  • .fade .show - For animation
  • .btn-close - The close button
  • data-bs-dismiss="alert" - Triggers close

✨ Dismissible Alert Examples

Here are dismissible alerts (click the X to close):

Success! Your changes have been saved.
Warning! You can dismiss this alert by clicking the X button.

💡 When to Use Dismissible Alerts

Best Use Cases
  • Notifications - Non-critical messages
  • Cookie banners - Consent messages
  • Welcome messages - First-time user tips
  • Temporary alerts - Messages that can be dismissed
When NOT to Use
  • Critical errors - Users must see
  • Form validation - Keep visible until fixed
  • System status - Important operational messages
