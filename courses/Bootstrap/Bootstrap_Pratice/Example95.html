📋 Bootstrap Navigation - Summary

Quick reference guide for all navigation classes.

📚 Navigation Classes

Class Description
.nav Base navigation class
.nav-tabs Tab-style navigation
.nav-pills Pill-style navigation
.nav-link Navigation link styling
.nav-item Wrapper for nav items
.active Currently selected item
.disabled Unclickable item
.flex-column Vertical/stacked layout
.nav-fill Proportional width
.nav-justified Equal width

🔻 Tab Content Classes

Class Description
.tab-content Container for tab panels
.tab-pane Individual tab panel
.fade Fade animation
.show Make visible initially
data-bs-toggle="tab" Enable tab functionality

🎯 Key Points

  • Use .nav as the base class
  • Use .nav-tabs for folder-style tabs
  • Use .nav-pills for button-style pills
  • Use .flex-column for vertical navigation
  • Use .nav-fill or .nav-justified for spacing
  • Add data-bs-toggle="tab" for working tabs
