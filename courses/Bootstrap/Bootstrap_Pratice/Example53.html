Learn Bootstrap Button Groups from scratch! This guide covers all button group options.

📚 What are Button Groups?

Button groups allow you to group multiple buttons together on a single line:

  • Unified appearance - Buttons appear connected as one unit
  • Shared borders - No gaps between buttons
  • Toolbar interfaces - Great for text formatting, navigation
  • Toggle buttons - Checkbox and radio functionality

Button groups are perfect for toolbars and related actions!

🎨 Bootstrap Button Group Classes

Here are the main button group classes:

.btn-group - Groups buttons horizontally (side by side)
.btn-group-vertical - Stacks buttons vertically
.btn-toolbar - Container for multiple button groups
.btn-group-sm / .btn-group-lg - Size entire groups
