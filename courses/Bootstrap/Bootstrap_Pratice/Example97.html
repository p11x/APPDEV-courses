Learn how to create split button dropdowns.

📚 Split Button Dropdown

Split buttons have two parts - one for action, one for menu:

  • .btn-group - Groups buttons together
  • .dropdown-toggle-split - Toggle with less padding
  • Left button does action, right button shows menu
