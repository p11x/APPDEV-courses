👥 Bootstrap Button Group Sizes

Learn how to size entire button groups at once.

📚 Button Group Size Classes

Size entire button groups using size classes:

  • .btn-group-sm - Small size for compact spaces
  • Default - Standard size (no class needed)
  • .btn-group-lg - Large size for emphasis

The size class goes on the .btn-group container!

💡 When to Use Each Size

Small (.btn-group-sm)
  • Toolbars and compact spaces
  • Table action buttons
  • Cards with limited space
  • Navigation bars
Large (.btn-group-lg)
  • Call-to-action sections
  • Landing pages
  • Hero buttons
  • Mobile-friendly interfaces
Default
  • Most common use case
  • General purpose groups
  • Forms and dialogs
