🃏 Bootstrap Cards - Colors

Learn how to style cards with colors.

📚 Colored Cards

Use Bootstrap's color classes to style cards:

  • .bg-primary - Blue card
  • .bg-success - Green card
  • .bg-danger - Red card
  • .bg-warning - Yellow card
  • .bg-info - Cyan card
  • .bg-dark - Dark card

Use .text-white for text color on dark backgrounds.
