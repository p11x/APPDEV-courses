📚 What are Outline Buttons?

Outline buttons have colored borders and text with a transparent background. They provide a modern, subtle look:

  • Colored border - Only the border is colored
  • Transparent background - No fill color
  • Hover effect - Background fills on hover
  • Less prominent - Great for secondary actions

🔄 Solid vs Outline Comparison

Compare solid and outline buttons side by side:

Solid Buttons

Colored background, white/dark text

Best for:

  • Main/Primary actions
  • Call-to-action buttons
  • Emphasizing important actions
Outline Buttons

Transparent background, colored border/text

Best for:

  • Secondary actions
  • Less prominent actions
  • Toolbars and button groups

💡 When to Use Outline Buttons

Best Use Cases
  • Secondary actions - Like "Cancel" or "Back"
  • Button groups - Multiple related buttons
  • Toolbars - Editor toolbars, action bars
  • Cards - Inside cards with colored backgrounds
  • Minimal designs - When you want less visual weight
