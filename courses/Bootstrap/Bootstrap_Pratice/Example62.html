📚 Alerts with Links

Use .alert-link class for links inside alerts:

  • Matches alert color - Links inherit the alert's color
  • Visual consistency - Links look part of the alert
  • Easy to notice - Stands out from regular text

Syntax: Link Text

Best Use Cases
