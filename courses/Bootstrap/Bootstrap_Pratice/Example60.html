📚 What are Bootstrap Alerts?

Bootstrap alerts are styled message boxes used to communicate important information:

  • Visual feedback - Colored boxes for different message types
  • Contextual colors - Success, info, warning, danger colors
  • Dismissible - Users can close them with X button
  • Rich content - Can include headings, links, and more

Alerts are perfect for user notifications - success messages, errors, warnings!

🎨 Bootstrap Alert Classes

Here are the main alert classes:

.alert-primary - Blue alert for general information
.alert-success - Green alert for success messages
.alert-info - Cyan alert for information
.alert-warning - Yellow alert for warnings
.alert-danger - Red alert for errors
