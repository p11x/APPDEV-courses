🔻 Bootstrap Dropdowns - Introduction

Learn the basics of Bootstrap dropdowns.

📚 What are Bootstrap Dropdowns?

Dropdowns are toggleable menus that display a list of links when clicked:

  • Compact - Multiple options in small space
  • Interactive - Requires JavaScript
  • Positioned - Uses Popper for correct placement

✨ Basic Dropdown

  • Action
  • Another action
  • Something else

📝 Key Dropdown Classes

Class Description
.dropdown Wrapper container
.dropdown-toggle Trigger button with arrow
.dropdown-menu Menu list container
.dropdown-item Individual menu item
