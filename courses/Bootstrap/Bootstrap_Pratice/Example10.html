🚀 Bootstrap Grid - Nesting Grids

📚 What is Nesting?

You can put a .row inside a column to create nested grids. Each nested row still has 12 columns to work with!

Key Point: Nested grids are incredibly powerful for creating complex layouts!

📝 Quick Reference
  • Put .row inside .col - Creates nested grid
  • Each nested row - Still has 12 columns
  • Use .mt-* or .mb-* - Add spacing between nested rows
  • Unlimited nesting levels - Can nest as deep as needed
