🚀 Bootstrap Grid - Specific Column Widths

📚 What are Specific Column Widths?

Use .col-* (where * is 1-12) to specify exactly how many of the 12 columns a column should span. The numbers in each row should add up to 12!

Key Point: Unlike .col (responsive), these columns are always side-by-side regardless of screen size.

📝 Quick Reference
  • .col-* - Fixed width (1-12), always side-by-side
  • Numbers must add up to 12 in each row
  • Works on all screen sizes (not responsive)
  • Common widths: .col-3 (25%), .col-4 (33%), .col-6 (50%), .col-8 (67%), .col-9 (75%)
