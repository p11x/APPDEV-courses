🚀 Column Offset

Create space between columns using offset classes!

📚 What is Column Offset?

Offset classes let you push columns to the right by adding empty columns. It's like having invisible columns!

Formula: offset + column = 12

✨ Centered Single Column

Using .offset-4 .col-4 - offset-4 + col-4 = centered!

col-4 offset-4

✨ Offset with Two Columns

Using .offset-2 .col-4 - starts after 2 empty columns
