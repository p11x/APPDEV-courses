🔻 Bootstrap Dropdowns - Directions

Learn how to create dropdowns that open in different directions.

📚 Drop Directions

Control which direction the dropdown opens:

  • .dropend - Opens to the right
  • .dropstart - Opens to the left
  • .dropup - Opens upward
  • (default) - Opens downward
