🖼️ Bootstrap Rounded Images

Learn how to add rounded corners to images.

📚 What are Rounded Images?

The .rounded class adds rounded corners to all four corners of an image:

  • Border-radius: 0.375rem (about 6px)
  • Subtle, modern look
  • Works with any image size

💡 When to Use Rounded Images

Best Use Cases
  • Cards - Images in card components
  • Buttons - Small icon images
  • Gallery thumbnails - Image previews
  • Profile pictures - User avatars (use rounded-circle)
Design Tips
  • Use .rounded for subtle softness
  • Use .rounded-circle for profile photos
  • Match rounded corners to other elements on page
