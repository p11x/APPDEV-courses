🚀 Responsive Grid Design

Create different layouts for different screen sizes!

📚 What is Responsive Design?

Responsive design means your website looks good on ALL devices - phones, tablets, and desktops. Bootstrap makes this easy with responsive breakpoints!

Bootstrap Breakpoints:

  • xs - Phones (< 576px) - Default, no prefix
  • sm - Large phones (≥ 576px) - Use .col-sm-*
  • md - Tablets (≥ 768px) - Use .col-md-*
  • lg - Laptops (≥ 992px) - Use .col-lg-*
  • xl - Desktops (≥ 1200px) - Use .col-xl-*

💡 Mobile First Approach

Bootstrap uses mobile first design. This means:

  • Classes without prefix (like .col-6) apply to ALL screen sizes
  • Responsive classes (like .col-md-6) apply to that size AND LARGER
  • Start with mobile, then add larger screen layouts

Example: .col-md-4 means "4 columns on medium screens and larger"
