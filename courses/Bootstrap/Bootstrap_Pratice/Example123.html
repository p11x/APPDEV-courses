📝 Bootstrap List Groups

Learn how to create flexible list components.

📚 What are List Groups?

Bootstrap list groups are flexible components for displaying sequences of content:

  • .list-group - Container wrapper
  • .list-group-item - Individual item
  • .list-group-item-action - Clickable items
  • .active - Active/selected item
  • .disabled - Disabled item

🔧 Common Uses

  • Sidebars and navigation menus
  • Activity feeds and notifications
  • Displaying collections of related items
  • Contact lists and user directories
