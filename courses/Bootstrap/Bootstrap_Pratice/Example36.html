🎨 Responsive Breakpoints

Bootstrap provides responsive breakpoints for different screen sizes:

  • .table-responsive - Always horizontal scroll
  • .table-responsive-sm - Breakpoint at 576px (small)
  • .table-responsive-md - Breakpoint at 768px (medium)
  • .table-responsive-lg - Breakpoint at 992px (large)
  • .table-responsive-xl - Breakpoint at 1200px (extra large)
  • .table-responsive-xxl - Breakpoint at 1400px (extra extra large)

💡 When to Use Responsive Tables

Best Practices
  • Always use for wide tables - Prevents layout breaking on mobile
  • Consider column priority - Show most important columns first
  • Use breakpoints wisely - Only add scrolling when needed
  • Test on real devices - Ensure usability
Alternative Approaches
  • Stack data vertically on mobile
  • Use cards instead of tables on small screens
  • Hide less important columns
