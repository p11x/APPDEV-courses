💡 Pro Tips

  • Always Add ARIA - Use role="group" and aria-label for accessibility
  • Use for Related Actions - Group buttons that do similar things
  • Consistent Styling - Use same color/size within a group
  • Vertical for Sidebars - Great for mobile and sidebar navigation
  • Toolbars for Complex UI - Combine multiple groups
  • Toggle for Forms - Use .btn-check for filter forms

✅ Best Practices

  1. Accessibility First - Always add role and aria-label
  2. Group Related Items - Don't random group unrelated buttons
  3. Consistent Colors - Use same color within groups
  4. Proper Sizing - All buttons in group should be same size
  5. Mobile Consideration - Use vertical groups on mobile
  6. Clear Labels - Use icon-only buttons with aria-labels
