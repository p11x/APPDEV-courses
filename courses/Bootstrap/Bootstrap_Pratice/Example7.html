🚀 Bootstrap Grid - Understanding Breakpoints

📚 Bootstrap Breakpoints Explained

Each responsive class applies to that breakpoint and larger. For example, .col-md applies to md (768px+), lg (992px+), xl (1200px+), and xxl (1400px+).

Breakpoint Class Prefix Min Width Device Type
Extra Small .col- None (always) Phones
Small .col-sm- 576px Large Phones / Small Tablets
Medium .col-md- 768px Tablets
Large .col-lg- 992px Laptops
Extra Large .col-xl- 1200px Desktops
Extra Extra Large .col-xxl- 1400px Large Desktops

📝 Quick Reference
  • .col-sm-* - Small screens 576px+
  • .col-md-* - Medium screens 768px+
  • .col-lg-* - Large screens 992px+
  • .col-xl-* - Extra large 1200px+
  • Higher breakpoints include all lower ones
