🚀 Bootstrap Grid - Horizontal Alignment

📚 What is Horizontal Alignment?

Use .justify-content-* on the row to align columns horizontally. This controls how columns are distributed within their container!

Key Point: Works with flexbox to position columns left, center, or spaced out!

📝 Quick Reference
  • .justify-content-start - Align left (default)
  • .justify-content-center - Center
  • .justify-content-end - Align right
  • .justify-content-around - Space around
  • .justify-content-between - Space between
  • .justify-content-evenly - Space evenly
  • Add responsive prefix: .justify-content-md-center, .justify-content-lg-between
