🖼️ Bootstrap Image Thumbnails

Learn how to create thumbnail-style images with borders.

📚 What are Image Thumbnails?

The .img-thumbnail class creates an image with:

  • Rounded corners (like .rounded)
  • 1px gray border
  • White padding (0.25rem)
  • Perfect for image galleries and previews

💡 When to Use Thumbnails

Best Use Cases
  • Image galleries - Grid of clickable thumbnails
  • Product listings - E-commerce product images
  • Portfolio - Work samples preview
  • Blog posts - Featured images
Design Tips
  • Use consistent image sizes in galleries
  • Add hover effects for interactivity
  • Consider using .img-fluid with thumbnails
