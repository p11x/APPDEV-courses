📚 Button Color Variants

Bootstrap provides 9 button color variants, each with a semantic meaning:

  • Primary - Main actions (blue)
  • Secondary - Secondary actions (gray)
  • Success - Positive actions (green)
  • Danger - Destructive actions (red)
  • Warning - Caution/Warnings (yellow)
  • Info - Information (cyan)
  • Light - Light/Subtle actions (white)
  • Dark - Dark actions (black)
  • Link - Looks like a text link
