📊 Bootstrap Table Head Styles

Learn how to style table headers with contextual colors.

📚 Table Head Classes

Bootstrap provides classes to style the element with contextual colors:

  • .table-light - Light/white header
  • .table-dark - Dark header

You can also use contextual table row classes on header cells:

  • .table-primary - Blue header
  • .table-success - Green header
  • .table-danger - Red header
  • .table-warning - Yellow header
  • .table-info - Cyan header
  • .table-secondary - Gray header
