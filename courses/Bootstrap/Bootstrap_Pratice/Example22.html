🚀 Text Colors & Contextual Styles

Add meaning to your text with Bootstrap's color classes!

📚 Bootstrap Text Colors

Bootstrap provides semantic text colors that convey meaning. Use them to highlight different types of content!

✨ Primary & Secondary Colors

.text-primary - Primary brand color (blue)

.text-secondary - Secondary color (gray)

✨ Semantic Colors

Use colors to convey status or meaning:

.text-success - Success! (green)

.text-danger - Error/Danger (red)

.text-warning - Warning (yellow)

.text-info - Information (cyan)

✨ Neutral Colors

For body text and subtle styling:

.text-dark - Dark gray (almost black)

.text-body - Default body text color

.text-muted - Muted/gray text

.text-black-50 - Black at 50% opacity

.text-white-50 - White at 50% opacity

✨ Background Colors

Add colored backgrounds to elements:

.bg-primary
.bg-secondary
.bg-success
.bg-danger
.bg-warning
.bg-info
.bg-light
.bg-dark
