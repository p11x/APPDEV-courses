📋 Bootstrap Dropdowns - Summary

Quick reference guide for all dropdown classes.

📚 Dropdown Classes

Class Description
.dropdown Wrapper container
.dropdown-toggle Trigger button with arrow
.dropdown-menu Menu list container
.dropdown-item Individual menu item
.dropdown-divider Separator line
.dropdown-header Section heading
.dropdown-menu-dark Dark themed menu

⬆️➡️⬅️⬇️ Direction Classes

Class Description
.dropup Opens upward
.dropend Opens to the right
.dropstart Opens to the left

🔘 Split Button Classes

Class Description
.btn-group Groups buttons together
.dropdown-toggle-split Split button toggle
.btn-sm Small button
.btn-lg Large button

🎯 Key Points

  • Use .dropdown as wrapper
  • Add data-bs-toggle="dropdown" to trigger
  • Use .dropdown-menu-dark for dark theme
  • Use .dropup, .dropend, .dropstart for directions
  • Use .dropdown-divider to separate items
  • Use .dropdown-header for section titles
