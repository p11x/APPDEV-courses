🚀 Bootstrap Grid - Row Columns (Vertical Layout)

📚 What are Row Columns?

Use .row-cols-* to control how many columns display per row. Unlike regular rows where columns go horizontally, row-cols makes them wrap vertically!

Key Point: This is the opposite of normal grid flow - columns fill top-to-bottom first!

📝 Quick Reference
  • .row-cols-* - Sets columns per row (1-6)
  • Columns flow vertically (top to bottom)
  • Great for galleries and card layouts
  • Can combine with responsive breakpoints
  • .row-cols-sm-2 - 2 cols at sm+
  • .row-cols-md-3 - 3 cols at md+
  • .row-cols-lg-4 - 4 cols at lg+
