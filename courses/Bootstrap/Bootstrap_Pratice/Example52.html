💡 Pro Tips

  • Always Start with .btn - This is the base class for all buttons
  • Use Primary for Main Action - Only one primary button per section
  • Outline for Secondary - Great for cancel/back buttons
  • Size Consistency - Use the same size for related buttons
  • Accessibility - Add aria-labels and role="button" for links
  • Mobile First - Use large buttons on mobile for better tapping

✅ Best Practices

  1. One Primary Button - Only one main action per form/section
  2. Clear Labels - Use action verbs (Save, Delete, Submit)
  3. Proper States - Always show disabled state when appropriate
  4. Size Appropriately - Large for CTAs, small for toolbars
  5. Color Meaning - Red for danger, green for success
  6. Consistency - Use the same button styles throughout your site
