2. Content Security Policy Awareness

Bootstrap 5 works without inline scripts. CSP headers should allow Bootstrap's CDN.

Content-Security-Policy:
  default-src 'self';
  script-src 'self' https://cdn.jsdelivr.net;
  style-src 'self' https://cdn.jsdelivr.net;
  img-src 'self' data:;
  connect-src 'self';

All Bootstrap components work with data-bs-* attributes — no inline event handlers needed.

Always use addEventListener instead of inline event attributes.

Use: button.addEventListener('click', doSomething)
