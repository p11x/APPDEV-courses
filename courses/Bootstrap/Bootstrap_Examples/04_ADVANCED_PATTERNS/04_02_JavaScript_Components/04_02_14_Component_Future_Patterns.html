Modern & Future JavaScript Patterns

Modern JavaScript patterns that complement Bootstrap 5 and prepare code for the future.

1. ES Module Import Pattern

Import individual Bootstrap components for smaller bundle sizes.

// Import only what you need (tree-shaking friendly)
import { Modal, Tooltip, Toast } from 'bootstrap';

// Or import the full bundle
import * as bootstrap from 'bootstrap';

// In a module environment:
const modal = new Modal(document.getElementById('myModal'));
modal.show();
Note: ES module imports require a bundler (Webpack, Vite, Rollup) or native ESM support. The CDN version used here attaches to the global bootstrap object.
