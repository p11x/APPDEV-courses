Performance Tools

Essential tools for measuring, monitoring, and optimizing web performance.

Browser Dev Tools

Built-in tools available in every modern browser.

Chrome DevTools

Performance tab provides flame charts, network waterfall, and Lighthouse integration.

  • Performance: Record & analyze runtime
  • Network: Resource loading waterfall
  • Lighthouse: Run audits inline
  • Coverage: Unused JS/CSS detection
Ctrl+Shift+I
Firefox DevTools

Performance panel with flame charts, memory profiling, and CSS performance insights.

  • Performance: Recording & markers
  • Network: Timing breakdown
  • Memory: Heap snapshots
  • Storage: Cache inspection
Ctrl+Shift+I
Safari Web Inspector

Timeline recording, network analysis, and WebKit-specific performance metrics.

  • Timelines: Record & analyze
  • Network: Request details
  • Storage: Cache & databases
  • Console: Performance API
Cmd+Option+I

NPM Packages

Install and integrate into your build pipeline.

web-vitals

Google's official library for measuring Core Web Vitals in real user sessions.

// Install
npm install web-vitals

// Usage
import { onLCP, onINP, onCLS } from 'web-vitals';
onLCP(console.log);
onINP(console.log);
onCLS(console.log);
webpack-bundle-analyzer

Visualize the size of webpack output files with an interactive zoomable treemap.

// Install
npm install --save-dev webpack-bundle-analyzer

// webpack.config.js
const { BundleAnalyzerPlugin } = require('webpack-bundle-analyzer');
plugins: [new BundleAnalyzerPlugin()]
size-limit

Calculate the real cost of your library for users, with CI budget enforcement.

// Install
npm install --save-dev size-limit @size-limit/file

// .size-limit.json
[{
  "path": "dist/*.js",
  "limit": "30 KB"
}]
@lhci/cli

Automate running Lighthouse for every commit, with performance budgets and assertions.

// Install
npm install --save-dev @lhci/cli

// Run
npx lhci autorun --collect.url=http://localhost:3000

CI/CD Integrations

Automate performance checks in your deployment pipeline.

GitHub Actions

Run Lighthouse on every PR with budget checks.

# .github/workflows/perf.yml
- uses: treosh/lighthouse-ci-action@v10
  with:
    budgetPath: ./budget.json
GitLab CI

Performance testing stage with Browser Performance Testing.

# .gitlab-ci.yml
performance:
  stage: test
  script:
    - npm run lighthouse
CircleCI

Performance orbs and custom workflows for Lighthouse.

# .circleci/config.yml
jobs:
  lighthouse:
    docker:
      - image: cimg/node
