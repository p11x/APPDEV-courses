Progressive enhancement through polyfills - adding missing browser features using JavaScript and CSS fallback strategies.

1. CSS Polyfills & Fallbacks

CSS Custom Properties (Variables)

Bootstrap 5 uses CSS custom properties extensively for theming. Browsers without support (IE 11) need fallback values.

background-color: var(--brand-light, #e7f1ff);

The fallback value (#e7f1ff) after the comma is used when the browser doesn't support CSS variables.

Aspect Ratio Fallback

The aspect-ratio property is relatively new. Use padding-bottom hack for older browsers.

16:9 Aspect Ratio

Padding-bottom hack:
height: 0; padding-bottom: 56.25%;
Creates a responsive 16:9 container that works in all browsers.

:focus-visible Fallback

:focus-visible allows differentiating keyboard focus from mouse focus. Not supported in older browsers.

Keyboard focus shows a ring, mouse click doesn't (in supporting browsers). Uses @supports selector(:focus-visible) for detection.

2. JavaScript Polyfills

Array Methods

Array.from(), Array.find(), Array.includes(), Array.flat() 

// Array.from() polyfill for IE 11
if (!Array.from) {
  Array.from = function(arrayLike) {
    return [].slice.call(arrayLike);
  };
}
// Array.includes() polyfill
if (!Array.prototype.includes) {
  Array.prototype.includes = function(searchElement) {
    return this.indexOf(searchElement) !== -1;
  };
}
Object Methods

Object.assign(), Object.entries(), Object.values(), Object.keys() 

// Object.assign() polyfill for IE 11
if (typeof Object.assign !== 'function') {
  Object.assign = function(target) {
    for (var i = 1; i < arguments.length; i++) {
      var source = arguments[i];
      for (var key in source) {
        if (Object.prototype.hasOwnProperty
            .call(source, key)) {
          target[key] = source[key];
        }
      }
    }
    return target;
  };
}
String Methods

String.includes(), String.startsWith(), String.endsWith(), String.repeat() 

// String.includes() polyfill
if (!String.prototype.includes) {
  String.prototype.includes = function(search, start) {
    if (typeof start !== 'number') start = 0;
    return this.indexOf(search, start) !== -1;
  };
}
DOM API Methods

Element.closest(), Element.matches(), NodeList.forEach() 

// Element.closest() polyfill for IE 11
if (!Element.prototype.closest) {
  Element.prototype.closest = function(selector) {
    var el = this;
    while (el && el.nodeType === 1) {
      if (el.matches(selector)) return el;
      el = el.parentElement;
    }
    return null;
  };
}

Conditional Polyfill Loading Pattern

Load polyfills only when needed, using feature detection:

<!-- Load polyfills conditionally -->
<script>
  // Only load fetch polyfill if fetch is not natively supported
  if (!window.fetch) {
    document.write('<script src="https://cdn.jsdelivr.net/npm/whatwg-fetch@3.6.2/dist/fetch.umd.js"><\/script>');
  }
  
  // Only load Promise polyfill if needed
  if (typeof Promise === 'undefined') {
    document.write('<script src="https://cdn.jsdelivr.net/npm/es6-promise@4.2.8/dist/es6-promise.auto.min.js"><\/script>');
  }
  
  // Check for CSS Custom Properties support
  if (!CSS || !CSS.supports || !CSS.supports('color', 'var(--test)')) {
    // Load css-vars-ponyfill
    document.write('<script src="https://cdn.jsdelivr.net/npm/css-vars-ponyfill@2.4.9/dist/css-vars-ponyfill.min.js"><\/script>');
  }
</script>

<!-- Then load your main application code -->
<script src="app.js"></script>

5. Bootstrap 5 Polyfill Requirements

Important: Bootstrap 5 dropped IE 11 support. The following information is for reference if you need to support legacy browsers.
What Bootstrap 5 Uses (Modern)
  • CSS Custom Properties - theming system
  • CSS Grid - utility classes
  • Flexbox - grid system core
  • ES6 JavaScript - let, const, arrow functions
  • ES6 Classes - component architecture
  • Arrow functions - event handlers
  • Template literals - string building
  • destructuring - config objects
What IE 11 Would Need
  • core-js - ES6+ polyfills
  • regenerator-runtime - async/await
  • css-vars-ponyfill - CSS variables
  • CustomEvent polyfill - event dispatch
  • classList polyfill - class manipulation
  • Promise polyfill - async operations
  • Fetch polyfill - AJAX requests
  • Babel transpiler - ES6 to ES5
