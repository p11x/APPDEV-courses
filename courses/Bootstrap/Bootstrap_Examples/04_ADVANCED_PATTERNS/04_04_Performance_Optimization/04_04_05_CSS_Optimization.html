CSS Optimization

Reduce CSS payload and eliminate render-blocking stylesheets.

CSS Impact on Core Web Vitals

FCP
First Contentful Paint

Render-blocking CSS delays FCP. Inline critical CSS to paint content faster.

LCP
Largest Contentful Paint

Heavy CSS files delay the largest element. Minify and compress CSS.

CLS
Cumulative Layout Shift

Late-loading CSS causes layout shifts. Avoid FOIT/FOUT with proper loading.

Critical CSS Extraction

What is Critical CSS?

Critical CSS is the minimum set of styles needed to render the above-the-fold content. By inlining these styles, you eliminate a render-blocking HTTP request.

Steps:
  1. Identify above-the-fold content for each page/route
  2. Extract CSS rules that style that content
  3. Inline extracted CSS in a <style> tag
  4. Load full CSS asynchronously
Async CSS Loading

Load the full stylesheet without blocking render using this pattern:

<!-- Inline critical CSS -->
<style>
  /* Above-the-fold styles */
  .hero { min-height: 50vh; }
  .nav { display: flex; }
</style>

<!-- Load full CSS async -->
<link rel="stylesheet"
      href="bootstrap.min.css"
      media="print"
      onload="this.media='all'">
PurgeCSS Configuration

PurgeCSS scans your HTML/JS files and removes unused CSS selectors.

// postcss.config.js
module.exports = {
  plugins: [
    require('@fullhuman/postcss-purgecss')({
      content: [
        './src/**/*.html',
        './src/**/*.js'
      ],
      safelist: {
        standard: [/^modal/,
          /^tooltip/, /^show/]
      }
    })
  ]
};
Sass Selective Imports

Import only the Bootstrap modules you need in your Sass build:

// custom-bootstrap.scss

// Required: functions and variables
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";
@import "bootstrap/scss/mixins";

// Pick only what you need
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/type";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import "bootstrap/scss/utilities/api";

// Skip: carousel, accordion, etc.
// Savings: ~40% smaller CSS file

Prefer Utilities Over Custom CSS

Bootstrap 5 utilities are already in the CSS bundle. Using them avoids adding custom CSS rules.

Custom CSS (Avoid)
/* Custom CSS adds to bundle */
.card-custom {
  padding: 1.5rem;
  margin-bottom: 1rem;
  background: white;
  border-radius: 0.5rem;
  box-shadow: 0 2px 8px
    rgba(0,0,0,0.1);
}

Optimization Tools

PurgeCSS

Removes unused CSS by analyzing your content files.

npm i @fullhuman/postcss-purgecss
cssnano

Modular CSS minifier that compresses whitespace, colors, and selectors.

npm i cssnano
Critical

Extracts & inlines above-the-fold CSS for faster first paint.

npm i critical
UnCSS

Removes unused CSS from stylesheets by running pages through PhantomJS/Puppeteer.

npm i uncss
Lightning CSS

Rust-based CSS parser, minifier, and bundler. Extremely fast.

npm i lightningcss
Bundlephobia

Check npm package sizes before adding to your project.

bundlephobia.com
