Image Optimization

Serve the right image, at the right size, to every device.

Responsive Images

.img-fluid Class

Bootstrap's .img-fluid applies max-width: 100% and height: auto so images scale with their parent container.

<img class="img-fluid" ...>
srcset Attribute

The srcset attribute provides multiple image resolutions. The browser picks the best one based on device pixel density and viewport.

<img srcset="400w, 800w, 1200w" sizes="...">

Lazy Loading

The loading="lazy" attribute defers off-screen images until the user scrolls near them. This reduces initial page weight and improves Largest Contentful Paint (LCP).

Eager (Default)

Loads immediately. Use for above-the-fold hero images.

<img loading="eager">
Lazy Loaded

Deferred until near viewport. Use for below-the-fold content.

<img loading="lazy">
Lazy + Skeleton

Skeleton placeholder shown until image finishes loading.

CSS shimmer + loading="lazy"

Optimization Techniques

Explicit Dimensions

Always set width and height attributes. This prevents Cumulative Layout Shift (CLS) by reserving space before the image loads.

<img src="photo.jpg"
    width="800" height="400"
    class="img-fluid">
CLS: 0.00
Async Decoding

decoding="async" allows the browser to decode images off the main thread, preventing jank during page rendering.

<img src="photo.jpg"
    decoding="async">
Non-blocking decode
Fetch Priority

fetchpriority="high" on LCP images tells the browser to prioritize downloading the most important visual element.

<img src="hero.jpg"
    fetchpriority="high">

<img src="below-fold.jpg"
    fetchpriority="low">
Priority hints
Object Fit

Bootstrap 5 utility classes for object-fit control how images fill their containers.

.object-fit-cover / .object-fit-contain
Placeholder Colors

Use a dominant color as background-color on the image container to provide a visual placeholder while loading.

background-color: #dominant
CDN Image Transforms

Use image CDNs (Cloudinary, Imgix, Cloudflare Images) to auto-resize, convert formats, and optimize quality on the fly.

// Cloudinary URL-based transform
res.cloudinary.com/demo/
w_400,h_300,c_fill/
f_webp/photo.jpg
