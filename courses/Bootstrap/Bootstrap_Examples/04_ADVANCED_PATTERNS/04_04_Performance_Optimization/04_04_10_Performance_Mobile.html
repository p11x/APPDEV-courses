Adaptive Loading

Network-Aware Loading

Use the Network Information API to adapt content based on connection quality.

// Check connection quality
const conn = navigator.connection || {};
const effectiveType = conn.effectiveType; // 4g, 3g, 2g, slow-2g
const saveData = conn.saveData; // user enabled data saver

if (effectiveType === '4g' && !saveData) {
  // Load high-res images, video autoplay
  loadHighQuality();
} else {
  // Load low-res images, no video autoplay
  loadLowQuality();
}
Device Memory-Aware

Use the Device Memory Client Hint to serve lighter experiences on low-memory devices.

// Check available device memory (GB)
const memory = navigator.deviceMemory || 4;

if (memory <= 2) {
  // Low-end device: skip heavy animations
  disableAnimations();
  loadPlaceholderImages();
} else if (memory <= 4) {
  // Mid-range: moderate quality
  loadMediumQuality();
} else {
  // High-end: full experience
  loadFullExperience();
}
Required Client Hints Headers
# Server must request client hints via Accept-CH header
Accept-CH: Device-Memory, Downlink, ECT, RTT, Save-Data
Critical-CH: Device-Memory, ECT

Touch Target Optimization

WCAG 2.5.8 requires touch targets of at least 24x24 CSS pixels. Apple recommends 44x44pt and Google recommends 48x48dp.

Top: Good (48px). Bottom: Minimum (24px)
