Performance Monitoring

Measure real user metrics with Web Vitals and the Performance API.

Core Web Vitals

--
LCP

Largest Contentful Paint

Measures loading performance. The time until the largest visible element finishes rendering.

≤ 2.5s ≤ 4s > 4s
--
INP

Interaction to Next Paint

Measures responsiveness. The latency of the worst interaction during the page lifecycle.

≤ 200ms ≤ 500ms > 500ms
--
CLS

Cumulative Layout Shift

Measures visual stability. The sum of all unexpected layout shifts during the page session.

≤ 0.1 ≤ 0.25 > 0.25
--
FCP

First Contentful Paint

Time until the first DOM content is painted. Good: ≤ 1.8s

--
TTFB

Time to First Byte

Time until the first byte of the response is received. Good: ≤ 800ms

Performance API Examples

Navigation Timing
// Get navigation timing data
const nav = performance.getEntriesByType(
  'navigation'
)[0];

console.log({
  dns: nav.domainLookupEnd - nav.domainLookupStart,
  tcp: nav.connectEnd - nav.connectStart,
  ttfb: nav.responseStart - nav.requestStart,
  download: nav.responseEnd - nav.responseStart,
  domParse: nav.domInteractive - nav.responseEnd,
  domReady: nav.domContentLoadedEventEnd,
  fullLoad: nav.loadEventEnd
});
Resource Timing
// Analyze loaded resources
const resources = performance
  .getEntriesByType('resource');

resources.forEach(r => {
  console.log({
    name: r.name.split('/').pop(),
    type: r.initiatorType,
    size: r.transferSize,
    duration: Math.round(r.duration)
  });
});
