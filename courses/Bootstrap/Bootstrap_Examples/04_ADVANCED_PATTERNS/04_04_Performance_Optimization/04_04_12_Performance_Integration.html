Performance Integration

Embed performance optimization into every stage of your build pipeline.

Optimization Pipeline

Build Steps
Sass Compilation
Compile only imported Bootstrap modules
PurgeCSS
Remove unused CSS selectors
Autoprefixer
Add vendor prefixes for target browsers
Minification
Minify CSS (cssnano) and JS (Terser)
Compression
Generate .gz and .br pre-compressed files
Integrity Hash
Generate SRI hashes for CDN assets
PostCSS Config
// postcss.config.js
module.exports = {
  plugins: [
    // Autoprefixer for cross-browser compat
    require('autoprefixer'),

    // PurgeCSS to remove unused styles
    require('@fullhuman/postcss-purgecss')({
      content: ['./src/**/*.html',
        './src/**/*.jsx'],
      safelist: {
        standard: [/^modal/, /show$/]
      }
    }),

    // CSSnano for minification
    require('cssnano')({
      preset: 'advanced'
    })
  ]
};

Framework Integration

React + Bootstrap Optimization
// React.lazy for route-based code splitting
const Dashboard = React.lazy(
  () => import('./pages/dashboard')
);

// Suspense boundary for loading state
function App() {
  return (
    <Suspense fallback={<Spinner />}>
      <Routes>
        <Route path="/dashboard"
          element={<Dashboard />} />
      </Routes>
    </Suspense>
  );
}

// Only import needed Bootstrap components
import { Modal, Dropdown } from 'bootstrap';
Vue + Bootstrap Optimization
// Vue Router lazy loading
const routes = [{
  path: '/dashboard',
  component: () => import('./Dashboard.vue')
}];

// Dynamic component loading
const HeavyChart = defineAsyncComponent(() =>
  import('./components/HeavyChart.vue')
);

// In vite.config.js
build: {
  rollupOptions: {
    output: {
      manualChunks: {
        bootstrap: ['bootstrap']
      }
    }
  }
}
Angular + Bootstrap Optimization
// Angular lazy-loaded module
const routes: Routes = [{
  path: 'dashboard',
  loadChildren: () => import(
    './dashboard/dashboard.module'
  ).then(m => m.DashboardModule)
}];

// Angular.json optimization config
"budgets": [{
  "type": "initial",
  "maximumWarning": "200kb",
  "maximumError": "300kb"
}],
"optimization": {
  "scripts": true,
  "styles": {
    "minify": true,
    "inlineCritical": true
  }
}

Deployment Strategies

Cache Busting

Append content hash to filenames for immutable caching.

// Webpack output with content hash
output: {
  filename: '[name].[contenthash:8].js',
  chunkFilename: '[name].[contenthash:8].chunk.js'
}

// Result: main.a1b2c3d4.js
// Cache-Control: max-age=31536000, immutable
Edge Caching (CDN)

Deploy static assets to edge locations for lower latency.

# Cloudflare Cache-Control headers
/*.{js,css}
  Cache-Control: public, max-age=31536000,
    immutable

/*.html
  Cache-Control: public, max-age=0,
    must-revalidate

# Vercel vercel.json
{ "headers": [{
  "source": "/static/(.*)",
  "headers": [{
    "key": "Cache-Control",
    "value": "public, max-age=31536000,
      immutable"
  }]
}]}
