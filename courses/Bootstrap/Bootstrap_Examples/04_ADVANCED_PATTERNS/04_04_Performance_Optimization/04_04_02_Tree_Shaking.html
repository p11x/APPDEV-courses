Tree Shaking Bootstrap

Import only what you use. Eliminate dead code from your bundle.

How Tree Shaking Works

Bootstrap 5 provides ES module builds. Use import statements instead of require() to enable static analysis.

// ES module import - tree-shakeable
import { Tooltip, Dropdown } from 'bootstrap';

// CommonJS require - NOT tree-shakeable
const bootstrap = require('bootstrap');

Import specific components by name. The bundler marks unused exports as dead code.

// Only these 3 components will be in your bundle
import {
  Alert,
  Button,
  Modal
} from 'bootstrap';

// Carousel, ScrollSpy, Toast etc. are excluded

Webpack, Rollup, or Vite performs dead code elimination during production builds. Unused exports are never included in the output bundle.

// webpack.config.js
module.exports = {
  mode: 'production', // enables minification + tree shaking
  optimization: {
    usedExports: true, // mark unused exports
    sideEffects: false // trust package.json sideEffects
  }
};

Use bundle analysis tools to verify what was included. Tools like webpack-bundle-analyzer visualize the output.

// Install analyzer
npm install --save-dev webpack-bundle-analyzer

// Add to webpack config
const { BundleAnalyzerPlugin } = require('webpack-bundle-analyzer');
plugins: [new BundleAnalyzerPlugin()]

Bundler Configuration Examples

// webpack.config.js
module.exports = {
  mode: 'production',
  entry: './src/index.js',
  optimization: {
    usedExports: true,
    sideEffects: false,
    minimize: true
  },
  resolve: {
    mainFields: ['module', 'main']
  }
};
// vite.config.js
import { defineConfig } from 'vite';

export default defineConfig({
  build: {
    rollupOptions: {
      treeshake: true,
      output: {
        manualChunks: {
          bootstrap: ['bootstrap']
        }
      }
    }
  }
});
// rollup.config.js
import { nodeResolve } from '@rollup/plugin-node-resolve';
import terser from '@rollup/plugin-terser';

export default {
  input: 'src/index.js',
  output: {
    file: 'dist/bundle.js',
    format: 'es'
  },
  plugins: [nodeResolve(), terser()],
  treeshake: 'smallest'
};
