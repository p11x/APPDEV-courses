Bootstrap Contrast Fixes

Warning & Yellow Text

Bootstrap warning colors may not meet AA contrast. Use .text-dark with warning backgrounds.

text-dark (good) text-white (bad)
<!-- Good: dark text on warning -->
<div class="alert alert-warning
       text-dark">...</div>

<!-- Override via Sass -->
$warning-text: #664d03;
$warning-bg: #ffc107;
Custom Color Contrast

When creating custom colors, always verify contrast ratios.

// JavaScript contrast calculation
function getLuminance(r, g, b) {
  const [rs, gs, bs] = [r,g,b].map(c => {
    c /= 255;
    return c <= 0.03928
      ? c / 12.92
      : Math.pow((c + 0.055)/1.055, 2.4);
  });
  return 0.2126*rs + 0.7152*gs + 0.0722*bs;
}

function contrastRatio(l1, l2) {
  const lighter = Math.max(l1, l2);
  const darker = Math.min(l1, l2);
  return (lighter + 0.05) / (darker + 0.05);
}

Designing for Color Blindness

Approximately 8% of men and 0.5% of women have some form of color vision deficiency. Never rely on color alone to convey information.

Normal Vision
Error Warning Success
Colors + icons
Protanopia (no red)
Error Warning Success
Colors look similar
Deuteranopia (no green)
Error Warning Success
Hard to distinguish
Solution: Icons + Text
✗ Error ⚠ Warn ✓ OK
Color + icon + text
Non-Color Indicators Checklist
  • Use icons alongside color indicators
  • Add text labels (not just colored dots)
  • Use patterns/textures in charts
  • Underline links (not just color change)
  • Add borders to distinguish regions
  • Use position and shape for data points
  • Provide text alternatives for charts
  • Test with color blindness simulators
