Keyboard Navigation

Ensure every interaction is accessible without a mouse.

Keyboard Navigation Principles

All Interactive Elements Focusable

Every button, link, input, and interactive control must be reachable via Tab key.

<!-- Good: naturally focusable -->
<button>Click</button>

<!-- Bad: not focusable -->
<div onclick="doThing()">Click</div>
Visible Focus Indicator

Always show a visible focus ring. Use :focus-visible for keyboard-only focus styles.

/* Keyboard-only focus style */
:focus-visible {
  outline: 3px solid #0d6efd;
  outline-offset: 2px;
}

/* Remove default for mouse */
:focus:not(:focus-visible) {
  outline: none;
}
Logical Tab Order

Tab order should follow the visual reading order. Avoid positive tabindex values.

<!-- Correct: DOM order = tab order -->
<input id="first">
<input id="second">

<!-- Wrong: tabindex breaks order -->
<input tabindex="3">
<input tabindex="1">

Tab Order & tabindex

tabindex Value Behavior When to Use
tabindex="0" Adds element to natural tab order Custom interactive elements (e.g., div with role="button")
tabindex="-1" Programmatically focusable, not in tab order Modals, alerts, skip targets
tabindex="1+" Forces specific tab order (anti-pattern) Avoid. Use DOM ordering instead.

Roving Tabindex Pattern

In a toolbar or tablist, only one item is in the tab order (tabindex="0"). Arrow keys move focus between items. This reduces Tab stops and provides efficient keyboard navigation.

Roving Tabindex Toolbar Demo

Use Arrow Left/Right to move between buttons. Tab enters and exits the toolbar.

// Roving tabindex JavaScript
const toolbar = document.getElementById('rovingToolbar');
const items = [...toolbar.querySelectorAll('button')];

toolbar.addEventListener('keydown', (e) => {
  const current = document.activeElement;
  const idx = items.indexOf(current);
  let next;

  if (e.key === 'ArrowRight') next = items[(idx + 1) % items.length];
  if (e.key === 'ArrowLeft') next = items[(idx - 1 + items.length) % items.length];
  if (e.key === 'Home') next = items[0];
  if (e.key === 'End') next = items[items.length - 1];

  if (next) {
    e.preventDefault();
    current.tabIndex = -1;
    next.tabIndex = 0;
    next.focus();
  }
});
