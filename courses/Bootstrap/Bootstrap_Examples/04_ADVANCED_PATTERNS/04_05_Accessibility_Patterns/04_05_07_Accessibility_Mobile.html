Bootstrap Touch Utilities
/* Increase tap targets on touch devices */
@media (pointer: coarse) {
  .btn {
    padding: 0.75rem 1.25rem;
    min-height: 48px;
  }
  .nav-link {
    padding: 0.75rem 1rem;
  }
  .form-check-input {
    width: 1.5rem;
    height: 1.5rem;
  }
}

/* Spacing between adjacent targets */
.btn-group .btn + .btn {
  margin-left: 4px;
}

Zoom & Scaling

Never Disable Zoom
<!-- BAD: Disables zoom -->
<meta name="viewport"
  content="width=device-width,
  user-scalable=no,
  maximum-scale=1.0">

<!-- GOOD: Allows zoom -->
<meta name="viewport"
  content="width=device-width,
  initial-scale=1.0">

WCAG 1.4.4 requires text can be resized to 200% without loss of content.

Zoom-Friendly Design
  • Use relative units (rem, em, %) instead of px
  • Ensure content reflows at 200% zoom
  • No horizontal scrolling at 320px CSS width
  • Use Bootstrap responsive grid (col-* classes)
  • Test with browser zoom at 200%
  • Ensure modals work at high zoom levels

Reduced Motion

WCAG 2.3.3 requires animations can be disabled. Bootstrap 5 supports this via prefers-reduced-motion.

/* Bootstrap's built-in reduced motion support */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}

/* Custom: disable specific animations */
@media (prefers-reduced-motion: reduce) {
  .carousel .carousel-item {
    transition: none;
  }
  .fade { transition: none; }
}
