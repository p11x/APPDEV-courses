1. Basic Single-Property Transition

The simplest CSS transition targets a single property. Hover the card below to see background-color smoothly change over 0.4 seconds.

Hover Me
transition: background-color 0.4s ease-in-out
Syntax Breakdown
transition: background-color 0.4s ease-in-out;
/*           ^property     ^duration ^timing  */
  • transition-property: Which CSS property to animate
  • transition-duration: How long (0.4s = 400ms)
  • transition-timing-function: Speed curve (ease-in-out)

10. Transition Properties Summary

Property Values Description
transition-property none | all | property-name Which CSS property to animate
transition-duration 0s | 0.3s | 300ms How long the transition takes
transition-timing-function ease | linear | ease-in | cubic-bezier(...) Speed curve of the transition
transition-delay 0s | 0.1s | -0.2s Wait time before transition starts
transition property duration timing-function delay Shorthand for all four properties
