Cross-Browser Animation

Vendor prefixes, feature detection, progressive enhancement, and browser bug workarounds.

1. @supports Feature Detection

Use @supports to progressively enhance with animations. Browsers that don't support animations get a simpler but still functional experience.

Animation Support

@supports (animation: fadeIn 0.3s)

Backdrop Filter

@supports (backdrop-filter: blur(10px))

CSS Transforms

@supports (transform: translateX(0))

2. Progressive Enhancement Pattern

Base experience works without animations. Animations are layered on top as an enhancement for capable browsers.

4. Common Browser Animation Bugs

Safari Bugs
  • Flex clipping: Add overflow: visible to flex parent
  • Transform-origin: Use translateY() instead
  • position:fixed: Use absolute within fixed wrapper
  • 100vh: Use 100dvh or JS to calculate
Firefox Bugs
  • will-change leak: Remove after animation completes
  • SVG transforms: Apply transform-box: fill-box
  • Flex gap animation: Use margin instead
  • backdrop-filter: Enable in about:config (older versions)
Chrome Bugs
  • Subpixel jitter: Use translateZ(0)
  • Font rendering: Add -webkit-font-smoothing
  • will-change memory: Limit to 4-5 elements at once
  • Layer explosion: Avoid promoting too many elements
Edge/IE (Legacy)
  • IE11: No CSS animation — use JS fallback
  • Old Edge: Use -ms- prefixes
  • Graceful degradation: Content must still be visible
  • JS fallback: element.style.opacity = 1

6. JavaScript Feature Detection

CSS.supports() API
// Check if CSS animation is supported
if (CSS.supports('animation', 'fadeIn 0.3s')) {
    el.classList.add('animated');
} else {
    el.style.opacity = 1; // Fallback
}

// Check will-change support
if (CSS.supports('will-change', 'transform')) {
    el.style.willChange = 'transform';
}
Event-based Detection
// Detect animation event support
const animEnd = (function() {
    const el = document.createElement('div');
    const map = {
        'animation': 'animationend',
        'WebkitAnimation': 'webkitAnimationEnd',
        'MozAnimation': 'animationend',
        'OAnimation': 'oAnimationEnd'
    };
    for (let prop in map) {
        if (el.style[prop] !== undefined) {
            return map[prop];
        }
    }
})();

8. Autoprefixer Configuration

package.json browserslist
{
  "browserslist": [
    "> 0.5%",           /* Browsers with >0.5% global usage */
    "last 2 versions",  /* Last 2 versions of each browser */
    "not dead",         /* Exclude unsupported browsers */
    "not ie 11"         /* Exclude IE11 (optional) */
  ]
}
PostCSS Config
// postcss.config.js
module.exports = {
  plugins: [
    require('autoprefixer'),  // Adds vendor prefixes automatically
    require('cssnano')        // Minifies CSS
  ]
}
