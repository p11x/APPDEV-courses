11. Animation Shorthand Reference

Property Values Description
animation-name myAnimation Name of the @keyframes rule
animation-duration 0s | 0.5s | 500ms Time for one animation cycle
animation-timing-function ease | linear | cubic-bezier(...) Acceleration curve
animation-delay 0s | 0.5s Wait before starting
animation-iteration-count 1 | 3 | infinite Number of times to repeat
animation-direction normal | reverse | alternate Playback direction
animation-fill-mode none | forwards | backwards | both Styles before/after animation
animation-play-state running | paused Whether animation is playing
Shorthand order: animation: name duration timing-function delay iteration-count direction fill-mode play-state;
