5. PerformanceObserver Integration

Long Animation Frames API
// Detect frames longer than 50ms
const observer = new PerformanceObserver((list) => {
    for (const entry of list.getEntries()) {
        console.warn('Long frame:', {
            duration: entry.duration,
            startTime: entry.startTime,
            name: entry.name
        });
    }
});

observer.observe({
    type: 'long-animation-frame',
    buffered: true
});
Animation & Transition Events
// Track all animation events
const animObserver = new PerformanceObserver(
    (list) => {
        for (const entry of list.getEntries()) {
            if (entry.entryType === 'measure') {
                console.log(`${entry.name}: ${
                    entry.duration.toFixed(2)
                }ms`);
            }
        }
    }
);

animObserver.observe({ entryTypes: ['measure'] });

// Measure an animation
performance.measure('card-animate', {
    start: animStartTime,
    end: animEndTime
});

7. Chrome DevTools Performance Guide

Performance Panel
  1. Open DevTools → Performance tab
  2. Click Record (Ctrl+E)
  3. Interact with animations
  4. Stop recording
  5. Look for red (long tasks) in the flame chart
  6. Check "Summary" for layout/paint/composite time
Rendering Panel
  1. Open DevTools → Rendering tab
  2. Enable "Paint flashing" — see repainted regions
  3. Enable "Layer borders" — see GPU layers
  4. Enable "Frame Rendering Stats" — live FPS counter
  5. Enable "Scrolling performance issues"
