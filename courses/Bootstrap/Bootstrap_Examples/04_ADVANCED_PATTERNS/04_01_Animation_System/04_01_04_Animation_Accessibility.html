Animation Accessibility

Building inclusive animations that respect user preferences and WCAG guidelines.

3. Marquee / Scrolling Text

Continuous scrolling can cause motion sickness. Always pause on hover and disable completely for reduced-motion users.

This scrolling text pauses on hover and is completely disabled for users with reduced motion preference enabled.    |    WCAG 2.2 Success Criterion 2.3.3: Animation from Interactions.

4. aria-live for Dynamic Content

When content appears via animation, screen readers need to know about it. aria-live="polite" announces changes without interrupting.

5. Focus Management with Animations

When animated content appears, move keyboard focus to it so users can interact without losing their place.

6. WCAG Animation Guidelines

Criterion Level Requirement Solution
1.3.3 Sensory A Instructions don't rely solely on shape/animation Use text labels alongside visual cues
2.2.2 Pause/Stop/Hide A Moving content >5s can be paused Add pause button for carousels, marquees
2.3.1 Three Flashes A No content flashes more than 3x/second Avoid rapid flashing animations
2.3.3 Animation from Interactions AAA Users can disable non-essential motion Respect prefers-reduced-motion
4.1.2 Name, Role, Value A Dynamic content is announced Use aria-live regions

7. CSS Patterns for Reduced Motion

Nuclear Option (Disable Everything)
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
Targeted Approach (Specific Elements)
@media (prefers-reduced-motion: reduce) {
  .animated-card {
    animation: fadeIn 0.1s ease !important;
  }
  .carousel-item {
    transition: none !important;
  }
  .parallax {
    background-attachment: scroll !important;
  }
}

8. Bootstrap 5's Built-In Motion Support

Bootstrap 5 automatically respects prefers-reduced-motion for:
  • Carousel: Auto-advancement stops, transitions become instant
  • Modal: Fade transition is disabled
  • Collapse: Height animation becomes instant
  • Toast: Fade in/out is instant
  • Offcanvas: Slide transition is instant
  • Dropdown: No animation on open/close
  • Alert dismiss: Instant removal
