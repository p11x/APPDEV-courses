1. Scroll-Driven Animations Chrome 115+

animation-timeline: scroll() and view() link animation progress to scroll position — no JavaScript needed.

Syntax: animation-timeline: scroll() (scroll container) or animation-timeline: view() (element visibility). animation-range controls when the animation starts/ends.
animation-range: entry 0% entry 100% — Element fully enters viewport
animation-range: entry 0% exit 100% — From entering to fully exiting
animation-range: cover 0% cover 100% — While element is in viewport

2. @starting-style Chrome 117+

@starting-style defines the initial state for elements when they are first rendered, enabling entry transitions without JavaScript class toggling.

CSS
.card {
    opacity: 1;
    transform: translateY(0);
    transition: opacity 0.5s, transform 0.5s;

    @starting-style {
        opacity: 0;
        transform: translateY(20px);
    }
}

5. View Transitions API Chrome 111+

The View Transitions API enables smooth animated transitions between DOM states (page navigations, content swaps) with minimal JavaScript.

Content A

Click the button to swap this content with a view transition animation.

JavaScript
// View Transitions API
if (document.startViewTransition) {
    document.startViewTransition(() => {
        // Swap DOM content here
        container.innerHTML = newContent;
    });
} else {
    // Fallback: instant swap
    container.innerHTML = newContent;
}
