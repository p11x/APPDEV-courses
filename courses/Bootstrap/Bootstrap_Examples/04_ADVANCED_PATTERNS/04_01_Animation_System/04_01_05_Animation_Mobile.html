8. Mobile Animation Best Practices

Practice Why How
Shorter durations Mobile users expect snappy UI 200-400ms on mobile vs 400-600ms on desktop
Disable decorative animations Saves battery on mobile @media (max-width: 767px)
Touch feedback Confirms user action :active + touch-action: manipulation
Avoid infinite animations Battery drain on mobile Use animation-iteration-count: 1
IntersectionObserver More efficient than scroll events Only animate visible elements
scroll-snap Native swipe feel CSS scroll-snap-type
Safe area insets Respect iPhone notch env(safe-area-inset-*)
