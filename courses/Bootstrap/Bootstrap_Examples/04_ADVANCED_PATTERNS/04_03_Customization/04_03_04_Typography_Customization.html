Article Typography

This section demonstrates how custom typography works in a real article context using the .prose container.

Heading Level Two

Body text uses Inter at 1rem with a line height of 1.8 for comfortable reading. Headings use Playfair Display for a distinctive editorial feel.

Heading Level Three

Inline code uses JetBrains Mono at a slightly smaller size. The font stack includes fallbacks for systems without the custom font installed.

"Typography is the craft of endowing human language with a durable visual form." - Robert Bringhurst

Heading Level Four

The modular type scale uses a 1.25 (major third) ratio, creating harmonious size relationships. Each step up multiplies the base size by the ratio.
