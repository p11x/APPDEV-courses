Custom Builds

Custom Bootstrap Builds

Create lean, optimized Bootstrap builds by importing only what you need.

Why Custom Builds?

🚀
Performance

Remove unused CSS to reduce bundle size by up to 80%.

🎨
Customization

Override SCSS variables before compilation for deep control.

🔒
Consistency

Lock in your design system values at the SCSS level.

Project Structure

📁 scss/
📄 _variables.scss
📄 custom-bootstrap.scss (main entry)
📄 _overrides.scss
📄 _components.scss
📁 css/
📄 custom-bootstrap.css (compiled output)
📄 custom-bootstrap.min.css
📄 package.json
📄 vite.config.js

Build Steps

Install Bootstrap and Sass
npm install bootstrap@5.3.3 sass
Create Variables Override File
// scss/_variables.scss // Override BEFORE importing Bootstrap $primary: #4f46e5; $secondary: #64748b; $success: #059669; $danger: #dc2626; $font-family-sans-serif: 'Inter', system-ui, sans-serif; $border-radius: 0.5rem; $enable-shadows: true; $enable-gradients: true; // Grid customization $grid-breakpoints: ( xs: 0, sm: 576px, md: 768px, lg: 992px, xl: 1200px, xxl: 1400px );
Selective Component Import
// scss/custom-bootstrap.scss // 1. Import functions first @import "bootstrap/scss/functions"; // 2. Import your variable overrides @import "variables"; // 3. Import Bootstrap's variables @import "bootstrap/scss/variables"; @import "bootstrap/scss/variables-dark"; // 4. Import mixins @import "bootstrap/scss/maps"; @import "bootstrap/scss/mixins"; @import "bootstrap/scss/root"; // 5. Import ONLY the utilities you need @import "bootstrap/scss/utilities"; @import "bootstrap/scss/utilities/api"; // 6. Import ONLY the components you need @import "bootstrap/scss/reboot"; @import "bootstrap/scss/type"; @import "bootstrap/scss/containers"; @import "bootstrap/scss/grid"; @import "bootstrap/scss/buttons"; @import "bootstrap/scss/card"; // Skip: nav, navbar, modal, tooltip, etc. // 7. Your custom styles @import "overrides"; @import "components";
Compile with npm script
// package.json scripts: { "scripts": { "css:compile": "sass scss/custom-bootstrap.scss css/custom-bootstrap.css --no-source-map", "css:minify": "sass scss/custom-bootstrap.scss css/custom-bootstrap.min.css --style=compressed --no-source-map", "build": "npm run css:compile && npm run css:minify" } }

Bundle Size Comparison

Full Bootstrap: ~227 KB (unminified)
Common subset: ~102 KB
Minimal (grid + buttons): ~45 KB

Sizes are approximate uncompressed CSS. Minification and gzip further reduce transfer size by 70-85%.

Vite Configuration Example

// vite.config.js import { defineConfig } from 'vite'; export default defineConfig({ css: { preprocessorOptions: { scss: { // Auto-include variables in every SCSS file additionalData: `@import "./scss/_variables.scss";` } } }, build: { cssCodeSplit: true, minify: 'esbuild', rollupOptions: { output: { assetFileNames: 'assets/[name].[hash].css' } } } });
