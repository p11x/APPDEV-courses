How It Works
  • data-bs-theme attribute: Set on <html> to apply theme globally
  • OS Detection: Bootstrap auto-detects prefers-color-scheme
  • localStorage: Theme preference persists across sessions
  • CSS Variables: All components adapt via Bootstrap's variable system
  • Custom Themes: Define new themes with [data-bs-theme="name"] selectors
