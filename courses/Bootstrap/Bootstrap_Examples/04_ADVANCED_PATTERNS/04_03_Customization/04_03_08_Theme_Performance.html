CSS Loading Strategies

✕ Blocking Load
<link rel="stylesheet"
      href="theme.css">

Browser cannot render anything until this CSS is downloaded and parsed.

✓ Non-Blocking Load
<link rel="stylesheet" href="theme.css"
      media="print" onload="this.media='all'">
<noscript>
  <link rel="stylesheet" href="theme.css">
</noscript>

CSS loads without blocking. Switches to all media once loaded.

Lazy Theme Loading Pattern

// Lazy load theme CSS
function loadTheme(themeName) {
  // Check if already loaded
  if (document.querySelector(`link[data-theme="${themeName}"]`)) {
    return Promise.resolve();
  }

  return new Promise((resolve, reject) => {
    const link = document.createElement('link');
    link.rel = 'stylesheet';
    link.href = `/css/theme-${themeName}.css`;
    link.dataset.theme = themeName;
    link.onload = resolve;
    link.onerror = reject;
    document.head.appendChild(link);
  });
}

// Usage: load theme on demand
document.getElementById('theme-switcher')
  .addEventListener('click', async () => {
    await loadTheme('dark');
    document.documentElement.setAttribute('data-bs-theme', 'dark');
  });

Font Loading Optimization

Subset Fonts

Use unicode-range to load only the characters you need. Reduces font file by 50-90%. 

@font-face {
  font-family: 'Inter';
  src: url('inter-subset.woff2');
  unicode-range: U+0020-007F;
}
Preload Key Fonts

Preload fonts used in above-the-fold content to eliminate FOIT/FOUT. 

<link rel="preload"
  href="inter-v47.woff2"
  as="font"
  type="font/woff2"
  crossorigin>
WOFF2 Format

WOFF2 offers 30% better compression than WOFF. Supported by all modern browsers. 

@font-face {
  font-family: 'Inter';
  src: url('inter.woff2')
    format('woff2');
  font-display: swap;
}
