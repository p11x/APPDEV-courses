Retry Strategies

Immediate Retry

Retry immediately up to N times.

async function retry(fn, attempts) {
  for (let i = 0; i < attempts; i++) {
    try {
      return await fn();
    } catch (err) {
      if (i === attempts - 1) throw err;
    }
  }
}
Exponential Backoff

Increase delay between retries exponentially.

async function retryBackoff(fn, max) {
  for (let i = 0; i < max; i++) {
    try {
      return await fn();
    } catch (err) {
      if (i === max - 1) throw err;
      const delay = Math.pow(2, i) * 1000;
      await sleep(delay);
    }
  }
}
Jittered Retry

Add randomness to prevent thundering herd.

async function retryJitter(fn, max) {
  for (let i = 0; i < max; i++) {
    try {
      return await fn();
    } catch (err) {
      if (i === max - 1) throw err;
      const base = Math.pow(2, i) * 1000;
      const jitter = Math.random() * 1000;
      await sleep(base + jitter);
    }
  }
}

Circuit Breaker Pattern
Closed

Requests flow normally. Failures are counted.

Normal Operation
Open

Requests are blocked. Fallback is used.

Service Down
Half-Open

Limited test requests allowed through.

Testing Recovery 
class CircuitBreaker {
  constructor(threshold = 5, timeout = 30000) {
    this.threshold = threshold;
    this.timeout = timeout;
    this.failures = 0;
    this.state = 'CLOSED';
    this.nextAttempt = 0;
  }

  async call(fn) {
    if (this.state === 'OPEN') {
      if (Date.now() < this.nextAttempt) {
        throw new Error('Circuit is OPEN - using fallback');
      }
      this.state = 'HALF-OPEN';
    }

    try {
      const result = await fn();
      this.onSuccess();
      return result;
    } catch (error) {
      this.onFailure();
      throw error;
    }
  }

  onSuccess() {
    this.failures = 0;
    this.state = 'CLOSED';
  }

  onFailure() {
    this.failures++;
    if (this.failures >= this.threshold) {
      this.state = 'OPEN';
      this.nextAttempt = Date.now() + this.timeout;
    }
  }
}
