Implementation Checklist
Error Detection
  • ☐ Wrap risky operations in try/catch
  • ☐ Handle async/await errors properly
  • ☐ Register global error handlers
  • ☐ Handle unhandled promise rejections
  • ☐ Validate form inputs before submission
Error Reporting
  • ☐ Implement structured logging
  • ☐ Include context with error reports
  • ☐ Set up error monitoring service
  • ☐ Configure alert rules
  • ☐ Create error grouping strategy
User Experience
  • ☐ Show user-friendly error messages
  • ☐ Provide retry/action buttons
  • ☐ Implement error boundary components
  • ☐ Add loading and empty states
  • ☐ Test with network throttling
Recovery
  • ☐ Implement retry with backoff
  • ☐ Add circuit breaker for external APIs
  • ☐ Cache data for offline fallback
  • ☐ Queue failed requests for retry
  • ☐ Monitor error resolution rates
Quick Reference: Error Types
Error Type Common Cause Handling Pattern
TypeError Accessing property of undefined/null Optional chaining, null checks
ReferenceError Using undeclared variable Strict mode, linting
SyntaxError Invalid JSON, malformed code try/catch around JSON.parse
RangeError Number outside allowed range Input validation, bounds checking
NetworkError Failed HTTP requests Retry with backoff, fallback UI
ChunkLoadError Failed code splitting chunk load Retry, full page reload fallback
