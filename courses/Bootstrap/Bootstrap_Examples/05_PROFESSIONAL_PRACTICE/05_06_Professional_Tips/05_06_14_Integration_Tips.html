🔗 Integration Tips

Integrating Bootstrap 5 with React, Vue, Angular, and other frameworks.

General Principles

📌

CSS vs JS Packages

Use Bootstrap CSS as-is from CDN/npm. For JS interactions in frameworks, prefer framework-specific libraries over vanilla Bootstrap JS.

🛠

Component Wrappers

Create wrapper components for Bootstrap elements. This provides consistent API, easier testing, and single point of customization.

🔎

Avoid DOM Manipulation

Frameworks manage the DOM. Don't use vanilla Bootstrap JS (e.g., new bootstrap.Modal()) alongside framework reactivity.

1. React + Bootstrap

Recommended
react-bootstrap (Library)

Pre-built React components with Bootstrap styling. No direct DOM manipulation.

// Install npm install react-bootstrap bootstrap // Import CSS once in index.js import 'bootstrap/dist/css/bootstrap.min.css'; // Use components import { Button, Modal, Card } from 'react-bootstrap'; function MyComponent() { const [show, setShow] = useState(false); return ( <> <Card> <Card.Body> <Button variant="primary" onClick={() => setShow(true)} > Open Modal </Button> </Card.Body> </Card> <Modal show={show} onHide={() => setShow(false)}> <Modal.Header closeButton> <Modal.Title>Hello</Modal.Title> </Modal.Header> <Modal.Body>Content</Modal.Body> </Modal> </> ); }
CSS Only
Bootstrap CSS Only (Manual)

Use Bootstrap classes directly in JSX. Handle JS interactions manually.

// Install CSS only npm install bootstrap // Import in App.js import 'bootstrap/dist/css/bootstrap.min.css'; function MyComponent() { const [isOpen, setIsOpen] = useState(false); return ( <div className="container"> <div className="row"> <div className="col-md-6"> <div className="card"> <div className="card-body"> <button className="btn btn-primary" onClick={() => setIsOpen(!isOpen) } > Toggle </button> {isOpen && <div className="alert alert-info mt-3" > Toggled content </div> } </div> </div> </div> </div> </div> ); }
React Tip: Use className instead of class. Use htmlFor instead of for. react-bootstrap handles these automatically.

2. Vue + Bootstrap

Recommended
Bootstrap-Vue-Next (Vue 3)

Vue 3 components for Bootstrap 5:

// Install npm install bootstrap bootstrap-vue-next // main.js import 'bootstrap/dist/css/bootstrap.min.css'; import { createBootstrap } from 'bootstrap-vue-next'; app.use(createBootstrap()); // Component.vue <template> <BCard title="Hello Vue"> <BButton variant="primary" @click="showModal = true" > Open Modal </BButton> <BModal v-model="showModal"> Modal content here </BModal> </BCard> </template> <script setup> import { ref } from 'vue'; const showModal = ref(false); </script>
CSS Only
Bootstrap CSS + Vue Directives

Use Bootstrap classes with Vue's reactivity:

<template> <div class="container"> <div class="card" :class="{ 'border-success': isActive }" > <div class="card-body"> <button class="btn" :class="btnClass" @click="toggle" > {{ buttonText }} </button> </div> </div> </div> </template> <script setup> import { ref, computed } from 'vue'; const isActive = ref(false); const btnClass = computed(() => isActive.value ? 'btn-success' : 'btn-outline-secondary' ); const buttonText = computed(() => isActive.value ? 'Active' : 'Inactive' ); const toggle = () => isActive.value = !isActive.value; </script>

3. Angular + Bootstrap

Recommended
ng-bootstrap

Angular widgets built from scratch with Bootstrap CSS. No jQuery, no Bootstrap JS dependency.

// Install npm install @ng-bootstrap/ng-bootstrap // app.module.ts import { NgbModule } from '@ng-bootstrap/ng-bootstrap'; @NgModule({ imports: [NgbModule] }) // component.html <ngb-accordion> <ngb-panel title="Panel 1"> <ng-template ngbPanelContent> Accordion content </ng-template> </ngb-panel> </ngb-accordion> <button class="btn btn-primary" (click)="open(content)" > Launch Modal </button>
CSS Only
Bootstrap CSS + Angular

Add Bootstrap CSS to angular.json or import directly:

// angular.json styles array "styles": [ "node_modules/bootstrap/dist/css/bootstrap.min.css", "src/styles.css" ] // component.html <div class="container"> <div class="row"> <div *ngFor="let item of items" class="col-md-4 mb-3" > <div class="card" [class.border-primary]="item.featured" > <div class="card-body"> {{ item.name }} </div> </div> </div> </div> </div>
