2. Component Documentation Template

Component: Custom Alert
Stable
Description

An extended Bootstrap alert with dismiss animation, auto-close timer, and icon support. Built on top of Bootstrap's .alert component.

Live Preview
This is a custom alert with an icon.
Warning! This alert can be dismissed.
Usage
<div class="alert alert-success d-flex align-items-center" role="alert"> <svg class="bi flex-shrink-0 me-2" width="24" height="24"> <use xlink:href="#check-circle-fill"/> </svg> <div>Success message here.</div> </div>
Options
OptionTypeDefaultDescription
variantString'primary'Bootstrap color: primary, success, danger, warning, info
dismissibleBooleanfalseShow close button
autoCloseNumber0Auto-dismiss after N ms (0 = disabled)
iconStringnullSVG icon ID to display
Accessibility
  • Always include role="alert" for screen reader announcement
  • Dismiss button needs aria-label="Close"
  • Use aria-live="polite" for non-critical alerts
  • Use aria-live="assertive" for critical error alerts
