2. Cascade Layers (@layer)

Cascade layers let you control specificity order without !important.

How Cascade Layers Work
/* Define layer order (later = higher priority) */
@layer reset, bootstrap, overrides;

@layer reset {
    /* Low priority: base resets */
    *, *::before, *::after {
        box-sizing: border-box;
    }
}

@layer bootstrap {
    /* Medium: Bootstrap styles (from CDN) */
}

@layer overrides {
    /* High priority: your overrides */
    .card {
        border-radius: 1rem;
    }
}
