Pattern: Form Group

Metadata
Category:Molecules
Version:2.0.0
Accessibility
  • Always pair labels with inputs via for/id
  • Mark required fields visually and with required
  • Use aria-describedby for help text
<div class="mb-3">
    <label for="email" class="form-label">Email <span class="text-danger">*</span></label>
    <input type="email" class="form-control" id="email" required>
    <div class="form-text">Helper text</div>
</div>
Pattern: Modal Dialog
Modal Title

Modal body content goes here.

Variants
  • modal-sm - Small (confirmations)
  • modal-lg - Large (complex forms)
  • modal-fullscreen - Full screen
  • data-bs-backdrop="static" - Non-dismissible
Best Practices
  • Focus is trapped inside modal when open
  • Pressing Escape closes modal by default
  • Use static backdrop for destructive actions
<button data-bs-toggle="modal" data-bs-target="#myModal">Open</button>
<div class="modal fade" id="myModal" tabindex="-1">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">...</div>
            <div class="modal-body">...</div>
            <div class="modal-footer">...</div>
        </div>
    </div>
</div>
