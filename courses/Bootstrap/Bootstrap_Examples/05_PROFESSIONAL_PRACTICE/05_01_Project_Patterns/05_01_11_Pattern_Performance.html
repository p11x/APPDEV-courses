Use native loading="lazy" for below-the-fold images.

Eager (above fold)
<img src="hero.jpg">
Lazy (below fold)
<img loading="lazy">
Lazy + aspect-ratio
<img loading="lazy" style="aspect-ratio:16/9">

4. CSS Performance Tips

Bad (slow selector)

.navbar .nav .nav-item .nav-link .badge { }

Good (fast selector)

.nav-badge { }

Only apply will-change to elements that actually animate:

/* Good: Only on elements being animated */
.modal.fade .modal-dialog {
    will-change: transform;
}

/* Bad: Applied globally */
* { will-change: transform; } /* DON'T */

transform and opacity are the only properties that don't trigger layout/paint:

/* Good: GPU-accelerated */
.element:hover { transform: translateY(-2px); }

/* Bad: Triggers layout */
.element:hover { top: -2px; margin-top: -2px; }

  • Batch DOM reads and writes
  • Use requestAnimationFrame for visual updates
  • Avoid reading layout properties (offsetHeight, getBoundingClientRect) in loops
  • Use content-visibility: auto for off-screen sections

5. JavaScript Performance

Event Delegation Pattern

Instead of attaching listeners to each button, use delegation:

Bad: Individual listeners

document.querySelectorAll('.btn')
    .forEach(btn => {
        btn.addEventListener('click', handler)
    });

Good: Event delegation

document.addEventListener('click', e => {
    if (e.target.matches('.btn')) {
        handler(e);
    }
});
