2. Cross-Browser Form Controls

Bootstrap 5 normalizes form controls across browsers.

3. Vendor Prefix Handling

Bootstrap uses Autoprefixer in its build process. For custom CSS, include fallbacks:

/* Autoprefixer handles these automatically in build */
.example {
    /* Flexbox - fully supported without prefixes in 2026 */
    display: flex;

    /* Smooth scrolling */
    scroll-behavior: smooth;

    /* Sticky positioning */
    position: sticky;
    top: 0;

    /* aspect-ratio */
    aspect-ratio: 16 / 9;

    /* object-fit */
    object-fit: cover;
}

/* Manual fallbacks when needed */
.gradient-bg {
    background: -webkit-linear-gradient(#eee, #333);
    background: linear-gradient(#eee, #333);
}

4. CSS Feature Detection (@supports)

Use @supports to provide fallbacks for newer CSS features.

/* Base styles for all browsers */
.card {
    border: 1px solid #dee2e6;
    border-radius: 0.375rem;
}

/* Enhanced styles for browsers supporting container queries */
@supports (container-type: inline-size) {
    .card-container {
        container-type: inline-size;
    }
    @container (min-width: 400px) {
        .card { flex-direction: row; }
    }
}

/* Fallback for browsers without :has() */
.form-group { margin-bottom: 1rem; }

@supports selector(:has(*)) {
    .form-group:has(.is-invalid) {
        margin-bottom: 1.5rem;
    }
}

5. Common Cross-Browser Issues & Fixes

/* Use dvh (dynamic viewport height) instead of vh */
.full-height {
    height: 100vh;       /* Fallback */
    height: 100dvh;      /* Dynamic - accounts for Safari address bar */
}

/* Or use JavaScript workaround */
.full-height {
    height: calc(100vh - env(safe-area-inset-bottom));
}

/* Firefox uses :-moz-focusring, Chrome uses :focus-visible */
/* Bootstrap 5 handles this with :focus-visible which works in both */
.btn:focus-visible {
    outline: 2px solid var(--bs-btn-focus-shadow-color);
    outline-offset: 2px;
}

/* iOS Safari auto-zooms on inputs smaller than 16px font-size */
/* Bootstrap 5 sets font-size: 1rem (16px) on form controls by default */
/* If overriding, ensure minimum 16px: */
.form-control {
    font-size: max(16px, 1rem);
}

/* smooth scroll */
html { scroll-behavior: smooth; }

/* Respect user preference for reduced motion */
@media (prefers-reduced-motion: reduce) {
    html { scroll-behavior: auto; }
}
