1. Branching Strategies

Git Flow
main ─────●────────────────●──────────── │ ↑ release ─────┼──────●────────┼────── │ ↑ │ develop ──●──┼──●───┼───●────┼──●── │ │ ↑ │ ↑ │ │ feature ───┼──●──┘ │ │ │ │ │ │ │ │ hotfix ────┼───────┼───┼────●──┘
BranchPurpose
mainProduction-ready code
developIntegration branch
feature/*New features
release/*Release preparation
hotfix/*Emergency production fixes
GitHub Flow (Simpler)
main ──●────●────●────●────●── ↑ ↑ ↑ ↑ feature ─●── │ │ │ │ │ │ feature ──────●────┘ │ │ feature ─────────────────────●
# GitHub Flow commands # 1. Create feature branch git checkout -b feature/card-component # 2. Make changes and commit git add . git commit -m "feat: add card component with hover effects" # 3. Push and create PR git push -u origin feature/card-component # Create Pull Request on GitHub # 4. After review, merge to main git checkout main git pull origin main git branch -d feature/card-component
Branch Naming Convention
# Format: type/short-description feature/card-hover-effects feature/responsive-navbar fix/modal-z-index fix/form-validation-error chore/update-bootstrap docs/api-documentation refactor/scss-architecture test/component-unit-tests

2. Commit Conventions

Conventional Commits Format
# Format: type(scope): description # Examples: feat: add dark mode toggle component feat(navbar): add mobile hamburger animation fix(modal): resolve z-index conflict with dropdown fix(forms): correct validation error display docs: update SCSS customization guide style: format code with prettier refactor(grid): simplify column generation mixin test(cards): add unit tests for card component chore: update bootstrap to 5.3.3 perf(images): add lazy loading to gallery
Commit Types
TypeDescription
featNew feature
fixBug fix
docsDocumentation
styleFormatting (no code change)
refactorCode restructure
testAdding/updating tests
choreBuild/tool changes
perfPerformance improvement
ciCI/CD changes
Good vs Bad Commits
Bad Commit Messages
  • fix stuff
  • update
  • wip
  • changes
  • fix bug
  • asdfgh
Good Commit Messages
  • fix(navbar): resolve collapse animation on iOS
  • feat(cards): add image overlay variant
  • docs: add SCSS customization examples
  • refactor(grid): extract breakpoint mixins
  • test(forms): add validation unit tests
  • chore: update bootstrap to 5.3.3
useful Git Config
# Set default branch name git config --global init.defaultBranch main # Set default editor git config --global core.editor "code --wait" # Useful aliases git config --global alias.co checkout git config --global alias.br branch git config --global alias.st status git config --global alias.lg "log --oneline --graph --all"
