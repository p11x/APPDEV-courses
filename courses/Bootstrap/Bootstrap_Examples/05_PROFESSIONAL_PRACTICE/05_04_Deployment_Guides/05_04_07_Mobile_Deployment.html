Responsive Images

Using srcset for Mobile Optimization
<img src="image-800.jpg" srcset="image-400.jpg 400w, image-800.jpg 800w, image-1200.jpg 1200w" sizes="(max-width: 576px) 100vw, (max-width: 992px) 50vw, 33vw" alt="Description" loading="lazy" class="img-fluid">
Bandwidth savings: A 400px image is typically 50-80% smaller than a 1200px image. On slow mobile networks, this means 2-5x faster loading.
