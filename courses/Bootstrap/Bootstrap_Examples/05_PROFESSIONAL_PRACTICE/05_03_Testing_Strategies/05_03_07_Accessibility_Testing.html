WCAG 2.1 Principles

1

Perceivable

Content must be presentable in ways users can perceive. Test: images have alt text, videos have captions.

POUR
2

Operable

UI components must be operable. Test: all interactive elements reachable via keyboard.

POUR
3

Understandable

Information must be understandable. Test: error messages are clear, labels are descriptive.

POUR
4

Robust

Content must be robust. Test: valid HTML, proper ARIA usage, works with assistive tech.
