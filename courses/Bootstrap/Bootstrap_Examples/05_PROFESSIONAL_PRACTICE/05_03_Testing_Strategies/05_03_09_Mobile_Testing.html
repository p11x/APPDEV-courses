Mobile-Specific Test Scenarios

Meta Viewport Checks
  • width=device-width present in meta tag
  • initial-scale=1.0 prevents auto-zoom
  • No maximum-scale=1.0 (accessibility issue)
  • No user-scalable=no (blocks zoom)
Content Scaling
  • Text readable without zooming at 320px
  • Images scale within container
  • No horizontal scrolling at any width
  • Touch targets remain accessible when zoomed

Network Conditions
  • Test on 3G throttled connection
  • Verify lazy loading works
  • Check offline behavior (PWA)
  • Measure Time to Interactive (TTI)
Resource Optimization
  • Images use srcset/sizes attributes
  • CSS/JS minified and compressed
  • Critical CSS inlined
  • Non-critical resources deferred

Screen Reader
  • VoiceOver (iOS) navigation test
  • TalkBack (Android) test
  • ARIA labels on interactive elements
  • Logical reading order verified
Motor Accessibility
  • All targets minimum 44x44px
  • Sufficient spacing between targets
  • No hover-only interactions
  • Keyboard/switch navigation works
