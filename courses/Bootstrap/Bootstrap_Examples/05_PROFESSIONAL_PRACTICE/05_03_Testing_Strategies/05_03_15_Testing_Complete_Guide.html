Testing Pyramid

The testing pyramid guides the balance between different test types. More unit tests at the base, fewer E2E tests at the top.

E2E Tests
~10% | Slow | High confidence
Integration Tests
~20% | Medium speed | Medium confidence
Unit Tests
~70% | Fast | Focused confidence
Unit Tests

Test individual functions and components in isolation. Fast execution, easy to debug.

  • Component rendering
  • Utility function logic
  • State management
  • Event handlers
Jest / Vitest
Integration Tests

Test component interactions, API contracts, and data flows between modules.

  • Component composition
  • API endpoint testing
  • Form submissions
  • Navigation flows
React Testing Library
E2E Tests

Test complete user journeys in real browsers. Highest confidence but slowest.

  • User registration flow
  • Checkout process
  • Cross-browser rendering
  • Responsive layouts
Playwright / Cypress

Test Templates

Bootstrap Component Test Template
// tests/components/Modal.test.js
import { render, screen, fireEvent }
  from '@testing-library/react';
import Modal from './Modal';

describe('Modal Component', () => {
  // Rendering tests
  it('renders when open', () => {
    render(<Modal isOpen title="Test" />);
    expect(
      screen.getByText('Test')
    ).toBeInTheDocument();
  });

  // Interaction tests
  it('closes on button click', () => {
    const onClose = jest.fn();
    render(
      <Modal isOpen onClose={onClose} />
    );
    fireEvent.click(
      screen.getByLabelText('Close')
    );
    expect(onClose).toHaveBeenCalled();
  });

  // Accessibility tests
  it('has correct ARIA attributes', () => {
    render(<Modal isOpen title="Test" />);
    expect(
      screen.getByRole('dialog')
    ).toHaveAttribute(
      'aria-modal', 'true'
    );
  });
});
E2E Test Template (Playwright)
// tests/e2e/navigation.spec.ts
import { test, expect }
  from '@playwright/test';

test.describe('Navigation', () => {
  test('responsive navbar toggle',
    async ({ page }) => {
    // Set mobile viewport
    await page.setViewportSize({
      width: 375, height: 667
    });

    // Navigate to page
    await page.goto('/');

    // Verify navbar is collapsed
    const toggler = page.locator(
      '.navbar-toggler'
    );
    await expect(toggler).toBeVisible();

    // Click to expand
    await toggler.click();

    // Verify menu is visible
    await expect(
      page.locator('.navbar-nav')
    ).toBeVisible();
  });
});
