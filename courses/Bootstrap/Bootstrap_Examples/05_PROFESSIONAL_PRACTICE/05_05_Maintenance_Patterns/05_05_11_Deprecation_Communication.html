Subject: [Important] Bootstrap 5 Deprecation Notice - Action Required

Dear Developer,

We are writing to inform you about upcoming deprecations in Bootstrap that may affect your projects.

Affected Features:
  • .badge-pill - Removed in v5.0
  • .media - Deprecated in v4.3, removal in v5.5
  • .form-group - Deprecated in v5.0, removal in v6.0
Recommended Actions:
  1. Review the migration guide
  2. Run the migration tool
  3. Test your application

Best regards,
Bootstrap Team

## [5.0.0] - 2021-05-05

### Breaking Changes
- Removed `.badge-pill` class. Use `.rounded-pill` instead.
- Removed `.badge-*` color classes. Use `.bg-*` instead.
- Removed jQuery dependency entirely.

### Deprecated
- `.media` class is deprecated. Use flexbox utilities.
- `.form-group` class is deprecated. Use `.mb-3`.
- `.form-row` class is deprecated. Use `.row .g-*`.

### Migration
Run `npx bootstrap-migrate --from 4 --to 5` to auto-fix most issues.
