API Reference Template
## .badge-pill {#badge-pill}

:::deprecated
**Deprecated since:** v4.0
**Will be removed in:** v5.0
:::

### Description
The `.badge-pill` class was used to create pill-shaped badges with more rounded corners.

### Why Deprecated
Bootstrap 5 introduced unified rounded utilities. The `.badge-pill` class is redundant
with the `.rounded-pill` utility which works across all components.

### Alternative
Use `.rounded-pill` instead:

```html

<span class="badge badge-pill badge-primary">Tag</span>


<span class="badge rounded-pill bg-primary">Tag</span>
```

### Migration
Run the following command to automatically replace:
```bash
npx bootstrap-codemod badge-pill
```
