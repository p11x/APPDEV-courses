Warning Details

Issue

The .badge-pill class was removed in Bootstrap 5.0.

Affected Code
<span class="badge badge-pill badge-primary">Tag</span>
Solution
<span class="badge rounded-pill bg-primary">Tag</span>
Migration Command
find . -name "*.html" -exec sed -i 's/badge-pill/rounded-pill/g' {} \;

Issue

The .media component is deprecated since v4.3.

Affected Code
<div class="media"><img class="mr-3"><div class="media-body">...</div></div>
Solution
<div class="d-flex"><img class="me-3"><div>...</div></div>

Issue

The .form-group class is deprecated since v5.0.

Affected Code
<div class="form-group"><label>Name</label><input></div>
Solution
<div class="mb-3"><label class="form-label">Name</label><input></div>
