Version Policy

Semantic Versioning

Bootstrap follows SemVer (Semantic Versioning):

X

Major

Breaking changes that require code modifications

Y

Minor

New features, backward compatible

Z

Patch

Bug fixes, backward compatible

Update Schedule

Critical Security Patches Immediate
Patch Updates Within 1 week
Minor Updates Within 1 month
Major Updates Planned migration

Support Policy

Bootstrap provides support for the latest minor release of each major version.
