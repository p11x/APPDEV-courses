Migration Tools

Essential tools and utilities for automating Bootstrap version migrations.

Available Tools

Bootstrap Migration CLI

Command-line tool that scans your project and applies migration rules automatically.

  • ✓ Class renaming
  • ✓ Attribute updates
  • ✓ CSS variable migration
  • ○ JS event renaming
npm install bootstrap-migrate
Visual Diff Tool

Compare before and after rendering to catch visual regressions during migration.

  • ✓ Screenshot comparison
  • ✓ Pixel-level diff
  • ✓ Multi-browser support
  • ✓ CI/CD integration
npm install backstopjs
Lint Rules

ESLint and Stylelint rules to detect deprecated Bootstrap usage in your code.

  • ✓ Real-time detection
  • ✓ IDE integration
  • ✓ Auto-fix suggestions
  • ✓ Custom rules
npm install eslint-plugin-bootstrap
Codemod Scripts

AST-based code transformation scripts for precise class and structure changes.

  • ✓ AST-based transforms
  • ✓ Preserves formatting
  • ✓ Batch processing
  • ○ Complex patterns
npx bootstrap-codemod v5
Dependency Checker

Analyze your project dependencies for compatibility with target Bootstrap version.

  • ✓ Version compatibility
  • ✓ Conflict detection
  • ✓ Update suggestions
  • ✓ Security audit
npm audit && npm outdated
Documentation Generator

Generate migration documentation and changelogs based on your changes.

  • ✓ Auto-generate docs
  • ✓ Change reports
  • ○ PDF export
  • ○ Wiki integration
bootstrap-docs --generate
