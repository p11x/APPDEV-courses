Deprecation Timeline

Critical .badge-pill removed
Removed in v5.0

The .badge-pill class has been removed. Use .rounded-pill instead for pill-shaped badges.

Migration: Replace .badge-pill with .rounded-pill
Warning .media deprecation
Removal in v5.3

The .media component is deprecated. Use flexbox utilities like .d-flex instead.

Migration: Convert to flexbox-based layout
Info .form-group planned removal
Removal in v6.0

The .form-group class will be removed in v6.0. Use spacing utilities instead.

Migration: Replace .form-group with .mb-3
Info .form-row planned removal
Removal in v6.0

The .form-row class will be removed. Use .row with gutter classes.

Migration: Replace .form-row with .row .g-3
Migrated .input-group-append removed
Successfully migrated

The .input-group-append wrapper is no longer needed. Place buttons directly in .input-group.
