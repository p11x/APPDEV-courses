Bootstrap Spacing - Complete Guide

The definitive reference for Bootstrap 5 spacing utilities: margins, padding, gap, responsive variants, and real-world patterns.

1. The Spacing Scale

Bootstrap uses a 6-point spacing scale (0-5) plus auto. All values use rem units.

KeyValuePixels*Common Use
00rem0pxRemove default spacing
10.25rem4pxMicro adjustments
20.5rem8pxSmall gaps
31rem16pxDefault (most common)
41.5rem24pxSection content
53rem48pxMajor sections
autoautoautoCentering (margin only)

* At default 16px root font size

CSS Custom Properties
:root {
  --bs-spacer-0: 0;
  --bs-spacer-1: 0.25rem;
  --bs-spacer-2: 0.5rem;
  --bs-spacer-3: 1rem;
  --bs-spacer-4: 1.5rem;
  --bs-spacer-5: 3rem;
}

2. Margin Utilities

Format: m{side}-{size}

Sides
m-*All sides
mt-*Top
mb-*Bottom
ms-*Start (left in LTR)
me-*End (right in LTR)
mx-*Left + Right
my-*Top + Bottom
Special Values
m-automargin: auto (centering)
mx-autoHorizontal center
m-n1 to m-n5Negative margins

5. Responsive Spacing

Format: {property}{side}-{breakpoint}-{size}

Base class applies to all screens. Breakpoint classes override at and above that breakpoint.

BreakpointAbbreviationMin Width Example
Extra Small(base)0pxp-2
Smallsm576pxp-sm-3
Mediummd768pxp-md-3
Largelg992pxp-lg-4
Extra Largexl1200pxp-xl-5
XXLxxl1400pxp-xxl-5

6. Common Spacing Patterns

Card Component
<div class="card p-4 mb-4">
  <h5 class="mb-3">Title</h5>
  <p class="mb-3">Content</p>
  <button class="mb-0">Action</button>
</div>
Card Title

Card content with standard spacing.

Form Fields
<div class="mb-3">
  <label class="form-label">Label</label>
  <input class="form-control">
</div>
Section Layout
<section class="py-5 my-5">
  <div class="container">
    <h2 class="mb-4">Title</h2>
    <div class="row g-4">
      <div class="col-md-4">...</div>
    </div>
  </div>
</section>
Navbar Spacing
<nav class="navbar px-3 py-2">
  <a class="navbar-brand me-auto">
    Brand
  </a>
  <ul class="navbar-nav ms-auto gap-2">
    <li class="nav-item">Link</li>
  </ul>
</nav>
