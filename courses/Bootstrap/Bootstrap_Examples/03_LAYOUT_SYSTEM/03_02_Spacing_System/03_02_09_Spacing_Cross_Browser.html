3. Margin Collapse Quirks

Vertical margins collapse between adjacent block elements. This is standard CSS behavior, consistent across browsers.

Bottom: mb-3 (16px)
Top: mt-3 (16px)

Gap is 16px (not 32px) because adjacent vertical margins collapse. This is correct CSS behavior.

4. RTL (Right-to-Left) Spacing

Bootstrap uses logical properties (start/end) that work correctly in both LTR and RTL layouts.

LTR Layout (English)
ms-3 (left)
me-3 (right)
RTL Layout (Arabic/Hebrew)
ms-3 (right in RTL)
me-3 (left in RTL)
Always use ms-*/me-* (margin-start/margin-end) instead of ml-*/mr-* (deprecated). Logical properties automatically adapt to text direction.

5. Safari-Specific Spacing Notes

Safari Flex Gap (Older Versions)

Safari < 14.1 did not support gap in flexbox layouts. Bootstrap's gap utilities work in Safari 14.1+.

/* For Safari < 14.1 fallback */
.flex-gap-fallback {
  display: flex;
  gap: 1rem; /* Works in modern browsers */
}

/* Fallback for older Safari */
@supports not (gap: 1rem) {
  .flex-gap-fallback > * + * {
    margin-left: 1rem;
  }
}
Safari 100vh Issue

On iOS Safari, 100vh includes the address bar, causing overflow. This can affect spacing in full-height layouts.

/* Fix for iOS Safari 100vh */
.full-height-section {
  min-height: 100vh;
  min-height: 100dvh; /* dvh accounts for browser chrome */
  padding: 3rem 0;
}

6. Legacy Browser Notes (IE 11)

Note: Bootstrap 5 does NOT support Internet Explorer 11. If you need IE support, use Bootstrap 4 or add CSS polyfills.
IE 11 Spacing Differences
  • ✗ No CSS custom property support (CSS variables)
  • ✗ No gap in flexbox
  • ✗ Different default box-sizing behavior
  • ✗ Logical properties not supported

7. Print Media Spacing

Spacing may render differently in print. Consider print-specific adjustments.

/* Print-specific spacing adjustments */
@media print {
  /* Reduce margins in print to save paper */
  .container {
    padding: 0;
    margin: 0;
  }

  /* Remove decorative spacing */
  .py-5 { padding-top: 1rem !important; padding-bottom: 1rem !important; }
  .my-5 { margin-top: 1rem !important; margin-bottom: 1rem !important; }

  /* Keep content spacing but reduce section spacing */
  .mb-5 { margin-bottom: 1.5rem !important; }
}
