Spacing Scale System

Bootstrap's spacing system provides a consistent set of utility classes based on a default spacing scale that can be customized via CSS variables.

1. The Spacing Scale

Each number represents a multiplier applied to the base spacing unit.

2. CSS Custom Properties

Bootstrap 5.3 exposes spacing as CSS custom properties for easy customization.

Default Spacing Variables
/* Bootstrap's default spacing CSS custom properties */
--bs-spacer: 1rem;                    /* Base unit (16px) */
--bs-spacer-0: 0;                     /* Scale 0 */
--bs-spacer-1: 0.25rem;               /* Scale 1 = 4px */
--bs-spacer-2: 0.5rem;                /* Scale 2 = 8px */
--bs-spacer-3: 1rem;                  /* Scale 3 = 16px */
--bs-spacer-4: 1.5rem;                /* Scale 4 = 24px */
--bs-spacer-5: 3rem;                  /* Scale 5 = 48px */
Customization: Override these variables on :root to change all spacing utilities globally without modifying Bootstrap's source.

Rem-Based

4. Choosing the Right Scale

Scale Value Best For Example
0 0rem Removing default spacing, tight layouts Collapse card gutters
1 0.25rem (4px) Fine adjustments, icon spacing Space between icon and text
2 0.5rem (8px) Small gaps, badge padding Badge inner padding
3 1rem (16px) Default/General purpose Form field spacing, card padding
4 1.5rem (24px) Section content, larger cards Card body padding, section gaps
5 3rem (48px) Major sections, page margins Page top/bottom margins, hero sections

7. Extending the Scale

You can extend Bootstrap's spacing scale using custom CSS.

Custom Spacing via Sass (for Bootstrap source users)
// Override default spacing map in _variables.scss
$spacer: 1rem;
$spacers: (
  0: 0,
  1: ($spacer * .25),
  2: ($spacer * .5),
  3: $spacer,
  4: ($spacer * 1.5),
  5: ($spacer * 3),
  6: ($spacer * 4),    // Custom: 4rem (64px)
  7: ($spacer * 6),    // Custom: 6rem (96px)
);
Custom Spacing via CSS Variables
/* Override Bootstrap's CSS custom properties */
:root {
  --bs-spacer-6: 4rem;   /* 64px */
  --bs-spacer-7: 6rem;   /* 96px */
}

/* Use with custom utility classes */
.m-6 { margin: var(--bs-spacer-6) !important; }
.p-7 { padding: var(--bs-spacer-7) !important; }

8. Consistency Patterns

Content Cards

Use p-3 for card body content.

Use mb-3 between form fields.

Consistent scale: 3
Section Containers

Use p-4 for section padding.

Use my-4 between sections.

Consistent scale: 4
Page Sections

Use p-5 for hero padding.

Use my-5 for page sections.

Consistent scale: 5
