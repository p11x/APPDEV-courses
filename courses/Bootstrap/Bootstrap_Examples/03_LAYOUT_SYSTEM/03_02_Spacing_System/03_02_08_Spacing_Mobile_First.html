Mobile-First Spacing

Bootstrap is mobile-first. Base spacing classes apply to all screens, then responsive variants override at larger breakpoints.

Watch the blue badge in the top-right corner as you resize. It shows your current breakpoint.

1. Mobile-First Concept

/* Mobile-first means: */
.class { property: small-value; }        /* Base = all screens */

@media (min-width: 576px) {
  .class-sm { property: medium-value; }  /* SM and up */
}

@media (min-width: 768px) {
  .class-md { property: larger-value; }  /* MD and up */
}

/* In Bootstrap HTML: */
<div class="p-2 p-md-4 p-lg-5">
  <!-- p-2: all screens -->
  <!-- p-md-4: 768px and up -->
  <!-- p-lg-5: 992px and up -->
</div>

8. Bootstrap Breakpoint Reference

Breakpoint Class Prefix Min Width Typical Device
Extra Small (none) 0px Phones (portrait)
Small sm- ≥ 576px Phones (landscape)
Medium md- ≥ 768px Tablets
Large lg- ≥ 992px Desktops
Extra Large xl- ≥ 1200px Large desktops
XXL xxl- ≥ 1400px Extra large screens
