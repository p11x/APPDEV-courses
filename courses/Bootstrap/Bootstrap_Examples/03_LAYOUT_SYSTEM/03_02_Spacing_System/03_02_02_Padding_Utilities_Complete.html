Padding Utilities - Complete Reference

Bootstrap padding utilities control the interior spacing of elements. Format: p{side}-{size} where size ranges from 0 to 5.

Spacing Scale: 0 = 0rem, 1 = 0.25rem, 2 = 0.5rem, 3 = 1rem, 4 = 1.5rem, 5 = 3rem

1. All-Side Padding (p-*)

Classes: p-0 through p-5

p-0 (0rem)
p-1 (0.25rem)
p-2 (0.5rem)
p-3 (1rem)
p-4 (1.5rem)
p-5 (3rem)

2. Side-Specific Padding

Top Padding (pt-*)

pt-4 (1.5rem top padding)
Content area expands down

Bottom Padding (pb-*)

pb-4 (1.5rem bottom padding)
Content area expands down

Start Padding (ps-*)

ps-5 (3rem left padding)

End Padding (pe-*)

pe-5 (3rem right padding)

3. Axis Padding (px-*, py-*)

Horizontal Padding (px-*)

px-4 (1.5rem left + right)

Vertical Padding (py-*)

py-4 (1.5rem top + bottom)

4. Responsive Padding

Format: p-{breakpoint}-{size} - applies at and above that breakpoint

p-2 / p-md-4 / p-lg-5 (responsive padding)
Resize browser to see changes
py-2 / py-md-4 / px-lg-5 (mixed responsive)
Tip: Responsive padding is essential for adapting content spacing to different screen sizes.

8. Padding vs Margin

With Padding (p-4)

Padding is INSIDE the element.
Background extends into padding.

With Margin (m-4)

Margin is OUTSIDE the element.
No background in margin area.
Key Difference: Padding is inside the border (part of the element). Margin is outside the border (space between elements). Padding has background; margin is transparent.
