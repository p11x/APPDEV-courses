Margin Utilities - Complete Reference

Bootstrap provides shorthand margin utility classes for controlling spacing. The format is m{side}-{size} where size ranges from 0 to 5 or auto.

Spacing Scale: 0 = 0rem, 1 = 0.25rem, 2 = 0.5rem, 3 = 1rem, 4 = 1.5rem, 5 = 3rem

1. All-Side Margins (m-*)

Classes: m-0 through m-5

m-1 (0.25rem)
m-2 (0.5rem)
m-3 (1rem)
m-4 (1.5rem)
m-5 (3rem)
m-auto

2. Horizontal Centering (mx-auto)

Use mx-auto with d-block or a set width to center elements

mx-auto + width: 300px (centered)
mx-auto + 50% width + max-width

3. Side-Specific Margins

Top Margin (mt-*)

mt-4 (top margin only)

Bottom Margin (mb-*)

mb-4 (bottom margin only)

Start Margin (ms-*)

ms-4 (start/left margin)

End Margin (me-*)

me-4 (end/right margin)

4. Axis Margins

Horizontal Axis (mx-*)

mx-3 (left + right margin)

Vertical Axis (my-*)

my-3 (top + bottom margin)

5. Negative Margins

Use negative values with m-n{size} (e.g., mt-n2)

Base element
mt-n3 (negative top margin)

6. Responsive Margins

Format: m-{breakpoint}-{size} - applies at and above that breakpoint. Breakpoints: sm (576px), md (768px), lg (992px), xl (1200px), xxl (1400px)

mt-2 / mt-md-4 / mt-lg-5 (responsive margins)
Progressive: ms-1 / ms-sm-2 / ms-md-3 / ms-lg-4 / ms-xl-5
Resize your browser to see the margin values change at different breakpoints!
