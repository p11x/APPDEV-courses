Powerful Features

Everything you need to build responsive websites

1
Responsive Design

Built-in responsive utilities make your site look great on any device. Use breakpoint classes to control layout at each screen size.

Grid System

12-column flexbox grid with five responsive tiers. Nest rows within columns for complex layouts.

Utility Classes

Hundreds of utility classes for spacing, display, flex, colors, and more - all with responsive variants.

Mobile First

Bootstrap is mobile-first. Styles apply to smallest screens first, then scale up with media queries.
