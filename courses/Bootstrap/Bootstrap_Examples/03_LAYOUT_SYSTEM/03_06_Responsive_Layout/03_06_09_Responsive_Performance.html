Responsive Performance Optimization

Techniques to keep responsive layouts fast and efficient.

1. Responsive Images with srcset

Serve appropriately sized images for each screen size.

srcset provides different image resolutions (400w, 800w, 1200w).
sizes tells the browser: 100% width on xs, 50% on md, 33% on lg+. 

<img
  src="image-800.jpg"
  srcset="
    image-400.jpg 400w,
    image-800.jpg 800w,
    image-1200.jpg 1200w"
  sizes="
    (max-width: 576px) 100vw,
    (max-width: 992px) 50vw,
    33vw"
  class="img-fluid"
>

8. Font Loading Strategy

Best Practices for Responsive Font Loading
  • font-display: swap - Shows fallback font immediately, swaps to web font when loaded
  • Preload critical fonts - <link rel="preload" href="font.woff2" as="font" crossorigin>
  • Subset fonts - Only load the characters you need
  • Use system fonts for body text - No loading delay
/* CSS for optimized font loading */
@font-face {
    font-family: 'CustomFont';
    src: url('font.woff2') format('woff2');
    font-display: swap; /* Shows fallback immediately */
    unicode-range: U+0000-00FF; /* Latin characters only */
}

/* Use system fonts for body text */
body {
    font-family: -apple-system, BlinkMacSystemFont,
    "Segoe UI", Roboto, sans-serif;
}

9. Critical CSS Loading

Critical CSS Strategy
  1. Inline critical CSS in the <head> for above-the-fold content
  2. Defer full CSS with media="print" onload="this.media='all'"
  3. Split CSS into critical and non-critical bundles
<!-- Critical CSS inlined in head -->
<style>
  /* Only above-the-fold styles */
  .hero { min-height: 100vh; }
  .navbar { position: sticky; }
</style>

<!-- Full CSS deferred -->
<link href="bootstrap.min.css"
  rel="stylesheet"
  media="print"
  onload="this.media='all'">

<!-- Noscript fallback -->
<noscript>
  <link href="bootstrap.min.css" rel="stylesheet">
</noscript>

10. Responsive Performance Checklist

Category Optimization Impact
Images Use srcset and sizes High - Serves correct image size
Images loading="lazy" for below-fold images High - Defers unnecessary loads
Images Use <picture> for art direction Medium - Better visual quality
Images Use WebP/AVIF formats High - 30-50% smaller files
CSS Critical CSS inlined in head High - Faster first paint
CSS Defer non-critical CSS High - Faster page load
Layout Use CSS grid auto-fit (no JS) Medium - Less JS overhead
Layout Minimize DOM nesting Medium - Faster rendering
JS Defer Bootstrap JS High - Non-blocking load
UX Skeleton loaders during load Medium - Better perceived speed
UX Conditional content per breakpoint Medium - Less data on mobile
Fonts font-display: swap Medium - No invisible text
