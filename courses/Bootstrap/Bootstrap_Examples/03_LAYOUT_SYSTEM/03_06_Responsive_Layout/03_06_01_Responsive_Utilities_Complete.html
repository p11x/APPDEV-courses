Responsive Utilities - Complete Reference

Resize the browser window to see responsive changes at each breakpoint.

1. Display Utilities (d-*)

d-{breakpoint}-{display} controls visibility and display type per breakpoint

d-none d-sm-block: Hidden on xs, visible on sm+
d-none d-md-block: Hidden on xs/sm, visible on md+
d-none d-lg-block: Hidden on xs/sm/md, visible on lg+
d-block d-sm-none d-xl-block: Visible xs, hidden sm-md-lg, visible xl+
d-print-none: This box is visible on screen but hidden when printing

2. Text Alignment Utilities

text-{breakpoint}-{start|center|end} changes text alignment per breakpoint

text-start text-md-center text-lg-end: Left on xs/sm, Center on md, Right on lg+

text-truncate: This text will be truncated with an ellipsis if it overflows its container width

3. Float Utilities

float-{breakpoint}-{start|end|none} for responsive floating

float-start float-md-end float-lg-none

Parent element uses .clearfix to contain floated children. The box floats left on xs/sm, right on md, and has no float on lg+.

4. Overflow Utilities

overflow-{auto|hidden|visible|scroll} with responsive variants

overflow-auto
Content that may overflow will trigger scrollbars only when needed. This is a long text to demonstrate the overflow behavior.
overflow-scroll
Scrollbars are always visible regardless of content length.

7. Responsive Spacing (Margin & Padding)

m/p-{breakpoint}-{size}: 0-5 or auto, with directional variants

mt-3 mt-md-5 mx-auto
Top margin increases from 1rem to 3rem at md breakpoint. Horizontally centered.
p-2 p-lg-4: Padding increases from 0.5rem to 1.5rem at lg breakpoint
ms-3 me-md-4: Start margin 1rem, end margin 1.5rem on md+
mt-n3 / negative margin: Overlaps the parent padding
