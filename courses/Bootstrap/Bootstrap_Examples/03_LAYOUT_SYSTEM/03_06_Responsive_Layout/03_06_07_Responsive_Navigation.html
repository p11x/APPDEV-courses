10. Navigation Pattern Summary

Pattern Class Behavior
Navbar collapses at lg navbar-expand-lg Full nav on ≥992px, hamburger on smaller
Navbar collapses at md navbar-expand-md Full nav on ≥768px, hamburger on smaller
Always collapsed No navbar-expand-* Always shows hamburger (good for mobile-first)
Offcanvas nav offcanvas offcanvas-start Slides in from left side
Vertical sidebar d-none d-lg-block Hidden on small, shown as sidebar on lg+
Responsive pills flex-column flex-sm-row Stacked on xs, horizontal on sm+
Scrollable tabs flex-nowrap overflow-auto Horizontal scroll on mobile
