6. Fluid Typography with CSS clamp()

Custom CSS for text that smoothly scales between min and max sizes.

Fluid Heading (1.5rem to 3rem)

Fluid Subheading (1rem to 1.5rem) - Resize the browser to see the text scale smoothly.

This body text uses fluid typography that scales from 0.9rem on small screens to 1.15rem on large screens. The transition is smooth and proportional to the viewport width, providing optimal readability at every screen size. 

/* CSS for fluid typography */
.fs-fluid-1 { font-size: clamp(1.5rem, 1rem + 2.5vw, 3rem); }
.fs-fluid-2 { font-size: clamp(1rem, 0.8rem + 1vw, 1.5rem); }
.fs-fluid-body { font-size: clamp(0.9rem, 0.85rem + 0.25vw, 1.15rem); }

15. Responsive Typography Strategy Summary

Technique When to Use Example
Bootstrap headings (h1-h6) Standard semantic headings <h1>Title</h1>
Display headings Hero sections, large titles .display-1 through .display-6
Font size utilities Quick sizing without heading semantics .fs-1 through .fs-7
Lead text Introductory paragraphs .lead
CSS clamp() Smooth fluid scaling between sizes font-size: clamp(1rem, 2vw, 2rem)
max-width (ch units) Optimal reading line length max-width: 65ch
text-break Prevent long word overflow .text-break
Responsive alignment Text that changes alignment per breakpoint .text-start .text-md-center
