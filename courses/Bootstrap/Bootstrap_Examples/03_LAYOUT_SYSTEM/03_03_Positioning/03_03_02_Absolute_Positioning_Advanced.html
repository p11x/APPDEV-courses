Absolute Positioning Advanced

Absolute positioning removes elements from the document flow and positions them relative to their nearest positioned ancestor.

1. How Absolute Positioning Works

position: absolute removes the element from normal flow. It's positioned relative to the nearest ancestor with a non-static position.

2. Containing Block Chain

An absolute element looks up the DOM tree for the nearest positioned ancestor. If none is found, it uses the initial containing block (viewport).

3. Z-Index & Stacking Context

z-index controls the stacking order of positioned elements. It only works on positioned elements (relative, absolute, fixed, sticky).

z-index: 1 (bottom)
z-index: 2 (middle)
z-index: 3 (top)
Bootstrap Z-Index Classes: z-0, z-1, z-2, z-3, z-n1 (negative), or use inline styles for specific values.
