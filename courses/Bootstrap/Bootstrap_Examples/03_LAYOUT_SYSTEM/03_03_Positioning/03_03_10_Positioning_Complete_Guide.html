CSS Positioning - Complete Guide

The definitive reference for CSS positioning with Bootstrap 5: static, relative, absolute, fixed, sticky, and all Bootstrap position utility classes.

1. Position Values Overview

Value In Flow Offset Reference Bootstrap Class Primary Use
static Yes N/A (offsets ignored) .position-static Default layout
relative Yes Its own normal position .position-relative Nudging, positioning context
absolute No Nearest positioned ancestor .position-absolute Overlays, badges, tooltips
fixed No Viewport .fixed-top, .fixed-bottom Navbars, footers
sticky Yes* Scroll container .sticky-top, .sticky-bottom Sticky headers, sidebars

* Sticky is in flow until the scroll threshold is reached, then behaves like fixed.

2. Static Positioning

Elements follow the normal document flow. Offset properties (top, right, bottom, left) have no effect.

<div class="position-static">
  <!-- Default behavior -->
  <!-- top/left/right/bottom: ignored -->
</div>

3. Relative Positioning

Offsets the element from its normal position. Original space is preserved. Creates a positioning context for absolute children.

<div class="position-relative">
  <!-- Offset from normal position -->
  <!-- Original space maintained in flow -->
  <!-- Creates context for absolute children -->
</div>

<!-- With Bootstrap offsets -->
<div class="position-relative top-50 start-50">
  <!-- 50% offset from normal position -->
</div>

4. Absolute Positioning

Removed from document flow. Positioned relative to nearest positioned ancestor.

<div class="position-relative">
  <!-- Parent must be positioned -->
  <div class="position-absolute top-0 end-0">
    Absolute child (top-right corner)
  </div>
</div>

<!-- Centering with absolute -->
<div class="position-absolute top-50 start-50
            translate-middle">
  Centered
</div>

5. Sticky Positioning

Hybrid of relative and fixed. Scrolls normally until hitting a threshold, then sticks.

<!-- Always sticky -->
<div class="sticky-top">Sticks at top</div>

<!-- Responsive sticky -->
<div class="sticky-md-top">
  Only sticky at md+
</div>

<!-- Custom offset -->
<div class="position-sticky" style="top: 20px;">
  Sticks 20px from top
</div>
Responsive Sticky Classes
  • .sticky-top - All sizes
  • .sticky-sm-top - 576px+
  • .sticky-md-top - 768px+
  • .sticky-lg-top - 992px+
  • .sticky-xl-top - 1200px+
  • .sticky-xxl-top - 1400px+

6. Fixed Positioning

Anchored to the viewport. Does not scroll. Removed from document flow.

<!-- Fixed navbar -->
<nav class="navbar fixed-top">...</nav>

<!-- Fixed footer -->
<div class="fixed-bottom">...</div>

<!-- Custom fixed position -->
<div class="position-fixed bottom-0 end-0 m-4">
  Fixed bottom-right
</div>

<!-- Remember body padding! -->
<body style="padding-top: 56px;">
Remember:
  • Add body padding for fixed navbars/footers
  • iOS Safari may have quirks with fixed
  • Use sticky where possible instead
