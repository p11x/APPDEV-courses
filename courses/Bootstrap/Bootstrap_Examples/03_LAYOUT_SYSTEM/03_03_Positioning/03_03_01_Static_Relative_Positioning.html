Static & Relative Positioning

The two most common CSS positioning values. Static is the default; relative lets you nudge elements from their natural position.

1. Static Positioning (Default)

position: static is the default. Elements are placed in the normal document flow. Top/bottom/left/right properties have NO effect.

Box 1 - position: static (default)
Box 2 - position: static with top:50px left:50px (IGNORED)
Box 3 - position: static
Key Point: Even though Box 2 has top: 50px; left: 50px;, these values are ignored because static positioning does not support offset properties.
Bootstrap Class
<div class="position-static">
  <!-- This element uses default static positioning -->
  <!-- top, right, bottom, left have no effect -->
</div>

2. Relative Positioning

position: relative positions the element relative to its normal position. The original space is preserved in the document flow.

Box 1 - Normal position (no positioning class)
Box 2 - position: relative; top: 20px; left: 40px
Box 3 - Notice Box 2's original space is preserved above
Important: position: relative does NOT remove the element from the document flow. The gap where it would normally be is preserved. Other elements act as if the relatively-positioned element is still in its original spot.
Bootstrap Class
<div class="position-relative">
  <!-- Offset from normal position -->
  <!-- Original space preserved in flow -->
</div>

<!-- With Bootstrap utility offsets -->
<div class="position-relative top-50 start-50">
  <!-- Offset 50% from top and left -->
</div>

3. Relative as Positioning Context

A relatively-positioned element creates a containing block for its absolutely-positioned children. This is essential for component-level positioning.

4. Bootstrap Offset Utilities with Relative

Bootstrap provides offset utility classes that work with position-relative.

top-0 start-0
top-0 end-0
top-50 start-50 translate-middle
bottom-0 start-0
bottom-0 end-0
Bootstrap Offset Classes
<div class="position-relative">
  <!-- Corners -->
  <div class="position-absolute top-0 start-0">Top Left</div>
  <div class="position-absolute top-0 end-0">Top Right</div>
  <div class="position-absolute bottom-0 start-0">Bottom Left</div>
  <div class="position-absolute bottom-0 end-0">Bottom Right</div>

  <!-- Center -->
  <div class="position-absolute top-50 start-50 translate-middle">
    Centered
  </div>

  <!-- Edge centers -->
  <div class="position-absolute top-0 start-50 translate-middle-x">
    Top Center
  </div>
</div>
