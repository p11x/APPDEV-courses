Positioning Mobile First

How to approach positioning with a mobile-first methodology. Start with mobile layout and progressively enhance for larger screens.

1. Mobile-First Positioning Strategy

Mobile: Use Normal Flow

On mobile, avoid absolute/fixed positioning. Let content flow naturally. Use normal document flow for stacking.

Header (flow)
Content (flow)
Footer (flow)
Desktop: Add Positioning

On larger screens, use absolute/fixed/sticky for enhanced layouts. Add positioning only when there's enough screen real estate.

Header (sticky)
Sidebar
Footer

2. Responsive Sticky Classes

Bootstrap provides responsive sticky classes that only activate at specific breakpoints.

ClassEffectMobileDesktop
.sticky-topAlways stickyStickySticky
.sticky-md-topSticky from md upNormal flowSticky
.sticky-lg-topSticky from lg upNormal flowSticky
 
<!-- Sidebar that is only sticky on desktop -->
<div class="col-md-3">
  <nav class="sticky-md-top">
    <!-- On mobile: scrolls normally -->
    <!-- On md+: sticks when scrolling -->
  </nav>
</div>

5. Mobile Touch Target Positioning

Minimum Touch Target: 44x44px
<!-- Positioned touch targets for mobile -->
<div class="position-relative">
  <!-- FAB (Floating Action Button) -->
  <button class="position-fixed bottom-0 end-0 m-3 rounded-circle
                  d-flex align-items-center justify-content-center"
          style="width: 56px; height: 56px; z-index: 1030;">
    +
  </button>

  <!-- Close button with adequate touch area -->
  <button class="btn-close position-absolute top-0 end-0 p-3"
          style="min-width: 44px; min-height: 44px;"
          aria-label="Close">
  </button>
</div>
