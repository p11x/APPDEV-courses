Display Utilities Complete Guide

1. How Display Utilities Work

Bootstrap display utilities set the CSS display property. They support responsive variants at each breakpoint.

.d-{value} for xs

.d-{breakpoint}-{value} for sm, md, lg, xl, xxl

Where value is: none, inline, inline-block, block, grid, table, flex, inline-flex
