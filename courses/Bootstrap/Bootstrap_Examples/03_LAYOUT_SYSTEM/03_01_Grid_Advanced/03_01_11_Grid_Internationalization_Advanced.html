5. CSS Logical Properties for i18n Grids

Replace physical properties (left, right, top, bottom) with logical equivalents (inline-start, inline-end, block-start, block-end) for automatic RTL support.
margin-inline-start

Replaces margin-left (LTR) / margin-right (RTL)

padding-inline-end

Replaces padding-right (LTR) / padding-left (RTL)

border-block-start

Replaces border-top (always top, regardless of direction)

inset-inline-start

Replaces left in position: absolute
