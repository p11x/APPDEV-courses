Category Best Practice Why

Responsiveauto-fit + minmax() first, media queries secondLess CSS, more fluid
NamingBEM for grid componentsSelf-documenting, maintainable
TokensCSS custom properties for grid configDRY, single source of truth
AccessibilityDOM order = visual orderScreen readers follow DOM
i18nLogical properties (inline-start/end)Automatic RTL support
