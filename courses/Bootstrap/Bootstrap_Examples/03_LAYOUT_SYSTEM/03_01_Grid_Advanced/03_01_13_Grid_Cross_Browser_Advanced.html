2. IE11-Safe Grid (Flexbox Fallback)

IE11 supports an old, incompatible grid spec. The safest approach is to use flexbox as the fallback and override with @supports for modern browsers.

6. Known Browser Bugs & Workarounds

Browser Bug Workaround
Safari < 14.1 gap doesn't work with flexbox Use CSS Grid (gap works) or margins for flexbox
Firefox Percentage gap calculated differently Use fixed units (px, rem) instead of %
Chrome < 57 grid-gap not supported Use gap (unprefixed) or margins
Safari < 10.1 auto-placement bug with implicit tracks Explicitly define grid-template-rows
Edge Legacy grid-template-areas with dense flow Avoid dense flow with named areas
IE 11 Entirely different grid spec Use flexbox fallback, not -ms-grid
