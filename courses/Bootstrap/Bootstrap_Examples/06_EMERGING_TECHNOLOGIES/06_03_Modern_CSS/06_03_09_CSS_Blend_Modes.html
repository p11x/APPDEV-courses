5. Blend Mode Reference

Mode Effect Common Use
multiplyDarkens (like stacking transparencies)Shadows, dark overlays
screenLightens (opposite of multiply)Highlights, light effects
overlayCombines multiply + screenContrast enhancement
darkenPicks darker pixel per channelShadow compositing
lightenPicks lighter pixel per channelHighlight compositing
color-dodgeBrightens backdropGlow effects
color-burnDarkens backdropDeep shadow effects
hard-lightStrong multiply/screenHarsh lighting
soft-lightSubtle multiply/screenGentle lighting
differenceSubtracts darker from lighterInversion, negative effects
exclusionLike difference, lower contrastSofter inversion
hueHue from blend, saturation+lum from backdropDuotone effects
saturationSaturation from blendColor intensity control
colorHue+sat from blend, lum from backdropColorization
luminosityLuminosity from blendGrayscale overlays
