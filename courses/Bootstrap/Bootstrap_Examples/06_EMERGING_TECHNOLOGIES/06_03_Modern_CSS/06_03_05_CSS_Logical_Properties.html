CSS Logical Properties

Writing-mode-aware layouts with inline/block and start/end

1. Physical vs Logical Mapping

Physical Property Logical Equivalent Description
margin-topmargin-block-startTop margin (block start)
margin-bottommargin-block-endBottom margin (block end)
margin-leftmargin-inline-startLeft margin (inline start in LTR)
margin-rightmargin-inline-endRight margin (inline end in LTR)
padding-toppadding-block-startTop padding
padding-leftpadding-inline-startLeft padding
widthinline-sizeSize along inline axis
heightblock-sizeSize along block axis
border-leftborder-inline-startBorder on inline start edge
topinset-block-startPosition offset from block start

2. Visual Comparison

Physical Properties

margin-left: 32px;
padding: 16px;
border-left: 4px;

In RTL, margin-left stays on the left (wrong side).

Logical Properties

margin-inline-start: 32px;
padding: 16px;
border-inline-start: 4px;

In RTL, margin-inline-start moves to the right automatically.

Logical properties adapt to writing-mode changes.

vertical-rl

inline:150px block:80px
