4. Animation Range Reference

Phase Start End Description
entry Element starts entering Element fully entered Animate during entrance
cover Element starts entering Element starts leaving Animate while visible
contain Element fully visible Element starts to leave Animate while fully in view
exit Element starts leaving Element fully left Animate during exit
