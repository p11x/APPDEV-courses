Rate Limit Headers

HTTP headers returned by APIs inform clients of their current rate limit status. Below simulates reading those headers.

Header Value Meaning
X-RateLimit-Limit 100 Max requests per window
X-RateLimit-Remaining 100 Requests left
X-RateLimit-Reset - Window reset (timestamp)
Retry-After - Seconds to wait if limited
X-RateLimit-Policy 100;w=60 Policy: limit;window_seconds
