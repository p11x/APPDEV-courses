API Integration Complete Guide

A full CRUD application demonstrating REST API calls, data tables, sorting, filtering, pagination, modals, toasts, search, and export.

Using JSONPlaceholder as a mock REST API — powered by Bootstrap 5.3.3

Key Concepts Demonstrated

1. Fetch API

Modern fetch() with async/await for GET, POST, PUT, and DELETE requests to a REST endpoint.

2. CRUD Operations

Create, Read, Update, Delete resources through modal forms and inline action buttons.

3. Data Table

Dynamic table with column sorting, client-side filtering, and responsive layout using Bootstrap table classes.

4. Modal Forms

Bootstrap modals serve as create/edit dialogs with form validation and submission handling.

5. Toast Notifications

Non-intrusive toast messages inform users of success, errors, and other events in real time.

6. Pagination

Client-side pagination controls slice the dataset and render page buttons dynamically.

7. Search

Real-time search filters the table by matching input against multiple fields simultaneously.

8. Export CSV

Convert the current dataset to CSV format and trigger a browser download for offline use.

9. Error Handling

Try/catch blocks, HTTP status checks, and user-friendly error messages ensure a resilient application.
