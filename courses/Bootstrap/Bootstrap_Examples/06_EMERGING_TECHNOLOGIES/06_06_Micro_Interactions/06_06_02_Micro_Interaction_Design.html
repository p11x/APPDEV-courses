Micro Interaction Design Principles

Applying Don Norman's design principles to Bootstrap UI

1. Affordance

Visual cues that suggest how elements should be used.

Pressable Buttons

Border thickness creates a "raised" look that invites pressing.

Grabbable Cards

Elevation on hover suggests the card can be moved.

Drag me

Hover to see elevation effect

2. Feedback

Every action needs immediate, visible response.

Click Confirmation

Button changes color and shows checkmark on click.

Loading State

Button shows spinner while processing.

3. Constraints

Limit actions to prevent errors and guide users.

Multi-Step Wizard with Constraints

Users can only proceed after completing current step.

4. Mapping

Controls should relate logically to what they control.

Volume Control (Vertical = Loudness)

Up = louder, Down = quieter. Natural spatial mapping.

5. Progressive Disclosure

Show only what's needed; reveal complexity gradually.
