PWA Core Concepts

Progressive Web Apps combine the best of web and native applications. Learn the fundamentals of building installable, offline-capable web apps with Bootstrap 5.

🌐 What is a Progressive Web App?

📡
Installable

Users can install the app on their device home screen without an app store.

🔌
Offline Capable

Works offline or on low-quality networks using service worker caching.

🔔
Engaging

Push notifications, home screen icons, and full-screen experience.

📄 Web App Manifest

The manifest.json file tells the browser how to behave when the PWA is installed.

manifest.json - Complete Example
{
    // App name displayed on home screen and splash screen
    "name": "My Progressive Web App",
    // Short name for constrained spaces
    "short_name": "My PWA",
    // Description for app stores and install prompts
    "description": "A Bootstrap 5 progressive web application",
    // URL opened when app launches
    "start_url": "/",
    // Display mode: standalone (no browser chrome)
    "display": "standalone",
    // Orientation lock: any, portrait, landscape
    "orientation": "any",
    // Background color for splash screen
    "background_color": "#ffffff",
    // Theme color for browser toolbar and title bar
    "theme_color": "#0d6efd",
    // Default language
    "lang": "en",
    // App icons for different sizes
    "icons": [
        {
            "src": "/icons/icon-72x72.png",
            "sizes": "72x72",
            "type": "image/png",
            "purpose": "any maskable"
        },
        {
            "src": "/icons/icon-96x96.png",
            "sizes": "96x96",
            "type": "image/png"
        },
        {
            "src": "/icons/icon-128x128.png",
            "sizes": "128x128",
            "type": "image/png"
        },
        {
            "src": "/icons/icon-192x192.png",
            "sizes": "192x192",
            "type": "image/png"
        },
        {
            "src": "/icons/icon-512x512.png",
            "sizes": "512x512",
            "type": "image/png"
        }
    ],
    // Categories for app store classification
    "categories": ["productivity", "utilities"],
    // Related applications for store listing
    "related_applications": [],
    "prefer_related_applications": false
}
Key Properties
  • name - Full app name
  • short_name - Short label (max 12 chars recommended)
  • start_url - Launch URL
  • display - standalone, fullscreen, minimal-ui, browser
  • icons - Array of icon objects (192x192 and 512x512 required)
HTML Link
<!-- Link to manifest file -->
<link rel="manifest" href="/manifest.json">

<!-- Theme color meta tag -->
<meta name="theme-color" content="#0d6efd">

<!-- Apple-specific meta tags -->
<link rel="apple-touch-icon" href="/icon-192.png">

PWA Registration - Basic Setup
// Register the service worker when the page loads
// This is the first step in converting a web app to a PWA

if ('serviceWorker' in navigator) {
    // Wait for the page to fully load before registering
    window.addEventListener('load', async () => {
        try {
            // Register the service worker file
            const registration = await navigator.serviceWorker.register('/sw.js', {
                // Scope determines which pages the SW controls
                scope: '/'
            });

            console.log('Service Worker registered:', registration.scope);

            // Listen for updates to the service worker
            registration.addEventListener('updatefound', () => {
                const newWorker = registration.installing;
                newWorker.addEventListener('statechange', () => {
                    if (newWorker.state === 'installed' && navigator.serviceWorker.controller) {
                        // New version available - show update prompt to user
                        showUpdateNotification();
                    }
                });
            });

        } catch (error) {
            console.error('Service Worker registration failed:', error);
        }
    });
}

// Show Bootstrap toast notification for updates
function showUpdateNotification() {
    const toast = document.createElement('div');
    toast.className = 'toast align-items-center text-bg-primary border-0 show position-fixed bottom-0 end-0 m-3';
    toast.setAttribute('role', 'alert');
    toast.innerHTML = `
        <div class="d-flex">
            <div class="toast-body">
                A new version is available! <a href="#" onclick="location.reload()" class="text-white">Refresh</a>
            </div>
            <button type="button" class="btn-close btn-close-white me-2 m-auto" data-bs-dismiss="toast"></button>
        </div>`;
    document.body.appendChild(toast);
}

📋 Quick Start Checklist

🚀 Getting Started with PWAs
  1. Create a manifest.json with app metadata and icons
  2. Add <link rel="manifest"> to your HTML
  3. Create a sw.js service worker file
  4. Register the service worker in your JavaScript
  5. Serve your site over HTTPS
  6. Test with Chrome DevTools > Application tab > Lighthouse
