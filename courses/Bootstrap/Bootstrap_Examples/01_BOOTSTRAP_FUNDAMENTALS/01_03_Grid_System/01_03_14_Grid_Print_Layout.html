5. Print CSS Best Practices

Save Ink
  • Remove backgrounds
  • Remove box shadows
  • Use black text
  • Remove decorative elements
  • Hide interactive UI
Improve Readability
  • Use 11-12pt font size
  • Show link URLs
  • Use serif fonts
  • Increase line height
  • Avoid tiny text
Page Management
  • Use break-inside: avoid
  • Add page breaks for sections
  • Set page margins
  • Use @page for size/orientation
  • Number pages if possible
@page Rule Example:
@page { size: A4 portrait; margin: 2cm; }
Sets paper size, orientation, and margins for printed pages.
