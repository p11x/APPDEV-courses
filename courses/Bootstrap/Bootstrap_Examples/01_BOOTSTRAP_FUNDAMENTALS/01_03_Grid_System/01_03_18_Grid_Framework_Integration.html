Grid Framework Integration

Integrating Bootstrap grid with Sass, build tools, and JavaScript frameworks.

1. Sass Grid Customization

Override Bootstrap's grid variables before importing the framework to customize columns, gutters, and breakpoints.

// _custom-grid.scss - Override Bootstrap grid defaults

// Change number of columns (default: 12)
$grid-columns: 16;

// Change gutter width (default: 1.5rem)
$grid-gutter-width: 2rem;

// Custom breakpoints
$grid-breakpoints: (
  "xs": 0,
  "sm": 576px,
  "md": 768px,
  "lg": 992px,
  "xl": 1200px,
  "xxl": 1400px,
  "xxxl": 1600px // Custom breakpoint
);

// Custom container max-widths
$container-max-widths: (
  "sm": 540px,
  "md": 720px,
  "lg": 960px,
  "xl": 1140px,
  "xxl": 1320px,
  "xxxl": 1500px
);

// Import Bootstrap after variable overrides
@import "bootstrap/scss/bootstrap";

2. Webpack Integration

Configure Webpack to compile Bootstrap's Sass and optimize grid CSS output.

webpack.config.js
const path = require('path');

module.exports = {
  entry: './src/index.js',
  module: {
    rules: [
      {
        test: /\.scss$/,
        use: [
          'style-loader',
          'css-loader',
          'sass-loader'
        ]
      }
    ]
  }
};
src/index.js
// Import only grid-related Sass
import './scss/custom-bootstrap.scss';

// custom-bootstrap.scss contains:
// @import "bootstrap/scss/functions";
// @import "bootstrap/scss/variables";
// @import "bootstrap/scss/mixins";
// @import "bootstrap/scss/grid";
// @import "bootstrap/scss/utilities";

3. Vite Integration

Vite has native Sass support. Import Bootstrap Sass directly in your entry file.

vite.config.js
import { defineConfig } from 'vite';

export default defineConfig({
  css: {
    preprocessorOptions: {
      scss: {
        additionalData: `
          $grid-columns: 12;
          $grid-gutter-width: 1.5rem;
        `
      }
    }
  }
});
src/main.js
// Vite auto-detects .scss files
import './style.scss';

// style.scss:
// @import "bootstrap/scss/bootstrap";

// Or import only what you need:
// @import "bootstrap/scss/grid";
// @import "bootstrap/scss/utilities";

4. React Integration

Use Bootstrap grid classes in React components. No special library needed.

React Component (JSX)
function GridLayout() {
  return (
    <div className="container">
      <div className="row">
        <div className="col-md-4">
          <p>Column 1</p>
        </div>
        <div className="col-md-4">
          <p>Column 2</p>
        </div>
        <div className="col-md-4">
          <p>Column 3</p>
        </div>
      </div>
    </div>
  );
}
react-bootstrap (Optional)
import { Container, Row, Col } from
  'react-bootstrap';

function GridLayout() {
  return (
    <Container>
      <Row>
        <Col md={4}>Column 1</Col>
        <Col md={4}>Column 2</Col>

}

5. Vue.js Integration

Bootstrap grid works seamlessly with Vue templates using standard class attributes.

Vue 3 Template
<template>
  <div class="container">
    <div class="row">
      <div
        v-for="item in items"
        :key="item.id"
        class="col-md-4 mb-3"
      >
        {{ item.name }}
      </div>
    </div>
  </div>
</template>
Bootstrap-Vue-Next (BSVN)
import { BContainer, BRow, BCol } from
  'bootstrap-vue-next';

// Composition API style
<BContainer>
  <BRow>
    <BCol md="4">
      Column 1
    </BCol>
    <BCol md="4">
      Column 2
    </BCol>
  </BRow>
</BContainer>

6. Angular Integration

Use Bootstrap grid with Angular's ngClass for dynamic class binding.

Angular Template
<div class="container">
  <div class="row">
    <div
      *ngFor="let item of items"
      [ngClass]="getColumnClass(item)"
    >
      {{ item.name }}
    </div>
  </div>
</div>
Component TypeScript
export class GridComponent {
  items = [...];

  getColumnClass(item: any): string {
    return item.featured
      ? 'col-md-8'
      : 'col-md-4';
  }
}

7. CSS Tree-Shaking (Import Only Grid)

Import only the grid module from Bootstrap's Sass to reduce CSS bundle size significantly.

Full Bootstrap CSS

~230 KB

Unminified, includes all components

@import "bootstrap/scss/bootstrap";
Grid + Utilities Only

~60 KB

Grid, flexbox, spacing, display utilities

@import "bootstrap/scss/grid";
@import "bootstrap/scss/utilities";
Grid Only

~25 KB

Just the grid system

@import "bootstrap/scss/grid";
// Tree-shaken Bootstrap import - grid only
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";
@import "bootstrap/scss/mixins";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/utilities/api";
