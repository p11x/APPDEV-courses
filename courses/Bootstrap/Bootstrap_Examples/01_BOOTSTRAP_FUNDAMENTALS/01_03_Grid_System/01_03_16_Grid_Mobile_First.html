Grid Mobile First

Mobile-first design principles, touch-friendly layouts, and progressive enhancement with Bootstrap grids.

Mobile First = Mobile Default. Base CSS targets mobile screens. Use col-* for mobile, then add col-md-*, col-lg-* for larger screens.

1. Mobile-First Column Stacking

Without breakpoint classes, columns stack on all screen sizes. Add col-md-* to go side-by-side on tablet+.

Stack → 2-col → 3-col

3. Responsive Gutters

Use responsive gutter classes like g-md-4 to increase spacing on larger screens where there's more room.

g-1 g-md-4
g-1 g-md-4
g-1 g-md-4

g-md-4 = 24px gutters on tablet+ (spacious)
