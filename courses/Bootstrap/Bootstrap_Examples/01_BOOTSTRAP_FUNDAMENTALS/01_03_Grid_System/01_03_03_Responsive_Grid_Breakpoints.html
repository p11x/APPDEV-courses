Responsive Grid Breakpoints

Using breakpoint-specific column classes to create responsive layouts.

1. Bootstrap 5 Breakpoints

BreakpointClassMin WidthDevice

2. How Responsive Classes Work

Responsive classes apply from the specified breakpoint and up. Classes without a breakpoint apply to all sizes.

1
2
3
<div class="row">
    <div class="col-sm-6 col-md-4">1</div>
    <div class="col-sm-6 col-md-4">2</div>
    <div class="col-sm-12 col-md-4">3</div>
</div>

/* Behavior:
   XS: full width (stacked)
   SM (576px+): 6-6-12 (two columns, third wraps)
   MD (768px+): 4-4-4 (three equal columns)
*/
