Grid Testing Strategies

Testing approaches for Bootstrap grid layouts: visual regression, responsive, unit, and accessibility testing.

1. Visual Regression Testing

Capture screenshots of grid layouts at specific viewport sizes and compare against baseline images to detect unintended changes.

2. Responsive Viewport Testing

Test grid layouts at each Bootstrap breakpoint to ensure columns stack/split correctly at every viewport size.

Test Column 1
col-12 col-md-6 col-lg-4
Test Column 2
col-12 col-md-6 col-lg-4
Test Column 3
col-12 col-md-12 col-lg-4
Playwright Test Script
const { test, expect } = require('@playwright/test');

// Test grid behavior at mobile viewport
test('grid stacks on mobile', async ({ page }) => {
  await page.setViewportSize({ width: 375, height: 667 });
  await page.goto('http://localhost:3000');

  const cols = await page.$$('#test-grid .col-12');

  // On mobile, columns should stack (each full width)
  for (const col of cols) {
    const box = await col.boundingBox();
    expect(box.width).toBeGreaterThan(300);
  }
});

// Test grid behavior at desktop viewport
test('grid splits on desktop', async ({ page }) => {
  await page.setViewportSize({ width: 1280, height: 800 });
  await page.goto('http://localhost:3000');

  const cols = await page.$$('#test-grid .col-lg-4');

  // On desktop, 3 columns should be side by side
  const box1 = await cols[0].boundingBox();
  const box2 = await cols[1].boundingBox();
  expect(box1.y).toBe(box2.y); // Same row
});

3. CSS Unit Tests (Jest + jsdom)

Write unit tests to verify that grid classes produce expected computed styles and DOM structure.

// grid.test.js
describe('Bootstrap Grid', () => {
  beforeEach(() => {
    document.body.innerHTML = `
      <div class="container">
        <div class="row">
          <div class="col-md-6" id="col1">A</div>
          <div class="col-md-6" id="col2">B</div>
        </div>
      </div>
    `;
  });

  test('row has flexbox display', () => {
    const row = document.querySelector('.row');
    const styles = window.getComputedStyle(row);
    expect(styles.display).toBe('flex');
  });

  test('columns are in the same row', () => {
    const col1 = document.getElementById('col1');
    const col2 = document.getElementById('col2');
    expect(col1.parentElement).toBe(col2.parentElement);
  });

  test('columns have correct class names', () => {
    const col1 = document.getElementById('col1');
    expect(col1.classList.contains('col-md-6')).toBe(true);
    expect(col1.classList.contains('col')).toBe(true);
  });

  test('row does not exceed 12 columns', () => {
    const cols = document.querySelectorAll('.row > [class*="col"]');
    let totalCols = 0;
    cols.forEach(col => {
      const match = col.className.match(/col-md-(\d+)/);
      if (match) totalCols += parseInt(match[1]);
    });
    expect(totalCols).toBeLessThanOrEqual(12);
  });
});

4. Grid Accessibility Testing

Ensure grid layouts are accessible: proper reading order, screen reader compatibility, and keyboard navigation.

Accessibility Checklist
✓ DOM order matches visual order
Don't use order-* classes that contradict reading order
✓ Content readable without CSS
Grid content should make sense linearly
✓ Sufficient color contrast
Text in grid items meets WCAG AA (4.5:1)
✓ Touch targets ≥ 44x44px
Interactive grid elements are easy to tap
✓ Responsive layout works at 200% zoom
No horizontal scrolling at 200% browser zoom
axe-core Test Script
const { AxeBuilder } = require('@axe-core/playwright');

test('grid has no accessibility violations',
  async ({ page }) => {
    await page.goto('http://localhost:3000');

    const results = await new AxeBuilder({ page })
      .include('.container')
      .analyze();

    expect(results.violations).toHaveLength(0);
  }
);

6. CI/CD Integration

Run grid tests automatically in your CI pipeline on every commit.

GitHub Actions Workflow
# .github/workflows/grid-test.yml
name: Grid Tests
on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
      - run: npm ci
      - run: npm run test:grid
      - run: npm run test:visual
package.json Scripts
{
  "scripts": {
    "test:grid": "jest grid.test.js",
    "test:visual": "backstop test",
    "test:a11y": "playwright test a11y",
    "test:responsive": "playwright test responsive",
    "test:all": "npm run test:grid && npm run test:visual && npm run test:a11y"
  }
}
