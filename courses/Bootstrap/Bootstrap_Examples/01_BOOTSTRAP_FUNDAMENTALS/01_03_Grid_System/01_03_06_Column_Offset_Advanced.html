Column Offset Advanced

Using offset classes to push columns to the right.

1. Basic Offsets (offset-*)

Offsets create empty space to the left of a column.

col-4
col-4 offset-4
col-3 offset-3
col-3 offset-3
col-6 offset-3 (centered)
col-md-4 offset-md-4
col-md-4

2. All Offset Values

offset-1
offset-2
offset-3
offset-4
offset-5
offset-6

3. Responsive Offsets

Use .offset-{breakpoint}-{n} for breakpoint-specific offsets.

col-md-4
col-md-4 offset-md-4
col-lg-3
col-lg-3 offset-lg-3
col-lg-3

4. Resetting Offsets at Breakpoints

Use .offset-md-0 to reset an offset at a specific breakpoint.

Reset offset on md
Col 2
<div class="col-sm-5 offset-sm-2 col-md-6 offset-md-0">
    Offset resets to 0 on md breakpoint
</div>

5. Using Margin Utilities Instead

Bootstrap 5 recommends .ms-auto and .me-auto as alternatives to offsets for pushing columns.

col-4
col-4 ms-auto
col-4 me-auto
col-4

6. Offset Classes Reference

ClassEffect
.offset-*Offset at all sizes
.offset-sm-*Offset at sm+ (576px+)
.offset-md-*Offset at md+ (768px+)
.offset-lg-*Offset at lg+ (992px+)
.offset-xl-*Offset at xl+ (1200px+)
.offset-xxl-*Offset at xxl+ (1400px+)
.offset-*-0Reset offset at breakpoint
