Basic Grid Structure

Understanding Bootstrap's 12-column flexbox grid system.

1. How the Grid Works

Bootstrap's grid uses three components:

  • .container - Wraps the grid and provides max-width constraints
  • .row - Horizontal group of columns
  • .col-* - Columns that hold content (12 per row)

5. The 12-Column Rule

Columns exceeding 12 units wrap to the next line:

col-8
col-6 (wraps! 8+6=14 > 12)
Note: When columns exceed 12, the extra columns wrap to a new line. This is intentional and useful for responsive layouts.
