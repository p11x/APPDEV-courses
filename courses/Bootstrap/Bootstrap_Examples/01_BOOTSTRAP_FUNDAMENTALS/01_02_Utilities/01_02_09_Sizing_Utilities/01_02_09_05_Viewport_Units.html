.min-vh-100 Section

This section uses .min-vh-100, meaning it will be at least as tall as the viewport, but can grow taller if content demands it.

This is different from .vh-100 which sets an exact height. With .min-vh-100, the element flexes with its content.

max-vh-100 for Scrollable Modals/Containers

.max-vh-100 is useful when you want a container to be at most the full viewport, but scrollable if content overflows.
