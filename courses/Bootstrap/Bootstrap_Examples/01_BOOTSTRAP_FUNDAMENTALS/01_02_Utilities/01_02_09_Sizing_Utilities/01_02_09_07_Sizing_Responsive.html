Responsive Sizing Utilities

Resize your browser window to see how responsive sizing classes change element widths at different breakpoints.

Responsive Width Classes

Apply width at specific breakpoints using .w-{breakpoint}-{size}.

Breakpoint behavior: The class applies at the specified breakpoint and above. Below that breakpoint, the default (auto) applies.
.w-100 .w-sm-75 .w-md-50 .w-lg-25
(Full on mobile, 75% on sm, 50% on md, 25% on lg)
.w-100 .w-md-50 .w-xl-25
(Full on mobile, 50% on medium, 25% on xl)

Responsive Height Classes

Height classes can also be made responsive, though this is less common.

.h-25 .h-md-50 .h-lg-100
This element changes height at different breakpoints. At sm: 25% of parent. At md: 50%. At lg and above: 100%.
