Max-Height Utilities

Use .mh-100 to set max-height: 100% on an element.

Basic Max-Height: .mh-100

The .mh-100 utility constrains an element so it does not exceed the height of its parent.

.mh-100 with overflow-auto
This content is constrained to the parent's 300px height. Even if we add lots of content here, it will scroll instead of expanding beyond the parent.

Quick Reference

Class CSS Output Description
.mh-100max-height: 100% !important;Element cannot exceed parent height

Usage Notes

  • The parent element must have an explicit height for .mh-100 to work
  • Pair with .overflow-auto or .overflow-scroll to handle content that exceeds the max-height
  • Useful for modals, sidebars, card bodies, and any container where content length is variable
