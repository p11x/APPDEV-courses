7. Common Patterns

Centered Form

<div class="d-flex justify-content-center">
    <div class="w-100" style="max-width: 400px;">
        <!-- form content -->
    </div>
</div>

Full-Height Hero

<div class="vh-100 d-flex align-items-center justify-content-center">
    <h1>Hero Content</h1>
</div>

Equal-Height Cards

<div class="row">
    <div class="col-md-4"><div class="card h-100">...</div></div>
    <div class="col-md-4"><div class="card h-100">...</div></div>
    <div class="col-md-4"><div class="card h-100">...</div></div>
</div>

Scrollable Sidebar

<div class="d-flex" style="height: 100vh;">
    <nav class="mh-100 overflow-auto" style="width: 250px;">...</nav>
    <main class="flex-grow-1 overflow-auto">...</main>
</div>

Responsive Image

<img src="photo.jpg" class="mw-100" alt="...">
<!-- or use img-fluid which includes max-width: 100% and height: auto -->
<img src="photo.jpg" class="img-fluid" alt="...">

8. Common Gotchas

Gotcha #1: Percentage height requires parent height

.h-100 does nothing if the parent has no explicit height. Use style="height: 500px" or .vh-100 on the parent.

Gotcha #2: !important overrides inline styles

.w-50 will override style="width: 80%" because Bootstrap utilities use !important.

Gotcha #3: w-auto on block elements

.w-auto on a block element makes it full-width (block behavior), not content-width. Use d-inline-block for content-width.

Gotcha #4: Viewport units and mobile browsers

100vh includes the browser's address bar on mobile. The actual visible area is smaller. Use dvh (dynamic viewport height) in modern browsers if needed.
