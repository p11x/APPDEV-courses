Max-Width on Embedded Content

Embedded content like videos should always use .mw-100 to remain responsive.

The 16:9 ratio container ensures the video stays responsive.

mw-100 vs w-100

.w-100 forces width to 100%. .mw-100 allows width up to 100% but not beyond.

.w-100:

Always 100% of parent (500px)

.mw-100:

Natural width 300px, max 500px

Quick Reference

Class CSS Output Description
.mw-100max-width: 100% !important;Element cannot exceed parent width

Common Use Cases

  • Responsive images: Prevent wide images from causing horizontal scrolling
  • Responsive tables: Wrap tables in a .mw-100 container with overflow-auto
  • Embedded videos: Ensure video players don't overflow their containers
  • Content readability: Limit text block width for optimal line length (~60-75 characters)
