Bootstrap 5 Width Utilities

Use .w-25, .w-50, .w-75, .w-100, and .w-auto to set the width of an element.

Percentage Width Classes

These classes set the width CSS property using percentages relative to the parent container.

.w-25 (25%)
.w-50 (50%)
.w-75 (75%)
.w-100 (100%)
.w-auto (auto)

Width on Different Elements

Width utilities can be applied to various HTML elements including divs, images, and inline elements.

Image with .w-50

Button with .w-100

Input with .w-75

Combining Width with Other Utilities

Width utilities are designed to work seamlessly with other Bootstrap utility classes.

.w-50 + .bg-primary + .p-3
.w-75 + .bg-success + .p-3
.w-25 + .bg-danger + .p-3
.w-100 + .bg-warning + .p-3

Width in Flexbox Layout

When used inside a flex container, width utilities override the flex item's default behavior.

.w-25
.w-50
.w-25

Auto Width Behavior

The .w-auto class resets the width to auto, which is useful in flex and grid layouts.

.w-auto (content-driven)
.w-auto
.w-auto with more text content
