Important: Parent Height Requirement

Percentage-based height utilities (.h-25 through .h-100) only work when the parent element has an explicit height set. Without it, the browser cannot calculate the percentage and the child may collapse to content height.

Use style="height: ...", min-height, or Bootstrap's .vh-100 on the parent.
