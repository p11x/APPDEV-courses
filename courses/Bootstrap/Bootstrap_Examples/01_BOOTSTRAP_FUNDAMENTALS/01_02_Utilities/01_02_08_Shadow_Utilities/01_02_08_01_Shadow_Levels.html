6. Shadow CSS Values

Class CSS box-shadow Value Use Case
(none) none Flat elements, minimal UI
.shadow-sm 0 1px 2px rgba(0,0,0,0.075) Subtle depth, buttons, inputs
.shadow 0 0.5rem 1rem rgba(0,0,0,0.15) Cards, modals, dropdowns
.shadow-lg 0 1rem 3rem rgba(0,0,0,0.175) Featured elements, hero sections
