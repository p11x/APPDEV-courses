5. Mobile Performance Tips

Recommendations
  • Use .shadow-sm for most mobile elements
  • Reserve .shadow for primary cards
  • Avoid .shadow-lg on lists with many items
  • Don't animate shadows on mobile
  • Use will-change sparingly
  • Test on actual devices, not just emulators
Avoid
  • .shadow-lg on 50+ list items
  • Animated shadows during scroll
  • Multiple overlapping shadow layers
  • Custom shadows with large blur values
  • Shadows on elements that resize often
  • Shadows combined with complex filters

6. Mobile Shadow Quick Reference

Element Recommended Shadow Reason
Cards .shadow-sm Subtle depth, good performance
Featured card .shadow Slight emphasis
Top navigation .shadow-sm Subtle separation
Bottom navigation .shadow Clear elevation from content
Modals/Sheets .shadow-lg Prominent overlay
List items .shadow-sm or none Performance on long lists
FAB button .shadow-lg Floating effect
