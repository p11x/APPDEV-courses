5. Text-Shadow (Not Box-Shadow)

Text with text-shadow

text-shadow is different from box-shadow. Bootstrap shadow utilities only apply box-shadow.

6. Compatibility Summary

Bottom line: Bootstrap 5 shadow utilities work consistently across all target browsers. box-shadow has been universally supported for over a decade.
