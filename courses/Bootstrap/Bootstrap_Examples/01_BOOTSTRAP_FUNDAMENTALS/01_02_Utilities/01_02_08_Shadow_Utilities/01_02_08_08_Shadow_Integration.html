10. Shadow + Responsive Design

Use display utilities to show/hide shadow elements at breakpoints:

<!-- Shadow only on medium+ screens -->
<div class="card shadow-none shadow-md">...</div>

<!-- Or use custom CSS with media queries -->
@media (min-width: 768px) {
    .responsive-shadow { box-shadow: 0 0.5rem 1rem rgba(0,0,0,0.15); }
}
