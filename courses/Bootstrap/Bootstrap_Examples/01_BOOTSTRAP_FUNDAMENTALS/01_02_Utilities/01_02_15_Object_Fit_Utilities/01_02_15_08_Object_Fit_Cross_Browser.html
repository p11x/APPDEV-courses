2. IE11 Fallback Pattern

If you must support IE11, use a background-image fallback:

/* Modern browsers */
.fit-cover img {
    object-fit: cover;
    width: 100%;
    height: 100%;
}

/* IE11 fallback using @supports */
@supports not (object-fit: cover) {
    .fit-cover {
        background-image: url('image.jpg');
        background-size: cover;
        background-position: center;
    }
    .fit-cover img {
        visibility: hidden;
    }
}

3. Edge Cases

Safari iOS: object-fit on <video> may not work with inline playback in older versions. Use playsinline attribute. 
<!-- Always add playsinline for iOS -->
<video class="object-fit-cover" autoplay muted loop playsinline>
    <source src="video.mp4" type="video/mp4">
</video>

4. CSS Feature Detection

/* Feature detection with @supports */
@supports (object-fit: cover) {
    .hero-image {
        object-fit: cover;
        object-position: center;
    }
}

/* JavaScript feature detection */
if ('objectFit' in document.documentElement.style) {
    // Browser supports object-fit
    document.documentElement.classList.add('objectfit');
} else {
    // Fallback needed
    document.documentElement.classList.add('no-objectfit');
}
