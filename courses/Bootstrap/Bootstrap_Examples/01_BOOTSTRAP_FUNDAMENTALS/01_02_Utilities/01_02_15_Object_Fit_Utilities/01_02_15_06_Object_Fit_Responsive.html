Object Fit Responsive

Adapting object-fit behavior across different screen sizes.

1. Responsive Container Heights

Adjust the container height at different breakpoints while keeping object-fit consistent.

.responsive-fit { height: 150px; }
@media (min-width: 768px) { .responsive-fit { height: 200px; } }
@media (min-width: 992px) { .responsive-fit { height: 300px; } }

2. Responsive Object Fit Values

Change object-fit behavior per breakpoint (e.g., contain on mobile, cover on desktop):

<!-- Show contain on mobile, cover on tablet+ -->
<img class="object-fit-contain d-md-none" src="...">
<img class="object-fit-cover d-none d-md-block" src="...">

3. Responsive Grid Gallery

Gallery items with consistent aspect ratios across screen sizes.

4. Responsive Object Position

Change the focal point based on screen size to show the most important part of the image:

/* Show different focal points per breakpoint */
.featured-img { object-position: center top; }
@media (min-width: 768px) { .featured-img { object-position: center; } }
