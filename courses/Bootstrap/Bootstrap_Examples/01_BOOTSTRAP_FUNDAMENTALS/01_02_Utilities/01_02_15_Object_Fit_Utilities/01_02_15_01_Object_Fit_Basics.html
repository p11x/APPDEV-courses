Object Fit Basics

Understanding the CSS object-fit property and Bootstrap 5's utility classes for it.

1. What is Object Fit?

The object-fit CSS property controls how <img> and <video> elements fill their container. It works like background-size but for HTML media elements.

Key Concept: When an image's dimensions don't match its container, object-fit determines how the image is scaled, cropped, or stretched.

3. Object Fit: Contain

Use .object-fit-contain to scale the image to fit entirely within the container, preserving aspect ratio. May show empty space.

<img src="image.jpg" class="w-100 h-100 object-fit-contain" alt="...">

4. Object Fit: Cover

Use .object-fit-cover to fill the entire container, cropping excess parts. No empty space visible.

<img src="image.jpg" class="w-100 h-100 object-fit-cover" alt="...">

5. Object Fit: Fill

Use .object-fit-fill to stretch the image to fill the container. This is the default browser behavior.

<img src="image.jpg" class="w-100 h-100 object-fit-fill" alt="...">

6. Object Fit: Scale Down

Use .object-fit-scale-down to behave like contain but never scale the image larger than its natural size.

<img src="image.jpg" class="w-100 h-100 object-fit-scale-down" alt="...">

7. Object Fit: None

Use .object-fit-none to keep the image at its natural size, which may overflow the container.

<img src="image.jpg" class="w-100 h-100 object-fit-none" alt="...">
