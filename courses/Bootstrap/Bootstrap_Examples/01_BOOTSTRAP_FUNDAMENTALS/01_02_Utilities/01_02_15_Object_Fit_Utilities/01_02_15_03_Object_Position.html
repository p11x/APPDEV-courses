Object Position

Controlling the focal point of images and videos using Bootstrap 5 object-position utilities.

1. What is Object Position?

The object-position CSS property determines the alignment of the image/video within its container when object-fit is cover, contain, or none.

Bootstrap provides these classes: .object-position-top, .object-position-center, .object-position-bottom.

2. Position: Center (Default)

Without object-position, the image is centered by default.

<img src="tall-image.jpg" class="w-100 h-100 object-fit-cover" alt="...">

3. Position: Top

Use .object-position-top to align the top of the image visible.

<img src="tall-image.jpg" class="w-100 h-100 object-fit-cover object-position-top" alt="...">

4. Position: Bottom

Use .object-position-bottom to align the bottom of the image visible.

<img src="tall-image.jpg" class="w-100 h-100 object-fit-cover object-position-bottom" alt="...">

6. Custom Position Values

You can use inline styles or custom CSS for more precise positioning:

left top

right top

50% 30%

left bottom

<!-- Custom position with inline style -->
<img src="image.jpg" class="w-100 h-100 object-fit-cover" style="object-position: 50% 30%;">

<!-- Custom CSS class -->
.object-position-top-right {
    object-position: top right;
}
