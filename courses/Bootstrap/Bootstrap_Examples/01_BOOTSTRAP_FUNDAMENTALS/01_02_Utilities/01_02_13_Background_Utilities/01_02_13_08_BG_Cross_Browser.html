Safari Fixed Background Fix

background-attachment: fixed doesn't work on iOS Safari. Use this workaround for parallax-like effects: 

/* Fix for iOS Safari */
@supports (-webkit-touch-callout: none) {
    .bg-fixed {
        background-attachment: scroll;
    }
}

Bootstrap CSS Variables for Backgrounds

Bootstrap 5 uses CSS custom properties (variables) for theming:

/* Bootstrap's CSS variables for backgrounds */
:root {
    --bs-primary: #0d6efd;
    --bs-success: #198754;
    --bs-body-bg: #fff;
    --bs-body-bg-rgb: 255, 255, 255;
}

/* Usage in custom CSS */
.custom-bg {
    background-color: var(--bs-primary);
}

IE11 Considerations

Bootstrap 5 dropped IE11 support. If you need IE11 compatibility, use Bootstrap 4 or add CSS variable polyfills like css-vars-ponyfill.
