Responsive Backgrounds

Adapting Backgrounds to Screen Size

Important Note

Bootstrap 5 does NOT include responsive background utility classes like .bg-md-primary. To achieve responsive backgrounds, use custom CSS with media queries, or JavaScript-based solutions.

CSS Media Query Approach

Resize your browser to see the background color change:

Responsive Background

Mobile: Light gray Tablet: Blue Desktop: Green XL: Purple 

.responsive-bg { background-color: #f8f9fa; }
@media (min-width: 768px) { .responsive-bg { background-color: #0d6efd; } }
@media (min-width: 992px) { .responsive-bg { background-color: #198754; } }

JavaScript Approach

Use JavaScript to dynamically apply Bootstrap background classes:

JS-Driven Background

Current: bg-light

function updateBg() {
    const el = document.getElementById('jsResponsiveBg');
    el.className = 'p-4 rounded text-center ';
    if (window.innerWidth >= 992) el.classList.add('bg-primary', 'text-white');
    else if (window.innerWidth >= 768) el.classList.add('bg-info', 'text-dark');
    else el.classList.add('bg-light');
}
window.addEventListener('resize', updateBg);
updateBg();
