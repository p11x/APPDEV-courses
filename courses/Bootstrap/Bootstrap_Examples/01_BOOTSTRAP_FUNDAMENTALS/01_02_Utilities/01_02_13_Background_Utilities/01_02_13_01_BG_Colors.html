Theme Color Backgrounds

Use .bg-{color} to apply Bootstrap's theme colors as backgrounds. These automatically adapt to dark mode when enabled.

.bg-primary
.bg-secondary
.bg-success
.bg-danger
.bg-warning
.bg-info
.bg-light
.bg-dark
.bg-body

Bootstrap provides subtle (lighter) versions of theme colors.
