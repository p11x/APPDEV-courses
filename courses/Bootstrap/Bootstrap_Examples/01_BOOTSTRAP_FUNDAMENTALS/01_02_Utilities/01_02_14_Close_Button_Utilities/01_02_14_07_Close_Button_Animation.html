Close Button Animation

Adding smooth transitions and animations to close button dismissals in Bootstrap 5.

1. Built-in Fade Animation

Fade animation: This alert uses Bootstrap's built-in .fade and .show classes. Click the X to see the fade-out effect. 
<!-- Fade classes: .fade .show -->
<div class="alert alert-info alert-dismissible fade show">
    Content here
    <button class="btn-close" data-bs-dismiss="alert"></button>
</div>

/* CSS Transition */
.fade {
    transition: opacity 0.15s linear;
}
.fade:not(.show) {
    opacity: 0;
}

2. Fade Slide Right

Slide Right: This alert slides to the right as it fades. 
.fade-slide-right {
    transition: opacity 0.3s ease, transform 0.3s ease;
}
.fade-slide-right.fade:not(.show) {
    opacity: 0;
    transform: translateX(20px);
}

3. Fade Slide Up

Slide Up: This alert slides up as it fades. 
.fade-slide-up {
    transition: opacity 0.3s ease, transform 0.3s ease;
}
.fade-slide-up.fade:not(.show) {
    opacity: 0;
    transform: translateY(-10px);
}

4. Scale Fade

Scale Fade: This alert scales down as it fades. 
.fade-scale {
    transition: opacity 0.3s ease, transform 0.3s ease;
}
.fade-scale.fade:not(.show) {
    opacity: 0;
    transform: scale(0.9);
}

5. Animated Close Button Hover

Rotate on hover

Pulse on hover

Spin on click

/* Rotate on hover */
.btn-close-animated {
    transition: transform 0.2s ease;
}
.btn-close-animated:hover {
    transform: rotate(90deg);
}

/* Pulse on hover */
@keyframes closePulse {
    0%, 100% { transform: scale(1); }
    50% { transform: scale(1.15); }
}
.btn-close-pulse:hover {
    animation: closePulse 0.5s ease infinite;
}
