5. Anatomy of a Close Button

<button 
    type="button"           
    class="btn-close"       
    aria-label="Close"      
    data-bs-dismiss="alert" 
></button>
  • type="button": Prevents form submission
  • class="btn-close": Applies Bootstrap close button styles
  • aria-label="Close": Provides accessible name for screen readers
  • data-bs-dismiss: Triggers dismissal of parent component
