Close Button Disabled

Controlling when close buttons should be disabled or non-interactive in your Bootstrap 5 applications.

1. Using HTML Disabled Attribute

Close button with disabled attribute:

This button has the disabled attribute applied.

Note: The HTML disabled attribute prevents interaction but Bootstrap's default CSS doesn't add a visible disabled appearance to .btn-close. You may need custom CSS.

2. Custom Disabled Class Approach

Normal close button:

Custom disabled close button:

/* Custom CSS for disabled close button */
.btn-close.disabled {
    opacity: 0.25;        /* Visual indication */
    pointer-events: none; /* Prevent clicks */
    user-select: none;    /* Prevent text selection */
}
