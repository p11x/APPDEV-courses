Basic Usage

Apply display classes directly to any HTML element.

<!-- Hidden element -->
<div class="d-none">Hidden</div>

<!-- Block element -->
<span class="d-block">This span is block-level</span>

<!-- Inline element -->
<div class="d-inline">This div is inline</div>

<!-- Inline-block element -->
<div class="d-inline-block" style="width:200px">Inline-block</div>

<!-- Flex container -->
<div class="d-flex">
    <div>Flex child 1</div>
    <div>Flex child 2</div>
</div>

Responsive Display

Format: d-{breakpoint}-{value}. Classes apply from the breakpoint and UP. 

<!-- Hidden on mobile, visible on md+ -->
<div class="d-none d-md-block">Desktop content</div>

<!-- Visible on mobile, hidden on md+ -->
<div class="d-block d-md-none">Mobile content</div>

<!-- Block on mobile, flex on lg+ -->
<div class="d-block d-lg-flex">Responsive layout</div>

Accessibility Guidelines

Rule Details
Use .visually-hidden for SR content Content hidden with d-none is removed from the accessibility tree
Update aria-expanded Toggle this attribute when showing/hiding content
Use aria-controls Connect trigger buttons to their target elements
Manage focus Move focus into shown content, return to trigger when hidden
Use aria-live Announce dynamic content changes to screen readers

Performance Tips

  • Use d-none for lazy loading: Hide heavy content until needed
  • Batch toggles: Toggle a parent container instead of individual children
  • Prefer opacity for animations: Use CSS transitions instead of toggling d-none
  • Purge unused classes: Use PurgeCSS or Bootstrap's Sass customization
  • Use .visually-hidden over d-none: When content should be accessible but not visible

Common Recipes

1. Responsive Navbar
<nav class="navbar navbar-expand-lg">
    <button class="navbar-toggler d-lg-none">☰</button>
    <div class="d-none d-lg-flex">Desktop links</div>
    <div class="collapse d-lg-none">Mobile menu</div>
</nav>
2. Show/Hide Toggle
<button onclick="this.nextElementSibling.classList.toggle('d-none')">
    Toggle
</button>
<div>Content to show/hide</div>
3. Responsive Table → Cards
<!-- Desktop table -->
<table class="d-none d-md-table">...</table>
<!-- Mobile cards -->
<div class="d-block d-md-none">Card view...</div>
4. Stacked Buttons on Mobile
<div class="d-grid d-md-flex gap-2">
    <button class="btn btn-primary flex-fill">Save</button>
    <button class="btn btn-secondary flex-fill">Cancel</button>
</div>
5. Sticky Footer
<body class="d-flex flex-column" style="min-height:100vh">
    <main class="flex-grow-1">Content</main>
    <footer>Footer</footer>
</body>
