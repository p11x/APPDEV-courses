Responsive Display Utilities

Bootstrap 5 responsive display utilities use breakpoint suffixes to control visibility and display type at specific screen widths.

Breakpoint Reference

Extra smallNone0px
Smallsm576px

Extra extra largexxl1400px

1. Visible Only on XS (Extra Small) Screens

Use d-block d-sm-none: visible by default, hidden at sm (≥576px) and above.

XS only — I'm only visible on screens narrower than 576px.
Classes: d-block d-sm-none
All others — I'm visible on sm screens and above.
Classes: d-none d-sm-block

Resize your browser to see the switch between elements.

2. Visible Only on MD (Medium) Screens

d-none d-md-block d-lg-none: hidden everywhere, shown only at md (768px–991px).

MD only — Visible between 768px and 991px only.
Classes: d-none d-md-block d-lg-none

This element is hidden on xs, sm, lg, xl, and xxl screens.

3. Visible on LG and Above (Desktop)

d-none d-lg-block: hidden on small/medium, visible on large screens.

Desktop only — I appear on screens ≥992px.
Classes: d-none d-lg-block
Mobile/Tablet — I appear below 992px.
Classes: d-block d-lg-none

4. Responsive Flexbox Layout

Switch from stacked (block) on mobile to side-by-side (flex) on medium screens and up.

Column 1
Stacked on mobile, flex item on md+
Column 2
Stacked on mobile, flex item on md+
Column 3
Stacked on mobile, flex item on md+

Classes: d-block d-md-flex — below 768px items stack vertically, above they sit in a flex row.

7. Multi-Breakpoint Visibility

Show an element only at certain breakpoint ranges.

SM + MD only — Visible from 576px to 991px.
d-none d-sm-block d-lg-none
XL + XXL only — Visible from 1200px+.
d-none d-xl-block
MD + LG + XL only — Visible from 768px to 1399px.
d-none d-md-block d-xl-none

8. Print Display Utilities

Control visibility when the page is printed using d-print-* classes.

Always visible — I show on screen and when printing.
Screen only — I'm visible on screen but hidden when printing.
d-print-none
Print only — I'm hidden on screen but appear when printing.
d-none d-print-block

Try pressing Ctrl+P to see the print preview.

9. Pattern Summary

Format: d-{breakpoint}-{value}

Key rule: Responsive classes apply from the specified breakpoint and UP. A class like d-md-none applies at md, lg, xl, and xxl.

Common patterns:
• Hide on mobile: d-none d-md-block
• Hide on desktop: d-block d-md-none
• Show on medium only: d-none d-md-block d-lg-none
• Switch from block to flex: d-block d-md-flex
