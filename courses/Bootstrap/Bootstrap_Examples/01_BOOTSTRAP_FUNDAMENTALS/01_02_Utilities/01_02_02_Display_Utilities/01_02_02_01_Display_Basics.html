Bootstrap 5 Display Utilities - Basics

Bootstrap display utilities let you control how elements are displayed using simple class names. These map directly to the CSS display property.

1. d-none — Hide Element

The d-none class sets display: none, completely removing the element from the layout.

I have d-none — I am hidden!
I am a normal visible box (no display class)

The first box has d-none and is completely removed from the page.

2. d-block — Block Display

d-block sets display: block, making the element take full width of its parent.

This is a <span> with d-block Another <span> with d-block

Both spans are displayed as block elements — each takes full width and starts on a new line.

3. d-inline — Inline Display

d-inline sets display: inline, so the element flows inline with text.

d-inline div 1
d-inline div 2
d-inline div 3

All three divs now flow inline with one another instead of stacking.

4. d-inline-block — Inline-Block Display

d-inline-block flows inline but respects width and height like a block element.

d-inline-block
150×100
d-inline-block
200×80
d-inline-block
120×120

Each element flows inline but respects its own width and height dimensions.

5. d-flex — Flexbox Container

d-flex sets display: flex, enabling the flexbox layout model.

Flex Item 1
Flex Item 2
Flex Item 3

Items are laid out in a row by default when using d-flex.

6. d-inline-flex — Inline Flex Container

d-inline-flex creates a flex container that itself behaves as an inline element.

Below are two inline-flex containers side by side:

A1
A2
B1
B2

Each container is inline — they sit next to each other in the flow.

7. d-table — Table Display

d-table mimics <table> behavior using divs.

Row 1, Cell 1
Row 1, Cell 2
Row 1, Cell 3
Row 2, Cell 1
Row 2, Cell 2
Row 2, Cell 3

Using d-table, d-table-row, and d-table-cell to create a table layout without <table> elements.

8. Visibility — invisible vs d-none

Bootstrap does not have d-invisible, but you can use the CSS class invisible.

Normal visible box
I am invisible — space reserved
Normal visible box below invisible element

The invisible box still takes up space in the document flow. Compare with d-none which removes it entirely.

9. Quick Reference

Class CSS Property Description
d-none display: none Hides the element completely, removes from flow
d-inline display: inline Element flows inline with text
d-block display: block Element takes full width, starts on new line
d-inline-block display: inline-block Flows inline, respects width/height
d-flex display: flex Creates a flex container
d-inline-flex display: inline-flex Inline flex container
d-table display: table Element behaves like <table>
d-table-row display: table-row Element behaves like <tr>
d-table-cell display: table-cell Element behaves like <td>
invisible visibility: hidden Hidden but space is preserved
