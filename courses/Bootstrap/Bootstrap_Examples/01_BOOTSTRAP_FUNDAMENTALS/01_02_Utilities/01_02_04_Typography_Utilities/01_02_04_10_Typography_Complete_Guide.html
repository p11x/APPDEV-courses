8. Text Wrap & Overflow

Wrap Utilities

.text-wrap: Long text wraps normally at the container edge.

.text-nowrap: Text stays on one line.

.text-truncate: Very long text is truncated with an ellipsis at the end.

Overflow Utilities

.overflow-auto: Scrollbars appear when content overflows.
.overflow-hidden: Content is clipped with no scrollbars.
.overflow-scroll: Scrollbars always visible.

10. Font Family

Default System Font

Bootstrap uses the OS system font by default. No custom fonts are loaded.

Monospace (.font-monospace)

This text uses .font-monospace for code-like appearance.
