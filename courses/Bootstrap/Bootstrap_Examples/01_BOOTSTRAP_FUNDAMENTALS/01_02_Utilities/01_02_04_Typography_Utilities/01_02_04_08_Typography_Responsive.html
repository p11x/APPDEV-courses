Fluid Typography (CSS clamp)

CSS clamp() provides smooth font scaling between viewport sizes.

This text uses CSS clamp(1rem, 2.5vw, 2.5rem) for fluid scaling. It smoothly grows and shrinks with the viewport width.

Bold fluid text for headings

How clamp() Works
font-size: clamp(minimum, preferred, maximum);

/* Example: */
font-size: clamp(1rem, 2.5vw, 2.5rem);
/* Minimum: 1rem (16px) */
/* Preferred: 2.5% of viewport width */
/* Maximum: 2.5rem (40px) */
