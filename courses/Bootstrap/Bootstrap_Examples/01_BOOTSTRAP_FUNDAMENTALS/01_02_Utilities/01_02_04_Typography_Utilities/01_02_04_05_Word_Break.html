Line Height Utilities

Use .lh-{value} to control line height (vertical spacing between lines).

.lh-1 (line-height: 1)

This paragraph has a line height of 1. Notice how the lines are very close together. This is useful for headings or display text where you want tight vertical spacing between lines. The descenders and ascenders of adjacent lines may overlap.

.lh-sm (line-height: 1.25)

This paragraph has a small line height of 1.25. It provides slightly more breathing room than lh-1 while still maintaining a compact appearance. Good for subheadings or secondary text.

.lh-base (line-height: 1.5 - Default)

This paragraph has the base line height of 1.5. This is the recommended line height for body text as it provides good readability. The default body line-height in Bootstrap is 1.5, which is ideal for most content.

.lh-lg (line-height: 2)

This paragraph has a large line height of 2. Notice the generous vertical spacing between lines. This is useful for large display text or when you want a more airy, open feel to your text.
