Typography Accessibility in Bootstrap 5

1. Semantic Heading Hierarchy

Headings must follow a logical nesting order for screen reader navigation.

Correct Hierarchy

H1: Page Title

H2: Section

H3: Subsection

Content goes here. Screen readers can navigate by heading level.

Incorrect Hierarchy

H1: Page Title

H4: Skipped H2 and H3!

Problem: Heading levels should not skip from h1 to h4.

Best Practice: Use semantic HTML tags (h1-h6) for structure, then apply Bootstrap classes for visual styling. Never use a tag just for its size.

3. Text Spacing for Readability

WCAG 1.4.12 requires text to remain readable with adjusted spacing.

Tight Spacing (Hard to Read)

This text has tight line-height, negative letter-spacing, and negative word-spacing. It becomes very difficult to read. Many users with cognitive disabilities struggle with tight text.

Proper Spacing (Easy to Read)

This text uses Bootstrap's default line-height (1.5) with standard letter and word spacing. It is comfortable to read for everyone including users with dyslexia or low vision.

4. Color Contrast for Text

Text must have sufficient contrast against its background.

Fail

Muted text on light background. Contrast ratio ~2.9:1.

Fail

Yellow text on white background. Contrast ratio ~1.5:1.

Fail

Info text on white background. Contrast ratio ~2.3:1.

Pass AA

Dark text on light background. Contrast ~16:1.

Pass AA

White text on dark background. Contrast ~16:1.

Pass AA

White text on primary. Contrast ~4.6:1.

9. All-Caps Text Considerations

CSS-transformed uppercase may be read letter-by-letter by some screen readers.

Use Sparingly

ALL-CAPS PARAGRAPHS ARE HARD TO READ. THE UNIFORM HEIGHT OF LETTERS REDUCES THE DISTINCTIVE SHAPES THAT HELP WITH WORD RECOGNITION. RESERVE FOR SHORT LABELS ONLY.

Better Approach

Normal case paragraphs are much easier to read. The varying letter heights create distinctive word shapes that aid recognition and comprehension.

ALL-CAPS FOR SHORT LABELS ONLY

Typography Accessibility Checklist

  • ✓ Use semantic heading hierarchy (h1-h6 in order)
  • ✓ Body text minimum 16px (Bootstrap default)
  • ✓ Line-height at least 1.5 for body text
  • ✓ WCAG AA contrast ratio (4.5:1 normal, 3:1 large text)
  • ✓ Links distinguishable without color alone (underline)
  • ✓ Visible focus indicators on interactive elements
  • ✓ Don't use all-caps for long text passages
  • ✓ Support text resizing up to 200% without breaking layout
