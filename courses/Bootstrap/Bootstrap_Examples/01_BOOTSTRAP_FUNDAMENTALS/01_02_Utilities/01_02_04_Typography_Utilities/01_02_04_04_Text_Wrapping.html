Overflow Utilities

Control content overflow behavior with .overflow-* classes.

.overflow-auto: Scrollbars appear only when content overflows. This container has a max-height, so vertical scrolling is enabled when this text is long enough. Keep writing to see the scrollbar appear.
.overflow-hidden: Content is clipped without scrollbars. You cannot scroll to see this hidden text. It simply gets cut off at the boundary.
.overflow-visible: Content overflows and is visible outside the container boundary.
.overflow-scroll: Scrollbars are always visible regardless of content size. Even short text will show scrollbar tracks.

Directional Overflow

.overflow-x-auto: Horizontal scroll only. This text is very long and stays on one line forcing horizontal scrolling.
.overflow-y-auto: Vertical scroll only. Add more content to see vertical scrollbar appear.
