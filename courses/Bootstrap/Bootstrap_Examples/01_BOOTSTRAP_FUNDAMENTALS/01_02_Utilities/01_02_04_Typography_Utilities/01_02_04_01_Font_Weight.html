Bootstrap 5 Font Weight Utilities

Font Weight Classes

Use .fw-{weight} to set font weight. Values range from 100 (thin) to 900 (black).

.fw-bold (700) - Bold text for emphasis

.fw-bolder - Relative bolder than parent

.fw-semibold (600) - Semi-bold for subheadings

.fw-medium (500) - Medium weight

.fw-normal (400) - Normal/default weight

.fw-light (300) - Light weight text

.fw-lighter - Relative lighter than parent
