Spacing & Accessibility

Proper spacing is critical for accessibility. It affects readability, touch target sizes, cognitive load, and overall usability for all users.

1. Touch Target Sizes (WCAG 2.5.5)
WCAG 2.5.5 Target Size (Enhanced): Touch targets should be at least 44×44 CSS pixels. Bootstrap's default button padding meets this, but icon buttons and small links often do not.
Poor: Cramped Targets

Buttons with no spacing between them:

.p-1 + btn-sm = tiny targets. Users with motor impairments will struggle.

Good: Spaced Targets

Buttons with gap-2 and adequate padding:

.gap-2 .px-4 .py-2 = 44px+ targets with 8px spacing.

2. Readability: Line & Paragraph Spacing
Poor: Tight Text

This paragraph has line-height: 1.0 and no bottom margin. Lines of text are cramped together making it very difficult to read especially for users with dyslexia or visual impairments. The lack of vertical rhythm creates a wall of text.

Another paragraph with the same tight spacing. There is no visual separation between paragraphs either, making it impossible to tell where one thought ends and another begins.

Good: Comfortable Text

This paragraph uses Bootstrap's default line-height of 1.5 and .mb-3 for paragraph spacing. Lines are clearly separated, making the text easy to read for everyone.

The second paragraph is visually distinct from the first, thanks to the bottom margin. This creates clear content hierarchy and improves readability for dyslexic users and screen magnification users.

3. Form Field Spacing
Poor: Cramped Form

.mb-1 — too little space between fields. Users may accidentally tap the wrong input.

Good: Spaced Form

.mb-4 = 1.5rem between fields — clear, accessible separation.

5. Cognitive Accessibility: Grouping

Use the Gestalt principle of proximity: items that are related should be close together; unrelated items should be further apart.

6. WCAG Spacing Requirements Summary
WCAG Criterion Level Spacing Requirement Bootstrap Class
1.4.8 Visual Presentation AAA Line spacing ≥ 1.5× font size, paragraph spacing ≥ 1.5× line spacing Default line-height: 1.5 + .mb-3
1.4.12 Text Spacing AA No loss of content when line height is 1.5×, paragraph spacing 2×, letter spacing 0.12×, word spacing 0.16× Use rem units (Bootstrap default)
2.5.5 Target Size AAA Touch targets ≥ 44×44 CSS pixels .px-4 .py-2 on buttons
2.5.8 Target Size (Min) AA Touch targets ≥ 24×24 CSS pixels, or sufficient spacing .gap-2 between targets
