Bootstrap 5 - Responsive Spacing

Apply spacing at specific breakpoints using the format: .m{side}-{breakpoint}-{size} or .p{side}-{breakpoint}-{size}

1. Bootstrap 5 Breakpoints
Breakpoint Class Infix Min Width Typical Devices
XS none (always applied) 0 Portrait phones
SM sm 576px Landscape phones
MD md 768px Tablets
LG lg 992px Small desktops
XL xl 1200px Desktops
XXL xxl 1400px Large desktops
2. How Responsive Spacing Works
Key concept: Responsive classes apply from the specified breakpoint and up. For example, .mt-md-3 applies margin-top: 1rem on screens 768px and wider. On smaller screens, the spacing falls through to any lower breakpoint class or the default.
.mt-0 .mt-md-3 .mt-lg-5
No top margin on xs, 1rem on md+, 3rem on lg+
3. Responsive Padding
.p-2 .p-md-4 .p-lg-5
Tight padding on small screens, large padding on desktop
.px-2 .px-md-5 .py-3
Horizontal padding grows at md breakpoint, vertical stays constant
