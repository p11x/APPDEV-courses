Bootstrap 5 - Spacing Scale

Bootstrap uses a base spacer of 1rem (16px) and multiplies it to produce a consistent scale. Understanding this scale is key to building visually harmonious layouts.

1. Spacing Value Reference
Size Key CSS Variable Value (rem) Approx. Pixels Multiplied By
0 --bs-spacer-0 0 0px $spacer * 0
1 --bs-spacer-1 0.25rem 4px $spacer * 0.25
2 --bs-spacer-2 0.5rem 8px $spacer * 0.5
3 --bs-spacer-3 1rem 16px $spacer * 1
4 --bs-spacer-4 1.5rem 24px $spacer * 1.5
5 --bs-spacer-5 3rem 48px $spacer * 3
Note: The $spacer Sass variable defaults to 1rem. You can customize this in your own _variables.scss to shift the entire scale.

Note the non-linear jump: from 1.5rem to 3rem (size 4 to 5). Size 5 is intentionally large for hero sections and major separators.

3. The "auto" Size

The auto keyword is a special spacing value that computes to margin: auto.

.mx-auto — centers horizontally
.me-auto — pushed to start
.ms-auto — pushed to end
4. CSS Custom Properties (Variables)

Each spacing size is available as a CSS custom property:

  • --bs-spacer-0: 0
  • --bs-spacer-1: 0.25rem
  • --bs-spacer-2: 0.5rem
  • --bs-spacer-3: 1rem
  • --bs-spacer-4: 1.5rem
  • --bs-spacer-5: 3rem
This element has margin-top: var(--bs-spacer-4) applied via inline CSS. This is equivalent to .mt-4.
5. Spacing Size Guidelines
Size Best For Example Use
0 Removing default spacing Tight component internals, flush edges
1 Micro-spacing Between icon and text, badge padding
2 Small spacing Between form controls, list items
3 Default/standard spacing Card padding, section gaps, general layout
4 Medium-large spacing Between sections, form groups
5 Large/hero spacing Page header, major section separators
auto Automatic distribution Centering, flexbox space distribution
