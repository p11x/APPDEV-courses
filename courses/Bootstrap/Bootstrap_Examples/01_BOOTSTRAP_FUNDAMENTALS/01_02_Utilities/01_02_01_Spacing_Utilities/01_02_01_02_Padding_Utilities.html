Bootstrap 5 - Padding Utilities

Bootstrap provides padding utility classes to control internal spacing. The format is: .p{side}-{size}

Yellow background = padding area   Blue background = content area

1. All-Side Padding (.p-{size})
.p-0 — No padding (0)
.p-1 — padding: 0.25rem
.p-2 — padding: 0.5rem
.p-3 — padding: 1rem
.p-4 — padding: 1.5rem
.p-5 — padding: 3rem
2. Directional Padding
.pt-4 — padding-top: 1.5rem
.pb-4 — padding-bottom: 1.5rem
.ps-4 — padding-start (left): 1.5rem
.pe-4 — padding-end (right): 1.5rem
.px-4 — padding-left & padding-right: 1.5rem
.py-4 — padding-top & padding-bottom: 1.5rem
