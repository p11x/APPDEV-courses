1. Class Syntax & Naming
General Pattern

.{property}{side}-{breakpoint}-{size}

Part Values Description
property m, p Margin or Padding
side t, b, s, e, x, y, (blank) Top, Bottom, Start, End, X-axis (s+e), Y-axis (t+b), All sides
breakpoint sm, md, lg, xl, xxl, (blank) Apply from this breakpoint and up
size 0–5, auto, n1–n5 Spacing size (see scale below)
Side abbreviations — LTR context:
  • s (start) = left in LTR layouts
  • e (end) = right in LTR layouts
  • x = both start + end (horizontal)
  • y = both top + bottom (vertical)
Examples:
  • .mt-3 — margin-top: 1rem (all breakpoints)
  • .px-md-4 — horizontal padding 1.5rem starting at md
  • .mb-n2 — negative margin-bottom: -0.5rem
  • .mx-auto — auto horizontal margins (centering)

7. Responsive Prefixes
Prefix Breakpoint Min Width Example CSS Applied At
(none) xs 0+ .mt-3 All screen sizes
sm sm ≥576px .mt-sm-3 ≥576px
md md ≥768px .mt-md-3 ≥768px
lg lg ≥992px .mt-lg-3 ≥992px
xl xl ≥1200px .mt-xl-3 ≥1200px
xxl xxl ≥1400px .mt-xxl-3 ≥1400px
Mobile-first: Classes without a breakpoint prefix apply at all sizes. Responsive classes apply from their breakpoint and up. Use smaller values first, then override upward: .mb-2 .mb-md-4 .mb-lg-5

9. Sass Customization
Override the default $spacer and $spacers map:
// In your custom _variables.scss (import BEFORE Bootstrap's _variables.scss)

// Change the base spacer (default: 1rem)
$spacer: 1.25rem;

// Override the full spacers map
$spacers: (
  0: 0,
  1: .25rem,
  2: .5rem,
  3: 1rem,
  4: 1.5rem,
  5: 3rem,
  6: 4rem,    // Custom size 6
  7: 6rem     // Custom size 7
);

// After compiling, you'll have .m-6, .p-6, .mt-7, .gap-7, etc.
Extend the spacers map (add sizes without replacing defaults):
// Using map-merge to add to existing spacers
$spacers: map-merge(
  $spacers,
  (
    6: 4rem,
    7: 6rem,
    8: 8rem
  )
);
Disable specific spacing utilities:
// In your custom utilities setup, remove padding utilities from generation:
$utilities: (
  "padding": null,  // This removes all .p-* classes
  // margin and gap remain available
);
