Align Items & Align Self Utilities

Bootstrap provides utilities for align-items (container-level) and align-self (item-level) to control vertical alignment along the cross axis.

1. align-items-start

Aligns all flex items to the top of the cross axis.

2. align-items-center

Centers all items along the cross axis — the most common vertical alignment.

4. align-items-stretch

Default behavior. Items stretch to fill the full height of the container along the cross axis.

Auto

5. align-items-baseline

Aligns items so their text baselines match, regardless of element heights. Useful when items have different font sizes.
