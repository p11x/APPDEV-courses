Flex Order Utilities

Bootstrap's order-* utilities control the visual order of flex items without changing the HTML source order. This is critical for accessibility since screen readers always follow DOM order.

1. Default Order (No Utilities)

Items display in their natural HTML source order.

First (DOM)
Second (DOM)
Third (DOM)

2. Numeric order-* Classes

Use order-1 through order-5 to set specific visual positions. order-0 resets to the default.

Red — order-2
Blue — order-3
Green — order-1
Green appears first visually because it has order-1, even though it's third in the DOM.

3. order-first & order-last

order-first sets order: -1 and order-last sets order: 6, pushing items to the extreme ends.

Red (DOM 1)
Blue — order-last
Green (DOM 3)
Yellow (DOM 4)
Purple — order-first

4. Responsive Order

Combine breakpoint suffixes to reorder at different screen sizes: order-md-1, order-lg-3, etc.

Sidebar
order-md-3 order-lg-2
Main
order-md-1 order-lg-3
Aside
order-md-2 order-lg-1

5. Real-World: Mobile-First Layout Reorder

A common pattern: content comes first on mobile (better for users), sidebar moves up on desktop.

Main Content

This column comes first on mobile (order-1) but shifts to second position on medium screens and up (order-md-2).

Sidebar

Appears second on mobile but first on desktop.

6. Order with Column Direction

Order also affects vertical stacking in flex-column layouts.

Header — order-3
Nav — order-1
Content — order-2
