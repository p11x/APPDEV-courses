Justify Content Utilities

Bootstrap's justify-content-* utilities control how flex items are distributed along the main axis (horizontal by default). These map directly to CSS justify-content.

1. justify-content-start

Aligns all items to the start (left in LTR) of the container.

4. justify-content-between

Distributes equal space around each item. Items at the edges get half the space compared to gaps between items.
