Responsive Flex Utilities

Every flex utility in Bootstrap supports responsive breakpoint suffixes. This enables mobile-first designs where flex behavior changes at sm (576px), md (768px), lg (992px), xl (1200px), and xxl (1400px).

1. Responsive d-* Display

d-flex applies flex at all sizes. Add d-md-block to switch to block layout on medium screens and up.
