Flex Wrap Utilities

The CSS flex-wrap property determines whether flex items are forced onto a single line or can wrap onto multiple lines. Bootstrap provides three utility classes: flex-nowrap, flex-wrap, and flex-wrap-reverse.

1. flex-nowrap (Default)

By default, d-flex uses flex-wrap: nowrap. All items stay on a single line. If the total width exceeds the container, items will overflow or shrink.

3. flex-wrap-reverse

flex-wrap-reverse causes items to wrap upward onto the previous line instead of downward. The flex direction axis remains the same, but the cross-axis is reversed.
