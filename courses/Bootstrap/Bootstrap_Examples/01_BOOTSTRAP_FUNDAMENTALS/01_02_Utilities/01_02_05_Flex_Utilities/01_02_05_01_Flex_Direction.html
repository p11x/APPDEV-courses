Flex Direction Utilities

Bootstrap 5 provides utility classes that map directly to the CSS flex-direction property. These classes control the primary axis along which flex items are laid out inside a flex container.

1. Default — d-flex (row)

Without any direction modifier, d-flex applies flex-direction: row. Items flow left-to-right.

2. flex-row

The flex-row class explicitly sets flex-direction: row. It produces the same layout as the default but makes intent clear in the markup.

3. flex-row-reverse

flex-row-reverse flips the axis so items are laid out right-to-left. This is useful for mirrored layouts without changing HTML order.

4. flex-column

flex-column sets flex-direction: column. Items stack vertically, each taking its own row.
