Float Integration

This guide demonstrates how to integrate Bootstrap float utilities with other Bootstrap systems: the grid, flexbox utilities, spacing classes, and components. Knowing when and how to combine these tools is key to efficient layout design.

1. Floats Inside Grid Columns

Article Title

The image is floated left inside a Bootstrap grid column. The grid handles responsive column widths while float handles text wrapping within the column. This separation of concerns keeps layouts maintainable.

Sidebar

Grid controls this column's width. Floats are not used here — just regular flow content.

2. Float + Flex Hybrid Layout

Flex handles the outer layout. Floats handle text wrapping inside flex children.

Feature Article

The outer container uses d-flex for responsive layout. Inside, the image is floated right with float-end for magazine-style text wrapping.

Sidebar

This column has a fixed flex basis. No floats needed.

3. Float + Spacing Utilities

Bootstrap's spacing utilities (m-*, p-*) work perfectly with floats to control gap between floated elements and surrounding content.

me-3 (margin-end)
ms-3 (margin-start)

The left box uses me-3 for right margin, and the right box uses ms-3 for left margin. These spacing utilities create the gap between the floated elements and the wrapping text.

4. Float + Display Utilities

Combining float with display utilities (d-none, d-block) creates complex responsive behaviors.

Visible md+ & floated

This box is hidden on mobile (d-none) and visible as a floated element on medium screens and up (d-md-block float-start). This pattern is useful for sidebar elements that only appear on larger screens.

When to Use Float vs. Flex/Grid

Use CaseUseWhy
Text wrapping around imagesFloatOriginal use case; simple and effective
Pull quotes / editorial layoutsFloatNatural text flow around elements
Full page layoutsGridTwo-dimensional control
Navigation barsFlexAxis-based alignment
Card gridsGridResponsive columns/rows
Centering contentFlexjustify-content + align-items
Sidebar + contentFlex or GridFlexible sizing and ordering
Badge/tag in a cornerFloat or PositionBoth work; float simpler for inline use
