Float Basics

Bootstrap provides utility classes for CSS float. While flexbox and grid are preferred for modern layouts, floats remain useful for wrapping text around images and legacy browser support.

1. What Is CSS Float?

The float property removes an element from the normal document flow and positions it to the left or right of its container. Other content flows around the floated element.

Key concept: Floated elements are taken out of normal flow. The surrounding inline content wraps around them, but the parent container may collapse if it doesn't account for the float.

2. float-start

float-start applies float: left. The element floats to the left edge, and surrounding text wraps along its right side.

5. Classic: Image with Wrapping Text

Floats were originally designed for exactly this scenario — wrapping text around images in a magazine-style layout. The image is floated to the left, and paragraph text flows naturally around it. This remains a valid and lightweight use case for float utilities even in modern web development.

Additional paragraphs continue wrapping until they clear below the floated element's bottom edge. This behavior is automatic and requires no JavaScript.
