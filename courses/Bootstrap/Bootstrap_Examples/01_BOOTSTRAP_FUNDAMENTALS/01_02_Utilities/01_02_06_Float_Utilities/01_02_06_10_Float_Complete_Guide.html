1. Float Basics

Three core classes control float direction.

float-start

float-start

float-end

float-end

Element stays in normal flow. No wrapping.

6. Float vs. Flexbox vs. Grid

Rule of thumb: Use floats for text wrapping. Use flex for 1D layouts. Use grid for 2D layouts. Use Bootstrap's grid system for responsive page structure.

7. Common Gotchas

Gotcha 1: Float inside flex items is ignored per CSS spec. <div class="d-flex"><div class="float-start">...</div></div> — the float has no effect.
Gotcha 2: Margin collapse doesn't work between a floated element and its parent. Use padding on the parent instead.
Gotcha 3: overflow: hidden or overflow: auto on the parent also contains floats (creates BFC), but may clip content or add scrollbars.
Gotcha 4: Floated elements don't stretch their parent's height. Always use clearfix or BFC.
