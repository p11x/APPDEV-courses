Screen Reader Utilities

Bootstrap provides classes specifically designed for screen reader accessibility.

1. .visually-hidden

Hides content visually but keeps it accessible to assistive technologies.

2. .visually-hidden-focusable

Hidden until focused via keyboard. Becomes visible when the user tabs to it.

Example: Tab to the link below to see it appear.

Skip to main content

The link is invisible until you press Tab to focus it.

<!-- Skip navigation pattern -->
<a class="visually-hidden-focusable" href="#main-content">
    Skip to main content
</a>

<nav>
    <!-- navigation items -->
</nav>

<main id="main-content">
    <!-- main content -->
</main>

5. aria-hidden vs .visually-hidden

aria-hidden="true"

Makes element invisible to screen readers but keeps it visually visible.

Use for:

  • Decorative images
  • Icons that duplicate text
  • Pure visual flourishes
*

This star icon is decorative, hidden from screen readers.

.visually-hidden

Makes element visually hidden but keeps it accessible to screen readers.

Use for:

  • Labels for icon buttons
  • Skip navigation links
  • Additional context text

The "+" is decorative. The hidden text provides the label.

6. Live Region Pattern

Combine .visually-hidden with aria-live for screen reader announcements.

Changes saved successfully!
<!-- Screen reader announcement (hidden) -->
<div class="visually-hidden" aria-live="polite" aria-atomic="true" id="srStatus"></div>

<!-- Visual notification (shown) -->
<div class="alert d-none" id="visualStatus">Saved!</div>

<script>
document.getElementById('srStatus').textContent = 'Changes saved';
document.getElementById('visualStatus').classList.remove('d-none');
</script>
