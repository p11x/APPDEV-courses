Visibility Utilities - Basics

Control the visibility of elements using Bootstrap's .visible and .invisible utility classes.

Basic Visibility Classes

Bootstrap provides two classes for controlling element visibility.

Class CSS Output Effect
.visible visibility: visible !important Element is visible (default behavior)
.invisible visibility: hidden !important Element is hidden but still takes up space

When to Use Each

Use CaseRecommended ClassReason
Toggle visibility without layout shift .invisible Preserves space, preventing layout reflow
Completely hide an element .d-none Removes from layout, frees up space
Placeholder for async content .invisible Reserves space until content loads
Responsive show/hide .d-{breakpoint}-none Responsive display utilities
Screen reader only content .visually-hidden Hidden visually but accessible to screen readers

CSS Property Reference

/* Bootstrap's visibility utilities */

.visible {
    visibility: visible !important;
}

.invisible {
    visibility: hidden !important;
}

/* Related Bootstrap classes */

.d-none {
    display: none !important;
}

.visually-hidden,
.visually-hidden-focusable:not(:focus):not(:focus-within) {
    position: absolute !important;
    width: 1px !important;
    height: 1px !important;
    padding: 0 !important;
    margin: -1px !important;
    overflow: hidden !important;
    clip: rect(0, 0, 0, 0) !important;
    white-space: nowrap !important;
    border: 0 !important;
}
