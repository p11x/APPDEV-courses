Responsive Visibility Utilities

Show or hide elements at specific breakpoints using Bootstrap's responsive display utilities.

Responsive Display Classes

Use .d-{breakpoint}-{value} to control visibility at different screen sizes.

Class Effect
.d-noneHidden on all screen sizes
.d-none .d-sm-blockHidden on xs, visible on sm+
.d-none .d-md-blockHidden on xs/sm, visible on md+
.d-none .d-lg-blockHidden on xs/sm/md, visible on lg+
.d-none .d-xl-blockHidden on xs/sm/md/lg, visible on xl+
.d-none .d-xxl-blockHidden on xs-xl, visible on xxl+
.d-sm-noneVisible on xs only, hidden on sm+
.d-md-noneVisible on xs/sm only, hidden on md+

Mobile-Only Content

Content that appears only on mobile devices (below sm breakpoint).

Mobile Only: This content is only visible on screens smaller than 576px.
Class: .d-block .d-sm-none
Tablet & Desktop: This content is visible on screens 576px and wider.
Class: .d-none .d-sm-block

Desktop-Only Content

Content designed for larger screens that is hidden on mobile and tablet.

Desktop-Only Feature

This section only appears on screens 992px and wider. It might contain complex data tables, side-by-side comparisons, or interactive charts that don't work well on small screens.

Class: .d-none .d-lg-block

Mobile/Tablet View: On smaller screens, show a simplified version or a link to the full desktop experience.
Class: .d-block .d-lg-none
