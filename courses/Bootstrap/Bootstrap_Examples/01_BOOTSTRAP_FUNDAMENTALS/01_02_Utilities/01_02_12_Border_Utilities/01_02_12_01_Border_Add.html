Adding Borders with Bootstrap 5 Utility Classes

Border Utility Classes

Bootstrap's .border utility adds a 1px solid light-gray border to elements. You can target specific sides using .border-top, .border-end, .border-bottom, and .border-start.

Border on All Sides

.border

All sides

.border .border-primary

Colored border

.border .border-success

Success color

.border .border-danger

Danger color

Border on Individual Sides

.border-top

Top only

.border-end

Right only

.border-bottom

Bottom only

.border-start

Left only

Combining Border Sides

Combine multiple border-side classes to create custom border configurations.

.border-top .border-bottom

Horizontal borders

.border-start .border-end

Vertical borders

.border-top .border-start .border-end

Three sides
