Bootstrap 5 Position Utilities - Complete Reference

Comprehensive reference of all Bootstrap 5 position utility classes.

1. Position Type Classes

Class CSS Property Description Demo
.position-static position: static Default positioning in document flow Default
.position-relative position: relative Relative to normal position, preserves space Common
.position-absolute position: absolute Relative to nearest positioned ancestor Common
.position-fixed position: fixed Relative to viewport, doesn't scroll Navbars
.position-sticky position: sticky Toggles between relative and fixed on scroll Headers

2. Position Value Classes

Category Classes CSS Property
Top .top-0, .top-50, .top-100 top: 0 | 50% | 100%
Bottom .bottom-0, .bottom-50, .bottom-100 bottom: 0 | 50% | 100%
Start .start-0, .start-50, .start-100 inset-inline-start: 0 | 50% | 100%
End .end-0, .end-50, .end-100 inset-inline-end: 0 | 50% | 100%
Translate .translate-middle, .translate-middle-x, .translate-middle-y transform: translate(-50%, -50%)

  • .top-{0|50|100}

Translate
  • .translate-middle
  • .translate-middle-x
  • .translate-middle-y
  • .translate-middle-{bp}
Key Rules
  1. position-absolute requires a position-relative ancestor
  2. position-fixed is relative to the viewport
  3. position-sticky needs a threshold (top/bottom/left/right)
  4. Use translate-middle for perfect centering with position
  5. Responsive format: .position-{breakpoint}-{value}
