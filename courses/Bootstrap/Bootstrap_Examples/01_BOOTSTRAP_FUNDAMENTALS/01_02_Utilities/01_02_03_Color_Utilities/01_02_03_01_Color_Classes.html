Bootstrap 5 Color Classes

Contextual Text Colors

Use .text-{color} classes to apply contextual colors to text.

.text-primary - Primary brand color

.text-secondary - Secondary color

.text-success - Success state color

.text-danger - Danger/error state color

.text-warning - Warning state color

.text-info - Informational color

.text-light - Light color for dark backgrounds

.text-dark - Dark color

.text-body - Default body text color

.text-muted - Muted/de-emphasized text

.text-white - White text on dark background

.text-black-50 - Semi-transparent black

.text-white-50 - Semi-transparent white
