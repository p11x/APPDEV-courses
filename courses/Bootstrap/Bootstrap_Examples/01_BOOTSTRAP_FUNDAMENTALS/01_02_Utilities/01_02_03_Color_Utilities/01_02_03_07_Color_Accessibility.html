Color Accessibility in Bootstrap 5

WCAG Contrast Ratios

Bootstrap's default color pairings meet WCAG AA standards. Below demonstrates contrast levels.

Pass AA
Dark on Light

.text-dark on .bg-light - Contrast ratio ~16:1. Excellent readability.

Pass AA
Light on Dark

.text-white on .bg-dark - Contrast ratio ~16:1. Excellent readability.

Pass AA
White on Primary

.text-bg-primary - Bootstrap auto-sets white text. Contrast ~4.6:1.

Pass AA
Dark on Warning

.text-bg-warning - Bootstrap uses dark text on yellow. Contrast ~4.8:1.

Pass AA
Dark on Info

.text-bg-info - Bootstrap uses dark text on cyan. Contrast ~3.1:1 for large text.

Fail AA
Muted on Light

.text-muted on .bg-light - Contrast ratio ~2.9:1. Below minimum for normal text.

Respecting User Preferences

Bootstrap respects prefers-reduced-motion and prefers-contrast.

System Settings: If your OS has "Reduce motion" enabled, Bootstrap's transitions are minimized. If "High contrast" is enabled, focus rings and borders become more prominent.
