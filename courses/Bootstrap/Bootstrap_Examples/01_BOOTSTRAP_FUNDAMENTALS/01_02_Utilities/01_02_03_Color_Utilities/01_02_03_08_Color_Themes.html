Color Scheme Meta Tag

The <meta name="color-scheme"> tag signals theme preference to the browser.

Browser Color Scheme

Add this meta tag to your <head> to support both light and dark:

<meta name="color-scheme" content="light dark">

This tells the browser that your page supports both color schemes, allowing OS-level dark mode preferences to be respected.
