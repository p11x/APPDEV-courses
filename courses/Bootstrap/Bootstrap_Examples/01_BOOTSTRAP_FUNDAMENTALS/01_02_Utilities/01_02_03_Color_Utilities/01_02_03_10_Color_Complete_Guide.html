8. CSS Custom Properties Reference

Bootstrap 5 exposes CSS custom properties for advanced theming.

Property Purpose Example
--bs-primary Primary color value Colored text
--bs-primary-rgb Primary RGB values for rgba() rgba(var(--bs-primary-rgb), 0.5)
--bs-text-opacity Text color opacity style="--bs-text-opacity: 0.5"
--bs-bg-opacity Background color opacity style="--bs-bg-opacity: 0.75"
--bs-border-opacity Border color opacity style="--bs-border-opacity: 0.25"
--bs-body-color Default body text color color: var(--bs-body-color)
--bs-body-bg Default body background background: var(--bs-body-bg)

9. Responsive Color Usage

Color utilities can be applied conditionally at different breakpoints.

Note: Bootstrap color utilities do not have built-in responsive variants by default, but you can combine them with responsive display utilities or use custom CSS media queries.
