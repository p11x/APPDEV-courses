Bootstrap 5 Gradient Utilities

Basic Gradient Backgrounds

Combine .bg-gradient with .bg-{color} for gradient effects.

.bg-primary .bg-gradient - Primary with gradient overlay
.bg-secondary .bg-gradient - Secondary with gradient overlay
.bg-success .bg-gradient - Success with gradient overlay
.bg-danger .bg-gradient - Danger with gradient overlay
.bg-warning .bg-gradient - Warning with gradient overlay
.bg-info .bg-gradient - Info with gradient overlay
.bg-dark .bg-gradient - Dark with gradient overlay

Custom CSS Gradients

Use inline styles or custom CSS for more complex gradients.

Linear Gradient - Primary to Secondary (135deg)
Linear Gradient - Success to Info (90deg)
Linear Gradient - Danger to Warning (180deg)
Multi-color Gradient - Blue to Purple to Pink to Red
Radial Gradient - Primary center, dark edge
Angled Gradient - Three colors at 45 degrees
