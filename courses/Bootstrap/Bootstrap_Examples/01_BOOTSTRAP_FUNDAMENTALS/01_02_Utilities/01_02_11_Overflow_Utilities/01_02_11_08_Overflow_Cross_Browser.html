Cross-Browser Overflow

Consistent Scrollbar Behavior Across Browsers

Browser Scrollbar Differences

  • Chrome/Safari/Edge (WebKit): Support ::-webkit-scrollbar pseudo-elements for custom styling
  • Firefox: Uses scrollbar-width and scrollbar-color properties
  • IE11: Uses -ms-overflow-style property
  • iOS Safari: Uses -webkit-overflow-scrolling: touch for momentum scrolling

Default Browser Scrollbars

Each browser renders scrollbars differently by default. These differences are normal and expected.

iOS Momentum Scrolling

On iOS, use -webkit-overflow-scrolling: touch for native-feeling momentum scrolling. In modern iOS (13+), overflow: auto already provides momentum scrolling by default. The property is mainly needed for older iOS versions.
