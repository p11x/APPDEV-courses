Understanding Bootstrap 5 Overflow Utility Classes

What is Overflow?

Overflow utilities in Bootstrap 5 control how content behaves when it exceeds the boundaries of its parent container. These utilities map directly to the CSS overflow property and its axis-specific variants (overflow-x and overflow-y).

overflow-auto

Auto Overflow Content

This content is taller than its container (300px vs 150px).

The .overflow-auto class adds a vertical scrollbar automatically.

Scrollbars only appear when needed - this is the recommended approach.

Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

<div class="overflow-auto">...</div>

overflow-hidden

Hidden Overflow Content

This content exceeds the container but is clipped.

You cannot scroll to see the rest of this content.

Use this when you intentionally want to hide excess content.

This text will be completely cut off and invisible.

<div class="overflow-hidden">...</div>

overflow-visible

Visible Overflow Content

Content spills out of the container boundaries.

This is the default CSS overflow behavior.

Notice how the gradient extends beyond the border.

<div class="overflow-visible">...</div>

overflow-scroll

Scroll Overflow Content

Scrollbars are always visible with this class.

Even short content shows scrollbar tracks.

Use sparingly - auto is usually better.

<div class="overflow-scroll">...</div>

Quick Reference

Bootstrap Class CSS Equivalent Description
.overflow-auto overflow: auto Scrollbars appear only when content overflows
.overflow-hidden overflow: hidden Content is clipped, no scrollbars
.overflow-visible overflow: visible Content spills outside container (default)
.overflow-scroll overflow: scroll Scrollbars always visible
.overflow-x-auto overflow-x: auto Horizontal scrollbar when needed
.overflow-y-auto overflow-y: auto Vertical scrollbar when needed
.overflow-x-hidden overflow-x: hidden Horizontal overflow clipped
.overflow-y-hidden overflow-y: hidden Vertical overflow clipped
.overflow-x-scroll overflow-x: scroll Horizontal scrollbar always visible
.overflow-y-scroll overflow-y: scroll Vertical scrollbar always visible
