Overflow Auto

Conditional Scrollbar Behavior in Bootstrap 5

How overflow-auto Works

The .overflow-auto class applies overflow: auto to an element. The browser will add scrollbars only when content exceeds the container's dimensions. This is the most user-friendly overflow option because scrollbars don't appear unnecessarily.
