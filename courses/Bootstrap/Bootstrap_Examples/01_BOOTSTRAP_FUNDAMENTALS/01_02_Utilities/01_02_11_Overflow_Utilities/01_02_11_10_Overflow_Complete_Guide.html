All Overflow Classes

Common Patterns

Pattern 1: Scrollable Card
<div class="card">
  <div class="card-header">Title</div>
  <div class="card-body overflow-auto"
       style="height:200px">
    Content scrolls
  </div>
  <div class="card-footer">Footer</div>
</div>
Pattern 2: Responsive Table
<div class="overflow-hidden 
     overflow-md-auto">
  <table class="table"
         style="min-width:800px">
    Wide table scrolls on md+
  </table>
</div>
Pattern 3: Image Crop
<div class="overflow-hidden"
     style="width:200px;height:200px">
  <img src="large.jpg" 
       class="img-fluid">
</div>
Pattern 4: Scrollable Modal
<div class="modal-body 
     overflow-y-auto"
     style="max-height:400px">
  Long modal content scrolls
</div>

Which Overflow to Use?

ScenarioRecommendedReason
Scrollable content areaoverflow-autoScrollbars only when needed
Image croppingoverflow-hiddenCleanly clips excess
Wide data table on mobileoverflow-x-autoHorizontal scroll for tables
Fixed-height card bodyoverflow-y-autoVertical scroll, no horizontal
Contain floated elementsoverflow-hiddenCreates BFC
Custom scrollbar implementationoverflow-hiddenHide native, use JS
