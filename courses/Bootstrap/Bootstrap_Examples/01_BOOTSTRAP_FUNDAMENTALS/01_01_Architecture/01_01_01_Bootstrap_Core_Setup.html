HTML5 Boilerplate

DOCTYPE Declaration

The <!DOCTYPE html> declaration tells the browser to render the page in HTML5 standards mode. Without it, browsers may use "quirks mode" which can cause rendering inconsistencies.

Viewport Meta Tag

The <meta name="viewport"> tag is essential for responsive design. Bootstrap's grid system, responsive utilities, and breakpoints all depend on this tag being present.

Without this tag: Mobile devices will render the page at desktop width (~980px) and zoom out, making everything tiny.
CDN Links

Bootstrap's CSS and JavaScript are loaded from a Content Delivery Network (CDN). The CDN serves files from servers geographically close to the user, reducing load times.

Setup Checklist
  • DOCTYPE declaration
  • HTML lang attribute
  • Charset meta tag
  • Viewport meta tag
  • Bootstrap CSS CDN
  • Bootstrap JS CDN
  • Custom CSS (after Bootstrap)
