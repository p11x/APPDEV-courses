3. Event Delegation

Attach one listener to parent instead of one per input. More efficient for large/dynamic forms.

4. Large Select Optimization

For large datasets, filter options with search rather than rendering thousands of <option> elements.

Only matching options are rendered. Reduces DOM size from 250+ to ~10 elements.
