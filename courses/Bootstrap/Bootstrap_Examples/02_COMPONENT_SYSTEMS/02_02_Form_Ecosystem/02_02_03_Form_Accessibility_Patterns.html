6. Keyboard Navigation

Avoid positive tabindex values. They break natural reading order.
Skip to main content
visually-hidden-focusable is visible only when focused (skip link pattern).

Keyboard tips: Use Tab to move forward, Shift+Tab backward, Space to toggle checkboxes, Enter to submit, Escape to close popups.
