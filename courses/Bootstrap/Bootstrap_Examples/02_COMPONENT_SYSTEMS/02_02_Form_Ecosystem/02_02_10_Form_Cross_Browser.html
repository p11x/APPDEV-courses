7. Autofill Style Override

Browsers apply their own background colors to autofilled inputs. Override with CSS.

Browsers add yellow/blue background for autofilled fields. Override with :-webkit-autofill CSS.
/* Override autofill background color */
input:-webkit-autofill,
input:-webkit-autofill:hover,
input:-webkit-autofill:focus {
    -webkit-box-shadow: 0 0 0 1000px white inset;
    transition: background-color 5000s ease-in-out 0s;
}
