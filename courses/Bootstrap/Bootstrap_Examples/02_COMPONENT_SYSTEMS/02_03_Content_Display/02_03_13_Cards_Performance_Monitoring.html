5. Performance Best Practices Summary

Images
  • Use loading="lazy" for below-fold images
  • Set explicit width/height or aspect-ratio
  • Use srcset for responsive images
  • Use modern formats (WebP, AVIF) with fallbacks
  • Use decoding="async"
Layout
  • Avoid layout shifts (CLS) with reserved space
  • Use skeleton screens for perceived performance
  • Minimize reflows on card interactions
  • Use content-visibility: auto for off-screen cards
  • Keep DOM depth shallow within cards
JavaScript
  • Defer non-critical card interactivity
  • Use IntersectionObserver for lazy loading
  • Debounce scroll/resize event handlers
  • Avoid layout thrashing (read then write DOM)
  • Use requestAnimationFrame for animations
