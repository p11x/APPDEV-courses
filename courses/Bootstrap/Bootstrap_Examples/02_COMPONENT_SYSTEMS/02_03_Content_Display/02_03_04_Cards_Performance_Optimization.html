Cards Performance Optimization

1. Lazy Loading Images

Native Lazy Loading

The loading="lazy" attribute is the simplest way to implement lazy loading. Supported in all modern browsers.

Placeholder Pattern

Using a placeholder container with aspect-ratio prevents layout shift when images load.

Error Fallback

Use onerror to show a fallback SVG when images fail to load. Prevents broken image icons.

4. Responsive Images with srcset

Responsive with picture/source

The <picture> element serves different image sizes based on media queries, reducing bandwidth on mobile.

Responsive with srcset/sizes

srcset with sizes lets the browser choose the optimal image based on viewport and device pixel ratio.

5. CSS Performance Tips

Async Decoding

decoding="async" lets the browser decode images off the main thread, preventing render blocking.

content-visibility

The content-visibility: auto CSS property skips rendering of off-screen content entirely. 

.card {
  content-visibility: auto;
  contain-intrinsic-size: 0 400px;
}
will-change

will-change hints the browser about upcoming changes for GPU-accelerated animations. 

.card:hover {
  will-change: transform;
  transform: translateY(-4px);
}
