6. Fullscreen Modals

Use .modal-fullscreen or responsive variants for edge-to-edge modals.

Fullscreen Modal

This modal takes up the entire viewport.

Use for immersive experiences, full-screen editors, or image viewers.

Responsive Fullscreen

This modal is fullscreen below the sm breakpoint (576px).

At larger viewports, it displays as a regular centered modal.

Available variants: .modal-fullscreen-sm-down, .modal-fullscreen-md-down, .modal-fullscreen-lg-down, .modal-fullscreen-xl-down, .modal-fullscreen-xxl-down

7. Static Backdrop

Prevent closing the modal by clicking outside or pressing Escape.

Static Backdrop

Try clicking outside this modal - it won't close.

Try pressing Escape - it won't close.

Only the close button or explicit dismiss actions work.

Useful for modals that require user action, like confirmations or mandatory forms.
