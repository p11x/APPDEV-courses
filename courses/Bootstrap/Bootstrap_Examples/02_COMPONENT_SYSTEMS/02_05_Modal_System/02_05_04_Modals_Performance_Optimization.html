5. Event Handler Optimization

Use event delegation instead of individual listeners for modal triggers.

Anti-pattern (Many Listeners)
// Bad: One listener per button
document.querySelectorAll('.btn').forEach(btn => {
    btn.addEventListener('click', handler);
});
Best Practice (Event Delegation)
// Good: One listener on parent container
document.addEventListener('click', function(e) {
    if (e.target.matches('[data-modal-trigger]')) {
        handleModalTrigger(e.target);
    }
});
