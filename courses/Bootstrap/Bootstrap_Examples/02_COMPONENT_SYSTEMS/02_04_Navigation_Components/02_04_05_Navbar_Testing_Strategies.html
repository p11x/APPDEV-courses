Navbar Testing Strategies

5.1 Structural / Unit Tests

Verify DOM structure, classes, and attributes:

Test Case Selector / Assertion Purpose
Navbar exists document.querySelector('.navbar') Confirms navbar renders
Brand link present .navbar-brand exists Brand element rendered
Toggler has ARIA attributes aria-controls, aria-expanded Accessibility compliance
Active link marked .nav-link.active + aria-current Current page indication
Dropdown has menu .dropdown-menu exists after toggle Dropdown structure
Interactive Test Runner

5.2 Behavioral / Interaction Tests

Test user interactions like clicking, toggling, and keyboard navigation:

Test Case Action Expected Result
Toggler opens collapse Click .navbar-toggler .show added to .navbar-collapse
Toggler closes collapse Click toggler again .show removed from .navbar-collapse
Dropdown opens on click Click .dropdown-toggle .show added to .dropdown-menu
Escape closes dropdown Press Escape key .show removed from .dropdown-menu
Search form submits Fill input, click submit Form submit event fires
Behavioral Test Runner

5.3 Responsive / Viewport Tests

Test navbar behavior across different viewport sizes:

Viewport Width Expected Behavior
Mobile < 992px Toggler visible, collapse hidden
Tablet 768px - 991px Toggler visible (for expand-lg)
Desktop ≥ 992px All nav items visible inline
Responsive Test Runner

5.4 Accessibility Tests

Verify ARIA attributes, keyboard navigation, and screen reader support:

Test Case Assertion
Nav has aria-label nav[aria-label] exists
Toggler has aria-controls button[aria-controls] references valid ID
Toggler has aria-expanded button[aria-expanded] exists
Active link has aria-current .active[aria-current="page"] exists
Dropdown toggle has aria-haspopup dropdown-toggle[aria-haspopup] exists
All links have href Every nav-link[href] is defined

5.5 E2E Test Examples (Cypress / Playwright)

Cypress Example:
// cypress/e2e/navbar.cy.js
describe('Navbar Tests', () => {
    beforeEach(() => {
        cy.visit('/');
    });

    it('renders the navbar', () => {
        cy.get('.navbar').should('be.visible');
        cy.get('.navbar-brand').should('contain', 'TestBrand');
    });

    it('toggles mobile menu', () => {
        cy.viewport(375, 667); // iPhone SE
        cy.get('.navbar-toggler').click();
        cy.get('#testNav').should('have.class', 'show');
        cy.get('.navbar-toggler').click();
        cy.get('#testNav').should('not.have.class', 'show');
    });

    it('opens dropdown menu', () => {
        cy.get('#navServicesToggle').click();
        cy.get('#navServicesMenu').should('have.class', 'show');
        cy.get('#serviceWeb').should('be.visible');
    });

    it('navigates on link click', () => {
        cy.get('#navAbout').click();
        cy.url().should('include', '/about');
    });

    it('has correct aria attributes', () => {
        cy.get('nav[aria-label]').should('exist');
        cy.get('.navbar-toggler').should('have.attr', 'aria-controls');
        cy.get('.nav-link.active').should('have.attr', 'aria-current', 'page');
    });
});
Playwright Example:
// tests/navbar.spec.js
import { test, expect } from '@playwright/test';

test.describe('Navbar', () => {
    test.beforeEach(async ({ page }) => {
        await page.goto('/');
    });

    test('renders navbar with brand', async ({ page }) => {
        await expect(page.locator('.navbar')).toBeVisible();
        await expect(page.locator('.navbar-brand')).toHaveText('TestBrand');
    });

    test('toggler opens on mobile', async ({ page }) => {
        await page.setViewportSize({ width: 375, height: 667 });
        await page.click('.navbar-toggler');
        await expect(page.locator('#testNav')).toHaveClass(/show/);
    });

    test('dropdown shows menu items', async ({ page }) => {
        await page.click('#navServicesToggle');
        await expect(page.locator('#navServicesMenu')).toHaveClass(/show/);
        await expect(page.locator('#serviceWeb')).toBeVisible();
    });

    test('keyboard navigation works', async ({ page }) => {
        await page.keyboard.press('Tab');
        await expect(page.locator('.navbar-brand')).toBeFocused();
        await page.keyboard.press('Tab');
        await expect(page.locator('#navHome')).toBeFocused();
    });
});
