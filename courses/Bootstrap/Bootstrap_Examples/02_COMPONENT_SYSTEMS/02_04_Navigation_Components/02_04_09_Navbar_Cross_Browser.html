9.3 Common Issues & Fixes

Safari Flexbox Gap (pre-14.1):
/* Flexbox gap not supported in older Safari */
/* Use margin fallback instead */
.navbar-nav {
    display: flex;
    /* gap: 1rem; Not supported in Safari <14.1 */
}
.navbar-nav .nav-item {
    margin-right: 1rem; /* Fallback */
}
@supports (gap: 1rem) {
    .navbar-nav { gap: 1rem; }
    .navbar-nav .nav-item { margin-right: 0; }
}
Safari 100vh Issue (Mobile):
/* Safari mobile includes address bar in 100vh */
/* Use dvh (dynamic viewport height) with fallback */
.full-screen-nav {
    height: 100vh;           /* Fallback */
    height: 100dvh;          /* Modern browsers */
}
/* Or use JavaScript */
const setVh = () => {
    document.documentElement.style.setProperty(
        '--vh', `${window.innerHeight * 0.01}px`
    );
};
window.addEventListener('resize', setVh);
Firefox Focus Ring:
/* Firefox adds its own focus ring */
/* Override with consistent custom focus */
.navbar .nav-link:focus {
    outline: 2px solid #0d6efd;
    outline-offset: 2px;
    box-shadow: none; /* Remove Firefox default */
}
/* Use :focus-visible to only show on keyboard nav */
.navbar .nav-link:focus:not(:focus-visible) {
    outline: none;
}
IE11 Graceful Degradation:
/* Bootstrap 5 does not support IE11 */
/* If needed, provide basic fallback */
@media all and (-ms-high-contrast: none) {
    /* IE11-specific overrides */
    .navbar { display: block; }
    .navbar-collapse { display: block; }
    .navbar-nav { display: block; }
    .nav-item { display: inline-block; }
}
