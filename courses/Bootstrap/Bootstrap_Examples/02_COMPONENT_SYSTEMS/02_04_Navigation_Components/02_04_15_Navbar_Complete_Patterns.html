R.9 Performance Quick Reference

  • Sticky nav: Use will-change: transform and contain: layout style paint
  • Scroll effects: Use requestAnimationFrame with { passive: true }
  • Images: SVG logos with explicit width/height
  • Lazy loading: Offcanvas/mega menu content loaded on first interaction
  • CSS: Avoid @import, use <link> tags
  • JavaScript: Bundle included at end of <body>
