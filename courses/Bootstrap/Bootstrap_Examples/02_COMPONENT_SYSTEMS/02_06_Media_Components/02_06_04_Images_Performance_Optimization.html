Images: Performance Optimization

Lazy loading, srcset, WebP format, responsive sizing, and CLS prevention

1. Native Lazy Loading

The loading="lazy" attribute tells the browser to defer loading off-screen images.

<!-- Native lazy loading - simplest approach -->
<img src="photo.jpg" loading="lazy" alt="Description">

<!-- Eager (default) for above-the-fold images -->
<img src="hero.jpg" loading="eager" alt="Hero image">
Tip: Always use loading="eager" (or omit) for above-the-fold images like hero banners. Use loading="lazy" for images below the fold.

2. Responsive Images with srcset

Provide multiple image sizes so the browser loads only what it needs.

<!-- srcset with width descriptors -->
<img src="photo-400.jpg"
     srcset="photo-400.jpg 400w,
             photo-800.jpg 800w,
             photo-1200.jpg 1200w"
     sizes="(max-width: 576px) 100vw,
            (max-width: 992px) 50vw,
            33vw"
     alt="Responsive image">

<!-- srcset with pixel density descriptors -->
<img src="photo-1x.jpg"
     srcset="photo-1x.jpg 1x,
             photo-2x.jpg 2x,
             photo-3x.jpg 3x"
     alt="Retina-ready image">
Art Direction with <picture>
<!-- Different image for different viewports -->
<picture>
  <source media="(min-width: 992px)" srcset="wide-desktop.jpg">
  <source media="(min-width: 576px)" srcset="tablet.jpg">
  <img src="mobile.jpg" alt="Art-directed responsive image">
</picture>

3. Modern Image Formats (WebP, AVIF)

Use <picture> to serve modern formats with fallbacks.

<!-- WebP with JPEG fallback -->
<picture>
  <source srcset="photo.avif" type="image/avif">
  <source srcset="photo.webp" type="image/webp">
  <img src="photo.jpg" alt="Image with format fallback">
</picture>

<!-- File size comparison (typical): -->
<!-- JPEG:  200 KB -->
<!-- WebP:  140 KB (30% smaller) -->
<!-- AVIF:  100 KB (50% smaller) -->

4. Preventing Layout Shift (CLS)

Always set explicit width and height attributes on images.

Bad - Causes Layout Shift
<!-- No dimensions = layout shift -->
<img src="photo.jpg" alt="...">
Good - No Layout Shift
<!-- Browser reserves space immediately -->
<img src="photo.jpg"
     width="800" height="600"
     style="max-width:100%; height:auto;"
     alt="...">
Bootstrap handles this: .img-fluid applies max-width:100% and height:auto, which respects the aspect ratio from your width/height attributes.

6. Fetch Priority Hints

Use fetchpriority="high" for critical above-the-fold images.

<!-- High priority for hero/LCP image -->
<img src="hero.jpg"
     fetchpriority="high"
     alt="Hero banner image">

<!-- Low priority for non-critical images -->
<img src="sidebar-ad.jpg"
     fetchpriority="low"
     loading="lazy"
     alt="Advertisement">
