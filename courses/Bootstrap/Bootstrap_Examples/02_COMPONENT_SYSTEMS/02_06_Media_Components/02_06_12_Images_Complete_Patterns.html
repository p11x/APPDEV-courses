5. Complete Best Practices Summary

Always Do
  • Use .img-fluid for responsive images
  • Set width and height attributes
  • Write descriptive alt text
  • Use loading="lazy" below the fold
  • Use <picture> for format fallbacks
  • Use srcset for responsive sizes
  • Use WebP/AVIF with JPEG fallback
  • Compress images in build pipeline
  • Use semantic <figure> for captioned images
Never Do
  • Load 2000px images for mobile displays
  • Omit alt attribute
  • Use filename as alt text
  • Forget width/height (causes CLS)
  • Use images for text content
  • Disable lazy loading on all images
  • Serve uncompressed PNG/JPEG
  • Use CSS background for content images
  • Ignore image performance budgets
