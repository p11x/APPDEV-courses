Button Maintenance Guide

Patterns for long-term maintainability, consistent naming, and clear documentation.

1. Button Component Inventory

Component Class Usage Example
Primary Action .btn .btn-primary Main form submit, confirm
Secondary Action .btn .btn-secondary Cancel, back, dismiss
Danger Action .btn .btn-danger Delete, destructive
Outline Action .btn .btn-outline-* Less prominent actions
Icon Button .btn .btn-icon Compact actions
Link Button .btn .btn-link Navigation, tertiary

2. Naming Conventions

CSS Class Naming (BEM-like)
/* Block: the component */
.app-btn { }

/* Element: child of block */
.app-btn__icon { }
.app-btn__label { }

/* Modifier: variant of block */
.app-btn--primary { }
.app-btn--large { }
.app-btn--loading { }

/* State: temporary condition */
.app-btn.is-active { }
.app-btn.is-disabled { }
.app-btn.is-loading { }
data-* Attribute Naming
/* Action: what the button does */
data-action="submit-form"
data-action="delete-item"

/* Target: what it acts on */
data-target="#myModal"
data-target-item="user-123"

/* Test ID: for automated testing */
data-testid="btn-save-changes"
data-testid="btn-cancel-modal"
JavaScript Function Naming
// Verb-noun pattern for handlers
function handleSubmit() { }
function handleDeleteClick() { }
function handleToggleState() { }

// "get" for data retrieval
function getButtonState() { }

// "set" for state mutation
function setButtonLoading() { }
function setButtonDisabled() { }

4. Button Lifecycle Documentation

/**
 * Button State Lifecycle
 *
 * [IDLE] --click--> [LOADING] --success--> [SUCCESS] --timeout(2s)--> [IDLE]
 *                  [LOADING] --error--> [ERROR] --retry--> [LOADING]
 *                  [LOADING] --timeout(30s)--> [TIMEOUT] --retry--> [LOADING]
 *
 * State transitions:
 * - IDLE: Default state, interactive
 * - LOADING: Disabled, spinner visible, aria-busy="true"
 * - SUCCESS: Green background, checkmark, 2s auto-reset
 * - ERROR: Red background, error icon, retry button
 * - TIMEOUT: Yellow background, refresh prompt
 */

6. Recommended File Organization

src/
  components/
    buttons/
      Button.tsx              # Base Button component
      ButtonGroup.tsx         # Button Group wrapper
      IconButton.tsx          # Icon-only button variant
      SplitButton.tsx         # Split dropdown button
      ToggleButton.tsx        # Toggle state button
      index.ts                # Public exports
      buttons.module.css      # Scoped styles
      buttons.test.tsx        # Unit tests
      buttons.stories.tsx     # Storybook stories
      __snapshots__/          # Visual regression baselines

  styles/
    _tokens.scss              # Design tokens (colors, spacing)
    _buttons.scss             # Global button overrides

  types/
    buttons.ts                # TypeScript interfaces
    // ButtonVariant, ButtonSize, ButtonState, etc.
