Button Future Trends

Modern CSS: nesting, color-mix, container queries, :has(), OKLCH, cascade layers.

1. CSS Nesting

Native CSS nesting removes Sass dependency for simple nesting.

.btn-modern {
    background: oklch(0.55 0.2 264);
    &:hover { background: oklch(0.45 0.2 264); }
    &.is-loading { opacity: 0.7; }
}

2. color-mix() Dynamic Colors

One class per variant, hover/active derived from base via color-mix.

background: color-mix(in oklch, var(--base-color) 80%, white);

3. Container Queries

Drag the container edge to resize. Buttons adapt to container, not viewport.

@container btn-area (max-width: 200px) {
    .btn-text { display: none; }
}

4. :has() Parent Selector

Container gains shadow when any child button is active.

5. OKLCH & Relative Color Syntax

background: oklch(55% 0.2 264);
/* Relative: derive from var */
background: oklch(from var(--brand) calc(l + 0.1) c h);

6. CSS Logical Properties

padding-block: 0.625rem;   /* block axis */
padding-inline: 1.25rem;   /* inline axis */
border-start-start-radius: 0.5rem;
margin-inline-end: 0.5rem;

7. CSS Cascade Layers

@layer base, components, overrides;
@layer components { .btn { background: purple; } }
@layer overrides { .btn.override { background: orange; } }
