# Installation Guide

## Overview

This guide walks you through installing FastAPI and all necessary dependencies for development. We'll cover multiple installation methods and best practices for dependency management.

## Quick Installation

### Minimum Installation

For a basic FastAPI setup, you only need two packages:

```bash
# Create and activate virtual environment (recommended)
python -m venv fastapi_env

# Windows
fastapi_env\Scripts\activate

# macOS/Linux
source fastapi_env/bin/activate

# Install FastAPI with all optional dependencies
pip install "fastapi[standard]"

# This includes:
# - fastapi: The framework itself
# - uvicorn: ASGI server to run the application
# - pydantic: Data validation (included as dependency)
```

### Verify Installation

```bash
# Check installed versions
pip show fastapi
pip show uvicorn

# Or check all at once
pip list | grep -i fastapi
pip list | grep -i uvicorn

# Quick test - create a test file
echo 'from fastapi import FastAPI
app = FastAPI()

@app.get("/")
async def root():
    return {"message": "FastAPI installed successfully!"}' > test_install.py

# Run the test
uvicorn test_install:app --reload

# Visit http://localhost:8000 to see the response
# Visit http://localhost:8000/docs to see auto-generated documentation
```

## Detailed Installation Options

### Option 1: pip (Recommended for Learning)

```bash
# Step 1: Create project directory
mkdir my_fastapi_project
cd my_fastapi_project

# Step 2: Create virtual environment
python -m venv venv

# Step 3: Activate virtual environment
# Windows PowerShell:
venv\Scripts\Activate.ps1

# Windows CMD:
venv\Scripts\activate.bat

# macOS/Linux:
source venv/bin/activate

# Step 4: Upgrade pip (recommended)
python -m pip install --upgrade pip

# Step 5: Install FastAPI with standard extras
pip install "fastapi[standard]"

# Step 6: Verify installation
python -c "import fastapi; print(f'FastAPI version: {fastapi.__version__}')"
```

### Option 2: Poetry (Recommended for Production)

Poetry provides better dependency management and project structure:

```bash
# Install Poetry (if not installed)
# Windows (PowerShell):
(Invoke-WebRequest -Uri https://install.python-poetry.org -UseBasicParsing).Content | python -

# macOS/Linux:
curl -sSL https://install.python-poetry.org | python3 -

# Create new project with Poetry
poetry new my_fastapi_project
cd my_fastapi_project

# Add FastAPI with standard extras
poetry add "fastapi[standard]"

# Activate virtual environment
poetry shell

# Run your app
poetry run uvicorn main:app --reload

# Example pyproject.toml generated by Poetry:
"""
[tool.poetry]
name = "my-fastapi-project"
version = "0.1.0"
description = ""
authors = ["Your Name <your.email@example.com>"]

[tool.poetry.dependencies]
python = "^3.9"
fastapi = {extras = ["standard"], version = "^0.100.0"}

[tool.poetry.dev-dependencies]
pytest = "^7.0"
httpx = "^0.24.0"

[build-system]
requires = ["poetry-core>=1.0.0"]
build-backend = "poetry.core.masonry.api"
"""
```

### Option 3: Pipenv

```bash
# Install Pipenv
pip install pipenv

# Create project
mkdir my_fastapi_project
cd my_fastapi_project

# Install FastAPI
pipenv install "fastapi[standard]"
```

## Understanding Dependencies

### Core Dependencies

```bash
# These are installed automatically with FastAPI:
# - starlette: Web framework core
# - pydantic: Data validation
# - pydantic-core: Pydantic performance core
# - typing-extensions: Backported typing features
# - anyio: Async compatibility layer

# Check what's installed
pip show fastapi
# Shows: Requires: starlette, pydantic, typing-extensions, anyio
```

### Standard Dependencies

```bash
# Installing with [standard] adds these optional packages:
# - uvicorn[standard]: ASGI server with performance extras
# - fastapi-cli: Command-line interface
# - httpx: HTTP client for testing
# - pydantic-settings: Settings management
# - jinja2: Template rendering
# - python-multipart: Form data parsing

# You can install these separately if needed:
pip install uvicorn[standard]
pip install python-multipart  # For file uploads
pip install jinja2            # For templates
```

### Development Dependencies

```bash
# Testing tools
pip install pytest pytest-asyncio httpx

# Code quality tools
pip install black mypy ruff

# Documentation tools
pip install mkdocs mkdocs-material

# Database tools (if needed)
pip install sqlalchemy alembic
pip install databases[asyncpg]  # For async PostgreSQL
pip install databases[aiosqlite]  # For async SQLite
```

## Project Structure Setup

### Basic Project Structure

```bash
# Create project structure
mkdir -p my_fastapi_project/{app,tests,docs}

# Create basic files
touch my_fastapi_project/app/__init__.py
touch my_fastapi_project/app/main.py
touch my_fastapi_project/tests/__init__.py
touch my_fastapi_project/tests/test_main.py
touch my_fastapi_project/requirements.txt
touch my_fastapi_project/.env

# Resulting structure:
"""
my_fastapi_project/
├── app/
│   ├── __init__.py
│   └── main.py          # FastAPI application
├── tests/
│   ├── __init__.py
│   └── test_main.py     # Test files
├── docs/                # Documentation
├── requirements.txt     # Dependencies
├── .env                 # Environment variables
└── README.md
"""
```

### Application Entry Point

```python
# app/main.py - Basic FastAPI application
from fastapi import FastAPI

# Create FastAPI instance
# This is the main entry point of your application
app = FastAPI(
    title="My FastAPI Project",
    description="A sample FastAPI application",
    version="0.1.0"
)

@app.get("/")
async def root():
    """Root endpoint - returns welcome message"""
    return {"message": "Welcome to My FastAPI Project"}

@app.get("/health")
async def health_check():
    """Health check endpoint for monitoring"""
    return {"status": "healthy"}

# This file can be run directly:
# uvicorn app.main:app --reload
```

## Running Your Application

### Development Server

```bash
# Basic run
uvicorn app.main:app

# With auto-reload (development)
uvicorn app.main:app --reload

# Specify host and port
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# With more workers (production-like)
uvicorn app.main:app --workers 4

# Enable debug mode
uvicorn app.main:app --reload --log-level debug
```

### Configuration Options

```bash
# Create a run script for consistency
cat > run.sh << 'EOF'
#!/bin/bash
# Development run script

export APP_MODULE="app.main:app"
export HOST="0.0.0.0"
export PORT="8000"
export RELOAD="--reload"
export LOG_LEVEL="info"

uvicorn $APP_MODULE \
    --host $HOST \
    --port $PORT \
    $RELOAD \
    --log-level $LOG_LEVEL
EOF

# Make executable
chmod +x run.sh

# Run with script
./run.sh
```

### Environment Variables

```python
# app/config.py - Configuration management
from pydantic_settings import BaseSettings
from functools import lru_cache

class Settings(BaseSettings):
    """
    Application settings loaded from environment variables.
    FastAPI automatically loads .env files with pydantic-settings.
    """
    app_name: str = "My FastAPI App"
    app_version: str = "0.1.0"
    debug: bool = False
    database_url: str = "sqlite:///./app.db"
    secret_key: str = "your-secret-key-here"

    class Config:
        env_file = ".env"
        env_file_encoding = "utf-8"

@lru_cache()
def get_settings():
    """Cached settings instance"""
    return Settings()

# Usage in main.py
from fastapi import FastAPI, Depends
from .config import get_settings, Settings

app = FastAPI()

@app.get("/info")
async def info(settings: Settings = Depends(get_settings)):
    """Endpoint showing app configuration"""
    return {
        "app_name": settings.app_name,
        "version": settings.app_version,
        "debug": settings.debug
    }
```

## Troubleshooting

### Common Installation Issues

#### Issue 1: Permission Denied
```bash
# Error: Permission denied

# Solution 1: Use --user flag
pip install --user fastapi

# Solution 2: Use virtual environment (recommended)
python -m venv venv
source venv/bin/activate
pip install fastapi

# Solution 3: Fix permissions (Linux/macOS)
sudo chown -R $(whoami) ~/.local/lib/python3.11/site-packages
```

#### Issue 2: Module Not Found
```bash
# Error: ModuleNotFoundError: No module named 'fastapi'

# Solution 1: Check if you're in the right virtual environment
which python
# Should point to your venv

# Solution 2: Reinstall in current environment
pip install --force-reinstall fastapi

# Solution 3: Check Python version compatibility
python --version
# Ensure it's 3.9+
```

#### Issue 3: uvicorn Not Found
```bash
# Error: command not found: uvicorn

# Solution 1: Install uvicorn separately
pip install uvicorn[standard]

# Solution 2: Use python -m
python -m uvicorn app.main:app --reload

# Solution 3: Check PATH
pip show uvicorn
# Note the Location, ensure Scripts directory is in PATH
```

#### Issue 4: Version Conflicts
```bash
# Error: pip dependency resolution failed

# Solution 1: Use constraints file
pip install fastapi --constraint constraints.txt

# Solution 2: Force reinstall all
pip install --force-reinstall --no-cache-dir fastapi

# Solution 3: Clean install
pip uninstall fastapi pydantic starlette
pip install fastapi
```

```bash
# Install uvicorn with performance extras
pip install "uvicorn[standard]"

# - uvloop: Fast asyncio event loop (Unix only)
# - httptools: Fast HTTP parsing

```

```bash

pip install uvloop     # Fast event loop

# Run with gunicorn (production)
gunicorn app.main:app -w 4 -k uvicorn.workers.UvicornWorker
```
