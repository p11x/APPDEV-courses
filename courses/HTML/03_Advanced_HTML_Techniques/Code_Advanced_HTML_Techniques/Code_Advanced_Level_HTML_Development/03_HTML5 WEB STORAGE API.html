Methods

MethodDescription
setItem(key, value)Store data
getItem(key)Retrieve data
removeItem(key)Delete item
clear()Clear all
key(index)Get key by index
lengthNumber of items

Code Examples

// Store data
localStorage.setItem('username', 'John');
localStorage.setItem('preferences', JSON.stringify({theme:'dark'}));

// Retrieve data
var user = localStorage.getItem('username');
var prefs = JSON.parse(localStorage.getItem('preferences'));

// Delete data
localStorage.removeItem('username');
localStorage.clear();

// Check support
if (typeof(Storage) !== "undefined") {
    // Use storage
}

Storage Events

// Listen for storage changes
window.addEventListener('storage', function(e) {
    console.log('Key:', e.key);
    console.log('Old:', e.oldValue);
    console.log('New:', e.newValue);
    console.log('URL:', e.url);
});

Best Practices

  • Always handle storage quota exceeded errors
  • Use JSON.stringify/parse for objects
  • Check for browser support
  • Don't store sensitive data
  • Use meaningful key names
