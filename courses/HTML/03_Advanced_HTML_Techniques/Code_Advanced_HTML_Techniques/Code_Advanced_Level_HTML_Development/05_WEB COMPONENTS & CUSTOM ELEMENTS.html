Custom Element Lifecycle

CallbackDescription
constructor()Initialize element
connectedCallback()Added to DOM
disconnectedCallback()Removed from DOM
attributeChangedCallback()Attribute changed
adoptedCallback()Moved to new document

Custom Element Example

class MyElement extends HTMLElement {
    constructor() {
        super();
    }
    
    connectedCallback() {
        this.innerHTML = '
Hello World!
';
    }
}

customElements.define('my-element', MyElement);

Shadow DOM

class MyElement extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        shadow.innerHTML = `
            
            
Shadow content

        `;
    }
}

Best Practices

  • Use Shadow DOM for style encapsulation
  • Follow naming conventions (kebab-case)
  • Implement lifecycle callbacks properly
  • Use slots for content projection
  • Consider accessibility
