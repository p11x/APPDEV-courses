Custom Data Attributes (data-*)

Data attributes allow you to store custom data in HTML elements.

SyntaxDescription
data-idSingle word attribute
data-user-nameHyphenated attribute
data-userName camelCase conversion

JavaScript Access

// Get data
element.dataset.propertyName;

// Set data
element.dataset.propertyName = 'value';

// Remove data
delete element.dataset.propertyName;

// Check if exists
if (element.dataset.propertyName) { }

Note: Hyphens become camelCase (data-user-name → dataset.userName)

Complete Print Stylesheet

@media print {
    /* Hide non-essential elements */
    .no-print, nav, footer, .sidebar {
        display: none !important;
    }
    
    /* Reset body styles */
    body {
        font-size: 12pt;
        line-height: 1.5;
        color: #000;
        background: #fff;
    }
    
    /* Show link URLs */
    a[href^="http"]::after {
        content: " (" attr(href) ")";
    }
    
    /* Page breaks */
    .page-break {
        page-break-after: always;
    }
    
    /* Tables */
    table {
        border: 1px solid #000;
    }
    
    th, td {
        border: 1px solid #000;
        padding: 5px;
    }
}

Link Print Options

/* Show all link URLs */
a::after {
    content: " (" attr(href) ")";
}

/* Only show external links */
a[href^="http"]::after {
    content: " (" attr(href) ")";
}

/* Don't show links */
a::after {
    content: "";
}
