Email HTML vs Web HTML

Don't use:
  • External stylesheets
  • JavaScript
  • Modern CSS (flexbox, grid)
  • Background images (complex)
  • Forms
Do use:
  • Inline CSS
  • Table-based layouts
  • Web-safe fonts
  • HTML attributes
  • CSS reset

  • Always use inline CSS
