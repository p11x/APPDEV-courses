ContentEditable Attribute

The contenteditable attribute makes elements editable by the user.

ValueDescription
trueElement is editable
falseElement is not editable
inheritInherits from parent

ContentEditable JavaScript API

// Check if editable
element.isContentEditable;

// Get/Set edit mode
element.contentEditable = 'true';

// Events
element.addEventListener('input', function(e) {
    console.log('Content changed:', e.target.innerHTML);
});

element.addEventListener('blur', function(e) {
    console.log('User finished editing');
});

Iframe Security

Use the sandbox attribute to control what features are allowed in an iframe.

Sandbox ValueDescription
allow-formsAllow form submission
allow-modalsAllow modal dialogs
allow-orientation-lockAllow screen orientation
allow-pointer-lockAllow pointer lock
allow-popupsAllow popups
allow-popups-to-escape-sandboxAllow popups without sandbox
allow-presentationAllow presentations
allow-same-originTreat as same origin
allow-scriptsAllow JavaScript
allow-top-navigationAllow top navigation
allow-top-navigation-by-user-activationAllow with user action

X-Frame-Options Header

// .htaccess or server config
X-Frame-Options: DENY
X-Frame-Options: SAMEORIGIN
X-Frame-Options: ALLOW-FROM https://example.com

This prevents your site from being embedded in iframes on other sites.
