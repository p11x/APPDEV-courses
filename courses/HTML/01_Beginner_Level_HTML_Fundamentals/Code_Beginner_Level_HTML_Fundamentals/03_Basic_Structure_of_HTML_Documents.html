    Understanding HTML Document Structure

    Every valid HTML document follows a specific structure. This structure tells the browser how to interpret and display the content. Understanding this structure is fundamental to web development.

    Document Structure Overview:

    1. The DOCTYPE Declaration

    The declaration defines this document as HTML5. It must be the very first thing in your HTML document, before any HTML tags.

    Important: The DOCTYPE declaration is not an HTML tag. It's an instruction to the web browser about what version of HTML the page is written in. 
    


    Page Title

    2. The Element

    The element is the root element of an HTML page. It wraps all the content on the entire page. The lang attribute specifies the language of the document.
    Best Practice: Always include the lang attribute. It helps search engines and screen readers understand the language of your content.
