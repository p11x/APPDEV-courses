Semantic vs. Presentation Elements

HTML5 distinguishes between semantic (meaningful) and presentation elements.

Semantic Elements (Recommended)

  • - Important text (bold)
  • - Emphasized text (italic)
  • - Marked/highlighted text
  • - Deleted text (strikethrough)
  • - Inserted text (underline)

Presentation Elements (Deprecated)

  • - Bold text (no meaning)
  • - Italic text (no meaning)
  • - Underlined (no meaning)

1. Physical vs. Logical Text Formatting

Physical (Visual)

These tags describe how text should look:

  • - Bold
  • - Italic
  • - Underline
  • - Strikethrough

Logical (Semantic)

These tags describe what text means:

  • - Important
  • - Emphasized
  • - Deleted
  • - Inserted
Best Practice: Use semantic elements (, ) when possible. They provide meaning that helps search engines and screen readers.
