1. The Anchor Tag ()

The (anchor) tag creates hyperlinks. The essential attribute is href (Hypertext REFerence) which specifies the destination.

Basic Syntax:

Link Text

Complete Example:


   Click Here

2. Link Attributes Reference

Attribute Description Values
href URL of the page to link to URL, file path, anchor
target Where to open the link _self, _blank, _parent, _top
title Extra info shown on hover Any text
rel Relationship to the linked page noopener, noreferrer, nofollow
download Download instead of open Filename (optional)
hreflang Language of linked page Language code (en, es, fr)
Important: When using target="_blank", always add rel="noopener noreferrer" for security and performance.

3. Target Attribute Values

Value Description Example
_self Same tab/window (default) target="_self"
_blank New tab/window target="_blank"
_parent Parent frame target="_parent"
_top Full body of window target="_top"

6. Link Best Practices

Do's:

  • Use descriptive link text instead of "click here"
  • Always add rel="noopener noreferrer" with target="_blank"
  • Use absolute URLs for external links
  • Use relative URLs for internal links
  • Test links regularly to ensure they work
  • Use the title attribute for additional context

Don'ts:

  • Don't use "click here" as link text
  • Don't open external links in the same tab without warning
  • Don't have broken or dead links
  • Don't use links for everything - use buttons for actions

Good vs Bad Link Text:

❌ Bad ✅ Good
"Click here" for more info "Learn more about our services"
"Read more" (where to?) "Read more about HTML"
"Here" link "Download the PDF guide"
URL as link text Descriptive text
