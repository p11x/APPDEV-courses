Attribute Description Example

9. Image Best Practices

Do's:

  • Always include descriptive alt text
  • Use appropriate image formats for content
  • Optimize images for web (compress)
  • Use responsive images for different screen sizes
  • Specify width and height to prevent layout shift
  • Use lazy loading for below-the-fold images

Don'ts:

  • Don't use alt="" for decorative images only
  • Don't use images for important text
  • Don't use excessively large images
  • Don't forget to test images on different devices
