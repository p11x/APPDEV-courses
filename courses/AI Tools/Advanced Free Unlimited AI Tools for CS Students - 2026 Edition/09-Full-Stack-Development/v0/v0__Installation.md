# v0 - Installation Guide

## Web-Based Access

v0 is a web-based tool - no installation required.

### Step-by-Step:

1. **Visit v0**
   - Go to: https://v0.dev

2. **Sign In**
   - Click "Sign In" button
   - Use Vercel account or GitHub

3. **Start Creating**
   - Enter prompt in the text box
   - Press Enter to generate

---

## VSCode Extension (Optional)

For integration with local development:

1. **Open VSCode**
2. **Go to Extensions** (`Ctrl+Shift+X`)
3. **Search for "v0"**
4. **Install v0 Extension**

---

## Usage Example

```
Prompt: "Create a responsive navbar with logo, navigation links, and a CTA button"
```

The AI will generate a complete React component with Tailwind CSS.
