# Bolt.new - Key Features

## 1. Browser-Based IDE

- **Full Code Editor**: Professional IDE in your browser
- **File Management**: Create, edit, delete files
- **Terminal Access**: Built-in terminal for commands
- **Multi-file Support**: Work with complex projects

---

## 2. AI-Powered Development

- **Natural Language**: Describe what you want to build
- **Smart Code Generation**: AI writes the code for you
- **Real-time Suggestions**: Get AI help as you type
- **Error Fixing**: AI helps debug issues

---

## 3. Instant Preview

- **Hot Reload**: See changes instantly
- **Mobile Preview**: Test responsive designs
- **Debug in Browser**: Built-in dev tools

---

## 4. One-Click Deployment

- **Free Hosting**: Deploy with free subdomain
- **HTTPS**: Automatic SSL
- **Custom Domain**: Connect your own domain
- **Global CDN**: Fast loading worldwide

---

## 5. Code Export

- **Download Code**: Get full source files
- **GitHub Integration**: Push to your repositories
- **Local Development**: Run locally with npm
