# Mintlify - Requirements

## System Requirements

### For Web-Based Usage

| Component | Requirement |
|-----------|-------------|
| **Browser** | Modern browser (Chrome, Firefox, Safari, Edge) |
| **Internet** | Stable internet connection |
| **Account** | GitHub account (recommended) |

### For Local Development

| Component | Requirement |
|-----------|-------------|
| **Node.js** | Version 14 or higher |
| **npm** | Version 6 or higher |
| **Git** | For version control |
| **Code Editor** | VS Code recommended |

---

## Account Requirements

### Free Tier Features

- **GitHub Account**: Required for most features
- **Email**: For account creation
- **Public Repositories**: Unlimited for free tier
- **Private Repositories**: Limited on free tier

### Optional Integrations

- **GitHub**: Primary integration
- **Slack**: Notifications
- **Google Analytics**: Usage tracking

---

## Language Support

### Documentation Languages

- **Markdown**: Full support
- **MDX**: Supported
- **JavaScript**: For interactive components
- **TypeScript**: Supported

---

## Version Information

- **Latest Version**: Continuous updates
- **Platform**: Web-based SaaS
