# SQLite - Description

## Overview

SQLite is a lightweight, self-contained, serverless, zero-configuration SQL database engine. It is the most widely deployed database in the world, used by billions of devices. For students, SQLite provides an excellent way to learn SQL without needing to set up a complex database server.

## Official Website

- **Website**: [sqlite.org](https://www.sqlite.org)
- **Documentation**: [sqlite.org/docs.html](https://www.sqlite.org/docs.html)

## Key Characteristics

- Serverless - no separate process needed
- Zero-configuration - no setup required
- Single file database - portable
- ACID compliant
- Public domain - free to use
