# PostgreSQL - Requirements

## System Requirements

### Minimum Requirements
- CPU: 1 core
- RAM: 512 MB
- Disk: 100 MB for installation
- OS: Windows 10+, macOS 10.14+, Linux (Ubuntu 18.04+, CentOS 7+)

### Recommended Requirements
- CPU: 2+ cores
- RAM: 2+ GB
- Disk: 10+ GB for data

## Software Requirements

- Web browser for pgAdmin (optional)
- SSL libraries for encrypted connections

## Network Requirements

- Default port: 5432
- Must allow incoming connections if remote access needed
