# MongoDB Atlas - Functionalities

## Core Functionalities

- **CRUD Operations**: Create, Read, Update, Delete documents
- **Aggregation Pipeline**: Data transformation and analysis
- **Indexing**: Single field, compound, text, geospatial indexes
- **Query Optimization**: Explain plans, performance advisor
- **Data Import/Export**: Atlas Data Explorer, mongo tools

## Developer Tools

- **MongoDB Compass**: GUI for database interaction
- **Atlas CLI**: Command-line interface
- **Atlas Data API**: RESTful API access
- **Atlas GraphQL API**: GraphQL endpoint access

## Integration

- **MongoDB Drivers**: Node.js, Python, Java, Go, etc.
- **Atlas SQL Interface**: Query using SQL
- **Realm SDK**: Mobile database sync
