# MongoDB Atlas - Features

## Key Features

- Document-oriented storage (JSON-like documents)
- Flexible schema
- Auto-sharding for horizontal scaling
- Rich query language
- Secondary indexes
- Aggregation framework
- Geospatial queries
- Text search indexing
- Change streams for real-time data
- Atlas Search (basic)

## Management Features

- Real-time performance metrics
- Automated backup and restore
- Alert configurations
- Role-based access control
- Network peering
- Atlas CLI for automation
