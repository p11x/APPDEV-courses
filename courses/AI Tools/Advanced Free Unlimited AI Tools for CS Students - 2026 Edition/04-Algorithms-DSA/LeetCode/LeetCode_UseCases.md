# LeetCode - Student Use Cases

## Placement Interview Preparation

### Problem Solving Practice

**Use Case:** Preparing for campus placement interviews

**Recommended Approach:**
1. Start with Easy problems (solve 50+)
2. Progress to Medium problems (solve 50+)
3. Attempt Hard problems based on company requirements
4. Focus on company-tagged problems

**Free Resources:**
- LeetCode 75 study plan
- Top 150 Liked Questions
- Daily Challenge

### Mock Interview Practice

**Use Case:** Simulating real interview conditions

**How to Use:**
1. Use 2 free mock interviews wisely
2. Time yourself (30-45 minutes per problem)
3. Practice explaining your approach
4. Write clean, working code

### Resume Building

**Use Case:** Adding to resume

**Showcase:**
- Number of problems solved
- Contest ratings
- Discuss contributions

## LeetCode/Competitive Programming

### Daily Practice Routine

**Recommended Schedule:**
- 1-2 problems daily
- Alternate between topics
- Review solutions after solving

### Problem-Solving Strategies

**For Easy Problems:**
- Focus on optimal solution
- Learn patterns
- Practice edge cases

**For Medium Problems:**
- Multiple approaches
- Optimize time/space complexity
- Practice similar problems

**For Hard Problems:**
- Study patterns first
- Break into subproblems
- Learn from editorial

### Contest Participation

**Free Contest Access:**
- Weekly Contest (limited entries)
- Biweekly Contest (limited entries)
- Daily Challenge (full access)

## Best Practices for Free Tier

### Maximizing Free Resources

| Strategy | Implementation |
|----------|----------------|
| Focus on free problems | Solve 100+ free problems first |
| Use study plans | LeetCode 75, Blind 75 |
| Daily challenge | Build consistent habit |
| Discuss forum | Learn from community |
| Contest participation | Limited but valuable |

### Problem Selection Priority

1. **Daily Challenge** - One problem every day
2. **Study Plan** - Structured approach
3. **Topic-based** - Fill knowledge gaps
4. **Pattern-based** - Group similar problems
