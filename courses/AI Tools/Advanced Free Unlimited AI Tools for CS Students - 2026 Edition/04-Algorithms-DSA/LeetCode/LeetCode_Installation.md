# LeetCode - Installation Guide

## Access Method

LeetCode is a **browser-based platform** that requires no installation. Users can access it directly through their web browser or mobile app.

## Quick Start

### Step 1: Create Account

1. Visit https://leetcode.com
2. Click "Sign Up"
3. Choose sign-up method:
   - Email registration
   - GitHub account
   - Google account
   - LinkedIn account

### Step 2: Verify Email

- Check email for verification link
- Click verification link
- Complete profile setup

### Step 3: Start Practicing

1. Navigate to Problems section
2. Select difficulty level
3. Begin solving problems

## Mobile App

### Android App

```bash
# Google Play Store
Search "LeetCode"
Install "LeetCode - Interview Prep"
```

### iOS App

```bash
# App Store
Search "LeetCode"
Install "LeetCode - Interview Prep"
```

### Mobile Features

- Problem solving on mobile
- Daily challenge notifications
- Practice tracks
- Submission history

## Browser Extension (Optional)

### LeetCode Enhancement Suite

For enhanced features, users can install browser extensions:

```bash
# Chrome Web Store
Search "LeetCode Enhancement Suite"
Install extension
```

Features include:
- Company tag filtering
- Solution shortcuts
- Theme customization
