# VisuAlgo - Features

## Algorithm Visualizations

### Sorting Algorithms

| Algorithm | Description | Complexity Displayed |
|-----------|-------------|---------------------|
| Bubble Sort | Adjacent element comparison and swapping | O(n²) |
| Selection Sort | Finding minimum element repeatedly | O(n²) |
| Insertion Sort | Building sorted array one element at a time | O(n²) |
| Merge Sort | Divide and conquer sorting | O(n log n) |
| Quick Sort | Partition-based sorting | O(n log n) average |
| Heap Sort | Binary heap-based sorting | O(n log n) |

### Data Structures

| Structure | Operations Supported |
|-----------|---------------------|
| Array | Traversal, Search, Update |
| Linked List | Insert, Delete, Search, Reverse |
| Stack | Push, Pop, Peek |
| Queue | Enqueue, Dequeue, Peek |
| Binary Search Tree | Insert, Delete, Search, Traversal |
| Heap | Insert, Extract-min/max |

### Graph Algorithms

| Algorithm | Type | Visualization |
|-----------|------|---------------|
| Breadth-First Search (BFS) | Traversal | Level-by-level exploration |
| Depth-First Search (DFS) | Traversal | Deep exploration |
| Dijkstra's Algorithm | Shortest Path | Weighted shortest path |
| Minimum Spanning Tree (MST) | Graph Optimization | Kruskal's/Prim's |

### Dynamic Programming

| Problem | Concept Displayed |
|---------|------------------|
| Fibonacci | Recursive vs DP approach |
| Knapsack | Optimal substructure |
| Longest Common Subsequence | Table building |

### String Matching

| Algorithm | Purpose |
|-----------|---------|
| KMP | Pattern matching with failure function |
| Rabin-Karp | Hash-based matching |

## Interactive Features

### Animation Controls

| Control | Function |
|---------|----------|
| Play/Pause | Start or stop animation |
| Step Forward | Execute one step |
| Step Backward | Reverse one step |
| Speed Slider | Adjust animation speed (0.25x to 4x) |
| Random Data | Generate random input |
| Custom Input | Enter custom data values |

### Visualization Modes

- **Split Screen:** Code panel + visualization panel
- **Variable Tracking:** Watch variable values change
- **Recursion Tree:** Visualize recursive call stack
- **Step Counter:** Display current step number

### Educational Features

- Pseudocode display synchronized with animation
- Explanation panel for each step
- Time and space complexity display
- Best/Average/Worst case scenarios
