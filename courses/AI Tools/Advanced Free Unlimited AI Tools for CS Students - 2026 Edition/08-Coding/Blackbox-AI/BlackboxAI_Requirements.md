# Blackbox AI - Requirements

## System Requirements

### Minimum Requirements

| Component | Requirement |
|-----------|-------------|
| **OS** | Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+) |
| **RAM** | 4GB minimum, 8GB recommended |
| **Storage** | 500MB free space |
| **Internet** | Required for cloud features |

### IDE Compatibility

#### VS Code
- **Version**: VS Code 1.60.0 or higher
- **Extensions Support**: Enabled
- **Internet Connection**: Required

#### JetBrains IDEs
- **IntelliJ IDEA**: 2020.3+
- **PyCharm**: 2020.3+
- **WebStorm**: 2020.3+
- **Other JetBrains**: 2020.3+

#### Chrome Browser
- **Version**: Chrome 90+
- **OS**: Windows, macOS, Linux, Chrome OS

---

## Account Requirements

### Optional (Basic Features Work Without)

- **Email Account**: Optional for basic completions
- **GitHub Account**: Optional for enhanced features
- **Internet Connection**: Required for AI features

### Premium Features (Optional)

- **Paid Subscription**: For enhanced limits
- **API Access**: For custom integrations

## Network Requirements

### For Cloud Features

- **Internet**: Broadband recommended
- **Latency**: Under 100ms for best experience
- **Ports**: 443 (HTTPS) for all connections

### For Offline Use

- **Local Models**: Limited offline capability
- **Caching**: Some features work offline after initial use

---

## Performance Considerations

### Recommended for Best Performance

| Resource | Recommended |
|----------|-------------|
| **RAM** | 8GB or more |
| **CPU** | Multi-core processor |
| **Internet** | Stable 10+ Mbps connection |
| **GPU** | Not required (cloud-based) |
