# Blackbox AI - Installation

## Installation Methods

### Method 1: VS Code Extension (Recommended)

**Step-by-Step Installation:**

1. **Open VS Code**
   - Launch Visual Studio Code on your computer

2. **Access Extensions**
   - Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (macOS)
   - Or click the Extensions icon in the sidebar

3. **Search for Blackbox AI**
   - Type "Blackbox AI" in the search bar
   - Look for "Blackbox AI - Code Faster"

4. **Install Extension**
   - Click the "Install" button
   - Wait for installation to complete

5. **Activate**
   - Click "Reload" or restart VS Code
   - Look for Blackbox icon in sidebar

**Direct Link:**
```
https://marketplace.visualstudio.com/items?itemName=BlackboxAI.blackbox
```

---

### Method 2: JetBrains IDEs

**For IntelliJ, WebStorm, PyCharm, etc.:**

1. **Open IDE Settings**
   - Go to `File > Settings` (Windows/Linux)
   - Go to `Preferences` (macOS)

2. **Navigate to Plugins**
   - Click on "Plugins" in the sidebar
   - Click "Marketplace"

3. **Search and Install**
   - Search for "Blackbox AI"
   - Click "Install"

4. **Restart IDE**
   - Click "Restart IDE" to activate

**Direct Link:**
```
https://plugins.jetbrains.com/plugin/xxxxx-blackbox
```

---

### Method 3: Chrome Browser Extension

**For code search in browser:**

1. **Open Chrome**
   - Launch Google Chrome browser

2. **Visit Chrome Web Store**
   - Navigate to: https://chrome.google.com/webstore

3. **Search for Blackbox AI**
   - Type "Blackbox AI" in search

4. **Add to Chrome**
   - Click "Add to Chrome"
   - Click "Add extension" in dialog

5. **Pin to Toolbar**
   - Click puzzle icon in Chrome toolbar
   - Pin Blackbox AI for quick access

**Direct Link:**
```
https://chrome.google.com/webstore/detail/blackbox-ai-code-faster/xxxxxxxx
```

## Verification

### Test Installation

**Test 1: Check Extension Status**
- Look for Blackbox icon in VS Code sidebar
- Status should show "Connected" or "Active"

**Test 2: Try Completion**
- Open any code file
- Start typing code
- Suggestions should appear

**Test 3: Test Chat**
- Open Blackbox chat panel
- Ask a simple question
- Should receive response

## Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| Extension not loading | Reload VS Code or restart IDE |
| No suggestions appearing | Check extension is enabled |
| Login issues | Clear cache and try again |
| Slow performance | Disable other extensions temporarily |
