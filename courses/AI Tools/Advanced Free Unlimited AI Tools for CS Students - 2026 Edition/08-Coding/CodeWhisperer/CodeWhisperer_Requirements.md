# CodeWhisperer - Requirements

## System Requirements

### Hardware Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| **RAM** | 4 GB | 8 GB or more |
| **Storage** | 200 MB | 500 MB |
| **Processor** | Intel Core i3 / AMD Ryzen 3 | Intel Core i5 / AMD Ryzen 5 |
| **Internet** | Required | Stable connection |

### Supported Operating Systems

| OS | Version |
|----|---------|
| **Windows** | Windows 10, 11 (64-bit) |
| **macOS** | macOS 10.14 (Mojave) or later |
| **Linux** | Ubuntu 18.04+, Fedora 30+, Debian 10+ |

## Software Dependencies

### Required

- **Internet connection** for code suggestions
- **AWS account** (optional for individual tier)
- **Email** (for Builder ID - optional)

### Optional

- **AWS CLI** (for advanced AWS features)
- **Docker** (for local testing)
