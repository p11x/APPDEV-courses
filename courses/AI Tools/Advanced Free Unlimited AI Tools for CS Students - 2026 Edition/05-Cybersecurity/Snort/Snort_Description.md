# Snort - Description

## Overview

Snort is a free, open-source network intrusion detection system (NIDS) capable of performing real-time traffic analysis and packet logging on IP networks.

## Official Links

| Resource | Link |
|----------|------|
| Website | https://www.snort.org |
| Documentation | https://docs.snort.org |

## Free Features

- Network intrusion detection
- Packet logging
- Real-time traffic analysis
- Protocol analysis
- Content matching
- Community rules (free)
