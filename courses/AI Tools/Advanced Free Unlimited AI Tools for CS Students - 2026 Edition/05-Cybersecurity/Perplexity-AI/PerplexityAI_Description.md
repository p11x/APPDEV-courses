# Perplexity AI - Description

## Overview

Perplexity AI is an AI-powered search engine and research tool that provides direct answers with citations. It combines large language model capabilities with real-time web search to deliver accurate, sourced information.

## Official Resources

- **Website**: https://www.perplexity.ai
- **Documentation**: https://docs.perplexity.ai
- **GitHub**: https://github.com/perplexity-ai

## Why It's Valuable for CS Students

- Instant answers to technical questions
- Research assistance for projects
- Code debugging help
- Security vulnerability research
- Citation-backed responses for academic work
