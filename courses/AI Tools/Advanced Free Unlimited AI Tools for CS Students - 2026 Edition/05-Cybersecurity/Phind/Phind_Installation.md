# Phind - Installation

## Access Methods

### Web Browser

```bash
# Visit the official website
https://www.phind.com

# No installation required
# Access via any modern browser
```

### Chrome/Edge Extension

```bash
# Chrome Web Store
https://chrome.google.com/webstore/detail/phind-ai-search-assistant/

# Click "Add to Chrome" to install
```

### VS Code Extension

```bash
# Open VS Code
# Go to Extensions (Ctrl+Shift+X)
# Search for "Phind"
# Click Install
```

### Firefox Add-on

```bash
# Firefox Add-ons
https://addons.mozilla.org/en-US/firefox/addon/phind-ai-search/
```
