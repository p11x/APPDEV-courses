# Gemini AI Studio Description

## Overview

Google Gemini AI Studio (now called Google AI Studio) is a free, web-based IDE for prototyping and testing AI prompts with Google's Gemini models. It provides an intuitive interface for developers to experiment with AI models, build prototypes, and generate code.

## What Makes Gemini AI Studio Unique

Gemini AI Studio offers:

- Free access to Gemini models
- Prompt engineering interface
- API key generation
- Multi-modal inputs
- Streaming responses

## Target Users

Gemini AI Studio is ideal for:

- Students learning AI/ML
- Developers prototyping AI features
- Researchers testing prompts
- Anyone exploring generative AI

## Key Characteristics

| Attribute | Description |
|-----------|-------------|
| Type | Web-based AI IDE |
| Platform | Browser-based |
| Pricing | Free tier available |
| Models | Gemini Pro, Gemini Flash |
| API | REST API generation |

## Free Tier Features

| Feature | Free Tier |
|---------|-----------|
| Requests | 60/minute |
| Context | 1M tokens |
| Models | Gemini 1.5 Pro/Flash |
| API Keys | 2 free keys |

## Limitations

- Rate limiting on free tier
- Requires Google account
- Internet required
- Not for production without billing
