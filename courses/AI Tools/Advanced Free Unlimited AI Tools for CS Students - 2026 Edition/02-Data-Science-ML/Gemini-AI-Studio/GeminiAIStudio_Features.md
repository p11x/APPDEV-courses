# Gemini AI Studio Features

## Core Features

| Feature | Description |
|---------|-------------|
| Prompt Engineering | Visual interface for building and testing prompts |
| Model Selection | Choose between Gemini Pro and Gemini Flash |
| Multi-modal Input | Support for text, images, and code |
| Streaming Responses | Real-time output as generation happens |
| API Key Management | Generate and manage API keys |

## Model Options

### Gemini 1.5 Pro

- Context window: 1 million tokens
- Advanced reasoning
- Best for complex tasks
- Free tier: 60 requests/minute

### Gemini 1.5 Flash

- Context window: 1 million tokens
- Fast response times
- Optimized for speed
- Free tier: Higher rate limits
