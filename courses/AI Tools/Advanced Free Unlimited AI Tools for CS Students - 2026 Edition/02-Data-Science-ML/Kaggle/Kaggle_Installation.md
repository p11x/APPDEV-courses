# Kaggle Installation Guide

## Getting Started

Kaggle is web-based - no installation required.

### Step 1: Create Account

1. Go to [kaggle.com](https://www.kaggle.com)
2. Click "Register"
3. Sign up with email or Google

### Step 2: Explore

- Browse datasets
- Join competitions
- Start notebooks

## Kaggle CLI (Optional)

### Install Kaggle API

```bash
pip install kaggle
```

### Setup API Key

1. Go to Account > API
2. Click "Create New API Token"
3. Place `kaggle.json` in `~/.kaggle/`
