# Google Colab Requirements

## System Requirements

### Minimum Requirements

| Component | Requirement |
|-----------|-------------|
| Browser | Chrome, Firefox, Safari, or Edge |
| Internet | Stable connection |
| Google Account | Required |
| Storage | 15 GB Google Drive |

### Recommended Setup

| Component | Recommendation |
|-----------|----------------|
| Browser | Chrome (latest) |
| Internet | Broadband |
| RAM | 8 GB+ (system) |

## Account Requirements

### Google Account

A Google account is required:

- Create at: [accounts.google.com](https://accounts.google.com)
- Must have Google Drive access

### No Payment Required

- Free tier requires no credit card
- Pro/Pro+ are optional upgrades

## Technical Requirements

### Browser Compatibility

| Browser | Support |
|---------|---------|
| Chrome | Full |
| Firefox | Full |
| Safari | Full |
| Edge | Full |

### Network Requirements

| Requirement | Detail |
|------------|--------|
| Connection | HTTPS |
| Ports | 443 (HTTPS) |
| Bandwidth | 1 Mbps minimum |

## Resource Limits (Free Tier)

### Compute Limits

| Resource | Limit |
|----------|-------|
| GPU | T4 or K80 (when available) |
| RAM | ~13 GB |
| Session | 12 hours max |
| Idle Timeout | 90 minutes |

### Storage

| Type | Limit |
|------|-------|
| Notebook | Unlimited |
| Drive Storage | 15 GB |

## Getting Started Checklist

- [ ] Google Account
- [ ] Modern browser
- [ ] Stable internet
- [ ] Google Drive (optional for storage)
