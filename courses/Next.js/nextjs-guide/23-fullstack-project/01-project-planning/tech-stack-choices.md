## Prerequisites

## Do I Need This Right Now?
Understanding why we choose each technology.

## Our Stack

### Next.js 15
- App Router, Server Components
- Server Actions
- Latest features

### TypeScript
- Type safety
- Better DX

### Prisma + PostgreSQL
- Easy database access (see Section 05)
- Type-safe queries
- Great developer experience

### NextAuth v5
- Official Next.js auth solution
- Session management (see Section 10)
- Easy social logins

### Tailwind CSS
- Utility-first styling (see Section 07)
- Fast development
- Responsive

### Sentry
- Error monitoring (see Section 20)
- Performance tracking

## Alternatives

| Instead of | Could use |
|-----------|-----------|
| Prisma | Drizzle, raw SQL |
| NextAuth | Clerk, custom JWT |
| Tailwind | CSS Modules, styled-components |

## Summary
- Choose tools that work well together
- Consider learning curve
- Stick with mainstream choices
