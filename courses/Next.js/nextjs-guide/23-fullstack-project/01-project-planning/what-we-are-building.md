## Project: Task Manager App

We're building a full-featured Task Manager with:

### Features
- User authentication (sign up, sign in, sign out)
- Create, read, update, delete tasks
- Mark tasks as complete
- Real-time updates (see others' changes)
- Beautiful, responsive UI
- Error tracking
- Performance monitoring

### Technology Stack
- Next.js 15 (App Router)
- TypeScript
- NextAuth v5 (Authentication)
- Prisma + PostgreSQL (Database)
- Server Actions (Mutations)
- SSE (Real-time updates)
- Sentry (Error monitoring)
- Tailwind CSS (Styling)

## What You'll Learn

This project integrates everything from the guide:

1. **App Router** - File-based routing
2. **Server Components** - Default rendering
3. **Server Actions** - Form handling and mutations
4. **Authentication** - User sessions
5. **Data Fetching** - Prisma integration
6. **Real-time** - Server-Sent Events
7. **Styling** - Tailwind CSS
8. **Error Handling** - Error boundaries + Sentry
9. **Performance** - Optimizations
10. **Deployment** - Vercel
