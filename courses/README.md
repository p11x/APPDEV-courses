# APPDEV Courses

This folder contains course materials for the APPDEV courses.

## Folder Structure

The `courses` directory contains learning materials for various technologies:

### Programming Languages & Frameworks
- **Angular** - Angular Classroom Curriculum and Angular Project Guide
- **Java** - Java programming language materials
- **Java Script** - JavaScript Basics
- **Type Script** - TypeScript learning materials
- **.NET** - .NET framework courses

### Web Technologies
- **HTML** - Complete HTML course with 4 levels:
  - Beginner Level HTML Fundamentals
  - Intermediate Level HTML Development
  - Advanced HTML Techniques
  - HTML Projects
- **CSS** - CSS styling and design
- **Bootstrap** - Bootstrap framework
- **SpringBoot** - Spring Boot framework

### Databases & Tools
- **SQL Server** - Database management with SQL Server
- **Git Hub** - Git and GitHub version control

### Other Topics
- **AI Tools** - Artificial Intelligence tools and concepts

### Practice Materials
- **HTML practice** - HTML practice exercises
- **java practice** - Java practice exercises
- **Java Script/-p** - JavaScript practice exercises

## Getting Started

Navigate through the folders to access course materials for each topic.
