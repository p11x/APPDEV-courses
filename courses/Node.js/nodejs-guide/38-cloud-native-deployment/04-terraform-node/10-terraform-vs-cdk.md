# Terraform vs AWS CDK

## Comparison

| Feature | Terraform | AWS CDK |
|---------|-----------|--------|
| Language | HCL | TypeScript, Python |
| State | File-based | CloudFormation |
| Planning | terraform plan | cdk diff |
| Testing | Terratest | Jest/Pytest |
| Multi-cloud | Yes | AWS-first |
