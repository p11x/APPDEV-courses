# What is Flux

## Overview

Flux is a GitOps toolkit for Kubernetes that automates the deployment of applications. Part of the GitOps Toolkit, it provides a CLI-first approach to GitOps.

## Prerequisites

- Kubernetes basics

## Core Concepts

### Flux vs ArgoCD

| Aspect | Flux | ArgoCD |
|--------|------|--------|
| UI | Minimal | Full UI |
| CLI-first | Yes | No |
| Multi-tenancy | Strong | Good |
| Learning curve | Steeper | Easier |

## Quick Reference

| Tool | Description |
|------|-------------|
| flux bootstrap | Initialize Flux |
| flux get | View resources |
| flux reconcile | Force sync |
