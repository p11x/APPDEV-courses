# Docker Desktop Extensions

## Overview

Docker Desktop Extensions extend Docker's functionality with additional features. This guide covers what extensions are and how to use them.

## Prerequisites

- Docker Desktop installed
- Understanding of Docker basics

## Core Concepts

### What Are Extensions

- Add features to Docker Desktop
- Run in containers
- UI integrated into Docker Dashboard
- Can access Docker API

## Step-by-Step Examples

### Installing Extensions

```bash
# Install extension via CLI
docker extension install publisher/extension-name

# List available extensions
docker extension ls
```
