# Useful Extensions

## Overview

This guide covers recommended Docker Desktop extensions that enhance your development workflow.

## Portainer

- Web-based container management
- Visual interface for Docker
- Easy container/image management
- Install from Docker Desktop Marketplace

## Logs Explorer

- View container logs in GUI
- Search and filter logs
- Export logs to file
- Real-time log streaming

## Volumes Backup & Share

- Backup volumes to file
- Share volumes between containers
- Restore from backup
- Export/import functionality

## Snyk

- Security vulnerability scanning
- Image scanning
- Find vulnerabilities in dependencies
- Security recommendations

## Resource Usage

- Visual resource monitoring
- CPU, memory, disk usage
- Container-level breakdown
- Historical data

## Quick Reference

| Extension | Purpose |
|-----------|---------|
| Portainer | Management UI |
| Logs Explorer | Log viewing |
| Snyk | Security |
| Volumes Backup | Data management |
