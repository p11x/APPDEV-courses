# Latent Dirichlet Allocation (LDA)

## Learning Objectives

- Understand LDA fundamentals
- Create document-term matrices
- Apply LDA model
- Interpret topics

## Theory

LDA is a probabilistic model that discovers topics in documents. Each document is a mixture of topics, and each topic is a distribution over words. LDA assumes documents are generated by first choosing topic proportions, then selecting words from those topics.

## Step-by-Step

### Creating DTM

```r
library(tm)

# Create corpus
corp <- Corpus(VectorSource(texts))

# Clean corpus
corp <- tm_map(corp, content_transformer(tolower))
corp <- tm_map(corp, removePunctuation)
corp <- tm_map(corp, removeWords, stopwords("en"))

# Create DTM
dtm <- DocumentTermMatrix(corp)

# Remove sparse terms
dtm <- removeSparseTerms(dtm, 0.95)
```

### Running LDA

```r
library(topicmodels)

# Fit LDA model
k <- 5  # number of topics

lda_model <- LDA(dtm, k = k, 
                control = list(seed = 123))

# Get topics
topics(lda_model)

# Get word-topic probabilities
beta <- beta(lda_model)
terms(lda_model, 5)
```

## Code Examples

### Topic Interpretation

```r
# Top words per topic
tidy(lda_model, matrix = "beta") |>
  group_by(topic) |>
  top_n(10, beta) |>
  ungroup() |>
  arrange(topic, -beta)
```

### Document-Topic Proportions

```r
# Document proportions
tidy(lda_model, matrix = "gamma") |>
  group_by(document) |>
  top_n(1, gamma) |>
  ungroup()
```

### Optimal Topics

```r
# Calculate perplexity
library(dplyr)

perp <- numeric(20)

for (k in 2:20) {
  model <- LDA(dtm, k = k, seed = 123)
  perp[k] <- perplexity(model, dtm)
}

# Plot
plot(2:20, perp[-1], type = "b")
```

## Best Practices

1. **Preprocess**: Clean and transform text.

2. **Choose K**: Use cross-validation.

3. **Interpret**: Assign labels to topics.

4. **Validate**: Check coherence.

5. **Document**: Document topic meanings.
