## Theory

ARIMA (AutoRegressive Integrated Moving Average) is a widely used time series forecasting method. It combines autoregressive (AR), differencing (I), and moving average (MA) components. The model is specified as ARIMA(p, d, q) where p = AR order, d = differencing order, q = MA order.

Key concepts: stationarity, ACF (autocorrelation), PACF (partial autocorrelation), and model selection criteria like AIC and BIC.

## Step-by-Step

1. Visualize and test data for stationarity
2. Difference data if needed
3. Identify p and q using ACF/PACF
4. Fit ARIMA model
5. Check residuals
6. Make forecasts

## Code Examples

### Basic ARIMA Fit

```r
cat("===== ARIMA BASICS =====\n\n")

library(forecast)

# Sample time series
set.seed(42)
ts_data <- ts(arima.sim(n = 100, model = list(ar = 0.5, ma = 0.3)), 
              start = c(2024, 1), frequency = 12)

# Fit ARIMA model
fit <- auto.arima(ts_data)
cat("Best model:", fit$model, "\n")
cat("Coefficients:\n")
print(coef(fit))
cat("\nAIC:", AIC(fit), "\n")
cat("BIC:", BIC(fit), "\n")
```

### Manual ARIMA

```r
cat("\n===== MANUAL ARIMA =====\n\n")

# Fit specific ARIMA(1,1,1)
fit_manual <- arima(ts_data, order = c(1, 1, 1))
cat("ARIMA(1,1,1) fit:\n")
print(coef(fit_manual))

# Different orders
cat("\nARIMA(2,1,1):\n")
fit2 <- arima(ts_data, order = c(2, 1, 1))
print(AIC(fit2))
```

### Forecasting

```r
cat("\n===== FORECASTING =====\n\n")

# Make forecast
forecast_values <- forecast(fit, h = 12)
cat("12-step ahead forecast:\n")
print(forecast_values$mean)

# Plot
cat("\nPlot available with:\n")
cat("plot(forecast_values)\n")
```

### Model Diagnostics

```r
cat("\n===== DIAGNOSTICS =====\n\n")

# Check residuals
residuals <- residuals(fit)
cat("Residual statistics:\n")
cat("Mean:", mean(residuals), "\n")
cat("SD:", sd(residuals), "\n")

# Ljung-Box test
lb_test <- Box.test(residuals, lag = 10, type = "Ljung-Box")
cat("Ljung-Box p-value:", lb_test$p.value, "\n")

# Normality test
shapiro_test <- shapiro.test(residuals)
cat("Shapiro p-value:", shapiro_test$p.value, "\n")
```

### Stationarity Testing

```r
cat("\n===== STATIONARITY TESTS =====\n\n")

library(tseries)

# Augmented Dickey-Fuller test
adf_result <- adf.test(ts_data)
cat("ADF test p-value:", adf_result$p.value, "\n")

# KPSS test
kpss_result <- kpss.test(ts_data)
cat("KPSS test p-value:", kpss_result$p.value, "\n")
```

## Real-World Example: Sales Forecasting

```r
# Real-world: Sales forecasting
cat("===== SALES FORECAST =====\n\n")

# Sample monthly sales data
sales <- ts(c(100, 105, 108, 112, 115, 118, 120, 125, 130, 135, 
              138, 142, 145, 150, 155, 158, 162, 165, 170, 175),
            start = c(2022, 1), frequency = 12)

cat("Sales data (first 12 months):\n")
print(window(sales, 1, 12))

# Fit best ARIMA
cat("\nFitting auto.arima...\n")
sales_fit <- auto.arima(sales)
cat("Selected model:", sales_fit$model, "\n")
cat("Coefficients:\n")
print(coef(sales_fit))

# Forecast next 6 months
sales_fc <- forecast(sales_fit, h = 6)
cat("\n6-month forecast:\n")
print(sales_fc$mean)

# Forecast with confidence intervals
cat("\nLower 95% CI:\n")
print(sales_fc$lower[, 2])
cat("Upper 95% CI:\n")
print(sales_fc$upper[, 2])
```

## Best Practices

1. Check stationarity before modeling
2. Use auto.arima for automated selection
3. Check residuals for whiteness
4. Consider seasonal ARIMA (SARIMA)
5. Compare multiple models with AIC/BIC
6. Cross-validate with time series CV
