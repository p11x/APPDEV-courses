## Theoretical Background

Box plots (also known as box-and-whisker plots) are powerful visualizations that display the five-number summary of a distribution: minimum, first quartile (Q1), median, third quartile (Q3), and maximum. They provide a concise way to compare distributions across groups and identify outliers.

### The Five-Number Summary

1. **Minimum**: Smallest observation (excluding outliers)
2. **First Quartile (Q1)**: 25th percentile
3. **Median**: 50th percentile (center line)
4. **Third Quartile (Q3)**: 75th percentile
5. **Maximum**: Largest observation (excluding outliers)

### Box Plot Components

- **Box**: Spans from Q1 to Q3 (interquartile range, IQR)
- **Median Line**: Horizontal line at the median
- **Whiskers**: Extend to minimum and maximum (1.5 × IQR)
- **Outliers**: Points beyond 1.5 × IQR from quartiles
- **Notch**: Optional confidence interval around median

### When to Use Box Plots

1. **Comparing Distributions**: See shape differences between groups
2. **Identifying Outliers**: Points beyond whiskers
3. **Showing Spread**: Visualize IQR and range
4. **Skewness**: Asymmetry visible in box position

## Step-by-Step Implementation

### Step 1: Load Required Packages

```r
install.packages("ggplot2")
library(ggplot2)
```

### Step 2: Basic Box Plot with Base R

```r
# Using base R
boxplot(mtcars$mpg,
        xlab = "Miles per Gallon",
        main = "Box Plot of MPG",
        col = "steelblue",
        border = "black",
        notch = TRUE)
```

### Step 3: Grouped Box Plot

```r
# Box plot by group
boxplot(mpg ~ cyl, data = mtcars,
        xlab = "Number of Cylinders",
        ylab = "Miles per Gallon",
        main = "MPG by Cylinder Count",
        col = c("lightblue", "lightgreen", "lightpink"),
        border = "black")
```

### Step 4: Using ggplot2

```r
# Basic box plot with ggplot2
ggplot(mtcars, aes(x = factor(cyl), y = mpg)) +
  geom_boxplot(fill = "steelblue", color = "black") +
  ggtitle("MPG by Cylinder Count") +
  xlab("Cylinders") +
  ylab("Miles per Gallon") +
  theme_minimal()
```

### Step 5: With Notches and Points

```r
# Box plot with notches and jittered points
ggplot(mtcars, aes(x = factor(cyl), y = mpg, fill = factor(cyl))) +
  geom_boxplot(notch = TRUE, outlier.color = "red") +
  geom_jitter(alpha = 0.5, color = "darkgray") +
  ggtitle("MPG by Cylinders (with Notches and Points)") +
  theme_minimal()
```

### Step 6: Horizontal Box Plot

```r
# Horizontal orientation
ggplot(mtcars, aes(x = mpg, y = factor(cyl))) +
  geom_boxplot(fill = "lightblue") +
  ggtitle("MPG by Cylinders (Horizontal)") +
  xlab("Miles per Gallon") +
  ylab("Cylinders") +
  theme_minimal()
```

## Code Examples

### Example 1: Iris Data Analysis

This example shows petal length distributions.

```r
# Box plot by species
ggplot(iris, aes(x = Species, y = Petal.Length, fill = Species)) +
  geom_boxplot() +
  ggtitle("Petal Length by Species") +
  theme_minimal()

# Multiple box plots side by side
ggplot(iris, aes(x = Species, y = Petal.Length, fill = Species)) +
  geom_boxplot(alpha = 0.7) +
  facet_wrap(~Species, scales = "free") +
  ggtitle("Petal Length Distribution") +
  theme_minimal()
```

### Example 2: Comparing Multiple Variables

This example shows multiple variables in one plot.

```r
# Reshape iris for multiple comparisons
iris_long <- iris %>%
  gather(key = "measure", value = "value", -Species)

# Multiple box plots
ggplot(iris_long, aes(x = measure, y = value, fill = Species)) +
  geom_boxplot(position = "dodge") +
  ggtitle("Iris Measurements by Species") +
  theme_minimal() +
  theme(axis.text.x = element_text(angle = 45))
```

### Example 3: With Statistics

This example shows adding statistical annotations.

```r
# Create box plot with mean marker
ggplot(iris, aes(x = Species, y = Sepal.Width, fill = Species)) +
  geom_boxplot() +
  stat_summary(fun = mean, geom = "point", shape = 23, fill = "white", size = 3) +
  ggtitle("Sepal Width with Mean (diamond)") +
  theme_minimal()

# Add mean labels
ggplot(iris, aes(x = Species, y = Sepal.Width, fill = Species)) +
  geom_boxplot(alpha = 0.7) +
  stat_summary(fun = "mean", aes(label = round(..y.., 2)),
               geom = "text", vjust = -0.5) +
  ggtitle("Sepal Width with Mean Values") +
  theme_minimal()
```

## Best Practices and Common Pitfalls

### Best Practices

1. **Include Outliers**: Show all points beyond whiskers
2. **Use Notches**: For comparing medians visually
3. **Order Groups**: By median value for easier comparison
4. **Label Clearly**: Include units and group names
5. **Consider Points**: Add individual points for sample size

### Common Pitfalls

1. **Ignoring Sample Size**: Small samples need caution
2. **Not Using Notches**: Missing median comparisons
3. **Wrong Orientation**: Consider data context
4. **Inconsistent Scales**: Different scales confuse comparison
5. **Too Many Groups**: Consider faceting or limiting

## Performance Considerations

### When Box Plots Excel

| Data Type | Use Case |
|-----------|----------|
| Categorical groups | Compare distributions |
| Many points | Summarize large data |
| Skewed data | Show median vs mean |
| Outliers | Identify extreme values |

## Related Concepts

- **Violin Plots**: Combines box plot with density
- **Bean Plots**: Shows rug plot and density
- **Jitter Plots**: Shows all points
- **Histograms**: Shows full distribution
