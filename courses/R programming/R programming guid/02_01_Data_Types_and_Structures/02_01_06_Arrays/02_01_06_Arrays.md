# Arrays in R

## Learning Objectives

- Understand what arrays are and their role in R
- Create and manipulate multi-dimensional arrays
- Understand array indexing and operations
- Apply arrays in numerical computing and data analysis

## Theoretical Background

### What is an Array?

An array is a data structure that can store data in more than two dimensions. While matrices are 2D, arrays can have any number of dimensions. All elements in an array must be of the same type.

### Array Properties

- **Dimensions**: Vector of dimension sizes (use `dim()`)
- **Dimnames**: Optional names for each dimension
- **Length**: Total number of elements

## Code Examples

### Standard Example: Array Creation

```r
# Array creation and manipulation
cat("===== CREATING ARRAYS =====\n\n")

# 1. Create 3D array
# Array(data, dim, dimnames)
arr3d <- array(1:24, dim = c(2, 3, 4),
              dimnames = list(
                c("A", "B"),           # 2 rows
                c("X", "Y", "Z"),      # 3 columns
                c("Time1", "Time2", "Time3", "Time4")  # 4 "layers"
              ))

cat("3D Array (2x3x4):\n")
print(arr3d)

cat("\nDimensions:", dim(arr3d), "\n")
cat("Total elements:", length(arr3d), "\n")

# 2. Access array elements
cat("\n===== ACCESSING ARRAYS =====\n\n")
cat("Element [1,2,1]:", arr3d[1, 2, 1], "\n")
cat("First row, all columns, first time:", arr3d[1, , 1], "\n")
cat("All rows, second column:", arr3d[, 2, ], "\n")
```

**Output:**
```
[1] "===== CREATING ARRAYS =====\n\n"
[1] "\n"
```

**Comments:**
- Arrays extend matrices to multiple dimensions
- dim() sets dimensions and attributes
- Indexing uses comma-separated positions

### Real-World Example: Climate Data Arrays

```r
# Real-world: 4D array for climate data (location x time x variable x scenario)
set.seed(42)

# Dimensions: 3 locations x 12 months x 2 variables x 2 scenarios
climate_data <- array(
  data = runif(144, min = 10, max = 35),  # Random temperature data
  dim = c(3, 12, 2, 2),
  dimnames = list(
    location = c("CityA", "CityB", "CityC"),
    month = month.abb,
    variable = c("Temperature", "Precipitation"),
    scenario = c("Historical", "Future")
  )
)

cat("===== CLIMATE DATA ARRAY =====\n\n")
print(climate_data)

# Get temperature for all locations, months, historical
cat("\nHistorical Temperature:\n")
print(climate_data[, , 1, 1])

# Calculate annual average by location
cat("\nAnnual Average Temperature by Location:\n")
for(loc in c("CityA", "CityB", "CityC")) {
  avg_temp <- mean(climate_data[loc, , 1, 1])
  cat(" ", loc, ":", round(avg_temp, 2), "°C\n")
}
```
