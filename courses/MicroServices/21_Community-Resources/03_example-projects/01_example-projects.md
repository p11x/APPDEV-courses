# Example Projects

## Overview

Example projects are reference implementations that demonstrate microservices patterns. They provide practical code samples for learning and as starting points for new projects.

## Available Examples

### Basic Examples

- **User Service**: Simple REST API with database
- **Order Service**: Event-driven with message queue
- **Product Service**: With caching layer

### Advanced Examples

- **E-commerce Platform**: Complete microservices reference
- **Payment System**: With PCI-DSS compliance
- **Real-time Analytics**: Streaming architecture

### Technology-Specific

- Spring Boot Microservices
- Python FastAPI Services
- Go gRPC Services
- Node.js Services
