# Community Templates

## Overview

Community templates provide standardized patterns and configurations used across microservices projects. They ensure consistency and follow best practices.

## Template Types

### Configuration Templates

- Kubernetes manifests
- Docker configurations
- CI/CD pipelines

### Documentation Templates

- README templates
- API documentation
- Runbook templates

### Testing Templates

- Unit test templates
- Integration test configs
- Contract test setups
