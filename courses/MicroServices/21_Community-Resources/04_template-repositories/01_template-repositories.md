# Template Repositories

## Overview

Template repositories provide starter code for new microservices projects. They include standard configurations, dependencies, and patterns to accelerate development.

## Available Templates

### Language-Specific Templates

- **Python FastAPI Template**: FastAPI + Docker + Kubernetes
- **Java Spring Boot Template**: Spring Boot + Gradle + Jib
- **Go Template**: Standard layout + Docker
- **Node.js Template**: Express + TypeScript

### Multi-Service Templates

- **Microservices Starter**: Multiple services + API Gateway
- **Event-Driven Template**: With message queue integration

## Using Templates

```bash
# GitHub template
gh repo create my-service --template org/microservice-template
```
