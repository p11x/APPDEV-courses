# Technical Whiteboard Exercises

## Overview

Whiteboard exercises evaluate your ability to design systems visually. Practice drawing diagrams and explaining your thinking out loud.

## Common Exercises

### Exercise 1: Design a URL shortener

**Requirements**:
- Create short URLs
- Redirect to original
- Track click statistics

**Key Components**:
- API service
- Database (Redis/SQL)
- Analytics service
- Cache layer

### Exercise 2: Design a notification system

**Requirements**:
- Support multiple channels (email, SMS, push)
- Template-based messages
- Rate limiting

### Exercise 3: Design an e-commerce checkout

**Requirements**:
- Inventory management
- Payment processing
- Order management

## Drawing Tips

1. Start with high-level architecture
2. Add data flow with arrows
3. Include key components
4. Mention scale considerations
5. Discuss trade-offs
