# Workshop Materials for Microservices

## Overview

Workshop materials provide structured learning experiences for teams adopting microservices. These materials are designed for in-person or virtual workshops led by instructors.

## Workshop Modules

### Module 1: Microservices Fundamentals
- Duration: 2 hours
- Topics: Architecture, benefits, challenges

### Module 2: Service Design
- Duration: 3 hours
- Topics: Domain-driven design, API design

### Module 3: Communication Patterns
- Duration: 2 hours
- Topics: Sync/async, service mesh

### Module 4: Data Management
- Duration: 3 hours
- Topics: Database per service, sagas

### Module 5: Operations
- Duration: 2 hours
- Topics: Kubernetes, monitoring

## Hands-On Exercises

```yaml
# Exercise: Create a Simple Service
steps:

  - name: Add database
    task: Connect to PostgreSQL
  - name: Containerize

```
