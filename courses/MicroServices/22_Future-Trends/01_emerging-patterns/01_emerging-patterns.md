# Emerging Patterns

## Overview

New architectural patterns are emerging to address evolving requirements in microservices development. This guide covers patterns that are gaining traction in the community.

## Emerging Patterns

### 1. Ambient Mesh
Lightweight service mesh without sidecar proxies using eBPF.

### 2. Composable Architecture
Building applications from independent, reusable components.

### 3. Edge Computing
Processing data at the network edge closer to users.

### 4. Serverless Microservices
Combining serverless with microservices for specific use cases.
