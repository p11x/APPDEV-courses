# Abbreviations in Microservices

## Overview

Common abbreviations used in microservices documentation and communication.

## Common Abbreviations

- **App** - Application
- **Auth** - Authentication
- **Autoscaling** - Automatic scaling
- **Backend** - Server-side
- **BFF** - Backend for Frontend
- **CDN** - Content Delivery Network
- **Config** - Configuration
- **DB** - Database
- **Deploy** - Deployment
- **Dev** - Development
- **DNS** - Domain Name System
- **DPL** - Deployment
- **Env** - Environment
- **FE** - Frontend
- **Infra** - Infrastructure
- **LB** - Load Balancer
- **Logs** - Logging
- **Metrics** - Performance metrics
- **Monolith** - Monolithic application
- **Msg** - Message
- **Num** - Number
- **Ops** - Operations
- **Org** - Organization
- **Prod** - Production
- **Pub/Sub** - Publish/Subscribe
- **PVC** - Persistent Volume Claim
- **QoS** - Quality of Service
- **Replicas** - Number of instances
- **Req** - Request
- **Resp** - Response
- **Svc** - Service
- **TLS** - Transport Layer Security
- **TTL** - Time to Live
- **UID** - Unique Identifier
- **UUID** - Universal Unique Identifier
