# Microservices Acronyms

## Overview

Common acronyms used in microservices architecture and cloud-native development.

## Common Acronyms

| Acronym | Full Form | Description |
|---------|-----------|-------------|
| API | Application Programming Interface | Interface for communication |
| CI/CD | Continuous Integration/Continuous Deployment | Build and deployment automation |
| CQRS | Command Query Responsibility Segregation | Read/write separation pattern |
| DDD | Domain-Driven Design | Software design methodology |
| HA | High Availability | System uptime design |
| IaC | Infrastructure as Code | Infrastructure management |
| JWT | JSON Web Token | Token format for authentication |
| K8s | Kubernetes | Container orchestration |
| mTLS | Mutual TLS | Bidirectional authentication |
| OPA | Open Policy Agent | Policy engine |
| POD | Plain Old Documentation | K8s deployment unit |
| RBAC | Role-Based Access Control | Authorization model |
| RTO | Recovery Time Objective | Recovery time target |
| RPO | Recovery Point Objective | Data loss tolerance |
| SLA | Service Level Agreement | Service commitment |
| SLO | Service Level Objective | Internal target |
