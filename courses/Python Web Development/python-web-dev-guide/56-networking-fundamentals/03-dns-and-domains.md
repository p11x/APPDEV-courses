## DNS Record Types

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    DNS RECORD TYPES                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  A       — IPv4 address                                                    │
│  AAAA    — IPv6 address                                                    │
│  CNAME   — Canonical name (alias)                                          │
│  MX      — Mail exchange                                                   │
│  TXT     — Text records (verification, SPF)                                │
│  NS      — Name servers                                                    │
│  SOA     — Start of authority                                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Summary

- DNS translates domain names to IP addresses
- A, AAAA, CNAME are common record types
