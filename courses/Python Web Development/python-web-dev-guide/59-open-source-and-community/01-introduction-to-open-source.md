## Prerequisites

- Basic Python knowledge

## Introduction

Open source software is software where the source code is publicly available and can be modified and distributed. Python itself is open source, as are most of the libraries you'll use in your career.

Contributing to open source is one of the best ways to improve your skills, build your reputation, and give back to the community that provides you with powerful tools.

## Why Contribute

- Learn from reading quality code
- Build your portfolio
- Network with experienced developers
- Improve tools you use daily
- Give back to the community

## Summary

- Open source powers most modern software
- Contributions benefit both you and the community
