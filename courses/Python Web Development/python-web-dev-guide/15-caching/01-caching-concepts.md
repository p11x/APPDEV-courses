# Caching Concepts

## What You'll Learn
- Caching fundamentals
- When to use caching

## Prerequisites
- Completed GraphQL folder

## What Is Caching?

Caching stores frequently accessed data in fast storage for quick retrieval.

## Types of Cache

| Type | Speed | Persistence |
|------|-------|-------------|
| Memory | Fastest | Lost on restart |
| Redis | Fast | Persistent |
| Database | Slow | Persistent |
| CDN | Fast | Edge locations |

## Summary
- Caching improves performance
- Use for expensive operations
- Cache invalidation is important
