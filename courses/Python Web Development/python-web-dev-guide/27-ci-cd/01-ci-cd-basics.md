## What You'll Learn

## Prerequisites
- Completed containerization folder

## CI/CD Pipeline

```
Code → Build → Test → Deploy
```

## Stages

1. **Source**: Push to repo
2. **Build**: Compile/code analysis
3. **Test**: Unit, integration tests
4. **Deploy**: Staging → Production
