# Python Interpreters

## What You'll Learn

- Understanding Python interpreters
- CPython vs others
- When to use each

## Prerequisites

- Basic Python knowledge

## Types of Interpreters

| Interpreter | Description |
|------------|-------------|
| CPython | Default, C-based |
| PyPy | JIT compiled, fast |
| Jython | Runs on JVM |
| IronPython | Runs on .NET |

## Summary

- CPython is the standard
- PyPy can be faster for some code
