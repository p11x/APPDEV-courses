## Debugging Checklist

1. Understand the error message
2. Reproduce the issue consistently
3. Isolate the problematic code
4. Identify root cause
5. Implement fix
6. Test the fix
7. Check for side effects

## Prevention Tips

- Write tests
- Use type hints
- Code review
- Continuous integration

## Summary

- Follow systematic approach
- Document fixes
