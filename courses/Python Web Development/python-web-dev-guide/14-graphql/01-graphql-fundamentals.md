## What Is GraphQL?

GraphQL is a query language for APIs. Unlike REST where you get fixed endpoints, GraphQL lets clients request exactly the data they need.

## REST vs GraphQL

| Feature | REST | GraphQL |
|---------|------|---------|
| Endpoints | Multiple fixed URLs | Single endpoint |
| Data | Fixed structure | Client specifies needed fields |
| Over-fetching | Common | Avoided |
| Under-fetching | Multiple requests | Single request |

## Summary
- GraphQL provides flexible API queries
- Single endpoint serves all data needs
