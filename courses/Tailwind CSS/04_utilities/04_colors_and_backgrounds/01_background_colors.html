Background Colors in Tailwind CSS

Understanding Background Colors

Tailwind CSS provides a comprehensive color palette for background colors. The bg-* utilities set the background-color property. Colors include gray, red, orange, yellow, green, teal, blue, indigo, purple, pink, and more withShade variations from 50 to 950.

Standard Example: Background Color Classes


<div class="bg-red-500 p-4 text-white rounded mb-2">bg-red-500</div>
<div class="bg-blue-500 p-4 text-white rounded mb-2">bg-blue-500</div>
<div class="bg-green-500 p-4 text-white rounded mb-2">bg-green-500</div>
<div class="bg-purple-500 p-4 text-white rounded mb-2">bg-purple-500</div>


<div class="bg-red-50 p-4 rounded mb-2">bg-red-50 (light)</div>
<div class="bg-red-100 p-4 rounded mb-2">bg-red-100</div>
<div class="bg-red-200 p-4 rounded mb-2">bg-red-200</div>


<div class="bg-gray-800 p-4 text-white rounded mb-2">bg-gray-800</div>
<div class="bg-gray-900 p-4 text-white rounded mb-2">bg-gray-900 (near black)</div>

Output: Elements with various background colors from the Tailwind palette.

│                  COLOR SHADE REFERENCE                         │

Accessibility & Performance

Contrast Requirements:

  • Use at least 4.5:1 contrast for normal text (WCAG AA)
  • bg-blue-500/600 work well with white text
  • bg-gray-500/600/700 work well with white text

Best Practices:

  • Use semantic color names for meaning (success=green, error=red)
  • Darken hover states for interactive elements
  • Use 50-100 shades for subtle backgrounds

Performance:

  • Background colors render efficiently in CSS
  • Avoid expensive pseudo-element additions
  • Reduce repaints by using transform for animations
