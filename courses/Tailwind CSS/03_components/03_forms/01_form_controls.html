Form Controls

Form controls are the building blocks of user input. Tailwind CSS provides utilities to style input fields, textareas, select dropdowns, and other form elements. Proper styling improves usability and accessibility.

Standard Example: Basic Form Controls

<!-- Text Input -->
<div>
    <label class="block text-sm font-medium text-gray-700 mb-1">Name</label>
    <input type="text" class="w-full px-3 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-blue-500" placeholder="Enter name">
</div>

<!-- Textarea -->
<div>
    <label class="block text-sm font-medium text-gray-700 mb-1">Message</label>
    <textarea class="w-full px-3 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-blue-500" rows="3"></textarea>
</div>

<!-- Select -->
<div>
    <label class="block text-sm font-medium text-gray-700 mb-1">Country</label>
    <select class="w-full px-3 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-blue-500">
        <option>Select country</option>
        <option>USA</option>
    </select>
</div>
