Form Validation

Form validation provides visual feedback for valid/invalid states. Tailwind CSS makes it easy to style input validation with colors and icons.

Standard Example: Validation States

<!-- Valid Input -->
<div>
    <label class="block text-sm font-medium text-gray-700 mb-1">Email</label>
    <input type="email" class="w-full px-3 py-2 border border-green-500 rounded-lg bg-green-50" value="valid@example.com">
    <p class="mt-1 text-sm text-green-600">✓ Email looks good!</p>
</div>

<!-- Invalid Input -->
<div>
    <label class="block text-sm font-medium text-gray-700 mb-1">Email</label>
    <input type="email" class="w-full px-3 py-2 border border-red-500 rounded-lg bg-red-50" value="invalid">
    <p class="mt-1 text-sm text-red-600">✗ Please enter a valid email.</p>
</div>

<!-- Error with Icon -->
<div class="relative">
    <input type="text" class="w-full pl-10 pr-3 py-2 border border-red-500 rounded-lg bg-red-50 focus:outline-none">
    <div class="absolute inset-y-0 right-0 pr-3 flex items-center">
        <svg class="h-5 w-5 text-red-500" fill="none" stroke="currentColor" viewBox="0 0 24 24"><path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M12 8v4m0 4h.01M21 12a9 9 0 11-18 0 9 9 0 0118 0z" /></svg>
    </div>
    <p class="mt-1 text-sm text-red-600">This field is required</p>
</div>

✓ Email looks good!

✗ Please enter a valid email.

Real-World Example: Registration Form with Validation

<form class="space-y-4">
    <!-- Username -->
    <div>
        <label class="block text-sm font-medium text-gray-700 mb-1">Username</label>
        <input type="text" class="w-full px-3 py-2 border border-gray-300 rounded-lg focus:ring-2 focus:ring-blue-500" placeholder="johndoe">
    </div>
    
    <!-- Email with Validation -->
    <div>
        <label class="block text-sm font-medium text-gray-700 mb-1">Email</label>
        <input type="email" class="w-full px-3 py-2 border border-red-500 rounded-lg bg-red-50 focus:outline-none" value="invalid-email">
        <p class="mt-1 text-sm text-red-600">Please enter a valid email address</p>
    </div>
    
    <!-- Password Strength -->
    <div class="space-y-2">
        <label class="block text-sm font-medium text-gray-700 mb-1">Password</label>
        <input type="password" class="w-full px-3 py-2 border border-gray-300 rounded-lg" value="pass123">
        
        <!-- Strength Meter -->
        <div class="space-y-1">
            <div class="flex gap-1">
                <div class="h-1 flex-1 bg-red-500 rounded"></div>
                <div class="h-1 flex-1 bg-gray-200 rounded"></div>
                <div class="h-1 flex-1 bg-gray-200 rounded"></div>
                <div class="h-1 flex-1 bg-gray-200 rounded"></div>
            </div>
            <p class="text-sm text-red-600">Weak - use at least 8 characters</p>
        </div>
    </div>
    
    <div class="pt-2">
        <button type="submit" class="w-full py-2 bg-blue-600 text-white font-semibold rounded-lg hover:bg-blue-700">
            Create Account
        </button>
    </div>
</form>

Output Statement

After implementing form validation, you will have:

  • Valid and invalid input states
  • Error messages with icons
  • Password strength indicator
  • Visual validation feedback
