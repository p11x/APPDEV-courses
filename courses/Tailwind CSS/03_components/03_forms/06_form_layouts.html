Form Layouts

Form layouts determine how form elements are arranged. Common patterns include stacked layouts, grid layouts, inline forms, and multi-column forms.

Standard Example: Common Layouts

<!-- Stacked Layout -->
<div class="space-y-4">
    <div>
        <input type="text" class="w-full border p-2" placeholder="Field 1">
    </div>
    <div>
        <input type="text" class="w-full border p-2" placeholder="Field 2">
    </div>
    <div>
        <input type="text" class="w-full border p-2" placeholder="Field 3">
    </div>
</div>

<!-- Grid Layout -->
<div class="grid grid-cols-2 gap-4">
    <input type="text" class="border p-2" placeholder="First Name">
    <input type="text" class="border p-2" placeholder="Last Name">
    <input type="email" class="col-span-2 border p-2" placeholder="Email">
</div>

<!-- Inline Layout -->
<div class="flex gap-4 items-end">
    <div class="flex-1">
        <input type="text" class="w-full border p-2" placeholder="Search">
    </div>
    <button class="px-4 py-2 bg-blue-600 text-white rounded">Search</button>
</div>
