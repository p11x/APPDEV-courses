Select Styling

Select dropdowns can be styled with Tailwind to match your design system. While custom selects require additional JavaScript, basic styling works well.

Standard Example: Select Variations

<!-- Basic Select -->
<select class="w-full px-3 py-2 border border-gray-300 rounded-lg focus:ring-2 focus:ring-blue-500">
    <option>Select an option</option>
    <option>Option 1</option>
    <option>Option 2</option>
</select>

<!-- Select with Icon -->
<div class="relative">
    <select class="w-full px-3 py-2 border border-gray-300 rounded-lg appearance-none focus:ring-2 focus:ring-blue-500">
        <option>Choose a category</option>
    </select>
    <div class="absolute inset-y-0 right-0 pr-3 flex items-center pointer-events-none">
        <svg class="h-5 w-5 text-gray-400" fill="none" stroke="currentColor" viewBox="0 0 24 24">
            <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M19 9l-7 7-7-7" /></svg>
        </div>
    </div>

<!-- Multiple Select -->
<select multiple class="w-full px-3 py-2 border border-gray-300 rounded-lg focus:ring-2 focus:ring-blue-500 h-24">
    <option>Option A</option>
    <option>Option B</option>
    <option>Option C</option>
</select>

Output Statement

After implementing select styling, you will have:

  • Basic select dropdowns
  • Select with custom arrow icons
  • Multiple select inputs
  • Filter forms with selects
