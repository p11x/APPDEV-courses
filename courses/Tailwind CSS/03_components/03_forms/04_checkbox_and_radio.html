Checkbox and Radio

Checkboxes and radio buttons are essential form controls. Tailwind CSS provides default styling as well as custom styling options.

Standard Example: Basic Controls

<!-- Default Checkbox -->
<div class="flex items-center">
    <input type="checkbox" class="h-4 w-4 text-blue-600 border-gray-300 rounded">
    <span class="ml-2 text-gray-700">Remember me</span>
</div>

<!-- Default Radio -->
<div class="flex items-center space-x-4">
    <div class="flex items-center">
        <input type="radio" name="option" class="h-4 w-4 text-blue-600 border-gray-300">
        <span class="ml-2 text-gray-700">Option A</span>
    </div>
    <div class="flex items-center">
        <input type="radio" name="option" class="h-4 w-4 text-blue-600 border-gray-300">
        <span class="ml-2 text-gray-700">Option B</span>
    </div>
</div>

<!-- Toggle Switch -->
<div class="flex items-center">
    <button type="button" class="relative inline-flex h-6 w-11 items-center rounded-full bg-gray-200">
        <span class="inline-block h-4 w-4 transform rounded-full bg-white transition"></span>
    </button>
    <span class="ml-3 text-gray-700">Enable notifications</span>
</div>

Output Statement

After implementing checkboxes and radio buttons, you will have:

  • Default checkbox and radio controls
  • Checkbox groups for multiple selections
  • Radio groups for single selections
  • Proper labels and accessibility
