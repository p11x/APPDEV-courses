Card Variants

Card variants provide different visual styles for different contexts. Tailwind CSS enables creating bordered cards, outline cards, gradient cards, overlay cards, and interactive cards.

Standard Example: Card Variant Types

<!-- Bordered Card -->
<div class="max-w-sm bg-white border border-gray-200 rounded-lg p-6">
    <h3 class="font-bold">Bordered Card</h3>
</div>

<!-- Outline Card -->
<div class="max-w-sm border-2 border-dashed border-gray-300 rounded-lg p-6">
    <h3 class="font-bold text-gray-600">Outline Card</h3>
</div>

<!-- Gradient Card -->
<div class="max-w-sm bg-gradient-to-br from-purple-500 to-pink-500 rounded-lg p-6 text-white">
    <h3 class="font-bold">Gradient Card</h3>
</div>

<!-- Dark Card -->
<div class="max-w-sm bg-gray-900 rounded-lg p-6 text-white">
    <h3 class="font-bold">Dark Card</h3>
</div>
