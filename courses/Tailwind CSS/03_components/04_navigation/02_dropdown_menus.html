Dropdown Menus

Dropdown menus reveal additional navigation or actions when triggered. They are commonly used for user menus, submenus, and action buttons.

Standard Example: Basic Dropdown

<!-- Dropdown Button -->
<div class="relative inline-block">
    <button class="px-4 py-2 bg-gray-800 text-white rounded-lg hover:bg-gray-700">
        Options
        <svg class="w-4 h-4 inline-block ml-1" fill="none" stroke="currentColor" viewBox="0 0 24 24">
            <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M19 9l-7 7-7-7" /></svg>
    </button>
    <div class="absolute right-0 mt-2 w-48 bg-white rounded-lg shadow-xl border hidden">
        <a href="#" class="block px-4 py-2 text-gray-700 hover:bg-gray-100">Profile</a>
        <a href="#" class="block px-4 py-2 text-gray-700 hover:bg-gray-100">Settings</a>
        <div class="border-t"></div>
        <a href="#" class="block px-4 py-2 text-red-600 hover:bg-gray-100">Logout</a>
    </div>
</div>
