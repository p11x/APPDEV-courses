Breadcrumb

Breadcrumbs show the user's location in a site hierarchy. They improve navigation and help users understand where they are in the website structure.

Standard Example: Basic Breadcrumbs

<!-- Simple Breadcrumb -->
<nav class="flex" aria-label="Breadcrumb">
    <ol class="flex items-center space-x-2">
        <li>
            <a href="#" class="text-gray-600 hover:text-gray-900">Home</a>
        </li>
        <li>
            <span class="text-gray-400">/</span>
        </li>
        <li>
            <a href="#" class="text-gray-600 hover:text-gray-900">Products</a>
        </li>
        <li>
            <span class="text-gray-400">/</span>
        </li>
        <li>
            <span class="text-gray-900 font-medium">Current Page</span>
        </li>
    </ol>
</nav>
  1. Home
  2. /
  3. Products
  4. /
  5. Current Page
