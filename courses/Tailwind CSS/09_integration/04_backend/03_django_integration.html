Django + Tailwind CSS Integration

Overview

Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. Tailwind CSS integrates with Django through the static file pipeline. Django doesn't have a built-in CSS processor, so you use npm scripts or a Django package like django-tailwind to integrate PostCSS compilation during development.

Key Point: Django requires external tooling for Tailwind CSS. Use django-tailwind for seamless integration, or configure npm scripts directly for the build process.

Standard Example: Django Setup

Setting up Tailwind CSS with Django:

# Install Django-Tailwind (recommended)

pip install django-tailwind

 

# settings.py - add to INSTALLED_APPS

INSTALLED_APPS = [

# ...

'tailwind',

]

 

TAILWIND_APP = {

'path': os.path.join(BASE_DIR, 'static_src'),

}

 

# Alternative: Manual npm setup

npm init -y

npm install -D tailwindcss postcss autoprefixer

npx tailwindcss init

 

# tailwind.config.js

module.exports = {

content: [

"./templates/**/*.html",

"./**/*.py"

],

theme: { extend: {} },

plugins: []

};

 

# package.json scripts

"scripts": {

"dev": "tailwindcss -i ./static_src/input.css -o ./static/css/output.css --watch",

"build": "tailwindcss -i ./static_src/input.css -o ./static/css/output.css --minify"

}

 

# static_src/input.css

@tailwind

base;

@tailwind

components;

@tailwind

utilities;

Edge Cases & Considerations

  • django-debug-toolbar: Works with Tailwind for development debugging.
  • Django REST Framework: Works with DRF for API + Tailwind frontend clients.
  • Caching: Use Django's cache framework with Tailwind builds.
  • whitenoise: Serve static assets directly in production.

Performance Considerations

  • WhiteNoise: Serve static files efficiently in production.
  • CDN: Works with CloudFront and other CDNs.
  • Caching: Cache-bust using Django's static version.
  • Async: Consider Django Channels for real-time with Tailwind.

Accessibility Notes

  • Forms: Django forms generate proper labels for accessibility.
  • Error Messages: Display form errors inline with Tailwind.
  • Messages: Use django.contrib.messages with Tailwind styling.
  • Skip Links: Include skip-to-main in base template.

Cross-Browser Notes

  • Autoprefixer: npm handles vendor prefixes.
  • Python: Works on any Python-supported platform.
  • Testing: Test served static assets across browsers.
  • Production: Test with gunicorn/uwsgi servers.
