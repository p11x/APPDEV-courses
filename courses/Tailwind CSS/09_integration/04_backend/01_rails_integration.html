Ruby on Rails + Tailwind CSS Integration

Overview

Ruby on Rails is a full-stack web application framework that emphasizes convention over configuration. Tailwind CSS integrates with Rails through the asset pipeline (Sprockets) or the esbuild/webpacker options in Rails 7. The tailwindcss-rails gem provides the easiest integration for Rails applications.

Key Point: Rails 7+ with esbuild provides the modern default. Use the tailwindcss-rails gem for seamless integration, or configure PostCSS directly for the asset pipeline.

Standard Example: Rails 7 Setup

Setting up Tailwind CSS with Rails 7 using esbuild:

# Gemfile

gem "tailwindcss-rails"

 

# Install

bundle install

rails tailwindcss:install

 

# config/tailwind.config.js

const defaultTheme = require('tailwindcss/defaultTheme')

 

module.exports = {

content: [

'./app/views/**/*.html.erb',

'./app/components/**/*.html.erb',

'./app/helpers/**/*.rb',

'./app/javascript/**/*.js'

],

theme: {

extend: {}

},

plugins: []

}

 

# app/assets/stylesheets/application.css

@tailwind

base;

@tailwind

components;

@tailwind

utilities;

 

# app/layouts/application.html.erb

<%= stylesheet_link_tag "application", "data-turbolinks-track": "reload" %>

<%= javascript_include_tag "application", "data-turbolinks-track": "reload" %>

Edge Cases & Considerations

  • Turbo/Stimulus: Works well with Rails' Turbo and Stimulus for SPA-like features.
  • Active Storage: File uploads can use Tailwind-styled previews.
  • Action Mailer: Use Tailwind email plugins for mail templates.
  • Hotwire: Full compatibility with Hotwire for reactive features.

Performance Considerations

  • Asset Pipeline: Automatic fingerprinting and compression.
  • CDN Ready: Assets work with CloudFront and similar CDNs.
  • Turbo Drive: Fast page navigations with smart caching.
  • Cache Digests: Automatic cache invalidation.

Accessibility Notes

  • Form Helpers: Use Rails form helpers that generate proper labels.
  • Validation Errors: Associate errors with inputs for accessibility.
  • Flash Messages: Include aria-live for dynamic messages.
  • Skip Links: Add skip-to-main-content links.

Cross-Browser Notes

  • Asset Pipeline: Handles vendor prefixes via Autoprefixer.
  • Sprockets: Works on all modern browsers.
  • Testing: BrowserStack recommended for full testing.
  • Legacy: Can support IE11 with additional polyfills.
