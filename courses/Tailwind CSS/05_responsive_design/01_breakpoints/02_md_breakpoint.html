MD Breakpoint (768px)

Understanding the MD Breakpoint

The md breakpoint in Tailwind CSS targets screens with a minimum width of 768px. This breakpoint is commonly used for tablets in landscape mode, small laptops, and large smartphones. It's a crucial breakpoint for transitioning from mobile to desktop layouts.

When to Use MD Breakpoint

  • Switching from single-column to two-column layouts
  • Displaying sidebar navigation instead of hamburger menu
  • Increasing text sizes for better readability
  • Showing/hiding elements based on content importance

Standard Example: Layout Transition at MD

<!-- Base: single column layout (mobile) -->
<div class="flex flex-col md:flex-row">
    <!-- Sidebar/Content area -->
    <div class="w-full md:w-1/4 bg-gray-200 p-4">
        Sidebar content - full width on mobile, 1/4 on md+
    </div>
    <div class="w-full md:w-3/4 p-4">
        Main content - full width on mobile, 3/4 on md+
    </div>
</div>

<!-- Text alignment example -->
<p class="text-center md:text-left">
    Centered on mobile, left-aligned on md+ screens
</p>

<!-- Padding adjustment -->
<div class="p-4 md:p-8 lg:p-12">
    Padding increases progressively with screen size
</div>

Output Statement:

Below 768px: Vertical stack layout, all elements full width, centered text.
At 768px and above: Side-by-side layout (1/4 + 3/4 columns), left-aligned text.

Edge Cases and Best Practices

⚠️ Content Reflow Issues

When switching layouts at md, ensure content doesn't become orphaned or squeezed. Use flex-wrap and grid to control reflow.

✓ Performance Tip

Use md: prefix for loading heavy content only on larger screens to improve mobile performance.

🔍 Testing Tip

Test at exactly 768px width to catch edge cases where elements might appear misaligned or overlap.

♿ Accessibility

Ensure interactive elements remain accessible at both breakpoints. Maintain minimum 44px touch targets on mobile.

Common MD Breakpoint Patterns

  • flex-col md:flex-row - Stack vertically on mobile, row on desktop
  • w-full md:w-1/2 - Full width mobile, half width desktop
  • hidden md:block - Show element only from md and up
  • text-base md:text-lg - Increase font size on larger screens
  • p-4 md:p-6 lg:p-8 - Progressive padding increase
