Responsive Typography

Understanding Responsive Typography

Tailwind CSS provides comprehensive responsive typography utilities that control font size, weight, line height, letter spacing, and text alignment across different breakpoints. Typography is crucial for readability and visual hierarchy on any device.

Font Size Scale

text-xs - 0.75rem
text-sm - 0.875rem
text-base - 1rem
text-lg - 1.125rem
text-xl - 1.25rem
text-2xl - 1.5rem
text-3xl - 1.875rem
text-4xl - 2.25rem

Standard Example: Fluid Typography

<!-- Progressive text sizing -->
<h1 class="text-3xl md:text-4xl lg:text-5xl xl:text-6xl font-bold">
    Responsive Heading
</h1>

<!-- Body text with responsive size -->
<p class="text-base md:text-lg lg:text-xl text-gray-700">
    This paragraph adjusts size from 16px on mobile to 20px on large screens.
</p>

<!-- Line height with responsive spacing -->
<article class="leading-relaxed md:leading-loose">
    <p>Text with comfortable line height that adjusts for readability.</p>
</article>

<!-- Responsive text alignment -->
<p class="text-center md:text-left lg:text-justify">
    Text alignment changes across breakpoints for optimal presentation.
</p>

Output Statement:

Mobile: 48px heading, 16px body, comfortable leading, centered.
Tablet: 64px heading, 18px body, looser leading, left-aligned.
Desktop: 80px heading, 20px body, very loose leading, justified.

Typography Utility Reference

Font Size

  • text-xs to text-9xl
  • Responsive: text-sm md:text-base lg:text-lg

Font Weight

  • font-light (300), font-normal (400)
  • font-medium (500), font-semibold (600)
  • font-bold (700), font-black (900)

Line Height

  • leading-none - 1
  • leading-tight - 1.25
  • leading-normal - 1.5
  • leading-relaxed - 1.625
  • leading-loose - 2

Text Alignment

  • text-left, text-center
  • text-right, text-justify
  • Responsive: text-center md:text-left

Accessibility Guidelines

Minimum Text Size

Use at least 16px (text-base) for body text on mobile. Smaller text should only be used for captions, labels, or decorative elements.

Line Length

Optimal line length is 50-75 characters. Use max-width containers to prevent overly long lines on large screens.

Line Height

Use leading-relaxed or leading-loose for larger text blocks to improve readability.

Color Contrast

Ensure minimum 4.5:1 contrast ratio for normal text and 3:1 for large text.

Pro Tip: Use REM Units

Tailwind uses REM units for font sizes, which scales with the user's browser settings. This ensures text remains readable even when users have increased their default font size for accessibility. Always test with browser zoom and font size adjustments.
