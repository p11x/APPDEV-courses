Just-In-Time (JIT) Compiler Overview

What is JIT in Tailwind CSS?

The Just-In-Time (JIT) compiler is a new rendering engine introduced in Tailwind CSS v3.0 that generates CSS on-demand rather than generating a large static CSS file with all possible utilities. Instead of pre-generating thousands of utility classes, the JIT compiler only generates the CSS for the classes that you actually use in your project.

How JIT Works

Traditional Tailwind CSS (v2) used a post-CSS approach that generated a large CSS file containing all possible utility classes. This resulted in large file sizes even if you only used a small subset of utilities. The JIT compiler solves this by scanning your content files (HTML, JavaScript, Vue, React, etc.) and generating only the CSS needed for the classes you actually use.

Output Statement

With JIT enabled, a typical project that might have generated 3MB+ of CSS with classic Tailwind now generates only 10-50KB of CSS, containing only the utilities actually used in your templates.

Key Benefits of JIT

  • Smaller CSS file sizes - Only generates what's used
  • Faster build times - No scanning all possible utilities
  • All CSS features available by default - No need for purge
  • Arbitrary values support - Use any value without configuration
  • Better development experience - Faster hot reloads

Enabling JIT

JIT is enabled by default in Tailwind CSS v3.0+. For earlier versions, add this to your tailwind.config.js: 

module.exports = {
  mode: 'jit',
  // ... rest of config
}

Edge Cases and Considerations

  • Dynamic classes from string concatenation may not be detected
  • Classes used in JavaScript template literals need special handling
  • Content configuration must include all relevant file patterns
  • Some edge cases require safelisting specific classes

Performance Considerations

  • Initial build may be slower as it scans all content files
  • Watch mode provides fast incremental rebuilds
  • Content files should be properly configured for optimal scanning

Cross-Browser Notes

JIT-generated CSS is fully compatible with all modern browsers. The output CSS uses standard CSS features that work across Chrome, Firefox, Safari, and Edge. No additional polyfills are needed.
