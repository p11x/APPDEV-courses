Margin Utilities in Tailwind CSS

Understanding Margin

Margin creates external space around an element, pushing it away from adjacent elements. Key特性: margins collapse vertically.

Important: Margin collapse means when two vertical margins meet, they combine to the larger value instead of summing.

Standard Example: Basic Margin

<!-- Margin on all sides -->
<div class="m-4 bg-blue-500">
    <!-- 16px margin all around -->
</div>

<!-- Margin on specific axes -->
<div class="my-8 mx-4">
    <!-- 32px vertical, 16px horizontal -->
</div>

<!-- Individual sides -->
<div class="mt-4 mr-8 mb-4 ml-8">
    <!-- top, right, bottom, left -->
</div>

Real-World Example 1: Section Spacing

<section class="py-12">
    <!-- Vertical padding for section spacing -->
    <h2 class="mb-6">Section Title</h2>
    <p class="mb-4">First paragraph with margin-bottom.</p>
    <p class="mb-4">Second paragraph with margin-bottom.</p>
    <div class="mb-8">Content block with more space.</div>
</section>

Real-World Example 2: Flex Spacing

<!-- Use space-between for flex items -->
<div class="flex justify-between mb-8">
    <span>Left</span>
    <span>Right</span>
</div>

<!-- Or use space utilities (recommended) -->
<div class="flex space-x-4">
    <button>One</button>
    <button>Two</button>
    <button>Three</button>
</div>

Output Statement

  • m-0 to m-96 provide consistent margin scale
  • my- and mx- for axis-specific margins
  • mt-, mr-, mb-, ml- for individual sides
  • Margins collapse vertically between block elements

Edge Cases

  • Use first:mt-0 or last:mb-0 to override margin on first/last children
  • Adding padding to parent prevents margin collapse
  • Flex containers don't collapse margins between items
