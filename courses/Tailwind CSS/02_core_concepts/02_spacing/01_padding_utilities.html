Padding Utilities in Tailwind CSS

Understanding Padding

Padding creates internal space within an element, pushing content away from its edges. Unlike margin, padding is inside the border and affects the element's dimensions.

Axis-Based Padding

Use py- for vertical and px- for horizontal padding.

py-1 px-4
py-2 px-6
py-4 px-8

Standard Example: Button Padding

<!-- Small button padding -->
<button class="py-1 px-3 text-sm">
    Small
</button>

<!-- Medium button (default) -->
<button class="py-2 px-4">
    Medium
</button>

<!-- Large button padding -->
<button class="py-3 px-6 text-lg">
    Large
</button>

<!-- Icon with padding -->
<button class="p-3">
    <svg>...</svg>
</button>

Real-World Example 1: Card Component

<div class="bg-white rounded-lg shadow p-6">
    <!-- p-6 = 24px all around -->
    <h3 class="font-bold mb-2">Title</h3>
    <p class="text-gray-600">Content with good internal breathing room.</p>
</div>

<!-- Compact card variant -->
<div class="bg-white rounded p-3">
    <!-- p-3 = 12px for tighter spacing -->
    Compact content
</div>

Card Title

Content with good internal breathing room using p-6.

Output Statement

  • p-0 to p-96 provide consistent padding scale
  • py- and px- allow axis-specific padding
  • Padding is included in element dimensions (border-box)
  • Default scale: 0, 0.5, 1, 1.5, 2, 2.5, 3, 3.5, 4, 5, 6, 7, 8, 9, 10, 11, 12, 14, 16
