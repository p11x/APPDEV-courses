Spacing Variants in Tailwind CSS

Responsive Spacing Variants

Tailwind supports responsive spacing that changes based on breakpoints. Use the prefix: breakpoint:spacing.

Standard Example: Responsive Padding

<!-- Different padding at different sizes -->
<div class="
    p-4 
    md:p-8 
    lg:p-12
">
    Responsive padding example
</div>
Resize window to see padding change

Real-World Example: Responsive Container

<div class="
    px-4 
    sm:px-6 
    md:px-8 
    lg:px-12
    py-8 
    md:py-12
">
    Content adapts to screen size
</div>

State Variants: Hover and Focus

<!-- Padding on hover -->
<button class="
    p-4 
    hover:p-6
">
    Hover to expand padding
</button>

<!-- Margin on focus -->
<input class="
    p-4 
    focus:mb-2
">

Output Statement

  • sm:, md:, lg:, xl: prefixes for responsive spacing
  • hover: for mouse interaction spacing
  • focus: for keyboard interaction
  • Combine with base spacing for mobile-first approach
