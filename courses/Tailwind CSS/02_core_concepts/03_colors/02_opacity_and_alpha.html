Opacity and Alpha in Tailwind CSS

Opacity Utilities

Opacity utilities control the transparency of an element, with values from 0 (invisible) to 100 (fully opaque).

Standard Example: Using Opacity

<!-- Semi-transparent overlay -->
<div class="bg-black opacity-50">
    50% transparent black overlay
</div>

<!-- Image with overlay -->
<div class="relative">
    <img src="photo.jpg" class="w-full">
    <div class="absolute inset-0 bg-blue-500 opacity-75">
        Text on overlay
    </div>
</div>

<!-- Placeholder state -->
<div class="bg-gray-200 opacity-50">
    Loading placeholder
</div>
50% opacity overlay

Real-World Example 1: Backdrop Blur

<!-- Frosted glass effect -->
<div class="bg-white bg-opacity-90 backdrop-blur">
    Frosted glass panel
</div>

<!-- Modal overlay -->
<div class="fixed inset-0 bg-black bg-opacity-50">
    <!-- Modal content here -->
</div>
Frosted glass effect with backdrop-blur

Real-World Example 2: Interactive States

<button class="
    bg-blue-500 
    hover:opacity-90 
    active:opacity-75
    transition-opacity
">
    Button with opacity transitions
</button>

Output Statement

  • opacity-0 (0%) to opacity-100 (100%) available
  • Common values: 0, 5, 10, 20, 25, 30, 40, 50, 60, 70, 75, 80, 90, 95, 100
  • Use with bg-opacity-{value} for background color opacity
  • Combine with backdrop-blur for glass effects
