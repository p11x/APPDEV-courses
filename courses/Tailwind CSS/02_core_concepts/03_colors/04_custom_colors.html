Custom Colors in Tailwind CSS

Extending the Color Palette

You can add custom colors to your Tailwind config to use throughout your project.

Standard Example: Using Arbitrary Values

<!-- Arbitrary color value -->
<div class="bg-[#1e40af]">
    Custom hex color
</div>

<!-- Arbitrary RGB -->
<div class="bg-[rgb(30,64,175)]">
    RGB color value
</div>

<!-- Arbitrary with opacity -->
<div class="bg-[#1e40af]/50">
    50% opacity hex
</div>

tailwind.config.js Configuration

// tailwind.config.js
module.exports = {
    theme: {
        extend: {
            colors: {
                brand: {
                    50: '#eff6ff',
                    100: '#dbeafe',
                    500: '#3b82f6',
                    600: '#2563eb',
                    700: '#1d4ed8',
                },
                primary: '#3b82f6',
            }
        }
    }
}

Real-World Example: Brand Colors

<!-- Using custom brand color -->
<div class="bg-indigo-600 text-white p-4 rounded">
    Brand primary button
</div>

<!-- Using brand as text color -->
<h2 class="text-indigo-600 font-bold">
    Brand heading
</h2>
Brand button (using built-in indigo)

Output Statement

  • Use arbitrary values with square brackets: bg-[#hex]
  • Add custom colors in tailwind.config.js
  • Custom colors follow same shade pattern as built-in
  • Use / for opacity modifier with arbitrary values
