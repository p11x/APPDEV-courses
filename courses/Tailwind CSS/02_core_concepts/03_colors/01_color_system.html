Standard Example: Using Background Colors

<!-- Basic background color -->
<div class="bg-blue-500">
    Blue background
</div>

<!-- Light background -->
<div class="bg-gray-100">
    Light gray background
</div>

<!-- Dark background -->
<div class="bg-gray-900 text-white">
    Dark background with white text
</div>

Output Statement

  • 10 shades per color from 50 (lightest) to 900 (darkest)
  • bg-{color}-{shade} sets background color
  • Combine with text-{color}-{shade} for text color
  • Use lighter shades for backgrounds, darker for text
