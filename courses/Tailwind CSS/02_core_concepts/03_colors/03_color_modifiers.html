Color Modifiers in Tailwind CSS

State-Based Color Modifiers

Colors can be modified based on interactive states: hover, focus, active, and more.

Standard Example: Interactive States

<!-- Hover state -->
<div class="bg-blue-500 hover:bg-blue-600">
    Hover over me
</div>

<!-- Focus state -->
<input class="border focus:border-blue-500 focus:ring-2">

<!-- Multiple states -->
<button class="
    bg-green-500 
    hover:bg-green-600 
    focus:bg-green-700
    active:bg-green-800
">
    Multi-state button
</button>
Hover me

Responsive Color Modifiers

<!-- Different colors at different breakpoints -->
<div class="
    bg-blue-500 
    md:bg-purple-500 
    lg:bg-pink-500
">
    Resize to see color change
</div>
Resize window (blue → purple → pink)

Real-World Example: Interactive Card

<div class="
    bg-white border rounded-lg p-6
    hover:shadow-lg hover:border-blue-300
    transition-all duration-300
">
    <h3 class="font-bold">Card Title</h3>
    <p class="text-gray-600 hover:text-gray-900">
        Interactive card with hover effects.
    </p>
</div>

Card Title

Interactive card with hover effects.

Output Statement

  • hover: - changes on mouse hover
  • focus: - changes on keyboard focus
  • active: - changes on click/press
  • md:, lg: - responsive breakpoints
  • Combine modifiers: hover:focus:ring-blue-500
