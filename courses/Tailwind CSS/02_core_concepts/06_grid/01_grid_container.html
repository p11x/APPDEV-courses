Grid Container in Tailwind CSS

Creating a CSS Grid

Use grid to create a block-level grid container or inline-grid for an inline grid container.

Standard Example: Basic Grid

<!-- 3-column grid (12 equal columns by default) -->
<div class="grid">
    <div class="p-4 bg-blue-500 text-white">Item 1</div>
    <div class="p-4 bg-blue-500 text-white">Item 2</div>
    <div class="p-4 bg-blue-500 text-white">Item 3</div>
    <div class="p-4 bg-blue-500 text-white">Item 4</div>
    <div class="p-4 bg-blue-500 text-white">Item 5</div>
    <div class="p-4 bg-blue-500 text-white">Item 6</div>
</div>

<!-- Define column count explicitly -->
<div class="grid grid-cols-3">
    <div class="p-4 bg-green-500 text-white">1</div>
    <div class="p-4 bg-green-500 text-white">2</div>
    <div class="p-4 bg-green-500 text-white">3</div>
</div>

Output Statement

  • grid - creates block-level grid container
  • inline-grid - creates inline-level grid container
  • grid-cols-{n} - defines number of columns (1-12)
