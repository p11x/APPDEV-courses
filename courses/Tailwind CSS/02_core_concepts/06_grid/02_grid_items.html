Grid Items in Tailwind CSS

Grid Item Spanning

Control how many columns or rows a grid item spans using col-span-{n} and row-span-{n}.

Standard Example: Spanning Items

<div class="grid grid-cols-4 gap-2">
    <!-- Spans 1 column (default) -->
    <div class="p-4 bg-blue-500 text-white">1</div>
    
    <!-- Spans 2 columns -->
    <div class="col-span-2 p-4 bg-green-500 text-white">col-span-2</div>
    
    <!-- Spans 3 columns -->
    <div class="col-span-3 p-4 bg-purple-500 text-white">col-span-3</div>
    
    <!-- Full width -->
    <div class="col-span-4 p-4 bg-red-500 text-white">col-span-4 (full)</div>
</div>

Row Spanning

<div class="grid grid-cols-3 grid-rows-3 gap-2 h-64">
    <div class="row-span-2 bg-blue-500 text-white p-4">row-span-2</div>
    <div class="row-span-3 bg-green-500 text-white p-4">row-span-3</div>
    <div class="col-span-2 row-span-2 bg-purple-500 text-white p-4">2x2</div>
</div>
row-2
row-3
2x2

Output Statement

  • col-span-{1-12} - number of columns to span
  • row-span-{1-6} - number of rows to span
  • col-start-{1-13} - start at specific column line
  • row-start-{1-7} - start at specific row line
