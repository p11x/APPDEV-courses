Grid Template in Tailwind CSS

Defining Grid Columns and Rows

Use grid-template-columns and grid-template-rows to explicitly define your grid structure.

Standard Example: Column Templates

<!-- Fixed width columns -->
<div class="grid grid-cols-[100px_200px_100px]">
    <div class="bg-blue-500 p-4">A</div>
    <div class="bg-blue-500 p-4">B</div>
    <div class="bg-blue-500 p-4">C</div>
</div>

<!-- Mixed: fraction and fixed -->
<div class="grid grid-cols-[1fr_300px_1fr]">
    <div class="bg-green-500 p-4">1fr</div>
    <div class="bg-green-500 p-4">300px</div>
    <div class="bg-green-500 p-4">1fr</div>
</div>
1fr
300px
1fr

Row Templates

<!-- Define row heights -->
<div class="grid grid-rows-[100px_auto_100px] h-64">
    <div class="bg-purple-500 p-4 text-white">Header (100px)</div>
    <div class="bg-blue-500 p-4 text-white">Content (auto)</div>
    <div class="bg-purple-500 p-4 text-white">Footer (100px)</div>
</div>
Header
Content
Footer

Output Statement

  • grid-cols-{1-12} - predefined column counts
  • grid-cols-[values] - arbitrary column widths
  • fr unit - fractional space allocation
  • auto - content-based sizing
