Responsive Grid in Tailwind CSS

Responsive Grid Patterns

Use responsive prefixes to create grids that adapt to different screen sizes.

Standard Example: Adaptive Columns

<!-- Mobile: 1 col, Tablet: 2 cols, Desktop: 3-4 cols -->
<div class="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4">
    <div class="bg-blue-500 p-4 text-white rounded">Item 1</div>
    <div class="bg-blue-500 p-4 text-white rounded">Item 2</div>
    <div class="bg-blue-500 p-4 text-white rounded">Item 3</div>

Output Statement

  • sm:grid-cols-{n} - tablet breakpoint (640px+)
  • md:grid-cols-{n} - small desktop (768px+)
  • lg:grid-cols-{n} - desktop (1024px+)
  • xl:grid-cols-{n} - large desktop (1280px+)
  • Mobile-first: start with grid-cols-1 for smallest screens
