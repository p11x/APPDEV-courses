Justify Content in Tailwind CSS

Justify Content Utilities

Control alignment along the main axis (horizontal for flex-row, vertical for flex-col).

Standard Example: Justify Options

<!-- Start (default) -->
<div class="flex justify-start">
    <div class="p-2 bg-blue-500 text-white">A</div>
    <div class="p-2 bg-blue-500 text-white">B</div>
</div>

<!-- End -->
<div class="flex justify-end">
    <div class="p-2 bg-green-500 text-white">A</div>
    <div class="p-2 bg-green-500 text-white">B</div>
</div>

<!-- Center -->
<div class="flex justify-center">
    <div class="p-2 bg-purple-500 text-white">A</div>
    <div class="p-2 bg-purple-500 text-white">B</div>
</div>

<!-- Space between -->
<div class="flex justify-between">
    <div class="p-2 bg-red-500 text-white">A</div>
    <div class="p-2 bg-red-500 text-white">B</div>
</div>

<!-- Space around -->
<div class="flex justify-around">
    <div class="p-2 bg-yellow-500 text-white">A</div>
    <div class="p-2 bg-yellow-500 text-white">B</div>
</div>

Real-World Example: Navigation

Output Statement

  • justify-start - align to start (default)
  • justify-end - align to end
  • justify-center - center alignment
  • justify-between - space between items
  • justify-around - space around items
  • justify-evenly - equal space everywhere
