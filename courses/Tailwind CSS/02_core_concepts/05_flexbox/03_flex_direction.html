Flex Direction in Tailwind CSS

Flex Direction Utilities

Control the direction items are placed in the flex container - horizontally (row) or vertically (column).

Standard Example: Direction Options

<!-- Row (default) - left to right -->
<div class="flex flex-row">
    <div class="p-2 bg-blue-500 text-white">A</div>
    <div class="p-2 bg-blue-500 text-white">B</div>
    <div class="p-2 bg-blue-500 text-white">C</div>
</div>

<!-- Row reversed - right to left -->
<div class="flex flex-row-reverse">
    <div class="p-2 bg-green-500 text-white">A</div>
    <div class="p-2 bg-green-500 text-white">B</div>
    <div class="p-2 bg-green-500 text-white">C</div>
</div>

<!-- Column - top to bottom -->
<div class="flex flex-col">
    <div class="p-2 bg-purple-500 text-white">1</div>
    <div class="p-2 bg-purple-500 text-white">2</div>
    <div class="p-2 bg-purple-500 text-white">3</div>
</div>

<!-- Column reversed - bottom to top -->
<div class="flex flex-col-reverse">
    <div class="p-2 bg-red-500 text-white">1</div>
    <div class="p-2 bg-red-500 text-white">2</div>
    <div class="p-2 bg-red-500 text-white">3</div>
</div>

Real-World Example: Mobile-First Stack

<!-- Stack vertically on mobile, row on desktop -->
<div class="flex flex-col md:flex-row">
    <div class="p-4 bg-white shadow">Item 1</div>
    <div class="p-4 bg-white shadow">Item 2</div>
    <div class="p-4 bg-white shadow">Item 3</div>
</div>

Output Statement

  • flex-row - horizontal, left to right
  • flex-row-reverse - horizontal, right to left
  • flex-col - vertical, top to bottom
  • flex-col-reverse - vertical, bottom to top
