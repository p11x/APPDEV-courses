Flex Container in Tailwind CSS

Creating a Flex Container

Use flex or inline-flex to create a flex container. The difference: flex creates a block-level container, inline-flex creates an inline-level container.

Standard Example: Flex Containers

<!-- Block-level flex container (like div) -->
<div class="flex">
    <div class="p-4 bg-blue-500 text-white">1</div>
    <div class="p-4 bg-blue-500 text-white">2</div>
    <div class="p-4 bg-blue-500 text-white">3</div>
</div>

<!-- Inline flex container (like span) -->
<span class="inline-flex">
    <div class="p-4 bg-green-500 text-white">A</div>
    <div class="p-4 bg-green-500 text-white">B</div>
</span>
1
2

Real-World Example 1: Navigation Bar

    <div class="text-xl font-bold">Logo</div>

        <a href="#" class="text-gray-600 hover:text-gray-900">About</a>

    </div>
</nav>

Output Statement

  • flex - creates a block-level flex container
  • inline-flex - creates an inline-level flex container
  • Children become flex items automatically
