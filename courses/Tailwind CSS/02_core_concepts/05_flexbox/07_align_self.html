Align Self in Tailwind CSS

Align Self Utilities

Override the container's align-items setting for individual flex items. Useful for special positioning of specific items.

Standard Example: Individual Item Alignment

<!-- Auto (use container's align-items) -->
<div class="flex items-center h-24">
    <div class="p-2 bg-blue-500 text-white">Auto</div>
    <div class="self-start p-2 bg-green-500 text-white">Start</div>
    <div class="self-center p-2 bg-purple-500 text-white">Center</div>
    <div class="self-end p-2 bg-red-500 text-white">End</div>
    <div class="self-stretch p-2 bg-yellow-500 text-white">Stretch</div>
</div>
Auto Start Center End Stretch

Real-World Example: Form with Button

<!-- Button aligned to end, others centered -->
<div class="flex items-center h-24 bg-gray-100 p-4 rounded">
    <div class="flex-1">
        <h3 class="font-bold">Title</h3>
    </div>
    <button class="self-center px-4 py-2 bg-blue-500 text-white rounded">
        Action
    </button>
</div>

Title

Output Statement

  • self-auto - use container's align-items (default)
  • self-start - align to cross start
  • self-center - center on cross axis
  • self-end - align to cross end
  • self-stretch - stretch to fill container
