Align Items in Tailwind CSS

Align Items Utilities

Control alignment along the cross axis (vertical for flex-row, horizontal for flex-col). Each item stretches by default.

Standard Example: Align Options

<!-- Stretch (default) -->
<div class="flex items-stretch h-24">
    <div class="p-2 bg-blue-500 text-white">A</div>
    <div class="p-2 bg-blue-500 text-white">B</div>
</div>

<!-- Start -->
<div class="flex items-start h-24">
    <div class="p-2 bg-green-500 text-white">A</div>
    <div class="p-2 bg-green-500 text-white">B</div>
</div>

<!-- Center -->
<div class="flex items-center h-24">
    <div class="p-2 bg-purple-500 text-white">A</div>
    <div class="p-2 bg-purple-500 text-white">B</div>
</div>

<!-- End -->
<div class="flex items-end h-24">
    <div class="p-2 bg-red-500 text-white">A</div>
    <div class="p-2 bg-red-500 text-white">B</div>
</div>

Real-World Example: Centered Content

<!-- Centered card content -->

Output Statement

  • items-stretch - stretch to fill container (default)
  • items-start - align to cross start
  • items-center - center on cross axis
  • items-end - align to cross end
  • items-baseline - align on baseline
