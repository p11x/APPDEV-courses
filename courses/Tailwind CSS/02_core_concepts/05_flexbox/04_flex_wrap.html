Flex Wrap in Tailwind CSS

Flex Wrap Utilities

Control whether flex items wrap to additional lines or stay on a single line.

Standard Example: Wrap Options

<!-- No wrap (default) - items stay on one line -->
<div class="flex flex-nowrap">
    <div class="w-32 p-2 bg-blue-500 text-white">1</div>
    <div class="w-32 p-2 bg-blue-500 text-white">2</div>
    <div class="w-32 p-2 bg-blue-500 text-white">3</div>
    <div class="w-32 p-2 bg-blue-500 text-white">4</div>
</div>

<!-- Wrap - items wrap to new lines -->
<div class="flex flex-wrap">
    <div class="w-32 p-2 bg-green-500 text-white">1</div>
    <div class="w-32 p-2 bg-green-500 text-white">2</div>
    <div class="w-32 p-2 bg-green-500 text-white">3</div>
    <div class="w-32 p-2 bg-green-500 text-white">4</div>
</div>

<!-- Wrap reverse - wrap in reverse direction -->
<div class="flex flex-wrap-reverse">
    <div class="w-32 p-2 bg-purple-500 text-white">1</div>
    <div class="w-32 p-2 bg-purple-500 text-white">2</div>
    <div class="w-32 p-2 bg-purple-500 text-white">3</div>
</div>

Output Statement

  • flex-nowrap - single line (default)
  • flex-wrap - wrap to multiple lines
  • flex-wrap-reverse - wrap in reverse order
