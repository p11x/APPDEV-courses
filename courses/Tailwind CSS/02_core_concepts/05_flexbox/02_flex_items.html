Flex Items in Tailwind CSS

Flex Item Properties

Control how flex items grow, shrink, and size themselves within the flex container.

Standard Example: Flex Growth and Shrink

<!-- flex-1: grow to fill space, allow shrinking -->
<div class="flex">
    <div class="flex-1 bg-blue-500 p-4 text-white">flex-1</div>
    <div class="flex-1 bg-blue-500 p-4 text-white">flex-1</div>
</div>

<!-- flex-auto: initial size + grow/shrink -->
<div class="flex">
    <div class="flex-auto bg-green-500 p-4 text-white">flex-auto</div>
</div>

<!-- flex-none: no growing or shrinking -->
<div class="flex">
    <div class="flex-none bg-purple-500 p-4 text-white">fixed width</div>
</div>

<!-- flex-initial: shrink but don't grow -->
<div class="flex">
    <div class="flex-initial bg-red-500 p-4 text-white">shrink only</div>
</div>

Real-World Example: Content with Sidebar

<div class="flex">
    <!-- Main content: grows to fill space -->
    <div class="flex-1 bg-blue-50 p-4">
        Main Content Area
    </div>
    <!-- Sidebar: fixed width -->
    <div class="w-64 bg-gray-100 p-4">
        Sidebar
    </div>
</div>

Output Statement

  • flex-1 - grow to fill space, shrink if needed
  • flex-auto - grow and shrink, start from content size
  • flex-none - don't grow or shrink
  • flex-initial - shrink but don't grow
