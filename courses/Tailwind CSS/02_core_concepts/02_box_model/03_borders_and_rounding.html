Standard Example: Border Styles

<!-- Basic border with color -->
<div class="border-2 border-blue-500 p-8">
    <!-- 2px blue border -->
</div>

<!-- Border with radius -->
<div class="border border-gray-300 rounded-lg p-8">
    <!-- 1px gray border with rounded corners -->
</div>

<!-- Individual corner radius -->
<div class="rounded-t-lg rounded-b-none border-x-2 border-blue-500">
    <!-- Only top corners rounded -->
</div>

<!-- Circle with full rounded -->
<div class="w-24 h-24 rounded-full border-4 border-purple-500">
    <!-- Perfect circle using rounded-full -->
</div>

Output Statement

After implementing borders and rounding:

  • Elements have defined visual boundaries with border utilities
  • rounded-full creates perfect circles or pill shapes
  • Individual sides (rounded-t, rounded-l, etc.) provide corner control
  • Border colors can use any Tailwind color palette
  • Borders enhance or replace shadows for different visual effects

Edge Cases

  • border-b-0 removes bottom border while keeping others
  • rounded-full on rectangular element with same w/h = circle
  • Inset borders (border-t) can create specific-edge dividers
  • Combine with divide-y for list items

Performance

  • border and rounded are core utilities - no extra CSS
  • Use border-0 or rounded-0 to completely remove

Accessibility

  • Ensure focus-visible has visible border indicator
  • Don't remove outline without adding alternative focus style
  • Border can act as focus indicator (e.g., primary button)
