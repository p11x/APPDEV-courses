Line Height in Tailwind CSS

Line Height (Leading) Utilities

Line height controls vertical spacing between lines of text. It's crucial for readability.

leading-3 (0.75rem)

Tight line height. This is sample text to demonstrate very tight spacing between lines. See how close these lines are.

leading-tight (1.25)

Tight line height. Good for headings where you want less vertical space between lines.

leading-normal (1.5) - default

Normal line height. This is the default for body text. Provides good readability for most content.

leading-relaxed (1.625)

Relaxed line height. Better for longer passages where increased readability is important.

leading-loose (2)

Loose line height. Maximum spacing for very large text or accessibility purposes.

Standard Example: Different Line Heights

<!-- Headings with tight leading -->
<h1 class="text-3xl leading-tight font-bold">
    Tight Heading
</h1>

<!-- Body text with normal leading -->
<p class="leading-normal">
    Standard paragraph with normal line height...
</p>

<!-- Large body with relaxed leading -->
<p class="text-lg leading-relaxed">
    Larger text benefits from relaxed leading...
</p>

Real-World Example: Article Layout

<article class="max-w-2xl">
    <h1 class="text-4xl font-bold leading-tight mb-4">
        Article Title
    </h1>
    <p class="text-gray-500 text-sm leading-normal mb-6">
        By Author Name • 5 min read
    </p>
    <p class="text-lg text-gray-700 leading-relaxed">
        Long-form content benefits from larger text and relaxed line height
        for better readability and comprehension.
    </p>
</article>

Output Statement

  • leading-3 to leading-10 (numeric values)
  • leading-tight, leading-normal, leading-relaxed, leading-loose
  • Default is leading-normal (1.5)
  • Use leading-relaxed for improved readability in long text
