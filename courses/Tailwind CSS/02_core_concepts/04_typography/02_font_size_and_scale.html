Font Size and Scale in Tailwind CSS

Font Size Scale

Tailwind provides a comprehensive font size scale from xs (0.75rem) to 9xl (12rem).

text-xs - 0.75rem (12px)

text-sm - 0.875rem (14px)

text-base - 1rem (16px)

text-lg - 1.125rem (18px)

text-xl - 1.25rem (20px)

text-2xl - 1.5rem (24px)

text-3xl - 1.875rem (30px)

text-4xl - 2.25rem (36px)

text-5xl - 3rem (48px)

Standard Example: Using Font Sizes

<!-- Small text for captions -->
<p class="text-sm text-gray-500">
    Caption or helper text
</p>

<!-- Base for body text -->
<p class="text-base">
    Main paragraph content
</p>

<!-- Large for emphasis -->
<p class="text-lg font-semibold">
    Emphasized content
</p>

<!-- Headings -->
<h1 class="text-4xl font-bold">H1 Heading</h1>
<h2 class="text-3xl font-bold">H2 Heading</h2>
