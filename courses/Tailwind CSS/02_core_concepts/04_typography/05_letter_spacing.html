Letter Spacing in Tailwind CSS

Letter Spacing (Tracking) Utilities

Letter spacing controls the horizontal spacing between characters. Use it to create visual hierarchy or improve readability.

tracking-tighter (-0.05em)

Tighter letter spacing

tracking-tight (-0.025em)

Tight letter spacing

tracking-normal (0)

Normal letter spacing

tracking-wide (0.025em)

Wide letter spacing

tracking-wider (0.05em)

Wider letter spacing

tracking-widest (0.1em)

Widest letter spacing

Standard Example: Using Letter Spacing

<!-- All caps with wider spacing -->
<p class="text-sm tracking-widest uppercase">
    All Caps Label
</p>

<!-- Headings with tighter spacing -->
<h1 class="text-3xl font-bold tracking-tight">
    Heading Text
</h1>

<!-- Body text normal -->
<p class="tracking-normal">
    Standard body text...
</p>

Real-World Example: Navigation and Buttons

<!-- Navigation links -->
<nav class="flex space-x-6">
    <a href="#" class="text-sm font-medium tracking-wide">Home</a>
    <a href="#" class="text-sm font-medium tracking-wide">About</a>
    <a href="#" class="text-sm font-medium tracking-wide">Contact</a>
</nav>

<!-- Button with tracked text -->
<button class="
    px-6 py-3 
    bg-blue-600 text-white font-semibold 
    tracking-wider uppercase text-sm
">
    Learn More
</button>

Output Statement

  • tracking-tighter (-0.05em) to tracking-widest (0.1em)
  • tracking-tight = -0.025em, tracking-normal = 0
  • Use tracking-widest with uppercase for labels
  • Headings often look better with tracking-tight
