Font Weight in Tailwind CSS

Font Weight Scale

Tailwind provides font weights from thin (100) to black (900).

font-thin (100)

font-extralight (200)

font-light (300)

font-normal (400)

font-medium (500)

font-semibold (600)

font-bold (700)

font-extrabold (800)

font-black (900)

Standard Example: Typography Hierarchy

<!-- Heading hierarchy -->
<h1 class="text-4xl font-black">H1 - Boldest</h1>
<h2 class="text-3xl font-bold">H2 - Bold</h2>
<h3 class="text-2xl font-semibold">H3 - Semi-bold</h3>
<h4 class="text-xl font-medium">H4 - Medium</h4>

<!-- Body text -->
<p class="font-normal">Normal body text</p>
<p class="font-light">Light body text for large blocks</p>

H1 - font-black

H2 - font-bold

H3 - font-semibold

H4 - font-medium

Normal body - font-normal

Real-World Example: Card Typography

<div class="bg-white rounded-lg shadow p-6">
    <h3 class="text-xl font-bold text-gray-900 mb-2">
        Card Title
    </h3>
    <p class="text-base font-normal text-gray-600 mb-4">
        Description with normal weight.
    </p>
    <span class="text-sm font-medium text-blue-600">
        Link/Action
    </span>
</div>

Output Statement

  • font-thin (100) to font-black (900) available
  • font-normal = 400, font-bold = 700
  • Use font-medium/semibold for emphasis
