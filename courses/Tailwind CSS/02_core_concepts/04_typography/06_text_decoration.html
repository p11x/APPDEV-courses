Standard Example: Text Decoration Styles

<!-- Links with underline -->
<a href="#" class="underline text-blue-600">
    Link with underline
</a>

<!-- Deleted text with line-through -->
<span class="line-through text-gray-500">
    $99.00
</span>
<span class="text-green-600 font-bold">$79.00</span>

<!-- Decoration color -->
<p class="underline decoration-blue-500">
    Blue underline
</p>

Real-World Example: Link Styles

<!-- Primary link -->
<a href="#" class="text-blue-600 hover:underline">
    Learn more
</a>

<!-- Decorative link with decoration-offset -->
<a class="
    text-purple-600 
    underline decoration-2 
    underline-offset-4 
    hover:decoration-purple-400
">
    Hover underline effect
</a>
Learn more (hover to underline) With offset spacing

Output Statement

  • underline, overline, line-through, no-underline
  • decoration-{solid|double|dotted|dashed|wavy}
  • decoration-{1-8} for thickness
  • decoration-{color} for custom colors
  • underline-offset-{1-8} for spacing between text and underline
