## Overview

Know when to use real-time features vs regular HTTP requests.

## Core Concepts

**Use Real-time for:**
- Chat applications
- Live notifications
- Collaborative editing
- Real-time dashboards
- Gaming

**Use Regular HTTP for:**
- CRUD operations
- Form submissions
- One-time data fetches
