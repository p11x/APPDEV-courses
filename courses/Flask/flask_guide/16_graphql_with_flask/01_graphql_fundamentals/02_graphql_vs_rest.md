## Overview

Compare GraphQL and REST APIs.

## Comparison

| Feature | GraphQL | REST |
|---------|---------|------|
| Data fetching | Flexible | Fixed endpoints |
| Over-fetching | No | Common |
| Under-fetching | No | Common |
| Versioning | Not needed | Often needed |
| Caching | Complex | Simple |
| Learning curve | Steep | Easy |
