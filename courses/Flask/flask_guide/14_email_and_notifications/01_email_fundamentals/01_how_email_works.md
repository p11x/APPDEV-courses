## Overview

Email is fundamental to web applications for notifications, password resets, and marketing. This file explains how email works.

## Core Concepts

Email uses SMTP (Simple Mail Transfer Protocol):
1. Mail client → Mail server (via SMTP)
2. Mail server → Recipient's mail server (via SMTP)
3. Recipient → Their mail server (via POP3/IMAP)
