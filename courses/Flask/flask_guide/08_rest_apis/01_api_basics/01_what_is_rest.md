## Overview

**REST** (Representational State Transfer) is an architectural style for building APIs. REST APIs use HTTP methods to perform CRUD operations and return JSON data.

## Core Concepts

### REST Principles

- Use HTTP methods explicitly
- Stateless communication
- Return JSON responses
- Use resource-based URLs

## Code Walkthrough

### REST Example

```python
# REST API endpoints
# GET /users - List users
# GET /users/1 - Get user 1
# POST /users - Create user
# PUT /users/1 - Update user 1
# DELETE /users/1 - Delete user 1
```
