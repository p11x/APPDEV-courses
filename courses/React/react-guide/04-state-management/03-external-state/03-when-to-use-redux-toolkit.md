# When to Use Redux Toolkit

## Overview

While Zustand and Context work for many cases, Redux Toolkit is the go-to choice for large, complex applications requiring time-travel debugging, extensive middleware, and predictable state management.

## When to Use Redux

- Large applications with complex state logic
- Need for Redux DevTools time-travel
- Team familiar with Redux patterns
- Need for extensive middleware ecosystem
- Server-side rendering with state hydration

## Key Takeaways

- Redux Toolkit simplifies Redux setup
- Use for large, complex applications
- Context/Zustand work for simpler needs
