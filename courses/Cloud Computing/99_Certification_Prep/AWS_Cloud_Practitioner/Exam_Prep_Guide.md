# AWS Cloud Practitioner Exam Preparation Guide

## Exam Overview

**Duration**: 90 minutes  
**Questions**: 65 questions  
**Passing Score**: ~72% (subject to change)  
**Format**: Multiple choice and multiple response

## Domain Breakdown

### Domain 1: Cloud Concepts (26%)

- Cloud computing definitions
- AWS global infrastructure
- Cloud economics
- Benefits over traditional infrastructure

### Domain 2: Security and Architecture (25%)

- AWS shared responsibility model
- AWS security services
- Well-Architected Framework
- Identity and Access Management

### Domain 3: Technology (33%)

- Core AWS services (EC2, S3, Lambda, RDS, VPC)
- Networking (Route53, CloudFront)
- Database services (DynamoDB, Aurora)

### Domain 4: Billing and Pricing (16%)

- Pricing models (On-Demand, Reserved, Savings Plans)
- Cost management tools
- AWS Organizations

## Exam Tips

- Understand shared responsibility clearly
- Know differences between pricing models
- Understand basic networking concepts
- Be able to identify appropriate AWS services for scenarios

## Sample Practice Questions Example

**Q1**: A company wants to migrate to AWS with minimal cost and maximum flexibility. Which deployment model is most appropriate?

A) Dedicated hosting  
B) On-Premises hosting  
C) Cloud deployment  
D) Hybrid deployment

**Answer**: C - Cloud deployment

**Q2**: Which provides the HIGHEST durability for object storage?

A) S3 Standard  
B) S3 IA  
C) S3 One Zone-IA  
D) S3 Glacier

**Answer**: A - S3 Standard (99.999999999%)
