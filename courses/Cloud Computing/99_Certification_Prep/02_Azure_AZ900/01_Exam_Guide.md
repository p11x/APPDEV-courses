# WHY

The Azure Fundamentals certification validates foundational cloud knowledge. This exam guide helps you prepare systematically and pass on your first attempt.

## WHAT

### Exam Overview

- **Duration**: 60 minutes (varies by language)
- **Questions**: 40-60 questions
- **Format**: Multiple choice, multiple response, true/false, fill-in-blank
- **Passing Score**: 700/1000 (70%)

### Exam Strategy

1. **Read Carefully**: Understand what the question asks
2. **Eliminate Wrong Answers**: Remove obviously incorrect options
3. **Look for Keywords**: "most," "best," "lowest cost," "highest availability"
4. **Time Management**: ~1 minute per question
5. **Watch for "NOT"**: Pay attention to negative questions

## COMMON MISTAKES

### 1. Confusing PaaS and IaaS

**Mistake**: Not understanding the difference between Platform as a Service and Infrastructure as a Service.

**Correction**: 
- IaaS: You manage OS, apps (VM, VNet)
- PaaS: You manage apps, data (App Service, SQL DB)
- SaaS: You manage nothing (O365, Teams)

### 2. Azure Region Confusion

**Mistake**: Not understanding Azure regions, availability zones, and availability sets.

**Correction**: 
- Region: Geographic area
- Availability Zone: Physically isolated data centers
- Availability Set: Logical grouping for redundancy

### 3. Wrong Pricing Model

**Mistake**: Not understanding when to use different pricing options.

**Correction**: 
- Pay-as-you-go: Variable usage
- Reserved: 1-year commitment, up to 72% savings
- Spot: Interruptible workloads
