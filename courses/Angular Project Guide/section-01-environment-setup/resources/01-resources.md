## Quick Command Reference

### Node.js/npm
```bash
node -v              # Check Node version
npm -v               # Check npm version
npm install <pkg>   # Install a package locally
npm install -g <pkg> # Install a package globally
npm init             # Initialize new project
npm run              # Run scripts defined in package.json
```

### Angular CLI
```bash
ng new <project>     # Create new Angular project
ng serve             # Start development server
ng build             # Build for production
ng generate <type>   # Generate components, services, etc.
ng test              # Run unit tests
ng e2e               # Run end-to-end tests
```

### Maven
```bash
mvn -v               # Check Maven version
mvn clean            # Clean build
mvn compile          # Compile code
mvn package          # Package the application
mvn spring-boot:run # Run Spring Boot app
mvn test             # Run tests
```

### Git
```bash
git --version        # Check Git version
git init             # Initialize repository
git clone <url>      # Clone repository
git status           # Check status
git add .            # Stage all changes
git commit -m "msg"  # Commit changes
git push             # Push to remote
git pull             # Pull from remote
```

## System Requirements

### Minimum Requirements
- Windows 10 or later
- 4GB RAM (8GB recommended)
- 10GB free disk space
- 64-bit processor

### Recommended Requirements
- Windows 11
- 16GB RAM
- 20GB free disk space
- SSD storage

## Troubleshooting Common Issues

### Node.js Issues
- **Issue:** `npm is not recognized`
  - **Solution:** Restart computer after Node.js installation

### Java Issues
- **Issue:** `JAVA_HOME is not set`
  - **Solution:** Set JAVA_HOME environment variable manually

### Angular Issues
- **Issue:** `ng command not found`
  - **Solution:** Run `npm install -g @angular/cli` again

### Maven Issues
- **Issue:** Maven build is slow
  - **Solution:** Configure Maven mirror in settings.xml
