# ConApp1 - Beginner Console Application

ConApp1 is a beginner-level C# console application designed to teach fundamental programming concepts.

## Project Overview

| Attribute | Value |
|-----------|-------|
| **Project Type** | Console Application |
| **Framework** | .NET 8.0 |
| **Language** | C# |
| **Classes** | 24 |
| **Difficulty** | Beginner |

## Learning Path

This project is organized to take you from basic to intermediate C# concepts:

### 1. Getting Started
- **Basic Concepts** - Variables, data types, operators
  - See: [BASIC_CONCEPTS.md](./BASIC_CONCEPTS.md)

### 2. Control Flow
- **Control Flow** - If-else, switch, loops
  - See: [CONTROL_FLOW.md](./CONTROL_FLOW.md)

### 3. Pattern Programming
- **Pattern Printing** - Number and star patterns
  - See: [PATTERNS.md](./PATTERNS.md)

## Topics Covered

### Programming Fundamentals
- Variables and constants
- Data types (int, float, double, char, string, bool)
- Arithmetic, relational, and logical operators
- Type conversion (implicit and explicit)

### Control Flow
- If-else statements
- Nested if-else
- Switch expressions
- For loops
- While loops
- Do-while loops
- Break and continue

### Methods
- Method declaration and calling
- Parameters and arguments
- Return types
- Recursion
- Pass by value vs pass by reference
- Params array

### Pattern Printing
- Number patterns
- Star patterns
- Alphabet patterns
- Pyramid and diamond patterns

## Building the Project

### Build
```bash
cd ConApp1
dotnet build
```

### Run
```bash
dotnet run
```

### Release Build
```bash
dotnet publish -c Release
```
