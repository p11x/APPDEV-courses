# SQLite - Functionalities

## Core Functionalities

- **DDL Commands**: CREATE, ALTER, DROP tables
- **DML Commands**: SELECT, INSERT, UPDATE, DELETE
- **Joins**: INNER, LEFT, RIGHT, FULL, CROSS
- **Subqueries**: Nested queries supported
- **Aggregates**: COUNT, SUM, AVG, MIN, MAX, GROUP BY
- **Views**: Virtual tables
- **Triggers**: BEFORE/AFTER events

## Tools

- **sqlite3**: Command-line interface
- **DB Browser**: GUI for SQLite
- **VS Code Extension**: SQLite support
