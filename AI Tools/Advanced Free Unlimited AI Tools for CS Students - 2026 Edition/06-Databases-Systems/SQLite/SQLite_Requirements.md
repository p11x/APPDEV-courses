# SQLite - Requirements

## System Requirements

- CPU: Any modern processor
- RAM: Minimum 256 MB
- Disk: ~500 KB for installation
- OS: Windows, macOS, Linux, Android, iOS

## Storage Limits

- Maximum database size: 140 TB
- Maximum rows per table: 2^64
- Maximum columns: 2000+
- Maximum BLOB size: 140 TB
