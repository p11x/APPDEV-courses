# PostgreSQL - Functionalities

## Core Functionalities

- **SQL Query Execution**: Full SQL standard support with extensions
- **Data Types**: Native support for JSON, XML, UUID, arrays, hstore
- **Indexing**: B-tree, hash, GiST, GIN, BRIN indexes
- **Concurrency**: MVCC for high concurrent read/write performance
- **Procedural Languages**: PL/pgSQL, Python, Perl, Ruby support
- **Foreign Data Wrappers**: Connect to other databases
- **Full-Text Search**: Integrated text search engine
- **Window Functions**: Advanced analytical queries

## Advanced Capabilities

- **Partitioning**: Table partitioning for large datasets
- **Parallel Queries**: Query parallelization for performance
- **Extensions**: Extensible with extensions like PostGIS
- **Replication**: Streaming and logical replication
- **Backup/Restore**: pg_dump, pg_restore for data management
