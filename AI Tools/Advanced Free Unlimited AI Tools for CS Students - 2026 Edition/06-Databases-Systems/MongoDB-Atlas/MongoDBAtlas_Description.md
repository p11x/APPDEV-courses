# MongoDB Atlas - Description

## Overview

MongoDB Atlas is a fully-managed cloud database service for MongoDB. It provides a free tier (Atlas Free Tier) that allows students to deploy and use MongoDB databases in the cloud without any cost. This is ideal for learning NoSQL databases, building projects, and practicing for interviews.

## Official Website

- **Website**: [mongodb.com/atlas](https://www.mongodb.com/atlas)
- **Documentation**: [docs.atlas.mongodb.com](https://docs.atlas.mongodb.com/)

## Free Tier Features

- 512 MB storage
- Shared RAM
- Single node deployment
- Real-time cluster metrics
- Automated backups (free tier)
- Community support
