# Wireshark - Requirements

## System Requirements

- **Operating System**: Windows 10+, macOS 10.13+, Linux
- **RAM**: 4GB minimum, 8GB recommended
- **Storage**: 500MB for installation
- **Network Access**: Required for packet capture

## Free and Unlimited

- Wireshark is completely free and open-source
- No usage limits
- No registration required
- No paid features
