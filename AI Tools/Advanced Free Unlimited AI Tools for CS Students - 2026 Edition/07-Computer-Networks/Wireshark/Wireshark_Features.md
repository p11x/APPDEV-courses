# Wireshark - Features

## Key Features

- **Protocol Support**: TCP, UDP, HTTP, HTTPS, DNS, DHCP, FTP, and more
- **Packet Analysis**: Deep inspection of packet contents
- **Filtering**: Display filters and capture filters
- **Statistics**: Network statistics and traffic analysis
- **Coloring Rules**: Visual packet identification
- **VoIP Analysis**: Analyze voice over IP traffic
- **Export**: Export to various formats
