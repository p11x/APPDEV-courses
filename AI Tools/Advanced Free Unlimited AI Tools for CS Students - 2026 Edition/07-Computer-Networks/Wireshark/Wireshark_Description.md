# Wireshark - Description

## Overview

Wireshark is the world's most widely-used network protocol analyzer. It lets you see what's happening on your network at a microscopic level. For CS students, Wireshark is essential for understanding network protocols, troubleshooting network issues, and learning how data flows across networks.

## Official Website

- **Website**: [wireshark.org](https://www.wireshark.org)
- **Documentation**: [wireshark.org/docs](https://www.wireshark.org/docs/)

## Key Capabilities

- Deep inspection of hundreds of protocols
- Live capture and offline analysis
- Cross-platform support
- Industry standard
- VoIP analysis
