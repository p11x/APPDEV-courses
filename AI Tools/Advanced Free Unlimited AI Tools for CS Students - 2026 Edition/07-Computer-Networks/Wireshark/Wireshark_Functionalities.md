# Wireshark - Functionalities

## Core Capabilities

- **Live Capture**: Capture network packets in real-time
- **Protocol Decoding**: Decode hundreds of network protocols
- **Packet Filtering**: Filter by protocol, IP, port, and more
- **Statistics**: Generate network traffic statistics
- **Expert Analysis**: Identify network issues automatically

## Learning Applications

- Understand TCP/IP protocols
- Analyze HTTP/HTTPS traffic
- Debug network applications
- Study packet structure
