# Cisco Packet Tracer - Functionalities

## Learning Functions

- **Device Configuration**: Configure routers and switches via CLI
- **Topology Building**: Drag and drop network devices
- **Packet Simulation**: Visualize packet travel through network
- **Protocol Testing**: Test network protocols in simulation
- **Troubleshooting**: Practice network problem diagnosis

## Educational Benefits

- Hands-on networking experience
- Prepare for CCNA certification
- Understand network concepts visually
- Practice router/switch commands
