# Cisco Packet Tracer - Requirements

## System Requirements

- **Operating System**: Windows 10+, macOS 10.14+, Linux (Ubuntu 18.04+)
- **RAM**: 4GB minimum, 8GB recommended
- **Storage**: 2.5GB free space
- **Display**: 1024x768 resolution minimum

## Free Access Requirements

- Cisco NetAcad account (free)
- Internet connection for initial login
- Enrollment in any NetAcad course

## Limitations

- Requires periodic online verification
- Some advanced features may need paid courses
