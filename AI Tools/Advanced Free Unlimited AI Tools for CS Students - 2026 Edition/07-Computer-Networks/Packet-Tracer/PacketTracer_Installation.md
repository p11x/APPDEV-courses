# Cisco Packet Tracer - Installation

## Prerequisites

1. Create a free Cisco Networking Academy account at [netacad.com](https://www.netacad.com)
2. Enroll in any free course to access Packet Tracer

## Installation Steps

### Windows

1. Download Packet Tracer from NetAcad
2. Run the installer
3. Log in with NetAcad credentials

### macOS

1. Download Packet Tracer from NetAcad
2. Open the .dmg file
3. Drag to Applications folder

### Linux

1. Download the Linux version
2. Make executable: `chmod +x PacketTracer*`
3. Run: `./PacketTracer*`
