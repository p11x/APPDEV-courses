# Cisco Packet Tracer - Features

## Device Support

- Routers (Cisco 2911, 1941, etc.)
- Switches (2960, 3650, etc.)
- Wireless devices
- Firewalls
- Servers
- End devices

## Network Features

- VLAN configuration
- OSPF, EIGRP, RIP routing
- DHCP, DNS, HTTP services
- ACL configuration
- NAT configuration

## Simulation Mode

- Packet-by-packet simulation
- Event list visualization
- Traffic flow analysis
