# HackerRank - Description

## Overview

HackerRank is a competitive programming platform offering coding challenges across multiple domains including algorithms, data structures, databases, and more.

## Official Links

| Resource | Link |
|----------|------|
| Website | https://www.hackerrank.com |

## Free Features

- 3000+ coding problems across multiple difficulty levels
- Practice problems in 35+ programming languages
- Algorithm and data structure tutorials
- Interview preparation kits
- Certifications (some paid)
