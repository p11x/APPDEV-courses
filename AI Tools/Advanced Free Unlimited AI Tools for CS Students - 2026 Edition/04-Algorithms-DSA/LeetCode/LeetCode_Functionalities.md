# LeetCode - Functionalities

## Problem Solving Workflow

### 1. Problem Selection

Users can find problems through:
- Difficulty-based filtering
- Topic-based browsing
- Company-specific searches (Premium)
- Daily challenge participation

### 2. Code Execution

| Step | Description | Features |
|------|-------------|----------|
| Read Problem | Understand requirements | Examples, constraints |
| Write Code | Implement solution | Syntax highlighting |
| Run Tests | Verify functionality | Test cases provided |
| Submit | Submit solution | Performance metrics |

### 3. Submission System

- Automatic test case execution
- Runtime and memory usage reporting
- Comparison with other submissions
- Status: Accepted/Wrong Answer/Time Limit

## Practice Modes

### Daily Challenge

- One problem per day
- Free access
- Streak tracking
- Participation badges

### Mock Interviews

**Free Tier Limitations:**
- 2 mock interview sessions
- Limited peer availability
- Basic feedback

**Premium Features:**
- Unlimited mock interviews
- Real interview environment
- Detailed feedback

### Contest Participation

| Contest Type | Free Access | Frequency |
|--------------|-------------|-----------|
| Weekly | Limited entries | Weekly |
| Biweekly | Limited entries | Biweekly |
| Daily | Full access | Daily |
