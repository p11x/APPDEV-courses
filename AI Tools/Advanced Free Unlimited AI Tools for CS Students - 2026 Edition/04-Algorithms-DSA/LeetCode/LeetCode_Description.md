# LeetCode - Coding Practice Platform

## Overview

LeetCode is the leading online platform for coding interview preparation, offering hundreds of algorithm and data structure problems. It is widely used by students and professionals preparing for technical interviews at top technology companies.

## Official Resources

| Resource | URL |
|----------|-----|
| Main Website | https://leetcode.com |
| Problem Set | https://leetcode.com/problemset |
| Discuss Forum | https://leetcode.com/discuss |
| Explore | https://leetcode.com/explore |

## Tool Type

**Category:** Online Coding Practice Platform
**Access:** Browser-based with free and premium tiers
**Pricing:** Free tier available with limitations

## Target Audience

- Computer Science students preparing for campus placements
- Software engineers preparing for technical interviews
- Competitive programmers practicing algorithms
- Developers looking to improve problem-solving skills

## Free Tier Features

LeetCode offers a substantial free tier that allows meaningful practice:

| Feature | Free Tier | Premium Tier |
|---------|-----------|--------------|
| Practice Problems | 100+ | All 3000+ |
| Code Submissions | Limited | Unlimited |
| Problem Solving | All difficulties | All difficulties |
| Company Tags | Subset only | Full access |
| Mock Interviews | 2 free | Unlimited |
| Contest Participation | Limited | Full access |

## Why LeetCode for CS Students

1. **Industry Standard:** Used by major tech companies for screening
2. **Problem Variety:** Wide range of algorithm problems
3. **Discussion Forum:** Community solutions and explanations
4. **Company Questions:** Curated sets from top companies
5. **Mock Interviews:** Simulated interview environment

## Limitations of Free Tier

| Limitation | Workaround |
|-------------|------------|
| Limited problems | Focus on free problems, use other platforms |
| No company tags | Search problems by topic instead |
| Limited mock interviews | Use free sessions, practice with peers |
| Ads | Use ad blocker or tolerate |

## Premium Considerations

While premium offers more features, the free tier provides sufficient practice for:
- Completing 200+ quality problems
- Understanding core patterns
- Preparing for most interview processes
