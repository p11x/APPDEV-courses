# Builder.io - System Requirements

## Browser Requirements

| Browser | Minimum Version |
|---------|----------------|
| Chrome | 80+ |
| Firefox | 75+ |
| Safari | 14+ |
| Edge | 80+ |

---

## Account Requirements

- **Email Account**: Valid email for signup
- **GitHub Account** (optional): For social login
- **Google Account** (optional): For social login
- **Age**: 13+ (for account creation)

---

## Technical Requirements

### For Web-Based Usage:
- Internet connection (minimum 5 Mbps)
- Modern web browser
- No software installation needed

### For SDK Usage:
- Node.js 14+ (for npm packages)
- npm or yarn package manager
- React 16.8+ (for React SDK)
- Vue 2.6+ (for Vue SDK)

---
