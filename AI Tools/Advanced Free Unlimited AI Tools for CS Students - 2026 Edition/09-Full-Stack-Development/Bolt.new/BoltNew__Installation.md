# Bolt.new - Installation Guide

## Web-Based Access

Bolt.new is fully browser-based - no installation needed.

### Quick Start:

1. **Visit Bolt.new**
   - Go to: https://bolt.new

2. **Sign Up**
   - Click "Start Building" 
   - Create free account
   - No credit card required

3. **Begin Building**
   - Describe your project
   - AI generates code
   - Deploy with one click
