# Bolt.new - How It Works

## Development Process

### 1. Describe Your App
- Type what you want to build
- Example: "Create a task management app with user login"

### 2. AI Generates Code
- Bolt.new analyzes requirements
- Creates full-stack application
- Sets up frontend and backend

### 3. Edit & Customize
- Use browser-based IDE
- Make changes with AI assistance
- Preview in real-time

### 4. Deploy Instantly
- Click "Deploy" button
- Get live URL instantly
- Share with others
