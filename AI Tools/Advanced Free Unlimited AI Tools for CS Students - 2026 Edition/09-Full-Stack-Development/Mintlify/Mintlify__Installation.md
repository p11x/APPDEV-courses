# Mintlify - Installation

## Getting Started

### Method 1: Web-Based (Easiest)

1. **Create Account**
   - Visit https://mintlify.com
   - Sign up with GitHub or email

2. **Connect Repository**
   - Click "Start Documentation"
   - Connect your GitHub repository
   - Select repo to document

3. **Automatic Setup**
   - Mintlify creates config files
   - Deploys documentation automatically

### Method 2: Local Development

1. **Install CLI**
   ```bash
   npm install -g mintlify
   ```

2. **Initialize**
   ```bash
   mintlify init
   ```

3. **Run Locally**
   ```bash
   mintlify dev
   ```

### Method 3: VS Code Extension

1. **Open VS Code**
   - Go to Extensions Marketplace

2. **Search**
   - Search for "Mintlify"

3. **Install**
   - Click Install button

4. **Use**
   - Access Mintlify features in editor

## Configuration

### mintlify.json

Create configuration file:

```json
{
  "name": "Your Project",
  "logo": "/logo.png",
  "favicon": "/favicon.ico",
  "primaryColor": "#7C3AED"
}
```
