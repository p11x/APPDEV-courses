# Lovable - Installation Guide

## Web-Based Access

Lovable is a web-based platform - no installation needed.

### Step-by-Step:

1. **Visit Lovable**
   - Go to: https://lovable.dev

2. **Sign Up**
   - Click "Start Building" button
   - Connect with GitHub account
   - No credit card required

3. **Create Project**
   - Click "New Project"
   - Describe your app idea
   - Wait for AI to generate

---

## Quick Start Example

```
Prompt: "Create a todo app with user authentication and the ability to create, edit, and delete tasks"
```

Lovable will generate:
- React frontend with TypeScript
- Tailwind CSS styling
- Supabase backend
- User authentication
- Full CRUD for tasks
