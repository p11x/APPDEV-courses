# CodeWhisperer - Features

## Core Features

### 1. Real-Time Code Suggestions

- **Inline completions** as you type
- **Multi-line code suggestions** for entire functions
- **Context-aware recommendations** based on current code
- **Natural language to code** conversion

### 2. Code Generation

- Generate functions from comments
- Create boilerplate code automatically
- Build unit tests with single click
- Generate regex patterns from description

### 3. Security Scanning

- **Built-in security detector** identifies security vulnerabilities
- Scans for OWASP Top 10 vulnerabilities
- Provides remediation suggestions
- Flags hardcoded secrets and credentials

### 4. Reference Tracking

- Shows when suggestions match training data
- Links to open-source repositories
- Helps avoid copyright issues
- Promotes responsible AI use
