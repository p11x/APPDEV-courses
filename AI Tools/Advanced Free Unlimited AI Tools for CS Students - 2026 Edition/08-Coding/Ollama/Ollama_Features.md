# Ollama Features

## Core Features

### Local LLM Execution
- Run large language models directly on your personal computer
- No internet connection required after initial setup
- Complete privacy - your code and queries never leave your machine

### Multiple Model Support
- Pre-installed models: Llama 2, Mistral, CodeLlama, Gemma, and more
- Easy model switching between different AI models
- Model library with hundreds of available options

### Command-Line Interface
- Simple and intuitive CLI for interacting with AI models
- Quick prompt execution and response retrieval
- Scriptable for automation and integration

### System Integration
- Cross-platform support: macOS, Linux, and Windows
- REST API for building custom applications
- GPU acceleration support (CUDA for NVIDIA, Metal for Apple Silicon)

### Developer-Friendly
- Open Source under MIT License
- Active community and regular updates
- Extensive documentation and examples

## Advanced Features

### Model Customization
- Import custom GGUF models
- Adjust model parameters (temperature, top-p, etc.)
- Create and manage multiple model instances

### API Server
- Built-in HTTP server for programmatic access
- Integrate with other applications and services
- WebUI availability for graphical interface

### Resource Management
- Efficient memory usage with quantized models
- Adjustable context lengths
- Low resource mode for older computers
