# GitHub Pages Use Cases for CS Students

## 1. Portfolio Website

Host personal portfolio:

| URL Format | Description |
|------------|-------------|
| `username.github.io` | User site |
| `username.github.io/repo` | Project site |

## 2. Project Documentation

Host README as site:

- Auto-generate from Markdown
- Open source project docs
- API documentation

## 3. Academic Projects

Deploy coursework:

- Web development assignments
- Static prototypes
- Course notes

## 4. Resume/CV

Host online resume:

- HTML/CSS resume
- PDF download link
- Contact information

## 5. Open Source Projects

Showcase contributions:

- Project demos
- Live previews
- Documentation

## 6. Interview Preparation

Demonstrate skills:

- GitHub proficiency
- Web development
- Deployment knowledge
