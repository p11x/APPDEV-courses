# GitHub Pages Installation Guide

## Method 1: User Site

1. **Create Repository**
   - Create new repo: `username.github.io`
   - (Replace username with your GitHub username)

2. **Add Content**
   - Add `index.html`
   - Push to main branch

3. **Access**
   - Visit: `https://username.github.io`

## Method 2: Project Site

1. **Create Repository**
   - Create new repo or use existing

2. **Enable Pages**
   - Go to Settings → Pages
   - Select Source: main branch
   - Click Save

3. **Access**
   - Visit: `https://username.github.io/repo-name`

## Method 3: With Jekyll

```bash
# Create new Jekyll site
jekyll new my-site
cd my-site

# Edit _config.yml
# Push to GitHub

# Enable in Settings → Pages
```
