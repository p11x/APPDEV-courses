# GitHub Pages Features

## Core Features

### 1. Static Hosting

| Feature | Description |
|---------|-------------|
| HTML | Full HTML support |
| CSS | CSS and Sass |
| JavaScript | Client-side JS |
| Images | All image formats |

### 2. Jekyll Support

| Feature | Description |
|---------|-------------|
| Templates | Built-in themes |
| Markdown | Write in Markdown |
| Blog Support | Blogging features |
| Plugins | Jekyll plugins |

### 3. Custom Domains

| Feature | Description |
|---------|-------------|
| apex domains | example.com |
| subdomains | www.example.com |
| HTTPS | Automatic SSL |
| CDN | Cloudflare-backed |

### 4. GitHub Actions

| Feature | Description |
|---------|-------------|
| Custom Build | Build with Actions |
| Deployments | Automated deploys |
| Testing | Run tests before deploy |
