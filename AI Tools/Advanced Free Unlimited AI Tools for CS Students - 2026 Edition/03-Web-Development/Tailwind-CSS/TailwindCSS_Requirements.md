# Tailwind CSS Requirements

## System Requirements

### For Basic Usage (CDN)

| Requirement | Description |
|-------------|-------------|
| Browser | Any modern browser |
| Internet | Required for CDN |
| No installation | Works immediately |

### For Development

| Component | Requirement |
|-----------|-------------|
| Node.js | 12.x or higher |
| npm | 6.x or higher |
| OS | Windows, macOS, Linux |

## Development Tools

### Required Software

| Tool | Version | Purpose |
|------|---------|---------|
| Node.js | 12+ | Runtime |
| npm | 6+ | Package manager |
| Code Editor | Latest | VS Code recommended |

### Recommended VS Code Extensions

| Extension | Purpose |
|-----------|---------|
| Tailwind CSS IntelliSense | Autocomplete |
| PostCSS Language Support | Syntax highlighting |
| Prettier | Code formatting |

## Browser Support

| Browser | Support |
|---------|---------|
| Chrome | Full |
| Firefox | Full |
| Safari | Full |
| Edge | Full |

## Project Requirements

### For Build Tools

| Requirement | Description |
|-------------|-------------|
| Build Tool | Vite, Webpack, or PostCSS |
| Config File | tailwind.config.js |
| CSS Entry | input.css with directives |

### File Structure

```
project/
├── src/
│   ├── input.css
│   └── index.html
├── dist/
│   └── output.css
├── tailwind.config.js
├── postcss.config.js
└── package.json
```

## Getting Started Checklist

- [ ] Node.js installed
- [ ] Code editor (VS Code)

- [ ] Basic HTML/CSS knowledge
