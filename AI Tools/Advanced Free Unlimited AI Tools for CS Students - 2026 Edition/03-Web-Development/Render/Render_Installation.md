# Render Installation Guide

## Method 1: Web Dashboard

1. **Create Account**
   - Visit: https://dashboard.render.com
   - Click "Sign Up"
   - Use email or GitHub

2. **Deploy Web Service**
   - Click "New +" → Web Service
   - Connect GitHub
   - Select repository
   - Configure settings
   - Click "Create Web Service"

## Method 2: CLI

```bash
# Install Render CLI (if available)
npm install -g render-cli

# Or use curl
curl -sLO https://render.com/install.sh
```
