# Bootstrap Requirements

## System Requirements

### For CDN Usage

| Requirement | Description |
|-------------|-------------|
| Browser | Any modern browser |
| Internet | Required for CDN |
| No installation | Works immediately |

### For Development

| Component | Requirement |
|-----------|-------------|
| Node.js | 12.x or higher (optional) |
| npm | 6.x or higher (optional) |
| OS | Windows, macOS, Linux |

## Development Tools

### Required Software

| Tool | Version | Purpose |
|------|---------|---------|
| Browser | Latest Chrome/Firefox | Testing |
| Code Editor | Latest | VS Code recommended |

## Browser Support

| Browser | Support |
|---------|---------|
| Chrome | Full |
| Firefox | Full |
| Safari | Full |
| Edge | Full |
| Opera | Full |

## Dependencies

### JavaScript Dependencies

| Library | Purpose |
|---------|---------|
| Popper.js | Tooltips, popovers |
| Bootstrap JS | Interactive components |

**Note**: Use `bootstrap.bundle.min.js` which includes Popper.

## File Structure

```
project/

├── css/
│   └── styles.css (optional)
└── js/
    └── main.js (optional)
```
