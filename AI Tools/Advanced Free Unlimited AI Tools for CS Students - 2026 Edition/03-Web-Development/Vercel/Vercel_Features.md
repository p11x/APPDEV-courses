# Vercel Features

## Core Features

### 1. Automatic Deployments

| Feature | Description |
|---------|-------------|
| Git Integration | Auto-deploy on push |
| Preview Deployments | Every PR gets a URL |
| Production Deployments | Main branch = production |
| Rollbacks | One-click revert |

| Feature | Description |
|---------|-------------|
| Edge Network | 35+ regions |
| Caching | Automatic asset optimization |
| Compression | Brotli + Gzip |
| SSL | Free automatic HTTPS |

| Feature | Description |
|---------|-------------|

| Background Functions | Long-running tasks |
