# Netlify Features

## Core Features

### 1. Automated Deployments

| Feature | Description |
|---------|-------------|
| Git Integration | Auto-deploy on push |
| Preview Deployments | Each PR gets unique URL |
| Branch Deployments | Deploy specific branches |
| Instant Rollbacks | One-click revert |

### 2. Form Handling

| Feature | Description |
|---------|-------------|
| Zero Backend | HTML forms work |
| Spam Filtering | Built-in Akismet |
| File Uploads | Support for files |
| Notifications | Email/Slack alerts |

### 3. Serverless Functions

| Feature | Description |
|---------|-------------|
| Netlify Functions | AWS Lambda based |
| Edge Functions | Low-latency execution |
| Background Functions | Long-running tasks |

### 4. Identity & Auth

| Feature | Description |
|---------|-------------|
| Netlify Identity | User authentication |
| GoTrue | Open-source auth |
| Protected Routes | Role-based access |

### 5. Split Testing

| Feature | Description |
|---------|-------------|
| Branch-based | Test different branches |
| Traffic splitting | A/B testing |
| Analytics | Performance metrics |
