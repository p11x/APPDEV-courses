# Netlify Use Cases for CS Students

## 1. Portfolio Deployment

Deploy personal portfolio:

| Feature | Benefit |
|---------|---------|
| Free HTTPS | Secure SSL |
| Custom Domain | Personal branding |
| Global CDN | Fast loading |

## 2. Form Handling

No backend needed:

```html
<form name="contact" method="POST" data-netlify="true">
  <!-- form fields -->
</form>
```

## 3. Academic Projects

Deploy coursework:

- Web development assignments
- Static site projects
- Documentation sites

## 4. Hackathons

Quick deployment:

- Preview URLs for judges
- Form handling without backend
- Serverless API endpoints

## 5. Serverless Functions

Backend without servers:

```javascript
// functions/api.js
exports.handler = async (event) => {
  // Your backend logic
}
```

## 6. Interview Preparation

Demonstrate skills:

- Show live projects
- Explain serverless concepts
- Discuss CDN and performance
