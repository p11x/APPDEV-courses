## Mini-Project: Student Grade Tracker

After completing these topics, implement a student grade tracker application:

**Requirements:**
- Display student list with grades
- Use data binding for CRUD operations
- Implement pipes for grade formatting
- Use directives for conditional styling

## Assessment Checklist

Before proceeding to Intermediate Level, ensure you can:

- [ ] Create a new Angular project with CLI
- [ ] Understand TypeScript types, interfaces, and generics
- [ ] Build components with @Input and @Output
- [ ] Use all data binding types
- [ ] Apply control flow syntax (@if, @for, @switch)
- [ ] Create and use custom pipes
