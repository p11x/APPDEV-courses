## Project: Task Management Dashboard

This comprehensive project integrates all concepts learned:

### Features

- **Authentication**: Login, logout, JWT tokens, route guards
- **Task CRUD**: Create, read, update, delete tasks
- **State Management**: NgRx for centralized state
- **Material Design**: Complete UI with Angular Material
- **Search & Filter**: Full-text search, status/priority filters
- **Pagination**: Efficient data loading
- **Responsive Design**: Works on all screen sizes

### Tech Stack

- **Angular 17+**: Standalone components, signals, control flow
- **NgRx**: State management
- **Angular Material**: UI components
- **TypeScript**: Type safety
- **RxJS**: Reactive programming
