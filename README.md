## ⚙️ Auto-Commit Watcher

This repo has an automated file watcher that monitors for changes and
commits + pushes to GitHub automatically every 1 hour of inactivity.

| Action | Command |
|--------|---------|
| ▶️ Start manually | `python auto_commit.py` |
| ⛔ Stop watcher | `remove_windows_service.ps1` |
| 📄 View logs | `auto_commit.log` |
| ⏱️ Commit interval | 1 hour of inactivity |
| 🚀 Push | Immediate after every commit |

Auto-starts on Windows login via startup shortcut.
